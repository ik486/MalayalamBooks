6.21. ദൃശ്യം ലെയറുകൾ വിന്യസിക്കുക
6. “Image” മെനു  

6.21. ദൃശ്യം ലെയറുകൾ വിന്യസിക്കുക

Align Visible Layers കമാൻഡ് ഉപയോഗിച്ച്, വളരെ കൃത്യമായി കാണാവുന്ന പാളികൾ (“eye” ഐക്കൺ കൊണ്ട് അടയാളപ്പെടുത്തിയ ആ) സ്ഥാനം കഴിയും. നിങ്ങൾ പ്രത്യേകിച്ച് പല ചെറിയ പാളികൾ ഉണ്ടായിട്ടും ആനിമേഷനുകൾ, പ്രവർത്തിക്കുന്നു ചെയ്യുമ്പോൾ കൃത്യത ഈ ബിരുദം പ്രത്യേകിച്ചും ഉപകാരപ്രദമാണ്. Align Visible Layers ക്ലിക്കുചെയ്താൽ പാളികൾ വിന്യസിച്ചിരിക്കുന്ന എങ്ങനെ തിരഞ്ഞെടുക്കാൻ അനുവദിക്കുന്ന ഒരു ഡയലോഗ് പ്രദർശിപ്പിക്കുന്നു.

കുറിപ്പു്

GIMP 1.2 ൽ, വിന്യാസം സ്വതവേയുള്ള ബേസ് സ്റ്റാക്കിലുള്ള മുകളിൽ ദൃശ്യമായ പാളി ആയിരുന്നു. GIMP 2 സ്വതവേ അലൈൻമെന്റ് ബേസ് കാൻവാസ് വായ്ത്തലയാൽ ആണ്. നിങ്ങൾക്ക് തുടർന്നും ചിത്രം സ്റ്റാക്ക് ചുവടെ പാളി ന്, ഡയലോഗിൽ Use the (invisible) bottom layer as the base പരിശോധിച്ചുകൊണ്ട് കഴിയും അദൃശ്യ പോലും.

ചിത്രം 16.73. പാളി വേണ്ടിയുള്ള ഉദാഹരണം ചിത്രം

ഉദാഹരണത്തിന് ചിത്രം ഒരു വലിയ (150x150 പിക്സൽ) കാൻവാസിൽ നാലു പാളികൾ അടങ്ങിയിരിക്കുന്നു. റെഡ് സ്ക്വയർ 10x10 പിക്സൽ ആണ്, പച്ച സമകോണചതുർഭുജം 10x20 പിക്സലുകൾ ആണ് മഞ്ഞ സമകോണചതുർഭുജം 20x10 പിക്സലുകൾ ആണ്. Ignore lower layer ഓപ്ഷൻ ഡയലോഗും പരിശോധിക്കപ്പെട്ടിട്ടുമില്ല മുതൽ പശ്ചാത്തല പാളി (നീല, 100x100 പിക്സൽ), കൽപന പ്രകാരം ബാധിക്കില്ല. സ്വരൂപത്തിൽ പാളികൾ കാരണം കാൻവാസിൽ അവരുടെ സ്ഥാനം സ്റ്റാക്ക് അവരുടെ യഥാർത്ഥ ഓർഡർ അല്ലാതെ മറ്റൊരു ഓർഡർ തോന്നുന്നു ശ്രദ്ധിക്കുക. മഞ്ഞ പാളി ഇമേജിൽ മുകളിലെ പാളി ലേയറിന് രണ്ടാമത്തെ ഒന്നാണ്.


6.21.1. Activate the Command

  • You can access this command from the image menubar through Image → Align Visible layers. There is no default keyboard shortcut. If the image holds a single layer only, you get a message from GIMP telling that there must be more than one layer in the image to execute the command.

6.21.2. “Align Visible Layers” ഡയലോഗിന്റെ വിവരണം

ചിത്രം 16.75. The “Align Visible Layers” dialog


Horizontal Style; Vertical Style

ഈ ഓപ്ഷനുകൾ പാളികൾ പരസ്പരം ബന്ധത്തിൽ നീക്കി വേണം നിയന്ത്രിക്കുന്നു. തിരഞ്ഞെടുക്കാൻ നിങ്ങൾക്ക് കഴിയും:

  • None: ഉണ്ട് യഥാക്രമം, തിരശ്ചീനമായി മാറ്റമില്ലെന്നും ലംബമായതോ സ്ഥാനം ആയിരിക്കും.

  • Collect: കാണാവുന്ന പാളികൾ Horizontal base ആൻഡ് Vertical base ഓപ്ഷനുകൾ നിർണ്ണയിച്ചിരിക്കുന്നു എന്നു വഴിയിൽ, ക്യാൻവാസിൽ വിന്യസിച്ചിരിക്കുന്ന ചെയ്യും. നിങ്ങൾ Right edge ഒരു Horizontal base തെരഞ്ഞെടുത്താൽ, പാളികൾ ക്യാൻവാസ് നിന്നും അപ്രത്യക്ഷമാകും ചെയ്യാം. നിങ്ങൾ ക്യാൻവാസ് വിസ്താരം അവരെ വീണ്ടെടുക്കുന്നതിന് കഴിയും. നിങ്ങൾ Use the (invisible) bottom layer as the base ഐച്ഛികം ചെക്ക് ചെയ്താൽ, പാളികൾ ചുവടെ പാളിയുടെ മുകളിൽ ഇടത് മൂലയിൽ വിന്യസിച്ചിരിക്കുന്ന ചെയ്യും.

    ചിത്രം 16.76. തിരശ്ചീന “Collect” അലൈൻമെന്റ് (ക്യാൻവാസ് അറ്റത്തുള്ള)

    Original image with the layer stack

    അങ്ങനെ ഇടത്തു വേണമങ്കിലും ക്യാൻവാസ് ഇടത്തേയറ്റത്ത് അവസ്ഥയിലാണ് പാളികൾ തിരശ്ചീനമായി നീക്കി.


    ചിത്രം 16.77. തിരശ്ചീന “Collect” വിന്യാസം (ചുവടെ പാളി ന്)

    Original image with the layer stack

    അങ്ങനെ ഇടത്തു വേണമങ്കിലും ചുവടെ പാളി ഇടത്തേയറ്റത്ത് അനുസൃതമായി പാളികൾ തിരശ്ചീനമായി നീക്കി.


  • Fill (left to right); Fill (top to bottom): കാണാവുന്ന പാളികൾ നിങ്ങൾ യഥാക്രമം Horizontal base അല്ലെങ്കിൽ Vertical base, തിരഞ്ഞെടുത്ത എഡ്ജ് തക്കവണ്ണം കാൻവാസിൽ അവസ്ഥയിലാണ് ചെയ്യും. അങ്ങനെ അവർ പരസ്പരം ഓവർലാപ് ചെയ്യരുത് പാളികൾ, പതിവായി ക്രമീകരിച്ചിരിക്കുന്നത്. സ്റ്റാക്ക് ലെ മുകളിലെ പാളി സ്വരൂപത്തിൽ ഇടത്തേ (അല്ലെങ്കിൽ സന്ധിക്കു) സ്ഥാനം സ്ഥാപിച്ചിരിക്കുന്നു. സ്റ്റാക്ക് ലെ ചുവടെ പാളി ചിത്രത്തിന്റെ വലത്തേ (അല്ലെങ്കിൽ താഴെയുളള) സ്ഥാനം സ്ഥാപിച്ചിരിക്കുന്നു. മറ്റ് പാളികൾ ഈ രണ്ടു സ്ഥാനങ്ങൾ തമ്മിലുള്ള പതിവായി വയ്ക്കുന്നു. Use the (invisible) bottom layer as the base ഓപ്ഷൻ ചെക്ക് ചെയ്താല്, പാളികൾ ചുവടെ പാളി ഇതേ വായ്ത്തലയാൽ അവസ്ഥയിലാണ്.

    ചിത്രം 16.78. തിരശ്ചീന “Fill” വിന്യാസം (ക്യാൻവാസ്)

    Original image with the layer stack

    തിരശ്ചീന പൂരിപ്പിക്കൽ വിന്യാസം, ഇടത്തുനിന്ന് വലത്തേക്ക്, പരിശോധിച്ചില്ല Use the (invisible) bottom layer as the base ഓപ്ഷനുള്ള. സ്റ്റാക്ക് ലെ മുകളിലെ പാളി, പച്ച ഒന്നു ഇടത്തും എല്ലാ വഴി വയ്ക്കുന്നു. സ്റ്റാക്ക് ലെ ചുവടെ പാളി, ചുവന്ന ഒന്നു സ്ഥാപിക്കുന്നു വലതുഭാഗത്ത് ആണ് മഞ്ഞ പാളി മറ്റ് ഇടയിലാണ്.


    ചിത്രം 16.79. തിരശ്ചീന “Fill” വിന്യാസം (ചുവടെ പാളി)

    Original image with the layer stack

    ഒരേ പാരാമീറ്ററുകൾ മുൻ ഉദാഹരണം പോലെ, പക്ഷേ അടിസ്ഥാനമായി ഏറ്റവും കുറവ് (നീല) ലെവൽ.


  • Fill (right to left); Fill (bottom to top): ഈ ക്രമീകരണങ്ങൾ മുകളിൽ വിവരിച്ച പറഞ്ഞവ സമാനമായി പ്രവർത്തിക്കുന്നു, പക്ഷെ നിറയുന്നത് എതിർ ദിശയിൽ സംഭവിക്കുന്നത്.

    ചിത്രം 16.80. ലംബ “Fill” വിന്യാസം (ചുവടെ പാളി)

    Original image with the layer stack

    ലംബ “Fill” വിന്യാസം, താഴെ നിന്നും മുകളിലേക്കു്, അടിസ്ഥാനമായി ചുവടെ പാളി


    “Fill” ഓപ്ഷൻ ഉപയോഗിച്ച് സ്വരൂപത്തിൽ അവിടെ ചുരുങ്ങിയത് മൂന്ന് പാളികൾ ആയിരിക്കണം.
