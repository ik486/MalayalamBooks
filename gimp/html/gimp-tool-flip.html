4.9. ഫ്ലിപ്

ചിത്രം 14.133. Flip tool


ഫ്ലിപ് ഉപകരണം ഒന്നുകിൽ തിരശ്ചീനമായി അല്ലെങ്കിൽ ലംബമായി ലെയറുകളോ തിരഞ്ഞെടുക്കലുകൾ ഫ്ലിപ്പുചെയ്യുന്നതിനോ കഴിവ് നൽകുന്നു. ഒരു നിരക്കു .റിയാക്റ്റര് ചെയ്യുമ്പോൾ, ഒരു ഫ്ലോട്ടിംഗ് തിരഞ്ഞെടുക്കൽ ഒരു പുതിയ പാളി സൃഷ്ടിക്കപ്പെടുന്നു. നിങ്ങൾക്ക് പ്രതിഫലനം സൃഷ്ടിക്കാൻ ഈ ഉപകരണം ഉപയോഗിക്കാം.

4.9.1. ടൂൾ സജീവമാക്കുന്നു

You can access the Flip Tool in different ways:

  • From the image menu bar Tools/ Transform Tools Flip,

  • By clicking the tool icon: in Toolbox,

  • by using the Shift+F key combination.

4.9.2. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

Ctrl,

Ctrl നിങ്ങൾ തിരശ്ചീന, ലംബ ഫ്ലിപ്പുചെയ്യൽ തമ്മിലുള്ള മോഡുകൾ മാറ്റുക അനുവദിക്കുന്നു.

4.9.3. ഓപ്ഷനുകൾ

ചിത്രം 14.134. “Flip Tool” Options


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.
