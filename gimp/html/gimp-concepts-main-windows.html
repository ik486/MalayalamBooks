2. മെയിൻ വിൻഡോസ്

ജിമ്പ് യൂസര് ഇന്റര്ഫേസ് രണ്ടു മോഡുകൾ ഇപ്പോൾ ലഭ്യമാണ്: PLACEHOLDER,-100904

നിങ്ങൾ ആദ്യമായി ജിമ്പ് തുറക്കുമ്പോൾ, അത് സ്ഥിരമായി മൾട്ടി വിൻഡോ മോഡിൽ തുറക്കുന്നു. നിങ്ങൾ ഇമേജ് മെനു ബാറിലെ Windows → >Single-Window Mode വഴി സിംഗിൾ വിൻഡോ മോഡ് പ്രാപ്തമാക്കാൻ കഴിയും). ഈ ഓപ്ഷൻ പ്രവർത്തനക്ഷമമാക്കിക്കൊണ്ട് ജിമ്പ് നിറ്ത്തിയ ശേഷം ജിമ്പ് സിംഗിൾ വിൻഡോ മോഡ് അടുത്ത ആരംഭിക്കും.

മൾട്ടി വിൻഡോ മോഡ്

ചിത്രം 3.2. മൾട്ടി വിൻഡോ മോഡ് ചിത്രങ്ങളടങ്ങിയ ഒരു സ്ക്രീൻഷോട്ട്.


മുകളിൽ സ്ക്രീൻഷോട്ട് ഫലപ്രദമായി ഉപയോഗിക്കാൻ കഴിയുന്ന GIMP വിൻഡോസ് ഏറ്റവും അടിസ്ഥാന ക്രമീകരണത്തെ കാണിക്കുന്നു.

നിങ്ങൾ മിഡിൽ രണ്ടു പാനലുകൾ, ഇടത്, വലത്, ഒരു ചിത്രം വിൻഡോ നോട്ട്ട്. രണ്ടാമത്തെ ചിത്രം ഭാഗികമായും അണിഞ്ഞതുമാണ് ആണ്. ഇടത് പാനൽ ഒരുമിച്ചു പണിസഞ്ചി ആൻഡ് ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗ് ശേഖരിക്കുന്നു. വലത് പാനലിൽ ചുവടെയുള്ള മറ്റൊരു ഡോക്കിൽ പാളികൾ, ചാനലുകൾ, പാതകൾ, മൾട്ടി-ടാബ് ഡോക്കിൽ ഒരുമിച്ച് ചരിത്രം ഡയലോഗുകള് പൂർവസ്ഥിതിയിലാക്കാനോ നയപ്രഖ്യാപന, പാറ്റേണുകളും ഘനമായ ഡയലോഗുകൾ ഒരുമിച്ച് ശേഖരിക്കുന്നു. നിങ്ങൾ സ്ക്രീനിൽ ഈ പാനലുകൾ നീക്കാം. നിങ്ങൾക്ക് Tab കീ ഉപയോഗിച്ച് അവരെ മാസ്ക് കഴിയും.

  1. The Main Toolbox: ഉപകരണങ്ങൾ തിരഞ്ഞെടുക്കുന്നതിനായി ഉപയോഗിച്ച ഐക്കൺ ബട്ടണുകൾ ഒരു കൂട്ടം അടങ്ങിയിരിക്കുന്നു. സ്വതവേ, അത് മുൻവശത്തും പശ്ചാത്തല വർണ്ണങ്ങൾ അടങ്ങിയിരിക്കുന്നു. നിങ്ങൾ ബ്രഷ്, പാറ്റേൺ ഗ്രേഡിയന്റ് സജീവ ഇമേജ് ഐക്കണുകൾ ചേർക്കാൻ കഴിയും. പ്രാപ്തമാക്കുക, അധിക ഇനങ്ങൾ അപ്രാപ്തമാക്കാൻ Edit → Preferences → Toolbox ഉപയോഗിക്കുക.

  2. പ്രധാന പണിസഞ്ചി താഴെ ഡോക്കുചെയ്തിരിക്കുന്ന Tool options: നിലവിൽ തിരഞ്ഞെടുത്ത ഉപകരണം (ഈ കേസിൽ, നീക്കുക പ്രയോഗം) ഓപ്ഷനുകൾ കാണിക്കുന്ന ഒരു ടൂൾ ഉപാധികൾ ഡയലോഗ് ആണ്.

  3. Image windows: ഓരോ GIMP ചിത്രം തുറക്കുക മറ്റൊരു വിൻഡോയിൽ കാണിക്കുന്നു. നിരവധി ചിത്രങ്ങൾ ഒരേ സമയം, മാത്രമേ സിസ്റ്റത്തിന്റെ വിഭവങ്ങളും പരിമിതിയുളളവനായിരുന്നതെങ്കിൽ ലെ ഓപ്പൺ കഴിയും. നിങ്ങൾ GIMP ഉപയോഗപ്രദമായ ഒന്നും ചെയ്യാൻ കഴിയും മുമ്പ്, കുറഞ്ഞത് ഒരു ഇമേജ് വിൻഡോ തുറക്കുക ആവശ്യമാണ്. ചിത്രം ജാലകം നിങ്ങൾക്ക് വിൻഡോ റൈറ്റ് ക്ലിക്ക് ചെയ്ത് ലഭിക്കും ഏത് GIMP പ്രധാന കമാൻഡുകൾ (ഫയൽ, എഡിറ്റ്, തിരഞ്ഞെടുക്കുക ...), എന്ന മെനു താങ്ങി.

    ഒരു ചിത്രത്തിൽ വിൻഡോ കൈവരുകയാണ് കഴിയും. ആ സാഹചര്യത്തിൽ, ജിമ്പ് ഇമേജ് വിൻഡോയിൽ പൂർണ്ണ ചിത്രം കാണാൻ അനുവദിക്കുന്ന ഒരു ചുരുക്കിയ സൂം നില ഇമേജ് പ്രദർശിപ്പിക്കുന്നു. നിങ്ങൾ 100% സൂം തലത്തിലേക്ക് കളയുകയാണെങ്കിൽ, സ്ക്രോൾ ബാറുകൾ ഇമേജിൽ ഉടനീളം പാൻ അനുവദിച്ചുകൊണ്ട്, ദൃശ്യമാകും.

  4. Layers, Channels, Paths, Undo History ഡോക്കിൽ - ഡോക്കിൽ ഡയലോഗുകൾ ടാബുകൾ ശ്രദ്ധിക്കുക. ലെയറുകൾ ടാബ് തുറക്കുക: അത് നിലവിൽ സജീവ ചിത്രത്തിന്റെ പാളി ഘടന സൂചിപ്പിക്കുന്നത്, അതു പല ഭാഗത്ത് കൈകാര്യം അനുവദിക്കുന്നു. ഇത് ലയർ ഡയലോഗ് ഉപയോഗിക്കാതെ കുറച്ച് അടിസ്ഥാന കാര്യങ്ങൾ ചെയ്യാൻ സാധ്യമാണ് മാത്രമല്ല, മിതമായ പരിഷ്കൃതമായ GIMP ഉപയോക്താക്കൾക്ക് അത് നിസ്തുല എല്ലാകാലത്തും ലഭ്യമല്ല പാളിയുണ്ടാക്കും കണ്ടെത്താൻ.

  5. Brushes/Patterns/Gradients: നബി പാളി ഡയലോഗ് താഴെ ഡയലോഗ് നയപ്രഖ്യാപന, പാറ്റേണുകളും ഘനമായ നിയന്ത്രിക്കുന്നതിനുള്ള ഡയലോഗുകൾ (ടാബുകൾ) കാണിക്കുന്നു ഡോക്ക്.

സിംഗിൾ വിൻഡോ മോഡ്

ചിത്രം 3.3. സിംഗിൾ വിൻഡോ മോഡ് ചിത്രങ്ങളടങ്ങിയ ഒരു സ്ക്രീൻഷോട്ട്.


PLACEHOLDER,-100924: നിങ്ങൾ അവരുടെ മാനേജുമെന്റ് വ്യത്യാസങ്ങള് ഇതേ ഘടകങ്ങൾ കണ്ടെത്താൻ

ഇത് ഒരു ചുരുങ്ങിയ സജ്ജീകരണം ആണ്. വിവിധ ആവശ്യങ്ങൾക്കായി GIMP ഉപയോഗിക്കുന്ന ഡയലോഗുകളുടെ ഒരു ഡസനോളം മറ്റു തരത്തിലുള്ള മേൽ ഉണ്ട്, എന്നാൽ അവർ ആവശ്യമുള്ളപ്പോൾ ഉപയോക്താക്കൾക്ക് സാധാരണ അവരെ തുറന്ന് അവർ പൂർത്തിയാക്കി കഴിയുമ്പോൾ അവരെ അടയ്ക്കുക. വിജ്ഞാനമുള്ളയാൾ ഉപയോക്താക്കൾ പൊതുവേ (ടൂൾ ഓപ്ഷനുകൾ കൂടെ) പണിസഞ്ചി പ്രമാണിച്ചു ലെയറുകൾക്കുമായുള്ള എല്ലായ്പോഴും ഓപ്പൺ ഡയലോഗ്. പണിസഞ്ചി പല GIMP ഓപ്പറേഷൻസ് അത്യന്താപേക്ഷിതമാണ്. ടൂൾ ഓപ്ഷനുകൾ വിഭാഗം യഥാർത്ഥത്തിൽ സ്ക്രീൻഷോട്ടിൽ മെയിൻ ഉപകരണബോക്സിലേക്ക് ഡോക്ക് കാണിച്ചിരിക്കുന്ന വേവ്വേറെ ഡയലോഗ് ആണ്. അത് അവരുടെ ഓപ്ഷനുകൾ എങ്ങനെ സജ്ജമാക്കി കാണാൻ കഴിയാത്തതിൽ ഇല്ലാതെ ഫലപ്രദമായി ഉപകരണങ്ങൾ ഉപയോഗിക്കാൻ വളരെ ബുദ്ധിമുട്ടാണ്: വിജ്ഞാനമുള്ളയാൾ ഉപയോക്താക്കൾക്ക് എല്ലായ്പ്പോഴും അത് സജ്ജീകരിക്കാൻ ഈ വഴി ഞങ്ങൾക്കുണ്ട്. നിങ്ങൾ തലങ്ങളുണ്ട് ഒരു ചിത്രം ഉപയോഗിച്ച് പ്രവർത്തിക്കുമ്പോൾ പാളികളുടെ ഡയലോഗ് പ്ലേ വരുമ്പോൾ: നീ GIMP വിദഗ്ദ്ധന്മാർ ഏറ്റവും ഘട്ടങ്ങൾ അപ്പുറം അഭിമുഖീകരിക്കും ശേഷം, ഈ almost always എന്നാണ്. പ്രതിഭയെ നിങ്ങൾ സ്ക്രീനിൽ എഡിറ്റ് ചിത്രങ്ങൾ പ്രദർശിപ്പിക്കാൻ സഹായിക്കുന്നു; നിങ്ങളുടെ ശേഖരിക്കുന്നത് മുമ്പ് ചിത്രത്തിന്റെ വിൻഡോ അടയ്ക്കുകയാണെങ്കിൽ, GIMP നിങ്ങൾ ഫയൽ ക്ലോസ് ആവശ്യമുണ്ടോ ഇല്ലയോ ആവശ്യപ്പെടും.

Add Tab, Close Tab , അല്ലെങ്കിൽ Detach Tab തിരഞ്ഞെടുക്കുക.

തുടർന്നുവരുന്ന വിഭാഗങ്ങളിൽ അവർ അവർ എങ്ങനെ പ്രവർത്തിക്കുന്നുവെന്നും വിശദീകരിക്കുന്നതിലും, സ്ക്രീൻഷോട്ട് കാണിച്ചിരിക്കുന്ന വിൻഡോസ് ഓരോ ഘടകങ്ങൾ ഇതിലൂടെ നിങ്ങളെ. നിങ്ങൾ അവരെ, പ്ലസ് GIMP ചിത്രങ്ങൾ അടിസ്ഥാന ഘടന വിവരിക്കുന്ന ഭാഗം വായിക്കുക ചെയ്താൽ, അടിസ്ഥാന ഇമേജ് മാറ്റങ്ങൾ വൈവിധ്യമാർന്ന വേണ്ടി GIMP ഉപയോഗിക്കാൻ മതി സാധ്യത ഇല്ല. അതിനുശേഷം, നിങ്ങൾ സാധ്യമാകൂ എന്ന് കൂടുതൽ തന്ത്രപൂർവം സവിശേഷമായ കാര്യങ്ങൾ ഏതാണ്ട് അനന്തമായ എണ്ണം പഠിക്കാൻ ചുറുചുറുക്കോടെ (അല്ലെങ്കിൽ പരീക്ഷണം) എന്ന മാനുവൽ തയാറായി നോക്കാൻ കഴിയും. തമാശയുള്ള!

പണിസഞ്ചി GIMP ഹൃദയം ആണ്. ഇവിടെ നിങ്ങൾ അവിടെ കണ്ടെത്തും പറ്റി ഒരു ദ്രുത ടൂർ ആണ്.

സൂചന

പണിസഞ്ചിയിൽ, എന്തെങ്കിലും മൌസ് ചലിക്കുന്ന അതു ഒരു നിമിഷം വിശ്രമം അനുവദിച്ചുകൊണ്ട് GIMP മിക്ക ഭാഗങ്ങളിലും പോലെ, സാധാരണയായി കാര്യം വിവരിക്കുന്ന ഒരു “tooltip” പ്രദർശിപ്പിക്കുന്നു. ഹൃസ്വ കട്ട് കീകൾ പലപ്പോഴും ടൂൾടിപ് കാണിച്ചിട്ടുണ്ട്. പല കേസുകളിലും നിങ്ങൾ ഒരു ഇനം മൌസ് ഹോവർ മൌസ് കീഴെയുള്ള കാര്യം സഹായം നേടുന്നതിനുള്ള F1 കീ അമർത്തുക കഴിയും.

സ്വതവേ, മാത്രം മുൻവശം-പശ്ചാത്തല ഐക്കൺ ദൃശ്യമാണ്. നിങ്ങൾ Edit → Preferences → Toolbox വഴി ബ്രഷ്-ശ്രേണി-വിതറുക ഐക്കണുകളുടെയും സജീവ ഇമേജ് ഐക്കൺ ചേർക്കാൻ കഴിയും: ഉപകരണങ്ങൾ കോൺഫിഗറേഷൻ.

  1. Tool icons: ഈ ഐക്കണുകൾ ആവശ്യങ്ങൾക്ക് വൈവിധ്യമാർന്ന ഉപകരണങ്ങൾക്ക് സജീവമാക്കുന്നതിന് ഏത് ബട്ടണുകളുണ്ടു്: ചിത്രങ്ങളുടെ ഭാഗങ്ങൾ തിരഞ്ഞെടുത്ത് ഒരു ചിത്രം പെയിന്റിംഗ്, ഒരു ഇമേജ് രൂപാന്തരപ്പെടുത്തി മുതലായവ വിഭാഗം 1, “പണിസഞ്ചി” ഉപകരണങ്ങൾ മായി എങ്ങനെ ഒരു അവലോകനം നൽകുന്നു, ഓരോ ഉപകരണം Tools ലെ വ്യവസ്ഥാപിതമായി വിശദീകരിച്ചിട്ടുണ്ട് അധ്യായം.

  2. ഇവിടെ Foreground/Background colors: നിറം പ്രദേശങ്ങൾ നിങ്ങൾ അനേകം ഓപ്പറേഷനുകളിൽ പ്ലേ കയറി വരുന്നു GIMP ന്റെ നിലവിലെ മുന്നിലും പശ്ചാത്തലത്തിലും നിറങ്ങൾ, കാണിക്കും. അവരിൽ ഒന്നുകിൽ ഒന്നിൽ ക്ളിക്കു നിങ്ങൾക്ക് മറ്റൊരു നിറത്തിൽ മാറ്റാൻ അനുവദിക്കുന്ന ഒരു കളർ സെലക്ടർ ഡയലോഗിന് നൽകുന്നു. ഇരട്ട തലയുള്ള അമ്പ് ക്ലിക്കുചെയ്യുന്നത് രണ്ട് നിറങ്ങൾ അക്കങ്ങള്, താഴത്തെ ഇടത് മൂലയിൽ ചെറിയ ചിഹ്നത്തിൽ ക്ലിക്കുചെയ്തുകൊണ്ട് കറുപ്പും വെളുപ്പും അവരെ പുനഃസജ്ജമാക്കുന്നു.

  3. ഇവിടെ Brush/Pattern/Gradient: ചിഹ്നങ്ങൾ നിങ്ങൾക്ക് തെരഞ്ഞെടുക്കുന്നവ GIMP's കാണിക്കാൻ: നിങ്ങൾ (“painting” വഴിയിൽ മായ്ക്കുന്ന ആൻഡ് smudging, പോലുള്ള പ്രവർത്തനങ്ങൾ ഉൾപ്പെടുന്നു) ചിത്രത്തിൽ പെയിന്റ് അനുവദിക്കുന്ന എല്ലാ ഉപകരണങ്ങൾ ഉപയോഗിക്കുന്ന അന്തസ്സായിത്തന്നെയാണു; ഒരു ചിത്രത്തിന്റെ തിരഞ്ഞെടുത്ത പ്രദേശങ്ങൾ പൂരിപ്പിച്ച് ഉപയോഗിക്കുന്ന ശ്രേണി, വേണ്ടി; ഒരു ഓപ്പറേഷൻ നിറങ്ങൾ സുഗമമായി വ്യത്യസ്ത ശ്രേണി ആവശ്യമാണ് എപ്പോഴൊക്കെ പ്ലേ കടന്നു വരുന്നതും ഗ്രേഡിയന്റ്, വേണ്ടി. ഈ ചിഹ്നങ്ങൾ ഏതെങ്കിലും ക്ലിക്കുചെയ്താൽ ഇത് മാറ്റാൻ അനുവദിക്കുന്ന ഒരു ഡയലോഗ് വിൻഡോ ചെയ്യുന്നു.

  4. Active Image: GIMP ൽ, ഒരേസമയം നിരവധി ചിത്രങ്ങൾ പ്രവർത്തിക്കാൻ കഴിയും, എന്നാൽ ഏതു നിമിഷത്തിലും, ഒരു ചിത്രം മാത്രം “active image” ആണ്. ഇവിടെ നിങ്ങൾ സജീവ ചിത്രത്തിന്റെ ഒരു ചെറിയ വിശ്വപ്രസിദ്ധമായ പ്രാതിനിധ്യം കണ്ടെത്താൻ. നിലവിൽ തുറന്നിരിക്കുന്ന ചിത്രങ്ങൾ ഒരു ലിസ്റ്റ് ഒരു ഡയലോഗ് പ്രദർശിപ്പിക്കുന്നതിന് ഐക്കൺ ക്ലിക്ക് അത് സജീവ ഉണ്ടാക്കുവാൻ ഡയലോഗിൽ ഒരു ചിത്രം ക്ലിക്ക് ചെയ്യുക. സാധാരണ, ഒരു ഇമേജ് വിൻഡോ മൾട്ടി വിൻഡോ മോഡ്, അല്ലെങ്കിൽ സിംഗിൾ വിൻഡോ മോഡ് ഒരു ഇമേജ് ടാബിൽ അത് സജീവ ഉണ്ടാക്കുവാൻ ക്ലിക്ക് ചെയ്യുക.

    നിങ്ങൾ “Drop to an XDS file manager to save the image” കഴിയും. യുണിക്സ്-പോലുള്ള പ്രവർത്തക സംവിധാനങ്ങൾക്കായി X Window സിസ്റ്റം ഗ്രാഫിക്കല് ​​യൂസര് ഇന്റര്ഫേസ് ഒരു അധിക സവിശേഷത: XDS “X Direct Save Protocol” വേണ്ടി ചുരുക്കപ്പേരാണ്.

കുറിപ്പു്

ഓരോ ആരംഭത്തിൽ GIMP ഒരു ഉപകരണം (ബ്രഷ്), ഒരു നിറം, സ്വതവേ ഒരു ബ്രഷ് ഒരു പാറ്റേൺ എപ്പോഴും ഒരേ തെരഞ്ഞെടുക്കുന്നു. നിങ്ങൾ GIMP കഴിഞ്ഞ നിങ്ങളുടെ മുമ്പത്തെ സെഷനിൽ അവസാനിപ്പിക്കുന്ന സമയത്ത് ഉപയോഗിച്ച ഉപകരണം, നിറം, ബ്രഷ് ആൻഡ് പാറ്റേൺ തിരഞ്ഞെടുക്കുക ചെയ്യണമെങ്കിൽ, Preferences/Input Devices ലെ Save input device settings on exit പരിശോധിക്കുക.
സൂചന

പണിസഞ്ചി വിൻഡോ ഡയലോഗ് മുകളിൽ സഹിതം “Wilber's eyes” പ്രദർശിപ്പിക്കുന്നു. (toolbox- wilber no): നിങ്ങളുടെ gimprc പ്രമാണത്തെ താഴെ ലൈൻ ചേർത്തുകൊണ്ട് “Wilber's eyes” തുടച്ചുനീക്കുന്ന കഴിയും. അതു മാത്രം പണിസഞ്ചി ബാധിക്കുന്നു. നിങ്ങൾ ഒരു തുറന്ന ഇമേജ് ഇല്ല ചിത്ര വിൻഡോയിൽ കണ്ണു മാത്രമേ ദൃശ്യമാകൂ.
