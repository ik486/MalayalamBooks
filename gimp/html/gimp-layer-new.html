7.2. പുതിയ ലെയർ

New Layer കമാൻഡുകളുടെയും സജീവ പാളി മീതെ ചിത്രത്തിന്റെ പാളി സ്റ്റാക്ക് പുതിയ, ശൂന്യമായ പാളി ചേർക്കുന്നു. കമാൻഡ് പുതിയ പാളി വലിപ്പം വ്യക്തമാക്കാൻ സാധിക്കുന്ന ഒരു ഡയലോഗ് പ്രദർശിപ്പിക്കുന്നു.

7.2.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → New Layer.

7.2.2. Description of the “New Layer” Dialog

ചിത്രം 16.90. The “New Layer” dialog


ശീർഷകം “Create a new layer” കീഴിൽ ആ ചിത്രത്തിന്റെ പേര് കാണാൻ കഴിയും ഈ പുതിയ പാളി ശീർഷകം അതു ഒരു ലഘുചിത്രം അടുത്ത സൃഷ്ടിക്കുക. അതാണ് നിങ്ങൾ നല്ല ചിത്രം തിരഞ്ഞെടുത്തു എങ്കിൽ ഒന്നിലധികം ഇമേജ് തുറക്കുക അവിടെ കാണാൻ രസകരമായിരിക്കും.

Layer Name

പുതിയ ലെയർ പേര്. ഏതെങ്കിലും ഫങ്ഷണൽ പ്രാധാന്യം ഇല്ല; അതു കേവലം നിങ്ങൾ പാളി ഉദ്ദേശ്യം ഓർക്കാൻ സുഖ വഴി. സ്ഥിര പേര് “New Layer” ആണ്. നിങ്ങൾ തിരഞ്ഞെടുക്കുന്ന പേരുള്ള ഒരു പാളി ഇതിനകം നിലവിലുണ്ട് എങ്കിൽ, ഒരു നമ്പർ സ്വയം നിങ്ങൾക്ക് OK ബട്ടണില് ക്ലിക്ക് ചെയ്യുമ്പോൾ അത് അതുല്യമായ (ഉദാ “New Layer#1”) വേണ്ടി അതും പ്പെടുന്നു.

Width; Height

പുതിയ ലയർ തലങ്ങളും. ഡയലോഗ് ലഭ്യമാകുമ്പോൾ, മൂല്യങ്ങൾ ചിത്രത്തിന്റെ അളവുകളിലേക്ക് ആരംഭിക്കുന്നു. നിങ്ങൾ രണ്ടു ടെക്സ്റ്റ് ബോക്സുകൾ ഉപയോഗിച്ച് മാറ്റാം. നിങ്ങൾക്ക് വലതുവശത്ത് പുൾ-ഡൌൺ മെനുവിൽ യൂണിറ്റുകൾ മാറ്റാവുന്നതാണ്.

ലെയർ വർണ്ണം ഇനം

നിലവിലെ Foreground color, നിലവിലെ Background color, White ആൻഡ് Transparency: പാളി നിറയുന്നു ഖര നിറം നാലു ഓപ്ഷനുകൾ ഉണ്ട്.
