16.3. ഏലിയൻ ഗ്ലോ

16.3.1. പൊതു അവലോകനം

ചിത്രം 17.375. “Alien Glow” ഫിൽട്ടർ ഉദാഹരണം

“Alien Glow” ഫിൽട്ടർ

“Alien Glow” ലോഗോ


ഈ ഫിൽട്ടർ സജീവ പാളി ന്റെ ആൽഫ ചുറ്റും ഒരു വലിയ തിളക്കം ചേർക്കുന്നു.

ഫിൽട്ടർ മുകളിൽ ടെക്സ്റ്റ് പ്രാബല്യത്തോടെ ഒരു ലോഗോ സൃഷ്ടിക്കുന്ന “Alien Glow” സ്ക്രിപ്റ്റ് (ചിത്രം വിൻഡോയിൽ File → Create → Logos → Alien Glow), നിന്നാണ്.

മുന്നറിയിപ്പു്

ചിത്രം എല്ലായ്പ്പോഴും സജീവ പാളി വലിപ്പത്തിന് വലുപ്പം മാറ്റും.

16.3.2. ഫിൽട്ടർ സജീവമാക്കുക

നിങ്ങൾ Filters → Alpha to Logo → Alien Glow കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ ഈ ഫിൽട്ടർ കണ്ടെത്താം.

16.3.3. ഓപ്ഷനുകൾ

ചിത്രം 17.376. “Alien Glow” ഓപ്ഷനുകൾ


വലുപ്പം ഗ്ലോ (പിക്സൽ * 4)

ഈ യഥാർത്ഥത്തിൽ “Alien Glow” സ്ക്രിപ്റ്റ്-ഫു സ്ക്രിപ്റ്റ് ഫോണ്ട് വലിപ്പം തീരുമാനം. എന്നാൽ രണ്ടു മൂല്യങ്ങളെ ഈ വലിപ്പം ബന്ധപ്പെട്ട് ആയിരിക്കും: തിളക്കം “Glow size” / 30 വലുതായ ചെയ്യും, നിങ്ങൾക്ക് ചിലപ്പോൾ ഈ ഓപ്ഷനായി നിങ്ങളുടെ വസ്തുക്കളുടെ ഉയരം തിരഞ്ഞെടുക്കുക വേണം തൂവൽ ആരം “Glow size” / 4. ആണ് (“pixels * 4” അവഗണിക്കുക).

നിറം ജലിക്കുക

ഈ “eerie” തിളങ്ങുന്ന നിറം ആണ്. കേരളത്തിലും ഇത് പച്ച (63,252,0) ലേക്ക് സ്ഥിരസ്ഥിതികളെ എന്നാൽ Swatch ബട്ടണിൽ ക്ലിക്കിൽ നിങ്ങൾക്ക് എന്തെങ്കിലും നിറം തിരഞ്ഞെടുക്കാൻ ലഭ്യമാണ് നിറം സെലക്ടർ നൽകുന്നു.

16.3.4. ഫിൽറ്റർ വിശദാംശങ്ങൾ

ഒരു രാകേഷിന്റെ അന്യഗ്രഹ തിളക്കം പുനരുൽപ്പാദിപ്പിക്കുന്നതിനായുമുള്ള എളുപ്പമാണ്:

  • ആവശ്യമെങ്കിൽ സജീവ പാളി ഒരു selection from the alpha channel സൃഷ്ടിക്കുക.

  • താഴെ Gradient Blend ഉപയോഗിച്ച് തിരഞ്ഞെടുപ്പ് നിറയ്ക്കുക: SHAPE = രൂപപ്പെടുത്തിയ (ഗോളാകൃതി); ഗ്രേഡിയന്റ് = ബൾഗേറിയ ലേക്ക് FG (ആർജിബി), FG = ഇരുണ്ട ചാരനിറം (79,79,79), ബൾഗേറിയ = കറുത്ത.

  • ഒരു new layer (“Alien Glow”) താഴെ സൃഷ്ടിക്കുക. Extend ചെറുതായി നിരക്കു, feather അതു Glow color ഫിൽ.

  • കറുത്ത നിറഞ്ഞു പുതിയ ഒരു പശ്ചാത്തല ലെയർ സൃഷ്ടിക്കുക.
