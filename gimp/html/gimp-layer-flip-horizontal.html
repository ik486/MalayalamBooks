7.43. Flip Horizontally

Flip Horizontally കമാൻഡ് ഇടതു നിന്നും വലത്തോട്ട്, അതായത് തിരശ്ചീനമായി സജീവ പാളി ദിശമാറുന്നതുമായ. അതിൽ മാറ്റം പാളി തലങ്ങളും പിക്സൽ വിവരങ്ങൾ വിടുന്നു.

7.43.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Transform → Flip Horizontally.

7.43.2. Example

ചിത്രം 16.108. “Flip Layer Horizontally” പ്രയോഗിക്കുന്നു

Before applying the command

അതു ഫ്ലിപ്പുചെയ്ത ശേഷം പാളി. ചിത്രം പാളി കേന്ദ്ര vertical അച്ചുതണ്ട് സഹിതം പ്രതിഫലിക്കുകയും ചെയ്തു തോന്നുന്നു.
