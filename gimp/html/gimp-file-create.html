2.3. സൃഷ്ടിക്കാൻ

ചിത്രം 16.7. The “Create” submenu


ഈ മെനു ഇനം പണിസഞ്ചി മെനു GIMP മുൻ പതിപ്പുകൾ നിലനിന്നിരുന്നു ലോഗോകളും, ബട്ടണുകൾ, പാറ്റേണുകളുടെ ധാരാളം അടങ്ങുന്ന “Acquire” മെനു മാറ്റിസ്ഥാപിക്കുന്നു ...

ഈ കമാൻഡുകൾ കരുതില്ല GIMP സിസ്റ്റം പ്രവർത്തനങ്ങൾ ലേക്കുള്ള കോളുകൾ ചെയ്യുന്നു, നിങ്ങളുടെ സിസ്റ്റം ആശ്രയിച്ചിരിക്കും വ്യത്യാസപ്പെട്ടിരിക്കും.

2.3.1. Activate the Submenu

  • You can access this submenu from the Image menu bar through File → Create

2.3.2. ക്ലിപ്ബോർഡ് നിന്നും

നിങ്ങൾ ഒരു നിര പകർത്താനും, അത് ക്ലിപ്പ്ബോർഡിലേക്ക് പോകുന്നു. അപ്പോൾ നിങ്ങൾ അതു ഒരു പുതിയ ഇമേജ് തയ്യാറാക്കാം.

ഈ കമാൻഡ് Paste as new കമാൻഡ് പോലെ തന്നെ നടപടി ഉണ്ട്.

Print Screen കീബോർഡ് കീ സ്ക്രീൻ ക്യാപ്ചർ ചെയ്ത് ക്ലിപ്ബോർഡ് അത് ഇടുന്നു. ഈ കമാൻഡ് Screenshot ഡയലോഗ് വിൻഡോയിൽ “taking a screenshot of the entire screen” അതേ നടപടി ഉണ്ട്. Alt+Print Screen കീ കോമ്പിനേഷൻ അതിന്റെ അലങ്കാരങ്ങൾ കൊണ്ട് സ്ക്രീനിൽ സജീവ ജാലകം പിടിച്ചുപറ്റി ക്ലിപ്ബോർഡ് അത് ഇടുന്നു.

2.3.4. സ്ക്രീൻഷോട്ട്

ചിത്രം 16.8. The “Screenshot” window


Screenshot കമാൻഡ് രണ്ടു ഭാഗങ്ങൾ ഒരു ഡയലോഗ് തുറക്കുന്നു:

ഏരിയ
Take a screenshot of a single window

മൗസ് സൂചികയെ ഒരു ക്രോസ് മാറുന്നു. നിങ്ങൾ പിടിച്ചടക്കാൻ ആഗ്രഹിക്കുന്ന ഇമേജ് വിൻഡോയിൽ ക്ലിക്ക്. ഒരു പുതിയ ഇമേജ് സൃഷ്ടിക്കപ്പെട്ടിരിക്കുന്നത്. Include window decoration ഓപ്ഷൻ പരിശോധിക്കാതെ എങ്കിൽ, ശീർഷക ബാറും ഇമേജിന് ചുറ്റും നീല ഫ്രെയിം നീക്കംചെയ്യും.

Take a screenshot of the entire screen

നിങ്ങൾ ഒരു പോപ്പ് മെനു പിടിച്ചടക്കാൻ ഇത് ഉപകാരപ്പെടുന്നതാണ് ആണ്. ഒരു കാലതാമസം നിങ്ങൾ സ്ഥാനം പോപ്പ് മെനുവിൽ pull സമയം അങ്ങനെ, പിന്നെ അത്യാവശ്യമാണ്.

Include mouse pointer ഐച്ഛികം എങ്കിൽ, മൌസ് പോയിന്റർ അതിന്റെ ഐക്കൺ കൂടെ വരുന്നു പുറമേ എടുത്തത്. മൗസ് സൂചികയെ ഒരു പ്രത്യേക പാളിയായി ജനിക്കുന്നു. അങ്ങനെ നിങ്ങൾ ചിത്രം മറ്റൊരു സ്ഥലത്തേക്കു നീക്കാനാകും.

Select a region to grab

മൗസ് സൂചികയെ ഒരു ക്രോസ് മാറുന്നു. ചിത്രം വിൻഡോയിൽ ഒരു ചതുരാകൃതിയിലുള്ള നിര സൃഷ്ടിക്കാൻ ക്ലിക്ക് ചെയ്ത് ഡ്രാഗ് ചെയ്യുക. ഈ തിരഞ്ഞെടുപ്പ് ഒരു പുതിയ ഇമേജ് ആയി തുറക്കപ്പെടും. അതിന്റെ വലിപ്പം നിരക്കു വലുപ്പം ചേർച്ചപ്പെടുത്തുകയാണ്.

കാലതാമസം

മുഴുവൻ സ്ക്രീനും ഒരു സ്ക്രീൻഷോട്ട് എടുക്കൽ ചെയ്യുമ്പോൾ സ്ക്രീൻ ഈ കാലതാമസം ശേഷം ജനിക്കുന്നു. മറ്റ് സന്ദർഭങ്ങളിൽ, മൌസ് പോയിന്റർ ഈ കാലതാമസം ശേഷം ഒരു ക്രോസ് തിരിയുന്നു.
