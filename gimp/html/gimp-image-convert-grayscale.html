6.5. Grayscale mode

താങ്കൾക്ക് 0 255 (കറുത്ത) (വെള്ള), ചാര 256 തലങ്ങളിൽ ഉപയോഗിച്ച് ഗ്രേസ്കെയിൽ നിങ്ങളുടെ ചിത്രം പരിവർത്തനം Grayscale കമാൻഡ് ഉപയോഗിക്കാം.

6.5.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Image → Mode → Grayscale.
