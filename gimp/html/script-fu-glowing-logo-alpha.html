16. ലോഗോ ഫിൽട്ടറുകൾ ആൽഫ  

16.15. ഹോട്ട് തിളങ്ങുന്ന

16.15.1. പൊതു അവലോകനം

ചിത്രം 17.405. “Glowing Hot” ഫിൽട്ടർ ഉദാഹരണം

“Glowing Hot” ഫിൽട്ടർ

“Glowing Hot” ലോഗോ


ഈ ഫിൽട്ടർ (അല്ലാത്തതോ സുതാര്യമായ പിക്സൽ നിർവചിക്കപ്പെട്ട സജീവ പാളി ഈ പ്രദേശങ്ങളിൽ ആണ്) ആൽഫ ഒരു മരീചിക ഹോട്ട് മെറ്റൽ പ്രഭാവം ചേർക്കുന്നു.

ഫിൽട്ടർ ഒരു തിളങ്ങുന്ന ടെക്സ്റ്റ് ലോഗോ സൃഷ്ടിക്കുന്ന “Glowing Hot” സ്ക്രിപ്റ്റ് (ചിത്രം വിൻഡോയിൽ File → Create → Logos → Glowing Hot), (മുകളിൽ കാണുക) കഥകളിയുടെ.

മറ്റൊരു മെറ്റൽ താപനില പ്രതിനിധീകരിക്കുന്ന ഓരോ നിറം - - ഫിൽട്ടർ ചുവന്ന-ചൂടുള്ള ഒരു മഞ്ഞ-ഹോട്ട് ഒരു വെളുത്ത-ചൂടുള്ള പ്രദേശം അനുകരിക്കുന്നു; ആൽഫ ന്റെ ഔട്ട്ലൈൻ തിളങ്ങുന്ന കൂടി.

മുന്നറിയിപ്പു്

ചിത്രം എല്ലായ്പ്പോഴും സജീവ പാളി വലിപ്പത്തിന് വലുപ്പം മാറ്റും.

16.15.2. ഫിൽട്ടർ സജീവമാക്കുക

ഈ ഫിൽട്ടർ Filters → Alpha to Logo → Glowing Hot കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ കാണപ്പെടുന്നത്.

സജീവ പാളി ഒരു ആല്ഫാ ചാനൽ ഉണ്ട് എങ്കിൽ അരിപ്പ മാത്രമേ പ്രവർത്തിക്കൂ. അല്ലാത്തപക്ഷം, മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്.

16.15.3. ഓപ്ഷനുകൾ

ചിത്രം 17.406. “Glowing Hot” ഓപ്ഷനുകൾ


പ്രഭാവം വലുപ്പം (പിക്സൽ * 3)

ഈ യഥാർത്ഥത്തിൽ “Glowing Hot” ലോഗോ ഫോണ്ട് വലിപ്പം തീരുമാനം. മൂല്യം ആൽഫാ, ചുവന്ന മഞ്ഞ, വെള്ള നിറഞ്ഞു മുമ്പ് ഭദ്രമാക്കുന്ന അതിർത്തിയിൽ (സി.എഫ് വിഭാഗം 4.9, “തൂവല്”) വലുപ്പം കണക്കാക്കുവാൻ ഉപയോഗിക്കുന്നു. ഈ തൂവലുകളും നിറങ്ങൾ ഹോട്ട് മെറ്റൽ ഉണ്ടാക്കുനതിനു.

ചിത്രം 17.407. ഇഫക്റ്റ് വലുപ്പം ഉദാഹരണങ്ങൾ

ഇഫക്റ്റ് വലുപ്പം 50

ഇഫക്റ്റ് വലുപ്പം 350


പശ്ചാത്തല നിറം

ഈ “Background” പാളി നിറയ്ക്കാൻ ഉപയോഗിക്കുന്ന നിറം; അതു കറുത്ത (7,0,20) സ്ഥിര. നിങ്ങൾക്ക് മറ്റൊരു നിറം തിരഞ്ഞെടുക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ ബട്ടണിൽ, ഒരു കളർ സെലക്ടർ തുറക്കാൻ ക്ലിക്കുചെയ്യുക.

16.15.4. ഫിൽറ്റർ വിശദാംശങ്ങൾ

തിളങ്ങുന്ന പ്രഭാവം (ചുവന്ന-ചൂടുള്ള, മഞ്ഞ-ഹോട്ട്, വെളുത്ത-ചൂടുള്ള പ്രദേശം) സൃഷ്ടിക്കാൻ ആൽഫ കടൽപുറത്തെ തുടർന്ന് കടൽപുറത്തെ പ്രദേശത്ത് തൂവൽ വലുപ്പത്തിലും കളർ തീവ്രത കുറയുകയും ചുവന്ന നിന്ന് വൈറ്റ്, അതാത് നിറം നിറഞ്ഞിരിക്കുന്നു.

താഴെ ദൃഷ്ടാന്തം “hot metal” നിറങ്ങളും “Effect size” ശതമാനത്തിൽ വരുന്ന ഭദ്രമാക്കുന്ന അതിർത്തിയിൽ വീതി (ഈ മൂല്യങ്ങൾ അരിപ്പ യഥാർത്ഥത്തിൽ ഉപയോഗിക്കുന്നു ആകുന്നു) കാണിക്കുന്നു.

ചിത്രം 17.408. ഇഫക്റ്റ് വലുപ്പം

ഹോട്ട് മെറ്റൽ നിറങ്ങൾ അവരുടെ ബന്ധു തൂവല് വലുപ്പത്തിലും തിളങ്ങുന്ന


ഉദാഹരണത്തിന് ചിത്രങ്ങൾ ൽ ആൽഫാ ന്റെ ഔട്ട്ലൈൻ തിളങ്ങുന്ന കൂടി കാണാൻ കഴിയും. ഈ എവിടെ layer mode overlay ലേക്ക് സെറ്റ് ടോപ് പാളി പോലെ കറുത്ത നിറഞ്ഞ ഒരു ആൽഫ, കൂടെ കൈവരിക്കുന്നു. ഒരു കറുത്ത ഓവർലേ പാളി ഉപയോഗിക്കുന്നു ശുദ്ധമായ വെള്ള മാറില്ല, പക്ഷേ ഔട്ട്ലൈൻ ദൃശ്യമാവും ആൽഫ ന്റെ നാനാഭാഗങ്ങളിലുമുണ്ടായിരിക്കും നേരിയ നിറങ്ങൾ കറുപ്പിക്കുന്നു.
