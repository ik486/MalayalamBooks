5.6. കയറ്റുമതി ഫയൽ

GIMP -2,8 വച്ച് സംരക്ഷിക്കുക കമാൻഡ് എക്സ്.സി.എഫ് ഫോർമാറ്റിൽ ചിത്രങ്ങൾ സംരക്ഷിക്കുന്നു. കയറ്റുമതി കമാൻഡ് ഇപ്പോൾ വിവിധ ഫയൽ ഫോർമാറ്റുകൾ ഇമേജുകൾ സംരക്ഷിക്കാൻ ഉപയോഗിക്കുന്നു.

You can access to this command through File → Export As..., or from the keyboard by using the shortcut Ctrl+Shift+E.

ചിത്രം 15.77. കയറ്റുമതി ഇമേജ് ഡയലോഗ്


5.6.1. കയറ്റുമതി ഇമേജ് ഡയലോഗ്

ഈ ഫയൽ ബ്രൌസർ ഉപയോഗിച്ച് നിങ്ങൾക്ക് പേര് നേരിട്ട് പെട്ടിയിൽ ഫയൽനാമം വ്യാപന എഡിറ്റ് (സ്വതവേ “Untitled.png” ആണ്) അല്ലെങ്കിൽ പേര് ലിസ്റ്റിൽ ഒരു ഫയൽ തിരഞ്ഞെടുത്ത് കഴിയും. നിങ്ങൾക്ക് Save in Folder ഇമേജ് ലക്ഷ്യസ്ഥാനം പരിഹരിക്കണം. ആവശ്യമെങ്കിൽ നിങ്ങൾക്ക് പുതിയ ഒരു ഫോൾഡർ സൃഷ്ടിക്കാൻ കഴിയും.

ഫയൽ തെരഞ്ഞെടുക്കു. നിങ്ങൾ ഈ ഉപാധി വികസിപ്പിക്കുകയും, നിങ്ങൾ നിങ്ങളുടെ ഫയലിന് ഡ്രോപ്പ്-ഡൌൺ പട്ടികയിൽ ഒരു വിപുലീകരണം തിരഞ്ഞെടുക്കാം:

ഫയൽ ഫോർമാറ്റുകൾ ഡയലോഗുകൾ ??? വിവരിച്ചിരിക്കുന്നു.

5.6.2. എക്സ്പോർട്ടുചെയ്യുന്നു

എപ്പോൾ ഫയൽ നാമം ലക്ഷ്യസ്ഥാനം സജ്ജമാക്കിയിരിക്കുന്ന, Export ക്ലിക്ക് ചെയ്യുക. ഈ വ്യക്തമാക്കിയ ഫയൽ ഫോർമാറ്റ് വേണ്ടി കയറ്റുമതി ഡയലോഗ് തുറക്കുന്നു.

നിങ്ങൾ ഒരു നോൺ-എക്സ്.സി.എഫ് ഫയൽ ലഭ്യമായ, ഒരു പുതിയ ഇനം നിങ്ങൾക്ക് യഥാർത്ഥ ഫയൽ എഴുതുന്നതിന്, അതേ ഫോർമാറ്റിൽ ഫയൽ കയറ്റുമതി ചെയ്യുന്നതിന് അനുവദിക്കുന്നു, ഫയൽ മെനുവിലെ ലഭ്യമാകുന്നു. PLACEHOLDER,-100407

If you modify an image that you already have exported, the Export command in File menu is changed, allowing you to export file again in the same format.
