3.4. നിങ്ങളുടെ ആദ്യ സ്ക്രിപ്റ്റ്-ഫു സ്ക്രിപ്റ്റ്
3. ഒരു സ്ക്രിപ്റ്റ്-ഫു ട്യൂട്ടോറിയൽ  

3.4. നിങ്ങളുടെ ആദ്യ സ്ക്രിപ്റ്റ്-ഫു സ്ക്രിപ്റ്റ്

നിങ്ങളുടെ ശ്വാസം നിർത്താൻ മനസ്സിലാവുന്നതും ആവശ്യമില്ല? ഇല്ല? നിങ്ങളുടെ ആദ്യ സ്ക്രിപ്റ്റ്-ഫു സ്ക്രിപ്റ്റ് - പിന്നെ നിങ്ങളുടെ നാലാമത്തെ പാഠം മുന്നോട്ടുപോകും ചെയ്യട്ടെ.

3.4.1. ഒരു ടെക്സ്റ്റ് ബോക്സ് സ്ക്രിപ്റ്റ് സൃഷ്ടിക്കുന്നു

ഞാൻ GIMP പ്രകടനം ഏറ്റവും സാധാരണമായ ഓപ്പറേഷൻസ് ഒന്ന് ഒരു വെബ് പേജ് അതിൽ ചില ടെക്സ്റ്റ്, ഒരു ലോഗോ അല്ലെങ്കിൽ ഒരു പെട്ടിക്ക് സൃഷ്ടിക്കുന്നു. എന്നിരുന്നാലും, നിങ്ങൾ തികച്ചും നിങ്ങളെ പുറത്തു ആരംഭിക്കുമ്പോൾ പ്രാരംഭ ഇമേജുകൾ എങ്ങനെ വലിയ അറിയില്ല. ടെക്സ്റ്റ് നിങ്ങൾ ആഗ്രഹിക്കുന്ന ഫോണ്ട്, ഫോണ്ട് വലുപ്പം എത്ര സ്ഥലം നിറെക്കും അറിയില്ല.

സ്ക്രിപ്റ്റ്-ഫു മാസ്റ്റർ (വിദ്യാർത്ഥി) വേഗത്തിൽ ഈ പ്രശ്നം എളുപ്പത്തിൽ സ്ക്രിപ്റ്റ്-ഫു പരിഹരിച്ചിട്ടുണ്ട് ഒപ്പം ഓട്ടോമേറ്റഡ് എന്നു തിരിച്ചറിയണം ചെയ്യും.

നാം അതുകൊണ്ട് ശരിയായി ഉപയോക്തൃ ചെലവായ ടെക്സ്റ്റ് ഒരു വരി ചുറ്റും കേബിളിന്റെ fit വലുപ്പംമാറ്റിയത് ഒരു ചിത്രം സൃഷ്ടിക്കുന്ന സ്ക്രിപ്റ്റും വിളിച്ചു ടെക്സ്റ്റ് ബോക്സ്, സൃഷ്ടിക്കും. ഞങ്ങൾ ഉപയോക്തൃ ഫോണ്ട്, ഫോണ്ട് സൈസ്, ടെക്സ്റ്റ് നിറം തിരഞ്ഞെടുക്കാൻ അനുവദിക്കും.

3.4.2. എഡിറ്റിംഗ് നിങ്ങളുടെ സ്ക്രിപ്റ്റുകൾ സംഭരിക്കാൻ

ഇപ്പോൾ വരെ, ഞങ്ങൾ സ്ക്രിപ്റ്റ്-ഫു കൺസോളിൽ പ്രവർത്തിക്കുന്നതാണ്. എന്നാൽ, ഞങ്ങൾ എഡിറ്റിംഗ് സ്ക്രിപ്റ്റ് ടെക്സ്റ്റ് ഫയലുകൾ മാറുക പോകുന്നു.

നിങ്ങളുടെ സ്ക്രിപ്റ്റുകൾ സ്ഥാപിക്കുക എവിടെ മുൻഗണന ഒരു കാര്യം - നിങ്ങൾ GIMP ന്റെ സ്ഥിര സ്ക്രിപ്റ്റ് ഡയറക്ടറി ആക്സസ് ഇല്ലെങ്കിൽ, നിങ്ങൾ അവിടെ സ്ക്രിപ്റ്റുകൾ സ്ഥാപിക്കാൻ കഴിയും. എന്നിരുന്നാലും, ഞാൻ ഫാക്ടറി-ഇൻസ്റ്റാൾ സ്ക്രിപ്റ്റുകളിൽ നിന്നും വേർതിരിച്ചു അവരെ പ്രമാണിച്ചു എന്റെ സ്വന്തം സ്ക്രിപ്റ്റ് ഡയറക്ടറിയിൽ എന്റെ വ്യക്തിപരമായ സ്ക്രിപ്റ്റുകൾ സൂക്ഷിക്കുന്നതിനുമുള്ള ഇഷ്ടപ്പെടുന്നത്.

In the .gimp-2.8 directory that GIMP made off of your home directory, you should find a directory called scripts. GIMP will automatically look in your .gimp-2.8 directory for a scripts directory, and add the scripts in this directory to the Script-Fu database. You should place your personal scripts here.

3.4.3. ബാരെ എസ്സൻഷ്യൽസ്

ഓരോ സ്ക്രിപ്റ്റ്-ഫു സ്ക്രിപ്റ്റ് ഏത് സ്ക്രിപ്റ്റിന്റെ പ്രധാന പ്രവർത്തനമല്ല കുറഞ്ഞത് ഒരു ഫംഗ്ഷൻ, നിഷ്കർഷിക്കുന്നു. നിങ്ങൾ ജോലി ചെയ്യാൻ ഇവിടെയാണ്.

ഓരോ സ്ക്രിപ്റ്റ് അതിലൂടെ GIMP അത് ആക്സസ് ചെയ്യാൻ കഴിയും, നടപടിക്രമങ്ങൾ ഡാറ്റാ രജിസ്റ്റർ ചെയ്യണം.

ആദ്യം നാം പ്രധാന നിര്വ്വചിക്കുമ്പോള് കാണാം:

        (define (script-fu-text-box inText inFont inFontSize inTextColor))

ഇവിടെ നാം പിന്നീട് ചില ടെക്സ്റ്റ് അനുയോജ്യമായി നാല് പരാമീറ്ററുകൾ, ഒരു ഫോണ്ട്, ഫോണ്ട് വലുപ്പം, ടെക്സ്റ്റ് കളർ എടുക്കും എന്നും script-fu-text-box എന്ന ഒരു പുതിയ ഫങ്ഷൻ നിർവ്വചിക്കുകയുംഎന്നിട്ട് ഞങ്ങൾ. ഫങ്ഷൻ നിലവിൽ ശൂന്യമാണ് അപ്രകാരം ഒന്നും. ഇതുവരെ വളരെ നല്ല - ഒന്നും പുതിയ ഒന്നും ഫാൻസി.

3.4.4. കൺവെൻഷനുകൾ പേരുനൽകുന്നത്

പദ്ധതി ന്റെ പേരിടൽ കൺവെൻഷനുകൾ ഞാൻ ധർമ്മ നാമകരണ നിങ്ങൾ പിന്തുടർന്ന ഏത് ഹൈഫനുകളോ, കൂടെ ചെറിയ അക്ഷരങ്ങൾ താൽപര്യം തോന്നുന്ന. എങ്കിലും, ഞാൻ പരാമീറ്ററുകളുപോഗിച്ച് കൺവെൻഷനിൽ വിട്ടുമാറി ഞങ്ങൾ. ഞാൻ എന്റെ പാരാമീറ്ററുകൾ ചരങ്ങളും കൂടുതൽ വിവരണാത്മക പേരുകൾ ഇഷ്ടമായി, അങ്ങനെ ഞാൻ വേഗം അതിൽ സൃഷ്ടിച്ചു പകരം അവർ സ്ക്രിപ്റ്റ് കടന്നുപോയോരു മൂല്യങ്ങൾ മുൻപരിചയം കാണാം ഇങ്ങനെ പാരാമീറ്ററുകൾ വരെ പ്രിഫിക്സ് "ലെ" ചേർക്കുക. ഞാൻ സ്ക്രിപ്റ്റ് ഉള്ളിൽ നിർവചിക്കപ്പെട്ട വേരിയബിളുകള്ക്കായി പ്രിഫിക്സ് ഉപയോഗിക്കുന്ന "".

ഇത് വന്നതുകൊണ്ടു അവർ നടപടിക്രമങ്ങൾ ഡാറ്റാബേസ് ലിസ്റ്റുചെയ്ത കാണുമ്പോൾ എല്ലാവരും സ്ക്രിപ്റ്റ്-ഫു കീഴിൽ നിങ്ങൾക്ക് ഫംഗ്ഷനുകളും ലിസ്റ്റിംഗ് ഉള്ളപ്പോൾ കാണിക്കും, NAME നിങ്ങളുടെ സ്ക്രിപ്റ്റ് പ്രവർത്തിക്കുന്നു script-fu-abc GIMP കൺവെൻഷൻ തുടർന്ന്. ഇത് പ്ലഗ്-ഇന്നുകളെ നിന്ന് അവരെ വേർതിരിച്ചറിയാൻ സഹായിക്കുന്നു.

3.4.5. ഫംഗ്ഷൻ രജിസ്റ്റർചെയ്യുന്നു

ഇപ്പോൾ, GIMP കൂടെ ചടങ്ങിൽ രജിസ്റ്റർ ചെയ്യട്ടെ. ഈ ഫങ്ഷൻ script-fu-register വിളിക്കുന്നു ഇത് ചെയ്തത്. GIMP ഒരു ലിപിയിലുള്ള വായിക്കപ്പെടുന്നു, അത് നടപടിക്രമങ്ങൾ ഡാറ്റാ സ്ക്രിപ്റ്റ് രജിസ്റ്റർ ചെയ്ത, ഈ ഫംഗ്ഷൻ നടത്തും. നിങ്ങളുടെ സ്ക്രിപ്റ്റിലെ ഇഷ്ടമുള്ളേടത്ത് ഈ ഫംഗ്ഷൻ കോൾ കഴിയില്ല, എന്നാൽ സാധാരണയായി അത് എന്റെ മറ്റ് കോഡ് ശേഷം അവസാനം സ്ഥാപിക്കുക.

ഇവിടെ (ഞാൻ ഒരു മിനിറ്റിനകം അതിന്റെ എല്ലാ പരാമീറ്ററുകൾ വിശദീകരിക്കും) ഈ ചടങ്ങിൽ രജിസ്റ്റര് ലിസ്റ്റിംഗ് ഇവിടെയുണ്ട്:

  (script-fu-register
    "script-fu-text-box"                        ;func name
    "Text Box"                                  ;menu label
    "Creates a simple text box, sized to fit\
      around the user's choice of text,\
      font, font size, and color."              ;description
    "Michael Terry"                             ;author
    "copyright 1997, Michael Terry;\
      2009, the GIMP Documentation Team"        ;copyright notice
    "October 27, 1997"                          ;date created
    ""                     ;image type that the script works on
    SF-STRING      "Text"          "Text Box"   ;a string variable
    SF-FONT        "Font"          "Charter"    ;a font variable
    SF-ADJUSTMENT  "Font size"     '(50 1 1000 1 10 0 1)
                                                ;a spin-button
    SF-COLOR       "Color"         '(0 0 0)     ;color variable
  )
  (script-fu-menu-register "script-fu-text-box" "<Image>/File/Create/Text")

നിങ്ങളുടെ സ്ക്രിപ്റ്റ് ഡയറക്ടറിയിൽ ഒരു .scm സഫിക്സ് ഉപയോഗിച്ച് ഒരു ടെക്സ്റ്റ് ഫയലിൽ ഈ ഫംഗ്ഷനുകൾ സംരക്ഷിക്കാൻ എങ്കിൽ, Filters → Script-Fu → Refresh Scripts തിരഞ്ഞെടുക്കാൻ ഈ പുതിയ സ്ക്രിപ്റ്റ് File → Create → Text → Text Box ദൃശ്യമാകും.

നിങ്ങൾ ഈ പുതിയ സ്ക്രിപ്റ്റ് പ്രാർത്ഥിക്കുന്ന പക്ഷം അതു തീർച്ചയായും, ഒന്നും ചെയ്യില്ല, എന്നാൽ നിങ്ങൾ സ്ക്രിപ്റ്റ് രജിസ്റ്റർ ചെയ്യുമ്പോൾ നിങ്ങൾ സൃഷ്ടിച്ച നിർദ്ദേശങ്ങൾ കാണാൻ കഴിയും (ഞങ്ങൾ ചെയ്തു എന്നതിനെക്കുറിച്ചുള്ള കൂടുതൽ വിവരങ്ങൾ അടുത്ത മൂടിയിരിക്കുന്നു).

അവസാനമായി, നടപടിക്രമം ബ്രൌസർ (Help → Procedure Browser) പ്രാർത്ഥിക്കുന്ന പക്ഷം നിങ്ങൾ ഞങ്ങളുടെ സ്ക്രിപ്റ്റ് ഇപ്പോൾ ഡാറ്റാബേസ് പ്രത്യക്ഷപ്പെടുന്ന ശ്രദ്ധിക്കണം.

3.4.6. സ്ക്രിപ്റ്റ് രജിസ്റ്റർ നടപടികൾ

GIMP നമ്മുടെ സ്ക്രിപ്റ്റ് രജിസ്റ്റർ ചെയ്യുന്നതിന്, ഞങ്ങൾ ഫങ്ഷൻ script-fu-register, ഓരോ പാരാമീറ്റർ ഒരു വിവരണം, സ്ഥിര മൂല്യം സഹിതം വിളിക്കും ഏഴ് ആവശ്യമായ പാരാമീറ്ററുകൾ പൂരിപ്പിക്കുക ഞങ്ങളുടെ സ്ക്രിപ്റ്റിന്റെ സ്വന്തം പാരാമീറ്ററുകൾ ചേർക്കുക.

ആവശ്യമായ പാരാമീറ്ററുകൾ
