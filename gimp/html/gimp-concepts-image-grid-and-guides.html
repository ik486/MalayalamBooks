2. ഗ്രിഡുകളുടെ ഗൈഡുകൾ

ചിലപ്പോൾ നിങ്ങൾക്ക് നീ വളരെ കൃത്യമായി ഒരു ഇമേജ് എന്തെങ്കിലും സ്ഥാപിക്കുക ചെയ്യേണ്ട പല തവണ സംഭവിക്കും, അത് ഒരു മൗസ് ഉപയോഗിച്ച് എളുപ്പമല്ല അല്ലെന്ന് കണ്ടെത്തി ഉണ്ടായിരിക്കും. പലപ്പോഴും കീബോർഡ് ആരോ കീകൾ ഉപയോഗിച്ച് (നിങ്ങൾ Shift കീ ഹോൾഡ് എങ്കിൽ ഒരു സമയത്ത് ബാധിത വസ്തുവുമായി പിക്സൽ, അല്ലെങ്കിൽ 25 പിക്സൽ നീക്കാൻ ഏത്) മികച്ച ഫലങ്ങൾ ലഭിക്കും കഴിയും, പക്ഷേ GIMP പുറമേ സ്ഥാനനിർണ്ണയം മറ്റ് രണ്ടു എയ്ഡ്സ് നൽകുന്നു എളുപ്പം: ഗ്രിഡുകൾ ഗൈഡുകൾ.

ചിത്രം 12.27. ചുവടെയുള്ള ഉദാഹരണങ്ങൾ ഉപയോഗിക്കുന്ന ചിത്രം


2.1. ചിത്രം ഗ്രിഡ്

ചിത്രം 12.28. സ്ഥിര ഗ്രിഡ് ചിത്രം


ഓരോ ചിത്രം ഒരു ഗ്രിഡ് ഉണ്ട്. ഇത് എപ്പോഴും, പക്ഷേ സഹജമായി അതു നിങ്ങൾ ഇമേജ് മെനുവിൽ View → Show Grid ഓണാക്കാനും വഴി അത് സജീവമാക്കുന്നതിന് വരെ അല്ല ദൃശ്യമാണ്. നിങ്ങൾ ഗ്രിഡുകൾ പലപ്പോഴും സാന്നിധ്യം ആഗ്രഹിക്കുന്നുണ്ടെങ്കിൽ, പ്രിഫറന്സ് ഡയലോഗ് Image Window Appearance പേജിലെ "ഗ്രിഡ് കാണിക്കുക" പരിശോധിച്ചുകൊണ്ട് സ്ഥിര പെരുമാറ്റം മാറ്റാനാകും. (സാധാരണ മോഡ്, ഫുൾസ്ക്രീൻ മോഡ് പ്രത്യേക ക്രമീകരണങ്ങൾ ഉണ്ട് ശ്രദ്ധിക്കുക.)

സ്ഥിര ഗ്രിഡ് ഭാവം, നിങ്ങൾ GIMP ഇൻസ്റ്റോൾ സജ്ജമാക്കാൻ, പ്ലസ് ആകൃതിയിലുള്ള കറുത്ത crosshairs ഗ്രിഡ് ലൈനുകൾ ഇരുവരും ലംബമായി ഒപ്പം തിരശ്ചീനമായി ഓരോ 10 പിക്സൽ അകലത്തിലുള്ള കൂടെ ഗ്രിഡ് ലൈൻ മുൻഗണ ഉൾക്കൊള്ളുന്നു. നിങ്ങൾ മുൻഗണനകൾ ഡയലോഗിന്റെ Default Image Grid പേജ് ഉപയോഗിക്കുന്ന സ്ഥിര ഗ്രിഡ് ഇഷ്ടാനുസൃതമാക്കാം. നിങ്ങൾ മാത്രം നിലവിലെ ചിത്രം ഗ്രിഡ് ഭാവം മാറ്റണമെങ്കിൽ, നിങ്ങൾക്ക് ഇമേജ് മെനുവിൽ നിന്നും Image → Configure Grid തിരഞ്ഞെടുത്താൻ അങ്ങനെ ചെയ്യാൻ കഴിയും: ഈ Configure Grid ഡയലോഗിന് നൽകുന്നു.

ചിത്രം 12.29. ഒരു വ്യത്യസ്ത ഗ്രിഡ് രീതിയിൽ


മാത്രമല്ല ഒരു ഗ്രിഡ് ദൂരവും സ്പേഷ്യൽ ബന്ധങ്ങൾ വിധിക്കുന്നു സഹായകരമാണ്, അത് കൃത്യമായി ഗ്രിഡ് കാര്യങ്ങൾ വിന്യസിക്കുക നിങ്ങൾക്ക് അനുമതി കഴിയും, നിങ്ങൾ ഇമേജ് മെനുവിൽ View → Snap to Grid ടോഗിൾ എങ്കിൽ: ഈ പോയിന്റർ എബൌട്ട് ഏതെങ്കിലും ഗ്രിഡ് ലൈൻ ലേക്ക് "പാവോ" കാരണമാകുന്ന ഒരു നിശ്ചിത ദൂരം ഉള്ളിൽ. നിങ്ങൾ മുൻഗണനകൾ ഡയലോഗിന്റെ Tool Options പേജിലെ "സ്നാപ്പ് ദൂരം" ക്രമീകരണം വഴി സ്നാപ്പ് ദൂരം ഉമ്മരപ്പടി ഇഷ്ടമുള്ളരീതിയിലാക്കാം പക്ഷെ അധികമാളുകളും 8 പിക്സൽ സ്ഥിര മൂല്യം തികച്ചും തോന്നുന്നില്ല. (ഗ്രിഡിലെ ദൃശ്യമല്ല പോലും. എന്നിരിക്കിലും, നിങ്ങളെ ഇത് ചെയ്യണമെന്ന് എന്തുകൊണ്ടാണ് സങ്കൽപിക്കാൻ എളുപ്പമാണ് ഗ്രിഡിലേക്ക് സ്നാപ്പ് തികച്ചും സാധ്യമാണ് ശ്രദ്ധിക്കുക.)
