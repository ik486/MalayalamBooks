7.51. Layer to Image Size

Layer to Image Size കമാൻഡ് ചിത്രം കാര്യത്തിൽ പാളി ഉള്ളടക്കങ്ങളെ നീക്കാതെതന്നെ, ഇമേജ് അതിരുകൾ പൊരുത്തപ്പെടുന്നില്ല ലെയർ അതിരുകൾ മാറ്റുന്നു.

7.51.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Layer to Image Size.
