PLACEHOLDER,-100440: പണിസഞ്ചിയിൽ FG / ബി.ജി. പ്രദേശത്ത് നിന്ന് വിളിച്ചു ഡയലോഗ് ചിത്രം മെനുവിൽ നിന്നും വിളിച്ച അപേക്ഷിച്ച് വ്യത്യസ്തമാണ്

3.1.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Colors” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • ഒരു ഇമേജ് മെനുവിൽ നിന്ന്: Windows → Dockable Dialogs → Colors;

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Colors തിരഞ്ഞെടുത്ത്,

  • ഉപകരണബോക്സിലേക്ക് നിന്ന്: നിലവിലെ മുൻവശം അല്ലെങ്കിൽ പശ്ചാത്തല നിറം ക്ലിക്ക് ചെയ്യുക.

Windows മെനുവിൽ, കുറഞ്ഞത് ഒരു ഡയലോഗ് തുറന്നുതന്നെയിരിക്കുന്ന മാത്രമേ ഉണ്ട് detached windows ഒരു ലിസ്റ്റ് ഉണ്ട്. Windows → Colors: ഈ സാഹചര്യത്തിൽ, ഇമേജ്-മെനുവിൽ നിന്നും “Colors” ഡയലോഗ് ഉണ്ടാകാം.
