5.7. Navigation Window

Navigation Window കമാൻഡ് navigation window തുറക്കുന്നു. ഈ എളുപ്പത്തിൽ സൂം ലെവലുകൾ സജ്ജമാക്കാനും ചിത്രത്തിന്റെ ദൃശ്യമായ ഭാഗങ്ങൾ നീക്കാൻ ചിത്രത്തിന് നാവിഗേറ്റുചെയ്യുന്നതിനുള്ള അനുവദിക്കുന്നു. കണ്ടെത്താവുന്നതാണ് Navigation dialog അധ്യായത്തിൽ അത് ഉപയോഗിച്ച് കൂടുതൽ വിവരങ്ങൾ.

5.7.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through View → Navigation Window,

  • You can also access it more rapidly by clicking on the icon in the lower right corner of the image window.
