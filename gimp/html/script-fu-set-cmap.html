8.20. Colormap സജ്ജമാക്കുക

ചിത്രം 16.161. “Set Colormap” വിൻഡോ


ഈ കമാൻഡ് നിങ്ങളുടെ ഇൻഡെക്സ്ഡ് ചിത്രത്തിന്റെ കളർ മാപ്പ് മാറ്റി പകരം മറ്റൊരു പാലറ്റ് തിരഞ്ഞെടുക്കുക അനുവദിക്കുന്ന ഒരു ഡയലോഗ് തുറക്കുന്നു. ആദ്യം പാലറ്റ് സെലക്ടർ തുറക്കാൻ (ഇതുവരെ നിങ്ങളുടെ ചിത്രത്തിൻറെ കളർ മാപ്പ് അല്ലാത്ത) നിലവിലെ പാലറ്റ് പേര് ബട്ടൺ ലെ ക്ലിക്കിൽ:

ചിത്രം 16.162. “Palette Selection” ഡയലോഗ്


നിങ്ങൾ പിടികൂടാനുള്ള പാലറ്റ് തിരഞ്ഞെടുത്ത ഒരിക്കൽ, ഇമേജ് colormap മാറ്റിസ്ഥാപിക്കാൻ “Set Palette” ഡയലോഗിൽ OK ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.

8.20.1. കമാൻഡ് സജീവമാക്കുക

ഈ കമാൻഡ് Colors → Map → Set Colormap കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ കാണപ്പെടുന്നത്.
