3.16. ഡോഡ്ജ് / ബേൺ

ചിത്രം 14.97. Dodge tool


ഡോഡ്ജ് അല്ലെങ്കിൽ ബേൺ ഉപകരണം നിങ്ങളുടെ സ്വരൂപത്തിൽ നിറങ്ങൾ ലഘൂകരിച്ചു അല്ലെങ്കിൽ ഇരുണ്ട് നിലവിലെ ബ്രഷ് ഉപയോഗിക്കുന്നു. മോഡ് പിക്സൽ ഏത് തരം ബാധിച്ചു തീരുമാനിക്കും.

3.16.1. ടൂൾ സജീവമാക്കുന്നു

ഉപകരണം സജീവമാക്കുന്നതിന് വിവിധ സാധ്യതകളും ഉണ്ട്:

  • From the image-menu: Tools → Paint Tools → Dodge / Burn.

  • The Tool can also be called by clicking the tool icon: ,

  • or by using the Shift+D keyboard shortcut.

3.16.2. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

Ctrl,

ഡോഡ്ജ് അല്ലെങ്കിൽ ചുട്ടുകളയേണം തരം തമ്മിൽ ടോഗിൾ. Ctrl റിലീസ് ചെയ്യുന്നത് വരെ തരം സ്വിച്ച്ഡ് തുടരും.

മാറ്റം

Shift ഡോഡ്ജ് സ്ഥലമെല്ലാം അല്ലെങ്കിൽ ഉപകരണം ബേൺ വര മോഡിലേക്ക്. ഹോൾഡിംഗ് Shift ക്ലിക്ക് ചെയ്യുമ്പോൾ നേരായ വരിയിൽ Button1 ഡോഡ്ജ് അല്ലെങ്കിൽ ചുട്ടുകളയും. തുടർച്ചയായ ക്ലിക്കുകൾ ഡോഡ്ജ് തുടരാൻ അല്ലെങ്കിൽ അവസാനത്തെ വരിയുടെ അവസാനം നിന്നും ആ രേഖാഖണ്ഡം കത്തിഎരിയുന്നവരത്രെ.

3.16.3. ഓപ്ഷനുകൾ

ചിത്രം 14.98. “Dodge/Burn” tool options


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

Opacity; Brush; Dynamics; Dynamics Options; Apply Jitter; Hard Edge
പല അല്ലെങ്കിൽ എല്ലാ പെയിന്റ് ഉപകരണങ്ങൾ ബാധകമായ ഉപകരണം ഓപ്ഷനുകൾ ഒരു വിവരണം വേണ്ടി Common Paint Tool Options കാണുക.
ടൈപ്പ് ചെയ്യുക

ഡോഡ്ജ് പ്രഭാവം നിറങ്ങൾ കുറയുന്നു.

The burn effect darkens colors.

ശ്രേണി

മൂന്നു രൂപങ്ങൾ ഉണ്ട്:

  • Shadows പിക്സൽ ദാരുവില് പ്രഭാവംപ്രാവര്ത്തികമാക്കുക ഇടുങ്ങിയതാക്കുകയും.

  • Midtones ശരാശരി ടോൺ എന്ന പിക്സൽ പ്രഭാവം ഇടുങ്ങിയതാക്കുകയും.

  • Highlights ഭാരം പിക്സൽ പ്രാബല്യത്തിൽ ഇടുങ്ങിയതാക്കുകയും.

എക്സ്പോഷർ

എക്സ്പോഷർ ഒരു കൂടുതലോ കുറവോ ആക്ഷേപം ഫോട്ടോഗ്രാഫ് പോലെ എത്ര പ്രയോഗം പ്രഭാവം ശക്തമായ ആയിരിക്കും നിഷ്കർഷിക്കുന്നു. സ്വതേ സ്ലൈഡർ 50 എന്നാൽ 0 മുതൽ 100 ​​വരെ ഉണ്ടാകുന്നു.
