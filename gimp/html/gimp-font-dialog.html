3.7. ഫോണ്ടുകൾ സംഭാഷണം

ചിത്രം 15.60. ഫോണ്ടുകൾ ഡയലോഗ്


“Fonts” ഡയലോഗ് Text tool വേണ്ടി ഫോണ്ടുകൾ തിരഞ്ഞെടുത്ത് ഉപയോഗിക്കുന്നു. അത് ജിമ്പ് പ്രവർത്തിക്കുന്ന സമയത്ത് നിങ്ങളുടെ സിസ്റ്റം പുതിയ ഒന്ന് ചേർത്താൽ, ലഭ്യമായ ഫോണ്ടുകൾ ലിസ്റ്റ് പുതുക്കുക അനുവദിക്കുന്നു.

3.7.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Fonts” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • ഒരു ഇമേജ് മെനുവിൽ നിന്ന്: Windows → Dockable Dialogs → Fonts;

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Fonts തിരഞ്ഞെടുത്ത്,

  • ടെക്സ്റ്റ് ടൂളിനായി ടൂൾ ഓപ്ഷനുകൾ നിന്ന്. നിങ്ങൾ “Font” ബട്ടൺ ക്ലിക്ക്, ഒരു ഫോണ്ട്-സെലക്ടർ മാറികിട്ടി. ചുവടെ വലത് വശത്തുള്ള അമർത്തി പക്ഷം “Fonts” ഡയലോഗിന് സമാഹരിക്കുന്ന ഒരു ബട്ടൺ ആണ്.

Windows മെനുവിൽ, കുറഞ്ഞത് ഒരു ഡയലോഗ് തുറന്നുതന്നെയിരിക്കുന്ന മാത്രമേ ഉണ്ട് detached windows ഒരു ലിസ്റ്റ് ഉണ്ട്. Windows → Fonts: ഈ സാഹചര്യത്തിൽ, ഇമേജ്-മെനുവിൽ നിന്നും “Fonts” ഡയലോഗ് ഉണ്ടാകാം.

3.7.2. ഫോണ്ടുകൾ ഡയലോഗ് ഉപയോഗിച്ച്

നിങ്ങൾക്ക് ചെയ്യാൻ കഴിയുന്ന ഏറ്റവും അടിസ്ഥാന കാര്യം ക്ലിക്ക് ചെയ്തു് ഒരു ഫോണ്ട് തിരഞ്ഞെടുക്കുക എന്നതാണ് ഈ ഫോണ്ട് തുടർന്ന് ടെക്സ്റ്റ് പ്രയോഗം ഉപയോഗിക്കും. പകരം ക്ലിക്കുചെയ്ത് റിലീസ്, നിങ്ങൾ ഫോണ്ട് ഉദാഹരണത്തിന് (“Aa”) മേൽ സ്ഥാനങ്ങളിലാണ് പോയിന്റർ കൂടെ ഇടത് മൗസ് ബട്ടൺ അമർത്തിപ്പിടിക്കുക, ഒരു വലിയ പാഠം ഉദാഹരണം കാണിക്കുന്ന ഒരു ജാലകം പോപ്പ് അപ്പ് ചെയ്യും (“Pack my box with five dozen liquor jugs”).

ഗ്രിഡ് / പട്ടിക മോഡുകൾ

ചിത്രം 15.61. ഫോണ്ടുകൾ ഡയലോഗ്

ഗ്രിഡ് കാഴ്ചയിൽ സംഭാഷണം

ലിസ്റ്റ് കാഴ്ചയിൽ സംഭാഷണം


സൂചന

Ctrl+F ഒരു തിരയൽ ഫീൽഡ് തുറക്കുന്നു. പട്ടിക ആയി കാണുക; ഗ്രിഡായി കാണുക കാണുക

ഫോണ്ടുകൾ ഡയലോഗ് ടാബ് മെനുവിൽ View as Grid ആൻഡ് View as List തമ്മിലുള്ള തിരഞ്ഞെടുക്കാം. ഗ്രിഡ് മോഡിൽ, ഫോണ്ടുകൾ ചതുരാകൃതിയിലുള്ള അണിനിരന്നു നിരത്തിയിരിക്കുന്നത്. പട്ടിക മോഡിൽ അവർ ലംബമായി, ഓരോ വരി ഫോണ്ട് (“Aa”), ഫോണ്ട് പേര് പിന്നാലെ രൂപം ഒരു ഉദാഹരണം കാണിക്കുന്ന കൂടിയ ചെയ്യുന്നു.

ഫോണ്ട് ലിസ്റ്റ് പുതുക്കുക

ഡയലോഗ് ചുവടെ ഈ ബട്ടൺ അമർത്തിയാൽ സിസ്റ്റം ഫോണ്ട് ലിസ്റ്റ് rescanned കാരണമാകുന്ന. ജിമ്പ് പ്രവർത്തിക്കുന്ന സമയത്ത്, ടെക്സ്റ്റ് പ്രയോഗം അവരെ ആക്സസ് ചെയ്യാനായി നിങ്ങൾ പുതിയ ഫോണ്ടുകൾ ചേർക്കുക എങ്കിൽ ഈ ക്രമീകരണം സഹായിക്കും. നിങ്ങൾക്ക് ഫോണ്ട് ലിസ്റ്റ് ഫോണ്ട് ഡിസ്പ്ലേയിൽ വലത്-ക്ലിക്കുചെയ്ത് rescanned സ്ഥാപിക്കുന്ന, ഒപ്പം (സത്യത്തിൽ മെനുവിലെ മാത്രം തീരുമാനം) മാറികിട്ടി മെനുവിൽ നിന്ന് “Rescan Font List” തിരഞ്ഞെടുത്ത് കഴിയും.

സൂചന

ഡയലോഗ് ന്റെ ടാബ് മെനുവിന്റെ “Preview Size” സബ്മെനു ഉപയോഗിച്ച് ഡയലോഗിൽ ഫോണ്ട് പ്രിവ്യൂകളുടെ വലിപ്പം മാറ്റാൻ കഴിയും.
