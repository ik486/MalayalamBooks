2. പുതിയ ഫയലുകൾ സൃഷ്ടിക്കുന്നു

Create a new image ഡയലോഗ് തുറക്കാൻ File → New ഉപയോഗിക്കുക. ഫയൽ പ്രാരംഭ വീതിയും ഉയരവും പരിഷ്ക്കരിക്കുക അല്ലെങ്കിൽ സാധാരണ മൂല്യങ്ങൾ ഉപയോഗിക്കുക, ഒരു പുതിയ ഇമേജ് ഫയൽ സൃഷ്ടിക്കുക. Create a new image ഡയലോഗ് കുറിച്ചുള്ള കൂടുതൽ വിവരങ്ങൾ വിഭാഗം 2.2, “പുതിയ” കണ്ടെത്താൻ കഴിയും.
