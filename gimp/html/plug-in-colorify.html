8.33. Colourify

8.33.1. പൊതു അവലോകനം

ചിത്രം 16.185. Example for the “Colorify” filter

Original image

Filter “Colorify” applied


ഈ ഫിൽട്ടർ അത് ഒരു നിറമുള്ള ഗ്ലാസ് വഴി കാണുന്ന പോലെ ഒരു greyscaled ചിത്രം റെൻഡർ.

ഓരോ പിക്സലിനായുള്ള, അരിപ്പ RGB ചാനലുകൾ ശരാശരി മൂല്യം കണക്കാക്കുന്നു (ഈ Luminosity അടിസ്ഥാനമാക്കി ചിത്രം desaturating തുല്യമാണ്). കാരണമായി നിറം ഈ ശരാശരി മൂല്യം “colorify color” ഗുണിച്ചു.

അതുകൊണ്ട് ഈ ഫിൽട്ടർ മാത്രം ആർജിബി മോഡിൽ ചിത്രങ്ങൾ പ്രവർത്തിക്കുന്നു.

8.33.2. Activate the filter

This filter is found in the image window menu under Colors → Colorify.

8.33.3. ഓപ്ഷനുകൾ

ചിത്രം 16.186. “Colorify” filter options


ഒരു നിറം പാലറ്റ് പ്രത്യേകിച്ച് RGB നിറങ്ങൾ റെഡ്, ഗ്രീൻ, ബ്ലൂ, CMY നിറങ്ങൾ സിയാൻ, മജന്ത, മഞ്ഞ അടങ്ങുന്ന ലഭ്യമാണ്.

നിങ്ങൾ Custom Color Swatch ക്ലിക്കുചെയ്ത് നിങ്ങളുടേതായ നിറങ്ങളും തിരഞ്ഞെടുക്കാം.
