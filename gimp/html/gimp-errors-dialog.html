5.4. പിശക് കൺസോൾ

Error console സിംഗിൾ “GIMP Message” കൂടുതൽ സാധ്യതകൾ പ്രദാനം ചെയ്യുന്നു. ഈ ജിമ്പ് പ്രവർത്തിക്കുന്ന സമയത്ത് സംഭവിക്കുന്ന എല്ലാ പിശകുകൾ ഒരു ലോഗ് ആണ്. നിങ്ങൾ എല്ലാം ഈ രേഖ അല്ലെങ്കിൽ മാത്രമേ തിരഞ്ഞെടുത്ത ഭാഗം സംരക്ഷിക്കാൻ കഴിയും.

5.4.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Error Console” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • ഒരു ഇമേജ് മെനുവിൽ നിന്ന്: Windows → Dockable Dialogs → Error Console;

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും PLACEHOLDER,-100654 ഞെക്കി, Add Tab → Error Console തിരഞ്ഞെടുത്ത്.

5.4.2. “Error Console” സംഭാഷണം

ചിത്രം 15.74. “Error Console” ഡയലോഗ് വിൻഡോ


മായ്ക്കുക പിശകുകൾ

ബട്ടണില് രേഖ എല്ലാ പിശകുകൾ ഇല്ലാതാക്കുക അനുവദിക്കുന്നു.

മുന്നറിയിപ്പു്

ഈ പ്രവർത്തനം “undo” കഴിയില്ല.
എല്ലാ പിശകുകൾ സംരക്ഷിക്കുക

ഈ ബട്ടൺ നിങ്ങൾ മുഴുവൻ രേഖ സംരക്ഷിക്കാൻ അനുവദിക്കുന്നു. നിങ്ങൾക്ക് രേഖ (ക്ലിക്കിൽ-ആൻഡ്-വലിച്ചോ മൗസ് മിനാരങ്ങൾ അല്ലെങ്കിൽ Shift+Arrow keys കീ ഉപയോഗിക്കുന്നു) ഒരു ഭാഗം തിരഞ്ഞെടുത്ത് Shift കീ അമർത്തി മാത്രം ഇത് തിരഞ്ഞെടുത്ത ഭാഗം സംരക്ഷിക്കാൻ കഴിയും.

ഒരു ഡയലോഗ് വിൻഡോ Save Error Log to File പേരും ഈ ചിത്രം ലക്ഷ്യം ഡയറക്ടറി തിരഞ്ഞെടുക്കാൻ അനുവദിക്കുന്നു:

ചിത്രം 15.75. “Save Error Log to file” ഡയലോഗ് വിൻഡോ
