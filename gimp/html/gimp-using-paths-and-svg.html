5.7. പാത എന്നിവയും SVG ഫയലുകൾ

SVG, “Scalable Vector Graphics” നില്ക്കുക, vector graphics വേണ്ടി തോതിൽ പ്രശസ്തമായ ഫയൽ ഫോർമാറ്റ്, ഗ്രാഫിക്കല് ​​ഘടകങ്ങൾ raster graphics വ്യത്യസ്തമായി, ഒരു ചിത്രം-സ്വതന്ത്ര ഫോർമാറ്റിൽ പ്രതിനിധാനം ഏത് ഉണ്ടു; ഇതിൽ ഗ്രാഫിക്കല് ​​ഘടകങ്ങൾ പിക്സൽ അറേകളിൽ ആയി പ്രതിനിധീകരിക്കുന്നത്. ജിമ്പ് പ്രധാനമായും ഓപ്പൺജിഎല്ലിനെക്കുറിച്ചുള്ള പ്രോഗ്രാം ആണ്, പക്ഷേ പാത വെക്റ്റർ സ്ഥാപനങ്ങളുടെയോ ആകുന്നു.

ഭാഗ്യവശാൽ, പാതകൾ അവർ ജിമ്പ് പ്രതിനിധീകരിക്കുന്നത് ഏതാണ്ട് ഇതേ വിധത്തിൽ SVG ഫയലുകൾ പ്രതിനിധീകരിക്കുന്നത്. (യഥാർത്ഥത്തിൽ ഭാഗ്യം അതു ബന്ധമില്ല ഉണ്ട്: ജിമ്പ് മാർഗത്തിൽ കൈകാര്യം മനസ്സിൽ എസ്.വി.ജി. പാതകൾ കൂടെ ജിമ്പ് 2.0 വേണ്ടി മാറ്റിയെഴുതി ചെയ്തു.) ഈ അനുയോജ്യത ഏതെങ്കിലും വിവരം നഷ്ടപ്പെടാതെ SVG ഫയലുകളായി ജിമ്പ് പാതകളെ സംഭരിക്കാൻ സാധ്യമാക്കുന്നു. നിങ്ങൾ വഴികളിലേക്ക് ഡയലോഗിൽ ഈ ശേഷി ആക്സസ് ചെയ്യാൻ കഴിയും.

ഇത് ജിമ്പ് പോലുള്ള Inkscape അല്ലെങ്കിൽ Sodipodi രണ്ടു പ്രശസ്തമായ ഓപ്പൺ സോഴ്സ് വെക്റ്റർ ഗ്രാഫിക്സ് അപേക്ഷകൾ പോലുള്ള മറ്റ് പരിപാടികൾ, സംരക്ഷിച്ചിരിക്കുന്ന SVG ഫയലുകളിൽ നിന്നും പാത സൃഷ്ടിക്കാൻ കഴിയും എന്നാണ്. ആ പ്രോഗ്രാമുകൾ കൂടുതൽ ശക്തമാക്കുന്നു പാത-കൃത്രിമത്വം ഉപകരണങ്ങൾ ജിമ്പ് അപേക്ഷിച്ച് ഉള്ളതിനാൽ ഈ ഹൃദ്യമായ. നിങ്ങൾ വഴികളിലേക്ക് ഡയലോഗ് ഉപയോഗിച്ച് ഒരു SVG ഫയലിൽ നിന്നും ഒരു പാത്ത് ഇമ്പോർട്ടുചെയ്യാനാകും.

SVG ഫോർമാറ്റ് വെറും പാതകളെ അധികം മറ്റു പല ഗ്രാഫിക്കൽ ഘടകങ്ങൾ കൈകാര്യം: മറ്റു കാര്യങ്ങൾ, അതു ജിമ്പ് ഈ സ്ഥാപനങ്ങളുമായി ഒന്നും ചെയ്യാൻ കഴിയില്ല പോലുള്ള സ്ക്വയറുകൾ, ദീർഘചതുരങ്ങൾക്കോ, സർക്കിളുകൾ, ellipses, ബഹഭജതിെന മുതലായവ കണക്കുകൾ കൈകാര്യം, എന്നാൽ അതു പാതകൾ അവരെ ലോഡ് ചെയ്യാം.

കുറിപ്പു്

പാതകളെ ഉണ്ടാക്കുന്നു ജിമ്പ് SVG ഫയലുകൾ ചെയ്യാൻ കഴിയുന്ന കാര്യം മാത്രം അല്ല. ഇത് SVG ഫയലുകൾ സാധാരണപോലെ വഴിയിൽ ജിമ്പ് ചിത്രങ്ങളായി തുറക്കാൻ കഴിയും.
