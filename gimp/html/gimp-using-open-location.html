3.2. സ്ഥാനം തുറക്കുക

പകരം ഒരു ഫയൽ നാമത്തിന്റെ, നിങ്ങൾ ചിത്രത്തിനായി ഒരു URI (അതായത്, ഒരു വെബ് വിലാസം) ഉണ്ടെങ്കിൽ, അത് മെനു ഉപയോഗിച്ച്, ഒരു ഇമേജ് മെനുവിൽ നിന്നും File → Open Location… തിരഞ്ഞെടുത്തുകൊണ്ട് തുറക്കാൻ കഴിയും. ഈ URI പ്രവേശിക്കാൻ (അല്ലെങ്കിൽ ഒട്ടിക്കുക) അനുവദിക്കുന്ന ഒരു ചെറിയ ഡയലോഗ് ചെയ്യുന്നു.
