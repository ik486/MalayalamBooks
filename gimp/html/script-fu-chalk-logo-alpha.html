16.8. ചോക്ക്

16.8.1. പൊതു അവലോകനം

ചിത്രം 17.386. “Chalk” ഫിൽട്ടർ ഉദാഹരണം

യഥാർത്ഥ ചിത്രം

“Chalk” പ്രയോഗിച്ചു


ഈ ഫിൽട്ടർ സജീവ പാളി ഒരു ചോക്ക് ഡ്രോയിംഗ് പ്രഭാവം സൃഷ്ടിക്കുന്നു.

അതു ഉദാഹരണത്തിന്, നിങ്ങളുടെ നിര ഒരു ടെക്സ്റ്റ് നിന്നുമുള്ള ലോഗോ സൃഷ്ടിക്കുന്ന “Chalk” സ്ക്രിപ്റ്റ് (ചിത്രം വിൻഡോയിൽ File → Create → Logos → Chalk), കഥകളിയുടെ:

“Chalk” ലോഗോ

മുന്നറിയിപ്പു്

ചിത്രം എല്ലായ്പ്പോഴും സജീവ പാളി വലിപ്പത്തിന് വലുപ്പം മാറ്റും.

16.8.2. ഫിൽട്ടർ സജീവമാക്കുക

നിങ്ങൾ Filters → Alpha to Logo → Chalk കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ ഈ ഫിൽട്ടർ കണ്ടെത്താം.

16.8.3. ഓപ്ഷനുകൾ

ചിത്രം 17.387. “Chalk” ഓപ്ഷൻ


പശ്ചാത്തല നിറം

പശ്ചാത്തല നിറം നിങ്ങളെ ചോക്ക് കൂടെ വരയ്ക്കുന്നു “blackboard” നിറം ആണ്, പ്രതിഭയെ അത് കറുത്ത തുടർന്ന്. നിങ്ങൾക്ക് കളർ ബട്ടണില് ക്ലിക്ക് ചെയ്യുമ്പോൾ, നിറം സെലക്ടർ അപ്പ് ചെയുകയും വേറെ ഏതെങ്കിലും നിറം തിരഞ്ഞെടുക്കാം.

16.8.4. ഫിൽറ്റർ വിശദാംശങ്ങൾ

“Chalk” ഫിൽട്ടർ

  1. , പാളി ഒരു Gaussian blur ബാധകമാണ് പിക്സൽ spreads, ഒപ്പം ripples തിരശ്ചീനമായും ലംബമായും പാളി,

  2. Sobel എഡ്ജ് ഉപയോഗിച്ച് ശശ വേണമങ്കിലും അരിപ്പ കണ്ടെത്തി

    കുറിപ്പു്

    ചിലപ്പോൾ സോബൽ എഡ്ജ് ഇമേജ് വശങ്ങളും ചില ചപ്പുചവറുകളും ഉത്പാദിപ്പിക്കുന്നത് കണ്ടുപിടിക്കുന്നു.

  3. പ്രതല level വർദ്ധിപ്പിക്കുന്നു.

നിർഭാഗ്യവശാൽ നിങ്ങൾ ഉപകരണം അരിച്ചെടുക്കുന്ന ഓപ്ഷനുകൾ മാറ്റാൻ കഴിയില്ല. എന്നാൽ നിങ്ങള് ബന്ധപ്പെട്ട ഓപ്ഷനുകൾ വ്യത്യസ്ത മുകളിൽ പറഞ്ഞിരിക്കുന്ന രീതികൾ ഉപയോഗിച്ച് പടിപടിയായി പ്രക്രിയ ഘട്ടം പുനർനിർമ്മാണം. അപ്പോൾ നിങ്ങൾ കേവലം ഏതെങ്കിലും നിറം നിറഞ്ഞ ഒരു പശ്ചാത്തല പാളി ചേർക്കാം. അത്രയേയുള്ളൂ.
