5.7. ഡ്രോപ്പ് ഷാഡോ
5. വെളിച്ചവും ഷാഡോ ഫിൽട്ടറുകൾ  

5.7. ഡ്രോപ്പ് ഷാഡോ

5.7.1. പൊതു അവലോകനം

ചിത്രം 17.110. “Drop Shadow” ഫിൽട്ടർ ഉദാഹരണം

യഥാർത്ഥ ചിത്രം

“Drop Shadow” പ്രയോഗിച്ചു (വെള്ള പശ്ചാത്തല പാളി സ്വമേധയാ ചേർത്തു)


ഈ ഫിൽട്ടർ നിലവിലുള്ള തെരഞ്ഞെടുപ്പ് അല്ലെങ്കിൽ സജീവ നിരക്കു ഉള്ളതെങ്കിൽ ചിത്രം ഒരു ഡ്രോപ്പ്-നിഴൽ ചേർക്കുന്നു. ആ നിഴൽ പ്രദർശിപ്പിക്കുന്നതിനായി അത്യാവശ്യമാണ് ഓപ്ഷണൽ ഫിൽട്ടർ ചിത്രം മാറ്റുന്നു.

നിങ്ങൾക്ക് കളർ, സ്ഥാനം, നിഴൽ വലുപ്പം നിങ്ങൾക്ക് തിരഞ്ഞെടുക്കാം.

ഫിൽട്ടർ നിഴൽ കാണുന്നതിനായി ഒരു പശ്ചാത്തല പാളി ചേർക്കാൻ ശ്രദ്ധിക്കുക. നിഴൽ പശ്ചാത്തല സുതാര്യമാണ്. മുകളിൽ ഉദാഹരണം വെളുത്ത പശ്ചാത്തല അരിപ്പ സൃഷ്ടിച്ചത് ചെയ്തിട്ടില്ല, പകരം അത് നിങ്ങൾ നിഴൽ നോക്കാം പിന്നീട് ചേർത്തിട്ടുണ്ട്.

5.7.2. ഫിൽട്ടർ സജീവമാക്കുക

നിങ്ങൾ Filters → Light and Shadow → Drop Shadow വഴി ഇമേജ് മെനുവിൽ മെനുവിൽ ഈ ഫിൽട്ടർ കണ്ടെത്താം.

5.7.3. ഓപ്ഷനുകൾ

ചിത്രം 17.111. “Drop Shadow” ഫിൽട്ടർ ഓപ്ഷനുകൾ


X ഓഫ്സെറ്റ്, Y ഓഫ്സെറ്റ്

ഡ്രോപ്പ് ഷാഡോ അടങ്ങുന്ന പാളി ലംബമായി വൈ പിക്സൽ, എക്സ് പിക്സൽ തിരശ്ചീനമായി നീക്കും. അതിനാൽ, X, Y നിഴൽ ചിത്രം ബന്ധപ്പെട്ട് സ്ഥാപിക്കപ്പെടും എവിടെ നിർണ്ണയിക്കാൻ ഓഫ്സെറ്റ്. അതു തിരശ്ചീനമായി ലംബമായതോ ദിശയിൽ ദൂരത്തു പോലെ ഉയർന്ന മൂല്യങ്ങൾ നേരിയ രൂപത്തിലേക്ക് സാങ്കല്പികമായ ഉറവിടം ഉണ്ടാക്കി, കുറഞ്ഞ മൂല്യങ്ങൾ അത് ചിത്രത്തിന്റെ ആത്മബന്ധവും ഉണ്ടാക്കാം.

ഓഫ്സെറ്റുകളുണ്ടു് വൈ < 0 ഓഫ്സെറ്റ് എങ്കിൽ എക്സ് < 0, അല്ലെങ്കിൽ നിരക്കു മുകളിൽ ഓഫ്സെറ്റ് എങ്കിൽ നിരക്കു ഇടതുഭാഗത്തായി ഒരു നിഴൽ വഴിതെളിക്കുന്നു നെഗറ്റീവ് വരാം.

സജീവ നിരക്കു ഉണ്ടെങ്കിൽ, നിങ്ങൾ Allow resizing സ്വാധീനം കാണാൻ പ്രാപ്തമാക്കിയ ഉണ്ടായിരിക്കണം.

റേഡിയസ് മങ്ങിക്കുക

നിഴൽ ഉണ്ടാക്കിയ ശേഷം വ്യക്തമാക്കിയ റേഡിയസ് ഒരു Gaussian blur ഡ്രോപ്പ് ഷാഡോ യഥാതഥ ഭാവം കാരണമാകുന്നു നിഴൽ പാളി പ്രയോഗിക്കുന്നത്. ഇത് മങ്ങിക്കുന്നതിനോ നിഴലിനെ നീട്ടിയത് ശേഷം, Allow resizing പ്രാപ്തമാക്കാൻ വന്നേക്കാം.

നിറം

നിഴൽ ഏതെങ്കിലും നിറം ഉണ്ടായിരിക്കാം. വെറും ബട്ടൺ ക്ലിക്ക്, നിറം സെലക്ടർ മാറികിട്ടി ഒരു വർണ്ണം തിരഞ്ഞെടുക്കുക.

ചിത്രം 17.112. “Drop Shadow” നിറം ഉദാഹരണം


അതാര്യത

നിഴൽ ന്റെ അതാര്യത നിഴൽ അടങ്ങുന്ന പുതിയ പാളി വെറും അതാര്യത (വിഭാഗം 1.1, “ലെയർ പ്രോപ്പർട്ടീസ്” കാണുക). 80% ആയി സഹജമായി, എന്നാൽ നിങ്ങൾ 0 (സുതാര്യതയോടെ) മുതൽ 100 ​​(ഫുൾ ഒപാസിറ്റിയിൽ) ഇവിടെ മറ്റേതെങ്കിലും മൂല്യം തിരഞ്ഞെടുക്കാം. ഒരു ചിത്രം ഫിൽട്ടർ പ്രയോഗിച്ച ശേഷം നിങ്ങൾ layers dialog ഒപാസിറ്റി മാറ്റാനാകും.

വലിപ്പംമാറ്റിക്കൊണ്ടിരിക്കുന്നജാലകങ്ങളിലുള്ളവകാണിയ്ക്കുക അനുവദിക്കുക

പ്രവർത്തനക്ഷമമാക്കിയിട്ടുണ്ടെങ്കിൽ, അരിപ്പ ആ നിഴൽ വേണ്ടി ഇടമാക്കുന്നതിനും ആവശ്യമാണോ എന്ന് ചിത്രം വലുപ്പം ചെയ്യും. പുതിയ വലുപ്പം നിരക്കു വലിപ്പം, മങ്ങൽ ആരം നിഴൽ ഓഫ്സെറ്റുകളുടെ ആശ്രയിച്ചിരിക്കുന്നു.
