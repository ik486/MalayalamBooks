7.23.  Apply Layer Mask

Apply Layer Mask കമാൻഡ് നിലവിലെ പാളി പാളി മാസ്ക് ലയിപ്പിക്കുന്നു. ലെയർ മാസ്ക് സുതാര്യത വിവരങ്ങൾ ഇത് നിലവിലില്ല എങ്കിൽ സൃഷ്ടിക്കപ്പെട്ടു ആൽഫാ ചാനൽ കൈമാറ്റം ചെയ്യപ്പെടുമ്പോൾ, ലെയർ മാസ്ക് നീക്കം ചെയ്യുന്നു. സജീവ പാളി ഒരു പാളി മാസ്ക് ഇല്ലെങ്കിൽ, മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്. കൂടുതൽ വിവരങ്ങൾക്ക് Layer Masks വിഭാഗം കാണുക.

7.23.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Mask → Apply Layer Mask,

  • or from the pop-up menu you get by right-clicking on the active layer in the Layers Dialog.
