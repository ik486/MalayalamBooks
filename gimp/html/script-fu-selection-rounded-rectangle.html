4.15. Rounded Rectangle

ചിത്രം 16.39. Example of using Rounded rectangle on a selection

An image with a selection

After “Rounded rectangle”


“Rounded Rectangle” സ്ക്രിപ്റ്റ്-ഫു കമാൻഡ് വൃത്താകാരമോ കോണിലും ഒരു ചതുരാകൃതിയിലുള്ള സെലക്ഷനും നിലവിലുള്ള ഒരു നിര (ചതുരാകൃതിയിലുള്ള, ദീർഘവൃത്താകൃതിയിലുള്ള അല്ലെങ്കിൽ മറ്റ് ആകാരം) മാറ്റപ്പെടുന്നു. കോണിലും അകത്ത് (അകത്തേയ്ക്ക്) നേരെ അല്ലെങ്കിൽ പുറത്ത് (പൈപ്പിന്െറ) നേരെ വളഞ്ഞും കഴിയും. ഇത് ചെയ്യുന്നതിന്, കമാൻഡ് നിരക്കു എന്ന കോണിലും ചേർക്കുന്നു അല്ലെങ്കിൽ സർക്കിളുകളിൽ നീക്കം.

4.15.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menu bar through Select → Rounded Rectangle.

4.15.2. “Rounded Rectangle” സംഭാഷണം വിൻഡോ വിവരണം

ചിത്രം 16.40. The “Rounded Rectangle” dialog


റേഡിയസ് (%)

നിങ്ങൾക്ക് ഒരു സ്ലൈഡർ അല്ലെങ്കിൽ ടെക്സ്റ്റ് ഫീൽഡ് ഉപയോഗിച്ചും ശതമാനം വൃത്താകാരമോ മൂലയിൽ ആരത്തിന് നൽകാം. ഈ മൂല്യം ഉയരം അല്ലെങ്കിൽ വീതി ഒരു ശതമാനം ഏതാണോ കുറവ് അത് ആണ്.

അകത്തേയ്ക്ക്

ഈ കള്ളി പരിശോധിക്കുകയാണെങ്കിൽ, കോണിലും (അകത്ത് നേരെ വളഞ്ഞുമാണ്) പകരം പൈപ്പിന്െറ (പുറത്തുള്ള നേരെ വളഞ്ഞുമാണ്) അകത്തേയ്ക്ക് ആയിരിക്കും.
