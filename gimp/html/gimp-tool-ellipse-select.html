2.3. Ellipse Selection

ചിത്രം 14.14. Ellipse Select icon in the Toolbox


അണ്ഡാകൃതി തിരഞ്ഞെടുക്കൽ ഉപകരണം നിങ്ങളെ ആവശ്യമെങ്കില് ഉയർന്ന ഗുണമേന്മയുള്ള വിരുദ്ധ-അലിയാസിങ് കൂടെ ഒരു ചിത്രത്തിൽ നിന്ന് സർക്കുലർ ദീർഘവൃത്താകൃതിയുള്ള പ്രദേശങ്ങളും തിരഞ്ഞെടുക്കുക രൂപകല്പന. തിരഞ്ഞെടുക്കലുകൾ വിവരങ്ങൾക്ക് അവർ എങ്ങനെ ജിമ് Selections കാണാൻ ഉപയോഗിക്കുന്നു; എല്ലാ നിരക്കു ഉപകരണങ്ങൾ പൊതുവായുള്ള സവിശേഷതകൾ വിവരങ്ങൾക്ക് Selection Tools കാണുന്നു.

ഈ ഉപകരണം പുറമേ ഒരു ഇമേജിൽ ഒരു സർക്കിൾ അല്ലെങ്കിൽ എല്ലിപ്സ് തർജ്ജമ ഉപയോഗിക്കുന്നു. ഒരു നിറഞ്ഞു എല്ലിപ്സ് റെൻഡർ ദീർഘവൃത്താകാരമായി നിരക്കു സൃഷ്ടിക്കുക, തുടർന്ന് Bucket Fill tool ഉപയോഗിച്ച് ഫിൽ ചെയ്യുക. ദീർഘവൃത്താകൃതിയിലുള്ള ഔട്ട്ലൈൻ സൃഷ്ടിക്കുന്നതിന്, ലളിതവും ഏറ്റവും ഇഷ്ടാനുസരണം സമീപനം ദീർഘവൃത്താകൃതിയിലുള്ള നിരക്കു സൃഷ്ടിക്കുക എന്നതാണ് പിന്നെ അത് stroke. എന്നിരുന്നാലും, ഈ സമീപനം എലിയാസിംഗ് വിരുദ്ധ നിലവാരം പകരം ക്രൂഡ് ആണ്. ഉയർന്ന ഗുണമേന്മയുള്ള ഔട്ട്ലൈൻ പുറത്തെ ഒരു നിന്ന് അകത്തെ കുറച്ചാണ് വലുപ്പത്തിലും രണ്ടു ദീർഘവൃത്താകൃതിയിലുള്ള തിരഞ്ഞെടുക്കലുകൾ സൃഷ്ടിച്ചുകൊണ്ട് ലഭിക്കും; എന്നാൽ ഇത് എല്ലായ്പ്പോഴും ശരിയായ നേടുകയും എളുപ്പമാണ്. കമാൻഡ് Select → Border... എളുപ്പമാക്കുന്നു.

2.3.1. Activating the tool

You can access the Ellipse Selection Tool in different ways:

  • From the image menu bar Tools → Selection Tools → Ellipse Select;

  • പണിസഞ്ചിയിൽ റ്റൂൾ PLACEHOLDER,-105575 മാകാം,

  • കീബോർഡ് കുറുക്കുവഴി E ഉപയോഗിക്കുന്നതിലൂടെ.

2.3.2. Key modifiers

കുറിപ്പു്

അതേ വഴിയിൽ ഈ എല്ലാ ഉപകരണങ്ങൾ ബാധിക്കുന്ന മോഡിഫയർ കീകൾ സഹായത്തിന് Selection Tools കാണുക. ഈ ഉപകരണം മാത്രമായുള്ള മാത്രം ഇഫക്റ്റുകൾ ഓപ്ഷനുകൾ വിശദമായി ഇവിടെ ചെയ്യുന്നു.
Ctrl,

നിങ്ങളുടെ നിരക്കു തുടങ്ങുന്ന, പൂർത്തിയാകുമ്പോൾ വരെ ഇത് താഴേയ്ക്കാക്കുമ്പോൾ ശേഷം കീ അമർത്തുന്നത് പകരം ഉൾപ്പെട്ടിരിക്കാവുന്ന ദീർഘചതുരം ഒരു കോണിൽ, നിങ്ങളുടെ ആരംഭ പോയിന്റ് തിരഞ്ഞെടുത്തു ദീർഘവശങ്ങളിലേക്ക് കേന്ദ്രമായി ഉപയോഗിക്കാൻ കാരണമാകുന്നു. നിങ്ങൾ തിരഞ്ഞെടുപ്പ് നടത്താൻ തുടങ്ങുന്ന Ctrl കീ before അമർത്തുക, ലഭിക്കുന്ന നിരക്കു നിലവിലുള്ള നിരക്കു നിന്ന് കുറയ്ക്കപ്പെടുന്ന ശ്രദ്ധിക്കുക.

മാറ്റം

നിങ്ങളുടെ നിരക്കു തുടങ്ങുന്ന, പൂർത്തിയാകുമ്പോൾ വരെ ഇത് താഴേയ്ക്കാക്കുമ്പോൾ ശേഷം Shift കീ അമർത്തുന്നത് ഒരു സർക്കിൾ ആയിരിക്കാൻ നിരക്കു നിർബ്ബന്ധിക്കുന്നു. നിങ്ങൾ തിരഞ്ഞെടുപ്പ് നടത്താൻ തുടങ്ങുന്ന Shift കീ before അമർത്തുക, ലഭിക്കുന്ന നിരക്കു നിലവിലുള്ള നിരക്കു ചേർക്കും ശ്രദ്ധിക്കുക.

Ctrl,+മാറ്റം

ഇരുവരും കീകൾ അമർത്തുന്നത് നിങ്ങളുടെ ആരംഭ പോയിന്റിൽ കേന്ദ്രമാക്കി ഒരു സർക്കുലർ സെലക്ഷനും നൽകുന്ന രണ്ടു ഇഫക്റ്റുകൾ കൂടിച്ചേർന്നുണ്ടായ.

2.3.3. Tool handling

ചിത്രം 14.15. Example of Ellipse Selection.


ഈ ഉപകരണം തെരഞ്ഞെടുക്കുമ്പോഴും മൌസ് പോയിന്റർ ഉടൻ അത് ചിത്രത്തിന്റെ മീതെ പോലെ ഒരു സർക്കിൾ ഐക്കൺ കൊണ്ട് വരുന്നു. ഒരു വലിച്ച് ഡ്രോപ്പ് നിങ്ങൾ ഒരു ചതുരാകൃതിയിലുള്ള ബോക്സ് ഉള്ളിൽ ഒരു എല്ലിപ്സ് (അല്ലെങ്കിൽ ഒരു സർക്കിൾ) നേടുകയും അനുവദിക്കുന്നു. മൗസ് ബട്ടൺ വിശ്രമിക്കുന്ന, ഒരു അതിനെക്കുറിച്ച് (“marching ants”) ദീർഘവൃത്തം നിരക്കു നൽകിയിട്ടുണ്ട്. അതു ശ്രദ്ധയോടെ നിരക്കു ക്രമീകരിക്കുന്നതിന് ആവശ്യമായ അല്ല; നിങ്ങൾ പിന്നീട് എളുപ്പത്തിൽ വലുപ്പം മാറ്റാൻ കഴിയും.

പോയിന്റർ കാൻവാസിൽ മാറുമ്പോൾ, പോയിന്റർ, നിരക്കു വശങ്ങൾ മാറ്റുക. നിങ്ങൾക്ക് കൈപ്പിടികൾ ഉപയോഗിച്ച് നിരക്കു വലുപ്പം മാറ്റാൻ കഴിയും. ചതുരാകൃതിയിലുള്ള അധ്യായം ഉള്ളിൽ Tool handling കാണുക.

ചിത്രം 14.16. Tool Options for the Ellipse Select tool
