6.28. ചിത്രം പ്രോപ്പർട്ടീസ്
6. “Image” മെനു  

6.28. ചിത്രം പ്രോപ്പർട്ടീസ്

“Image Properties” കമാൻഡ് ചിത്രത്തിനായുള്ള വിവിധ ധാരാളം വിവരങ്ങൾ കാണിക്കുന്ന ഒരു ജാലകം തുറക്കുന്നു.

6.28.1. Activate the Command

  • You can access this command from the image menubar through Image → Image Properties,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Alt+Return ഉപയോഗിച്ച്.

6.28.2. ഓപ്ഷനുകൾ

ഉള്ള കിളിവാതിൽ മൂന്നു ടാബുകൾ തിരിച്ചിട്ടുണ്ട്.

6.28.2.1. “Properties” tab

ചിത്രം 16.85. “Properties” tab


Pixel dimensions

പിക്സലിൽ ഇമേജ് ഉയരം വീതി, ആ, ചിത്രത്തിന്റെ physical വ്യാപ്തി.

Print size

ചിത്രം നിലവിലെ യൂണിറ്റുകളിൽ, അച്ചടിക്കാൻ പാകത്തിൽ ഉണ്ടാകും വലുപ്പം കാണിക്കുന്നു. ഈ ചിത്രത്തിന്റെ logical വ്യാപ്തി. അത് ചിത്രത്തിന്റെ സ്ക്രീൻ റെസലൂഷൻ ശാരീരിക വലിപ്പം ആശ്രയിച്ചിരിക്കുന്നു.

മിഴിവ്

Shows the print resolution of the image in pixel per inch.

Color space

Shows the images color space.

ഫയലിന്റെ പേര്

Path and name of the file that contains the image.

File size

Size of the file that contains the image.

File type

ചിത്രം ഉൾപ്പെടുന്ന ഫയൽ ഫോർമാറ്റ്.

Size in memory

ചിത്രങ്ങൾ ജേണൽ ഉൾപ്പെടെയുള്ള കയറ്റികൊണ്ട് ചിത്രത്തിന്റെ റാം ഉപഭോഗം. ഈ വിവരം പുറമേ ചിത്രം വിൻഡോയിൽ പ്രദർശിപ്പിച്ചിരിക്കുന്നു. വലുപ്പം ഡിസ്കിലെ ഫയലിന്റെ വലുപ്പം നിന്ന് തികച്ചും വ്യത്യസ്തമാണ്. പ്രദർശിപ്പിച്ച ചിത്രം റാമിലേക്ക് കാരണം ആണ് GIMP വീണ്ടും പ്രക്രിയകൾക്കുള്ള മെമ്മറിയിൽ ചിത്രത്തിന്റെ ഒരു പകർപ്പ് സൂക്ഷിക്കുന്നു കാരണം.

Undo steps

നിങ്ങൾ ചിത്രത്തിൽ നിർവഹിച്ച നിങ്ങൾ പഴയപടിയാക്കാനുമാകും പ്രവർത്തനങ്ങൾ എണ്ണം. നിങ്ങൾ Undo History ഡയലോഗിൽ അവരെ കാണാൻ കഴിയും.

Redo steps

നിങ്ങൾ വീണ്ടും കഴിയുന്ന, പൂർവാവസ്ഥയിലാക്കിയ പ്രവർത്തനങ്ങളുണ്ട് എണ്ണം.

പിക്സൽ എണ്ണം; പാളികൾ എണ്ണം; ചാനലുകളുടെ എണ്ണം; പാതകൾ എണ്ണം

Well counted!

6.28.2.2. “Color profile” tab

ചിത്രം 16.86. “Color profile” tab


ഈ ടാബ് നിറം പ്രൊഫൈൽ ചിത്രം ഉപയോഗിച്ച് ജിമ്പ് ലോഡ് പേര് അടങ്ങിയിരിക്കുന്നു. സ്വതേ ബിൽറ്റ്-ഇൻ “sRGB” പ്രൊഫൈൽ ആണ്.

ചിത്രം 16.87. “Comments” tab
