2.7. Intelligent Scissors

ചിത്രം 14.27. Intelligent Scissors tool icon in the Toolbox


നേർമാർഗത്തിൽ ഉപകരണം ഉപയോഗിച്ച് പൊതുവായ ചില സവിശേഷതകൾ അതു lasso കൊണ്ട് പൊതുവായ ചില സവിശേഷതകൾ ഉണ്ട്, അതിന്റെ എല്ലാ സ്വന്തം ചില സവിശേഷതകൾ: ഇന്റലിജന്റ് കത്രിക പ്രയോഗം ഉപകരണങ്ങൾ രസകരമായ ഒരു കഷണം ആണ്. നിങ്ങൾ നാനാഭാഗങ്ങളിലുമുണ്ടായിരിക്കും ശക്തമായ നിറം-മാറ്റങ്ങൾ നിഷ്കർഷിച്ച പ്രദേശം തിരഞ്ഞെടുക്കാൻ ശ്രമിക്കുമ്പോൾ ഇത് സഹായിക്കുന്നു. കത്രിക ഉപയോഗിക്കാൻ, "നിയന്ത്രണ നോഡുകൾ" ഒരു കൂട്ടം സൃഷ്ടിക്കാൻ ക്ലിക്കുചെയ്യുക, കൂടാതെ അവതാരകർ അല്ലെങ്കിൽ നിയന്ത്രണ പോയിന്റ് നിലയിൽ നിങ്ങൾ തിരഞ്ഞെടുക്കാൻ ശ്രമിക്കുന്ന മേഖലയിലെ അറ്റങ്ങളിൽ പരാമർശിച്ചിരിക്കുന്നു. ഉപകരണം ഈ നിയന്ത്രണ നോഡുകൾ കടന്നുപോകുന്നത് തുടർച്ചയായ വക്രം, ഏതെങ്കിലും ഉയർന്ന തീവ്രത വേണമങ്കിലും കണ്ടെത്താൻ കഴിയും താഴെ ഉത്പാദിപ്പിക്കുന്നത്. നീ ഭാഗ്യവാനാണ് എങ്കിൽ, പ്രയോഗം കണ്ടെത്തുന്നു പാതയിൽ നിങ്ങൾ തിരഞ്ഞെടുക്കാൻ ശ്രമിക്കുന്ന കോണ്ടൂർ അനുയോജ്യമായി ചെയ്യും.

നിർഭാഗ്യവശാൽ, അവിടെ എതിക തിരഞ്ഞെടുക്കലുകൾ കേസുകൾ ധാരാളം പ്രെറ്റി ക്രൂഡ് കുറവായിരിക്കും എന്ന് ഫലം, ഈ ഉപകരണം വേണ്ടി എഡ്ജ് താഴെ യുക്തി ചില പ്രശ്നങ്ങളുണ്ട് തോന്നുന്നില്ല. അവരെ വൃത്തിയാക്കാൻ ഒരു നല്ല വഴി QuickMask മോഡിലേക്ക് സ്വിച്ചുചെയ്യുക, പ്രശ്നക്കാരായ ഭാഗങ്ങളിൽ വരയ്ക്കാൻ പെയിന്റ് ഉപകരണങ്ങൾ ഉപയോഗിക്കുക എന്നതാണ്. മൊത്തത്തിൽ, അധികപേരും, കാരണം അത് ബുദ്ധിയുള്ള എഡ്ജ് കണ്ടെത്തുന്നതിൽ ശേഷിയെ ഇല്ല പോലും പാത ഉപകരണം, കത്രിക അധികം ഉപയോഗപ്രദമായ കണ്ടെത്തിയാൽ, പാതകൾ അതു നിലനിൽക്കുകയുള്ളൂ ഉത്പാദിപ്പിക്കുന്നത് നിങ്ങൾ ഇല്ലാതാക്കുന്നതുവരെ, ഏതു സമയത്തും അടിമരാജ്യങ്ങളിൽ കഴിയും .

2.7.1. Activating the tool

You can access the Intelligent Scissors Tool in different ways:

  • From the image menu bar Tools → Selection Tools → Intelligent Scissors,

  • by clicking on the tool icon in the ToolBox,

  • by using the keyboard shortcut I.

2.7.2. Key modifiers

എല്ലാ നിരക്കു ടൂളുകൾ Shift, Ctrl, ഒപ്പം Alt കീകളുടെ സ്ഥിര പെരുമാറ്റം ??? വിശദീകരിച്ചിട്ടുണ്ട്.

എന്നാൽ, ഒരു പ്രത്യേക സ്വഭാവം ഉണ്ട് ഒരു നിര എഡിറ്റിംഗിൽ നിങ്ങൾ ഉപയോഗിക്കുന്നത് ആ നിങ്ങൾ ആദ്യം നോഡ് ചേർത്തിട്ടില്ല after ആണ് ഒരു കീ മോഡിഫയർ:

മാറ്റം

സ്വതവേ, auto-edge snap feature പ്രാപ്തമാക്കി: നിങ്ങൾ ക്ലിക്ക് മൌസ് പോയിന്റർ വലിക്കുക എപ്പോഴൊക്കെ, കത്രിക പ്രയോഗം ഒരു പുതിയ നിയന്ത്രണ നോഡ് നല്കുന്നതിനു അല്ലെങ്കിൽ നിലവിലുള്ള നോഡ് നീക്കുന്നതിനായി (നിറം മാറ്റം പരമാവധി എവിടെ) പരമാവധി ഗ്രേഡിയന്റ് പോയിന്റ് കണ്ടെത്തുന്നു.

ക്ലിക്കുചെയ്ത് ഈ സവിശേഷത അപ്രാപ്തമാക്കുന്നു ഇഴയ്ക്കുമ്പോൾ ഈ കീ ഹോൾഡിങ്, നിയന്ത്രിക്കുകയും നോഡ് മൌസ് പോയിന്റർ സ്ഥാനം സ്ഥാപിക്കുമെങ്കിലും.

2.7.3. Tool handling

ചിത്രം 14.28. Using Intelligent Scissors


മൌസ് കൊണ്ട്-ക്ലിക്ക് അവശേഷിക്കുന്നു ഓരോ തവണയും സ്വരൂപത്തിൽ വേണമങ്കിലും പിന്തുടരാൻ ശ്രമിക്കുന്ന ഒരു കർവ് കഴിഞ്ഞ നിയന്ത്രണ പോയിന്റ് ഘടിപ്പിച്ച ഒരു പുതിയ നിയന്ത്രണ പോയിന്റ്, സൃഷ്ടിക്കുക. ഫിനിഷ്, ആദ്യം പോയിന്റ് (നിങ്ങൾ ശരിയായ സ്പോട്ട് വരുമ്പോൾ കഴ്സർ മാറ്റങ്ങൾ സൂചിപ്പിക്കുന്നതിന്) ക്ലിക്ക് ചെയ്യുക. നിങ്ങൾ നിയന്ത്രണത്തിലാണ് നോഡുകൾ ഇഴച്ച് കർവ് ക്രമീകരിക്കാനും അല്ലെങ്കിൽ പുതിയ നിയന്ത്രണ നോഡുകൾ സൃഷ്ടിക്കാൻ ക്ലിക്കുചെയ്ത് കഴിയും. നിങ്ങൾ സംതൃപ്തിയുണ്ട്, ഒരു നിരക്കു അതിനെ പരിവർത്തനം എവിടെയും കർവ് അകത്ത് ക്ലിക്ക് ചെയ്യുക.

നിങ്ങൾ ഈ ഉപകരണം ഉപയോഗിച്ച് ക്ലിക്കുചെയ്യുമ്പോൾ നിങ്ങൾ പോയിന്റ് ഡ്രോപ്പ് മുകളിൽ പറഞ്ഞു. നിരക്കു അതിർത്തി ഈ നിയന്ത്രണ പോയിന്റ് നയിക്കുന്നതും. സൃഷ്ടിയുടെ സമയത്ത് ഓരോന്നും ആദ്യ കഴിഞ്ഞ ഒരു ഒഴികെ ക്ലിക്കുചെയ്ത് ഇഴച്ച് നീക്കാനാകും. നിങ്ങൾക്ക് ആദ്യത്തേത് കഴിഞ്ഞ പോയിന്റ് ക്ളിക്ക് ചെയ്യുമ്പോൾ നിരക്കു അടച്ചു. എപ്പോഴാണ് നിരക്കു അതിന്റെ നിലപാടനുസരിച്ച് പോയിന്ററിന്റെ രൂപം മാറ്റങ്ങൾ അടച്ചു: നുള്ളിൽ, അതിർത്തി മേലും പുറത്ത്. നിങ്ങൾക്ക് അതിർത്തി ക്ലിക്കുചെയ്തുകൊണ്ടോ ഓരോ കൺട്രോൾ പോയിന്റുകൾ (ലയിച്ച ആദ്യ, അവസാന പോയിന്റ്) നീക്കി പുതിയ പോയിന്റ് സൃഷ്ടിക്കുന്നത് നിരക്കു ക്രമീകരിക്കാൻ കഴിയും. നിങ്ങൾക്ക് അകത്തേക്ക് ക്ലിക്കുചെയ്യുമ്പോൾ നിരക്കു സാധൂകരിച്ചു ആണ്.

നിങ്ങൾ ഒന്നു മാത്രം നിരക്കു ലഭിക്കും ശ്രദ്ധിച്ചോ ഞങ്ങൾക്കുണ്ട്; നിങ്ങൾ ഒരു രണ്ടാം നിര സൃഷ്ടിക്കാൻ, ആദ്യം ഒന്നു നിങ്ങൾക്ക് രണ്ടാമത്തെ ഒറ്റ സാധൂകരിക്കാൻ വരുമ്പോൾ മായ്ച്ചു.

മുന്നറിയിപ്പു്

നിങ്ങൾ പൂർണ്ണമായും ക്രമീകരിച്ചുകൊണ്ട് നടക്കുന്ന വരെ കർവ് അകത്ത് ക്ലിക്ക് ചെയ്യാൻ ഉറപ്പാക്കുക. നിങ്ങൾ ഒരു നിര മാറ്റി കഴിഞ്ഞാൽ, നഷ്ടമല്ലാതൊന്നും തിരികെ പൂജ്യം എത്തുവാൻ, അത് നിങ്ങളുടെ മാറ്റണമെങ്കിൽ ആദ്യം മുതൽ വീണ്ടും കർവ് നിർമിക്കാൻ തുടങ്ങേണ്ടത്. നിങ്ങളുടെ ശ്രദ്ധാപൂർവം സൃഷ്ടിച്ച നിയന്ത്രണം നോഡുകളുടെ നഷ്ടപ്പെടും വീണ്ടും മറ്റൊരു ഉപകരണത്തിലേക്ക് മാറാനോ ഉറപ്പാക്കുക, അല്ലെങ്കിൽ. (എന്നാൽ നിങ്ങൾ ഇനിയും ഒരു പാതയിലേക്ക് നിങ്ങളുടെ നിരക്കു അഴകാർന്ന പാത ഉപകരണം ഉപയോഗിച്ച് അത് പ്രവർത്തിക്കാം.)

നിരക്കു നീക്കാൻ, Moving selections കാണുക.

2.7.4. ഓപ്ഷനുകൾ

ചിത്രം 14.29. Tool Options for the Intelligent Scissors


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

മോഡുകൾ; ആന്റിഎലിയാസിംഗ്; തൂവല് അറ്റങ്ങൾ
കുറിപ്പു്

ഈ എല്ലാ ഉപകരണങ്ങൾ സാധാരണമാണ് ഉപാധികൾക്ക് സഹായത്തിന് Selection Tools കാണുക. ഈ ഉപകരണം മാത്രമായുള്ള ഓപ്ഷനുകൾ മാത്രം ഇവിടെ വിശദീകരിച്ചു ചെയ്യുന്നു.
ഇന്ററാക്ടീവ് അതിർത്തി

ഈ ഓപ്ഷൻ പ്രാപ്തമാക്കിയിട്ടുണ്ടെങ്കിൽ, പ്ലേസ്മെന്റ് സമയത്ത് ഒരു നിയന്ത്രണ നോഡ് വലിച്ചിടുമ്പോൾ നിരക്കു ബൗണ്ടറി എടുത്ത ചെയ്യും പാതയിൽ സൂചിപ്പിക്കും. ഇത് പ്രാപ്തമാക്കിയിട്ടില്ല എങ്കിൽ, നോഡ് നിങ്ങൾ ചുറ്റും വലിച്ചോ സമയത്ത് ഒരു വര കഴിഞ്ഞ നോഡ് കണക്റ്റുചെയ്തുവെന്നത് കാണിക്കും, നിങ്ങൾ പോയിന്റർ ബട്ടൺ റിലീസ് വരെ നിങ്ങൾ തത്ഫലമായുണ്ടാകുന്ന പാത കാണുകയില്ല. സ്ലോ സിസ്റ്റങ്ങളിൽ, നിങ്ങളുടെ നിയന്ത്രണ നോഡുകൾ വ്യത്യാസപ്പെട്ടിരിക്കുന്നു, ഇത് ഒരു സ്പീഡ്-അപ്പ് ഒരു ബിറ്റ് നൽകും.
