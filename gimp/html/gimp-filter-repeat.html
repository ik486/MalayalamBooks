10.2. അവസാനം ആവർത്തിക്കുക

Repeat Last കമാൻഡ് കഴിഞ്ഞ തവണ ഇത് പ്രവർത്തിപ്പിക്കാൻ ചെയ്തു അതേ ക്രമീകരണം ഉപയോഗിച്ച് വീണ്ടും ഏറ്റവും സമീപകാലത്ത് വധിക്കപ്പെട്ട പ്ലഗ്-ഇൻ നടപടി നടത്തുന്നു. ഒരു ഡയലോഗ് അല്ലെങ്കിൽ അഭ്യർത്ഥന സ്ഥിരീകരണം കാണിക്കില്ല.

കുറിപ്പു്

ഈ കമാൻഡ് ഏറ്റവും സമീപകാലത്ത് പരിഗണിക്കാതെ അത് Filters മെനു അല്ലെങ്കിൽ എന്നത് plug-in എക്സിക്യൂട്ട് ആവർത്തിക്കുന്നു ദയവായി ശ്രദ്ധിക്കുക.

10.2.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Filters → Repeat filter,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Ctrl+F ഉപയോഗിച്ച്.
