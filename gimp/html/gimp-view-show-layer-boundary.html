5.10. Show Layer Boundary

Show Layer Boundary കമാൻഡ് പ്രാപ്തമാക്കുന്നു ഇമേജ് ജാലകത്തിൽ ഒരു പാളി ചുറ്റുന്ന മഞ്ഞ അതിനെക്കുറിച്ച് പ്രദർശിപ്പിക്കുന്നത് നിഷ്ക്രിയമാക്കുന്നു. പാളി ഇമേജ് വിൻഡോ ചെറുതാണ് വരുമ്പോൾ അതിനെക്കുറിച്ച് യഥാർത്ഥത്തിൽ മാത്രമേ കാണാനാകൂ. പാളി ഇമേജ് വിൻഡോ ആയി ഒരേ വലിപ്പം എപ്പോഴാണ്, പാളി അതിർത്തി ചിത്രം അതിർത്തിയിൽ മൂടപ്പെടാത്ത ആണ്.

നിങ്ങൾ Image Window Appearance ഡയലോഗിൽ പാളി ബൗണ്ടറിയിലേക്ക് സ്വതേ സജ്ജമാക്കാൻ കഴിയും.

5.10.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through View → Show Layer Boundary.
