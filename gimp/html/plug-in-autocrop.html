6.17. Autocrop Image

Autocrop Image കമാൻഡ് ഒരു ചിത്രത്തിൽ നിന്ന് അതിരുകൾ നീക്കംചെയ്യുന്നു. നിങ്ങൾ Crop ഉപകരണം ഉപയോഗിച്ച പോലെ അത് എല്ലാം ഒരേ നിറം ഏറ്റവും വലിയ സാധ്യത അതിർത്തി സജീവ പാളി പരിശോധിക്കയും തുടർന്ന് ഇമേജ് നിന്ന് ഈ പ്രദേശം കൃഷിചെയ്യുക.

ശ്രദ്ധിക്കുക

ഈ കമാൻഡ് മാത്രം അതിരുകൾ കണ്ടെത്താൻ ചിത്രത്തിന്റെ active layer ഉപയോഗിക്കുന്ന ശ്രദ്ധാപൂർവ്വം ശ്രദ്ധിക്കുക. മറ്റു പാളികൾ സജീവ പാളിയായി പരിധികൾ അതേ പരിധികൾ പ്രകാരം വലുപ്പം ചെയ്യുന്നു.

6.17.1. Activate the Command

  • You can access this command from the image menubar through Image → Autocrop Image.

6.17.2. Example

ചിത്രം 16.69. “Autocrop” example

ഈ ചിത്രം വരക്കുക ഉണ്ടാക്കിയ. ഒരു ചുവന്ന സ്ക്വയർ ഉപയോഗിച്ച് ഒരു, ഒരു പച്ച സ്ക്വയർ ഉപയോഗിച്ച് മറ്റൊരു; ഒരു മഞ്ഞ അർദ്ധസുതാര്യം പശ്ചാത്തലത്തിൽ ഇരുവരും. പച്ച പാളി സജീവമാണ്.

“Autocrop” പച്ച സ്ക്വയർ വലുപ്പം അതിൽ നിന്ന് ഒരു പാളി ചെയ്തിരിക്കുന്നു. മറ്റ് പാളികൾ പച്ച അതേ വലുപ്പം ക്രോപ്പുചെയ്യും ചെയ്തു. ചുവന്ന സ്ക്വയറിന്റെ ഒരു ചെറിയ ഭാഗം മാത്രമേ സൂക്ഷിച്ചിരിക്കുകയാണ്.
