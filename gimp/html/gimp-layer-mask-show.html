7.25. Show Layer Mask

Show Layer Mask കമാന്ഡ് നിങ്ങളെ പാളി അദൃശ്യ ചിത്രം തിരിഞ്ഞ് മെച്ചപ്പെട്ട മാസ്ക് കാണാൻ അനുവദിക്കുന്നു. മെനു എൻട്രി ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഒരു ചെക്ക് അതിനടുത്തായി ദൃശ്യമാകുന്നു പാളികളിൽ ലെയർ മാസ്ക് ലഘുചിത്ര ഡയലോഗ് ഒരു പച്ച അതിർത്തിയിൽ കാണിച്ചിരിക്കുന്ന. പാളി സ്വയം പുതുക്കപ്പെട്ടത് ഇല്ല; നിങ്ങൾ അത് ദൃശ്യമായ പിന്നീട് തിരിയാൻ കഴിയും.

7.25.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Mask → Show Layer Mask,

  • അല്ലെങ്കിൽ ലെയറുകൾ ഡയലോഗിൽ ലെയർ മാസ്ക് ലഘുചിത്ര ന് Alt കീ (ചില സിസ്റ്റങ്ങളിൽ Ctrl+Alt) സിംഗിൾ ക്ലിക്ക് ഇറക്കി പിടിച്ച്.

  • നിങ്ങൾ ലെയർ മാസ്ക് ലഘുചിത്ര വീണ്ടും Layer → Mask ഉപമെനുവുണ്ട് മെനു എൻട്രി അൺചെക്കുചെയ്ത് അല്ലെങ്കിൽ Alt -clicking (അല്ലെങ്കിൽ Ctrl+Alt -clicking) വഴി ഈ പ്രവർത്തനം പഴയപടിയാക്കാനുമാകും.
