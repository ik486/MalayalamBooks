16. ലോഗോ ഫിൽട്ടറുകൾ ആൽഫ

16.1. അവതാരിക

ചിത്രം 17.368. ആൽഫാ ലോഗോ ഫിൽറ്ററുകൾ മെനു


ഈ ഫിൽട്ടറുകൾ logo- generating script-fu scripts രൂപീകരിച്ചിരിക്കുന്നത്. അവർ സജീവ പാളി (ഒരു നോൺ പൂജ്യം ആൽഫ മൂല്യം കൂടെ പിക്സൽ, ആണ്) ആൽഫാ ചാനൽ പ്രത്യേക ഇഫക്റ്റുകൾ എല്ലാ തരത്തിലുള്ള ചേർക്കുക.

കുറിപ്പു്

മെനു ഇനങ്ങൾ അനുയോജ്യമായ ഫങ്ഷനുകളെ സജീവ പാളി ഒരു ആല്ഫാ ചാനൽ ഉണ്ടെങ്കിൽ മാത്രമേ പ്രവർത്തനക്ഷമമാണ്. നിങ്ങൾ മെനു ഇനങ്ങൾ ഗ്രേ ഔട്ട് എന്ന് കാണുകയാണെങ്കിൽ, add an alpha channel ശ്രമിക്കുക.

ഫിൽട്ടർ പ്രഭാവം എപ്പോഴും ആൽഫ മൂല്യങ്ങൾ പ്രകാരം പ്രയോഗിക്കും. ഏതെങ്കിലും പിക്സൽ alpha (പൂർണ്ണമായി ആഗിരണം ചെയ്യും) 255 0 വരെയുള്ള ഒരു മൂല്യം (സുതാര്യമായ) ഉണ്ട്. അത് 1 മുതൽ 254 ആൽഫ മൂല്യങ്ങൾ ഉപയോഗിച്ച് പിക്സലുകൾ ചില (അല്ലെങ്കിൽ എല്ലാ) മാത്രമാണ് partially ഒരു ഫിൽട്ടർ പ്രയോഗിക്കാൻ സാധ്യമാണ്.

ഈ ഭാഗികമായോ selecting പിക്സൽ സമാനമായ നിങ്ങൾ ശ്രദ്ധിക്കും. സത്യത്തിൽ, ആന്തരികമായി ഈ ഫിൽട്ടറുകൾ എപ്പോഴും ഒരു selection from the alpha channel ഏത് നിരക്കു പ്രതിനിധാനം ചാനലിലേക്ക് ആൽഫ മൂല്യങ്ങൾ ട്രാൻസ്ഫർ ചെയ്ത്, അതിന് ശേഷം നിരക്കു പ്രവർത്തിക്കുക.

ഒരു നിരക്കു ഒരു “Alpha to Logo” അരിപ്പ പണി?

  1. സജീവ പാളി പശ്ചാത്തല പാളി, അല്ലെങ്കിൽ add an alpha channel, ഒരു ആല്ഫാ ചാനൽ ഉണ്ടോ എന്നു ഉറപ്പാക്കുക.

    സൂചന

    ലെയർ ഡയലോഗിൽ ഒരു പാളി നാമം ബോൾഡ് ആണെങ്കിൽ, ഈ പാളി ആൽഫ ചാനൽ ഉണ്ട്.

  2. നിരക്കു വിപരീത: Select → Invert.

  3. Edit → Clear: (വിപരീതമാക്കപ്പെടും) നിരക്കു നീക്കംചെയ്യുക.

  4. “Alpha to Logo” അരിപ്പ പ്രയോഗിക്കുക (ഫിൽട്ടറുകൾ തെരഞ്ഞെടുത്തത്ഒഴിവാക്കുക, നിങ്ങൾ നിരക്കു വീണ്ടും വിപരീത ആവശ്യമില്ല).
