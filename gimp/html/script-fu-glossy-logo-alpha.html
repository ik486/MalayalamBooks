16.14. മിനുക്കമുള്ള

16.14.1. പൊതു അവലോകനം

ചിത്രം 17.403. “Glossy” ഫിൽട്ടർ ഉദാഹരണം

“Glossy” ഫിൽട്ടർ

“Glossy” ലോഗോ


ഈ ഫിൽട്ടർ ആൽഫ ലേക്കുള്ള ഘനമായ പാറ്റേണുകളും ബാധകമാണ്. ഒരു ചെറിയ 3D പ്രഭാവം ഒരു കരീനയെ മാപ്പ് ഉപയോഗിച്ച് ചേർക്കും, ഫിൽട്ടർ ഒരു ഡ്രോപ്പ് നിഴൽ ചേർക്കുന്നു ഓപ്ഷണലായി.

കുറിപ്പു്

ഇതാ ഒരു ഭാഷ കുറുക്കുവഴിയായി ഞങ്ങൾ alpha നോൺ-സുതാര്യമായ പിക്സൽ നിർവചിക്കപ്പെട്ട സജീവ പാളി പ്രദേശത്തെ സൂചിപ്പിക്കാനും ഉപയോഗിക്കുന്നു. നിങ്ങൾ ഒരു നിര “by visibility” അതിനെ ചിന്തിക്കുന്നത്. സ്വാധീനം “to the alpha” ബാധകമാക്കുന്നത് വെറും സജീവ പാളി ദൃശ്യമായ എല്ലാ പിക്സൽ ഈ കളറില് എന്നാണ്.

അരിപ്പ സ്ഥിര ഓപ്ഷനുകൾ, ഇങ്ങനെ പേര് ഉപയോഗിക്കുമ്പോൾ (മുകളിൽ കാണുക) ഒരു ലോഗോ ഒരു തിളങ്ങുന്ന കാഴ്ച്ചപ്പാടുമായി സൃഷ്ടിക്കുന്ന “Glossy” സ്ക്രിപ്റ്റ് (ചിത്രം വിൻഡോയിൽ File → Create → Logos → Glossy), നിന്നാണ്.

സജീവ പാളി ഒരു ആല്ഫാ ചാനൽ ഉണ്ട് എങ്കിൽ ഈ ഫിൽട്ടർ മാത്രമേ പ്രവർത്തിക്കൂ. അല്ലാത്തപക്ഷം, മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്.

മുന്നറിയിപ്പു്

ചിത്രം എല്ലായ്പ്പോഴും സജീവ പാളി വലിപ്പത്തിന് വലുപ്പം മാറ്റും.

16.14.2. ഫിൽട്ടർ സജീവമാക്കുക

ഈ ഫിൽട്ടർ Filters → Alpha to Logo → Glossy കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ കാണപ്പെടുന്നത്.

16.14.3. ഓപ്ഷനുകൾ

ചിത്രം 17.404. “Glossy” ഓപ്ഷനുകൾ


ബ്ലൻഡ് ഗ്രേഡിയന്റ് (ടെക്സ്റ്റ്)

സ്വതവേ, ഫിൽട്ടർ ഒരു gradient blend കൊണ്ട് ആൽഫ നിറെക്കും. Swatch ബട്ടൺ ക്ലിക്കുചെയ്താൽ ഏതെങ്കിലും വ്യത്യാസത്തിനു തിരഞ്ഞെടുക്കാം എവിടെ ലളിതമായ gradient dialog, തുറക്കും. “Text” ഒരു ടെക്സ്റ്റ് നിന്നുമുള്ള ലോഗോ സൃഷ്ടിക്കുന്ന “Glossy” ലോഗോ, റഫർ, ഇവിടെ തരമുള്ളു.

Text gradient reverse ചെക്കുചെയ്തിട്ടുള്ളപ്പോൾ, ആൽഫാ ചുവടെ തുടങ്ങുന്ന ഒരു ഗ്രേഡിയന്റ് നാഴികകല്ലായി നിറയും.

ശ്രേണി (ടെക്സ്റ്റ്)

Use pattern for text instead of gradient ചെക്കുചെയ്തിട്ടുള്ളപ്പോൾ, ആൽഫാ ഒരു പാറ്റേൺ നിറയും. നിങ്ങൾ Browse ബട്ടൺ ക്ലിക്കുചെയ്ത് നിങ്ങളുടെ നിര ഒരു പാറ്റേൺ തെരഞ്ഞെടുക്കുന്നതിനായി ഒരു patterns dialog തുറക്കാൻ കഴിയും. അസഹ്യപ്പെടുത്തി ഇടതുവശത്ത് പ്രിവ്യൂ പ്രദേശം നിലവിലെ പാറ്റേൺ പോപപ്പ് പ്രിവ്യൂ താനേ വന്നുകൊള്ളും.

ബാഹ്യരേഖയുടെ വലുപ്പം

ഈ അതിർ ഒരു തരം, ആൽഫാ (വിശദാംശങ്ങൾ ചുവടെ കാണുക) ല് ഒരു പകർപ്പ് അടങ്ങുന്ന ഒരു പാളി മനസ്സിലായി വ്യാപ്തി.

ബ്ലൻഡ് ഗ്രേഡിയന്റ് (ഔട്ട്ലൈൻ); ശ്രേണി (ഔട്ട്ലൈൻ)

ജസ്റ്റ് സജീവ ലെയറിനായുള്ള “text” ഓപ്ഷനുകൾ പോലെ ഈ ഓപ്ഷനുകൾ (Use pattern for outline instead of gradient ചെക്കുചെയ്തിട്ടുള്ളപ്പോൾ) ഗ്രേഡിയന്റ് അല്ലെങ്കിൽ പാറ്റേൺ വ്യക്തമാക്കുക ഔട്ട്ലൈൻ പ്രദേശം നിറയ്ക്കാൻ ഉപയോഗിക്കുന്ന.

പാറ്റേൺ ഓവർലേ ഉപയോഗിക്കുക

ചെക്കിൻ ചെയ്തപ്പോൾ, ഔട്ട്ലൈൻ പാളി യഥാർത്ഥ, വലുതാക്കിയ അല്ല ആൽഫ അങ്ങനെ പാറ്റേൺ മുൻ ഉള്ളടക്കങ്ങൾ (പാറ്റേൺ അല്ലെങ്കിൽ ഗ്രേഡിയന്റ്) ലയിപ്പിക്കും, ഓവർലേ mode ഉപയോഗിച്ച് വ്യക്തമാക്കിയ പാറ്റേൺ നിറയും.

വീണ്ടും, Browse ബട്ടൺ ക്ളിക്ക് അമർത്തുന്നതിലൂടെ പ്രിവ്യൂ ഐക്കൺ നിലവിലെ പാറ്റേൺ പോപപ്പ് പ്രിവ്യൂ താനേ ഒരു patterns dialog തുറക്കും.

സ്വതേ bumpmap ക്രമീകരണങ്ങൾ

ഈ ഓപ്ഷൻ ഒന്നും, അരിപ്പ എപ്പോഴും ഒരു bump map ബാധകമാകും.

പശ്ചാത്തല നിറം

ഫിൽറ്റർ ചേർത്തു പശ്ചാത്തല പാളി നിറം. നിങ്ങൾക്ക് കളർ ബട്ടൺ ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഒരു color select dialog മാറികിട്ടി.

നിഴല്

അരിപ്പ ഓപ്ഷണലായി ഒരു drop shadow അടങ്ങുന്ന ഒരു പാളി സൃഷ്ടിക്കുന്നു. നിഴൽ പാളി വലതുവശത്ത് Shadow X offset പിക്സൽ നീക്കും ഒപ്പം Shadow Y offset ഇറങ്ങി പിക്സലുകൾ. പശ്ചാത്തല പാളി സജീവ പാളി വലിപ്പം കാക്കും സമയത്ത്, ഈ ചിത്രം വിശാലമാക്കുമ്പോൾ ശ്രദ്ധിക്കുക.

16.14.4. ഫിൽറ്റർ വിശദാംശങ്ങൾ

നിരവധി ഓപ്ഷനുകൾ ഇത് വളരെ സങ്കീര്ണ്ണമാക്കുക അരിപ്പ ഉണ്ടെന്നും എന്നാൽ യഥാർത്ഥത്തിൽ ഗംഭീരമായി ലളിതമാണ് ധാരണ വരാം. രസകരമായ ഭാഗം അരിപ്പ സജീവ പാളി ഔട്ട്ലൈൻ പാളി കൈകാര്യം എങ്ങനെ ആണ്:

സജീവ പാളിയായി, അരിപ്പ ഒരു selection from the alpha channel സൃഷ്ടിക്കുന്നു വ്യക്തമാക്കിയ ഗ്രേഡിയന്റ് നാഴികകല്ലായി അല്ലെങ്കിൽ പാറ്റേൺ ഉപയോഗിച്ച് തിരഞ്ഞെടുപ്പ് നിറയുന്നു.

അപ്പോൾ സജീവ താഴെയായി ഒരു പുതിയ “outline” പാളി സമാനമായ രീതിയിൽ സൃഷ്ടിക്കപ്പെടും: ആദ്യം, സജീവ പാളി ആൽഫ ഒരു തിരഞ്ഞെടുപ്പ് നടത്താൻ ഉപയോഗിക്കും. എന്നാൽ ഒരു ഗ്രേഡിയന്റ് അല്ലെങ്കിൽ പാറ്റേൺ ഉപയോഗിച്ച് തിരഞ്ഞെടുപ്പ് പൂരിപ്പിക്കുന്നതിന് മുമ്പ്, നിരക്കു Outline size പിക്സൽ enlarged ചെയ്യും.

നിങ്ങൾ ഒരേ പാറ്റേൺ അല്ലെങ്കിൽ വ്യത്യാസത്തിനു നാഴികകല്ലായി ഉപയോഗിച്ച് രണ്ടു പാളികൾ നിറഞ്ഞപ്പോൾ, തുടർന്നും ഒരു അതിർ (“outline”), കാണും

  • ഒരു 3D പ്രഭാവം ഒരു bump map പോലെ സജീവ പാളി ഉപയോഗിച്ച് ഔട്ട്ലൈൻ പാളി ബാധകമാകും;

  • സജീവ പാളി layer mode “Screen” ലേക്ക് സജ്ജമാക്കും.

കഴിഞ്ഞ (ഓപ്ഷണൽ) ഘട്ടം “overlay” പാളി mode ഉപയോഗിച്ച് ഒരു പാറ്റേൺ ഉപയോഗിച്ച് ഔട്ട്ലൈൻ പാളി, നിറയ്ക്കുക എന്നതാണ്. ഈ മുമ്പ് ഉപയോഗിച്ച പാറ്റേൺ അല്ലെങ്കിൽ ഗ്രേഡിയന്റ് കൂടെ പാറ്റേൺ സംയോജിപ്പിക്കും. ഓവർലേ മോഡ് ഉപയോഗിച്ച് ഫലമാണ് കുറിച്ച് കൂടുതലറിയാൻ, വിഭാഗം 2, “ലെയർ മോഡുകൾ” ലെ വിവരണം കാണുക.
