3.7. അന്തസ്സായിത്തന്നെയാണു

ചിത്രം 14.71. അന്തസ്സായിത്തന്നെയാണു


അന്തസ്സായിത്തന്നെയാണു പ്രയോഗം അവ്യക്തമായ ബ്രഷ് സ്ട്രോക്കുകൾ പെയിന്റ്. എല്ലാ സ്ട്രോക്കുകൾ നിലവിലെ ബ്രഷ് ഉപയോഗിച്ച് റെൻഡർ ചെയ്യുന്നു.

3.7.1. ടൂൾ സജീവമാക്കുന്നു

  • You can call the Paintbrush Tool in the following order, from the image-menu: Tools → Paint Tools → Paintbrush.

  • The Tool can also be called by clicking the tool icon:

  • or by using the P keyboard shortcut.

3.7.2. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

Ctrl,

ഈ കീ ഒരു Color Picker ലേക്ക് അന്തസ്സായിത്തന്നെയാണു മാറ്റുന്നു.

മാറ്റം

ഈ കീ വര മോഡിലേക്ക് അന്തസ്സായിത്തന്നെയാണു മതിയാവില്ല. ഹോൾഡിംഗ് Shift Button 1 ക്ലിക്ക് ചെയ്യുമ്പോൾ ഒരു വര സൃഷ്ടിക്കും. തുടർച്ചയായ ക്ലിക്കുകൾ കഴിഞ്ഞ വരിയുടെ അവസാനം നിന്നും ആ നേരായ ഡ്രോയിംഗ് തുടരും.

3.7.3. ഓപ്ഷനുകൾ

ചിത്രം 14.72. Paintbrush tool options


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

Mode; Opacity; Brush; Dynamics; Dynamics Options; Apply Jitter; Smoot Stroke; Incremental:
പല അല്ലെങ്കിൽ എല്ലാ പെയിന്റ് ഉപകരണങ്ങൾ ബാധകമായ ഉപകരണം ഓപ്ഷനുകൾ ഒരു വിവരണം വേണ്ടി Common Paint Tool Options കാണുക.
