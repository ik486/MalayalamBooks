4.1. ഉദ്ദേശം

GIMP നിരവധി ഓപ്ഷനുകൾ ഉപകരണങ്ങളും ഉള്ള ശക്തമായ ഇമേജ് എഡിറ്റിംഗ് പ്രോഗ്രാം ആണ്. എന്നിരുന്നാലും, ചെറിയ ചെറിയ ചുമതലകൾ വളരെ ഉചിതമാകുന്നു. താഴെ ട്യൂട്ടോറിയലുകൾ പൊതുവെ GIMP എല്ലാ െൻ കമ്പ്യൂട്ടർ ഗ്രാഫിക്സ് പഠിക്കാൻ ഇല്ലാതെ പൊതുവായ ഈ ചുമതലകൾ നേടാൻ ആഗ്രഹിക്കുന്നവർക്ക് നടപ്പിലാക്കുന്നത്.

ഭാഗ്യവശാൽ ഈ ട്യൂട്ടോറിയലുകൾ നിങ്ങളുടെ നിലവിലെ ദൗത്യം നിങ്ങളെ സഹായിക്കും മാത്രമല്ല, മാത്രമല്ല നിങ്ങളുടെ സമയം പ്രചോദനവും ഉണ്ടെങ്കിൽ, നിങ്ങൾ പിന്നീട് സങ്കീർണ്ണമായ ഉപകരണങ്ങളും രീതികളും പഠിക്കാൻ തയ്യാറാവുക.

ഈ ട്യൂട്ടോറിയൽ ആരംഭിക്കുക അറിയേണ്ടതെല്ലാം എല്ലാ കണ്ടെത്താനും തുറക്കാൻ എങ്ങനെ നിങ്ങളുടെ ചിത്രം. (ചിത്രം വിൻഡോയിൽ നിന്ന് File → Open).
