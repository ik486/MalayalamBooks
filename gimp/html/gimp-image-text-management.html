1. ടെക്സ്റ്റ് മാനേജ്മെന്റ്

ടെക്സ്റ്റ് ഉപകരണം ഉപയോഗിച്ച് നിയന്ത്രിക്കുന്നത്. പാഠ പെട്ടി വലിപ്പം കൊണ്ട് പാളി ഡയലോഗിൽ നിലവിലെ പാളി മുകളിൽ ഈ ഉപകരണം ടെക്സ്റ്റ് അടങ്ങുന്ന ഒരു പുതിയ പാളി, സൃഷ്ടിക്കുന്നു. പേര് വാചകം ആരംഭമാണ്.

ചിത്രം 9.1. ഒരു വാചക ഇനത്തിന്റെ ഉദാഹരണം

ഒരു വാചക ഇനത്തിന്റെ ഉദാഹരണം, ടെക്സ്റ്റ് പാളിയുടെ അതിർത്തി കാണിക്കുന്ന. (ഫോണ്ട്: ആദർശപൂർണ്ണമായ ബോൾഡ്)

പാളി ഡയലോഗ്, നിലവിലെ ആയിരുന്നു പാളി മുകളിലുള്ള പാഠ പാളി.


Text tool ക്രമേണ മെച്ചപ്പെട്ടു ആണ്. GIMP -2,8 ഉപയോഗിച്ച്, ഇപ്പോൾ ടെക്സ്റ്റ് നേരിട്ട് കാൻവാസിൽ എഡിറ്റുചെയ്യാം. ഒരു ടെക്സ്റ്റ് ടൂൾ ബോക്സ് ടെക്സ്റ്റ് ബോക്സിൽ മുകളിൽ ക്യാൻവാസ് ഓവർലേകളും ഏത് ചേർത്തു.

ഉടൻ text tool ഉപയോഗിച്ച് ക്യാൻവാസ് ക്ലിക്ക് പോലെ വെറും മുകളിൽ ഒരു അടച്ച ടെക്സ്റ്റ് ബോക്സ് ഒരു സെമി സുതാര്യമായ പ്രയോഗം ബോക്സ് നേടുകയും.

Text tool ഓപ്ഷനുകൾ വിഭാഗം 6.6, “ടെക്സ്റ്റ്” വിവരിച്ചിരിക്കുന്നു.

സ്ഥിര ബോക്സ് മോഡ് “Dynamic”, നിങ്ങൾ ഒരേസമയം ടെക്സ്റ്റ് ടൈപ്പുചെയ്തു തുടങ്ങാം. ടെക്സ്റ്റ് ബോക്സ് ക്രമേണ വിശാലമാക്കും. ഒരു പുതിയ ലൈൻ ചേർക്കാൻ Enter അമർത്തുക.

നിങ്ങൾ തിരഞ്ഞെടുക്കലുകളിൽ ചെയ്യുന്നതുപോലെ, ക്ലിക്കിൽ-ആൻഡ്-വലിച്ചോ enlarge the text box കഴിയും. ആ ബോക്സ് മോഡ് മാറുകയാണെങ്കിൽ "നിശ്ചിത" എന്ന ശ്രദ്ധിക്കുക. അത് ഒരു പുതിയ ലൈൻ ചേർക്കാൻ Enter അമർത്തുക.

edit text, നിങ്ങൾ ആദ്യം നിങ്ങൾ ക്ലിക്കിൽ-ആൻഡ്-വലിക്കുകയോ Shift+arrow key വഴി എഡിറ്റ് തുടർന്ന് വിഭാഗം 2, “ടെക്സ്റ്റ് ടൂൾ ബോക്സ്” എന്ന ഓപ്ഷനുകൾ ഉപയോഗിക്കാൻ ആഗ്രഹിക്കുന്ന ഭാഗം തിരഞ്ഞെടുക്കണം.

Instead of using the on-canvas text editing, you can use the text editor dialog described in വിഭാഗം 6.6.4, “ടെക്സ്റ്റ് എഡിറ്റർ”.

നിങ്ങൾ നീക്കാൻ ഉപകരണം ഉപയോഗിച്ച് ചിത്രത്തിൽ move the text കഴിയും: നിങ്ങൾ ഒരു പ്രതീക, അല്ലാതെ പശ്ചാത്തലത്തിൽ ക്ലിക്ക് ചെയ്യണം.

പ്ലസ് Ctrl+Shift+U കൂടെ Unicode characters ആവശ്യമുള്ള ചാർ എന്ന ഹെക്സാഡെസിമൽ യൂണിക്കോഡ് കോഡ് ലഭിക്കും. ദയവായി വിഭാഗം 6.6.4, “ടെക്സ്റ്റ് എഡിറ്റർ” കാണുക.

, Layer dialog വാചകം പാളി സജീവമാക്കുന്നത് തിരഞ്ഞെടുക്കുക text tool ഇമേജ് വിൻഡോയിൽ ടെക്സ്റ്റ് ക്ലിക്ക്: ടെക്സ്റ്റ് പാളി ഇപ്പോഴും നിലവിലുണ്ട്; മറ്റൊരു ഉപകരണം (ചുവടെ കാണുക) പുതുക്കപ്പെട്ടത് ചെയ്തിട്ടില്ലെങ്കിൽ നിങ്ങൾ edit the text later കഴിയും.

ടെക്സ്റ്റ് ലെയർ കൈകാര്യം

നിങ്ങൾ മറ്റേതെങ്കിലും പാളി അതേ വഴികളിൽ ഒരു ടെക്സ്റ്റ് ലയറിൽ ഓപ്പറേറ്റ് കഴിയും, അങ്ങനെ ചെയ്യുന്നത് പലപ്പോഴും നിങ്ങളുടെ സൃഷ്ടിയുടെ ഫലങ്ങൾ നഷ്ടപ്പെടാതെ ടെക്സ്റ്റ് എഡിറ്റ് കഴിവ് ഉപേക്ഷിക്കാതെ എന്നാണ്.

അത് ഒരു text- എഡിറ്റർ ഫോർമാറ്റിൽ വാചകം ഒരു അടങ്ങുന്നു: ടെക്സ്റ്റ് കൈകാര്യം idiosyncrasies ചില മനസ്സിലാക്കാൻ ഒരു ടെക്സ്റ്റ് പാളി നിങ്ങൾ കാണുന്ന പിക്സൽ ഡാറ്റ കൂടുതൽ വിവരങ്ങൾ അതിലുണ്ട് എന്ന് തിരിച്ച സഹായിച്ചേക്കാം. നിങ്ങൾ ടെക്സ്റ്റ് ഉപകരണം ഉപയോഗിച്ച് പോൾ മാറികിട്ടി ടെക്സ്റ്റ്-എഡിറ്റർ വിൻഡോയിൽ ഈ കാണാനാകും. നിങ്ങൾ ടെക്സ്റ്റ് മാറ്റംവരുത്തിയോ ഓരോ പ്രാവശ്യവും, ഇമേജ് പാളി നിങ്ങളുടെ മാറ്റങ്ങൾ പ്രതിഫലിപ്പിക്കാൻ ബാഹ്യ ആണ്.

ഉദാഹരണത്തിന്, തിരിക്കുന്നതിന്: ഇപ്പോൾ ഒരു ടെക്സ്റ്റ് പാളി, അതിന് ശേഷം ടെക്സ്റ്റ് പ്രയോഗം ഉള്പ്പെടുന്നില്ല ചില വിധത്തിൽ അത് ഓപ്പറേറ്റ് കരുതുക. പിന്നെ നിങ്ങൾ വന്ന് ഉള്ളടക്കം ഉപകരണം ഉപയോഗിച്ച് അത് എഡിറ്റ് ശ്രമിക്കുക കരുതുക. ഉടൻ പാഠം എഡിറ്റ് പോലെ text tool അതേസമയം നിർവഹിച്ച പ്രവർത്തനങ്ങൾ ഫലം തുടച്ച്, പാളി .Sunday, ചെയ്യും.

ഈ അപകടം വ്യക്തമായ അല്ലാത്തതിനാൽ, ടെക്സ്റ്റ് ടൂൾ നിന്ന് നിങ്ങളെ പരിരക്ഷിക്കാൻ ശ്രമിക്കുന്നു. നിങ്ങൾ ടെക്സ്റ്റ് എഡിറ്റ് ശ്രമിക്കുക ഒരു ടെക്സ്റ്റ് ലയറിൽ പിന്നീട് ഓപ്പറേറ്റ് ചെയ്തു ഒരു സന്ദേശം നിങ്ങളുടെ അയല്ക്കാര് പഴയപടിയാക്കാൻ എന്നു മുന്നറിയിപ്പ് മാറികിട്ടി, നിങ്ങൾക്ക് മൂന്ന് ഓപ്ഷനുകൾ നൽകുന്ന:

  • ഏതുവിധേനയും പാഠം എഡിറ്റ്;

  • റദ്ദാക്കുന്നതിന്;

  • നിലവിലുള്ള പാളി മാറ്റമില്ലാതെ, നിലവിലുള്ള പാളി അതേ വാചകം ഉപയോഗിച്ച് ഒരു പുതിയ ടെക്സ്റ്റ് ലയർ ഉണ്ടാക്കുക.

ചിത്രം 9.3. ഉള്ളടക്കം എഡിറ്റിംഗ് സന്ദർഭ മെനു


ടെക്സ്റ്റ് റൈറ്റ് ക്ലിക്ക് മെനു നേടുക. ഇത് ടെക്സ്റ്റ് എഡിറ്റർ ഡയലോഗ് ആ നിന്നും കുറച്ച് വ്യത്യസ്തമാണ്.

  • Cut, Copy, Paste, Delete: ഈ ഉപാധികൾ തിരഞ്ഞെടുത്ത ടെക്സ്റ്റ് കാര്യത്തിൽ. അവർ നീണ്ട വാചകങ്ങൾ ഒന്നും തിരഞ്ഞെടുത്തിട്ടില്ല പോലെ ഗ്രേ ഔട്ട് നിലനിൽക്കും. ക്ലിപ്ബോർഡ് വാചകം നിറഞ്ഞിരിക്കുന്നു എങ്കിൽ “Paste” സജീവമാകുന്നു.

  • Open text file: ഈ കമാൻഡ് പിടികൂടാനുള്ള ടെക്സ്റ്റ് ഫയൽ കണ്ടെത്താൻ കഴിയുന്ന ഒരു ഫയൽ ബ്രൌസർ തുറക്കുന്നു.

  • Clear: ഈ കമാൻഡ് ടെക്സ്റ്റ് തിരഞ്ഞെടുത്ത അല്ലെങ്കിൽ ഇല്ലാതാക്കുന്നു.

  • Path from text: ഈ കമാൻഡ് നിലവിലെ വാചകത്തിന്റെ രൂപരേഖകൾ നിന്ന് ഒരു പാത സൃഷ്ടിക്കുന്നു. ഫലം പ്രകടമാണ് അല്ല. നിങ്ങൾ പാത ഡയലോഗ് തുറന്ന് പാത ദൃശ്യമാക്കാൻ ഞങ്ങൾക്കുണ്ട്. അപ്പോൾ പാത സെലക്റ്റ് ടെക്സ്റ്റ് ക്ലിക്ക് ചെയ്യുക. ഓരോ കത്ത് ഇപ്പോൾ ഒരു പാത ഘടകം ഉപയോഗിച്ച് ചുറ്റും. അങ്ങനെ നിങ്ങൾ പാത്ത് നിയന്ത്രണ പോയിന്റുകൾ നീക്കി കത്തുകൾ ആകൃതി പരിഷ്കരിക്കാനും കഴിയും.

    ഈ കമാൻഡ് Layer → Text to Path സമാനമാണ്.

  • Text along path:

    ഈ ഓപ്ഷൻ ഒരു path നിലവിലുണ്ട് മാത്രമേ പ്രാപ്തമാക്കി. നിങ്ങളുടെ വാചക സൃഷ്ടിക്കുമ്പോൾ, തുടർന്ന് സൃഷ്ടിക്കാൻ അല്ലെങ്കിൽ ഇറക്കുമതി ഒരു പാത അതു സജീവവും. നിങ്ങൾ ടെക്സ്റ്റ് മുമ്പ് നിങ്ങളുടെ പാത ഉണ്ടാക്കി, പാത അദൃശ്യനായ മാറുന്നു നിങ്ങൾ മാർഗത്തിൽ ഡയലോഗിൽ ദൃശ്യമാക്കാൻ ഞങ്ങൾക്കുണ്ട്.

    Text along Path ബട്ടണിൽ ക്ലിക്ക്. ടെക്സ്റ്റ് പാതയിലൂടെ ഉരുകിയിരിക്കുന്നു. അക്ഷരങ്ങൾ അവരുടെ രൂപരേഖയിൽ കൊണ്ട് പ്രതിനിധീകരിക്കുന്നത്. അവരിൽ ഓരോ Path dialog പ്രത്യക്ഷപ്പെടുന്ന പുതിയ പാത ഒരു ഘടകമാണ്. എല്ലാ പാത്ത് ഓപ്ഷനുകൾ ഈ പുതിയ പാതയിലേക്ക് അപേക്ഷിക്കണം.

  • From Left to Right / From Right to Left: നിങ്ങളുടെ ഭാഷ എഴുതേണ്ട ദിശ പരിഹരിക്കാൻ.

  • Input Methods: രീതികൾ ചില ഭാഷകളിൽ ലഭ്യമാണ്. ഉദാഹരണത്തിന്, “Inuktitut” തിരഞ്ഞെടുത്ത്, നിങ്ങളുടെ കീബോർഡ് ഒരു ഇനുക്റ്റിടട്ട് കീബോർഡ് കയറി, താൽക്കാലികമായി അവസ്ഥയാണ്.
