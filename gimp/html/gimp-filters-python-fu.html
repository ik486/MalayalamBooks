10.5. The “Python-Fu” Submenu

ചിത്രം 16.202. The “Python-Fu” submenu


സ്വതവേ ഈ ഉപമെനുവുണ്ട് വെറും python-ഫു കൺസോൾ അടങ്ങിയിരിക്കുന്നു.

പൈത്തൺ-ഫു GIMP വേണ്ടി പ്ലഗിനുകളുടെ എഴുത്തു അനുവദിച്ചുകൊണ്ട് libgimp ഒരു റാപ്പർ ആയി പ്രവർത്തിക്കണം Python ഘടകങ്ങൾ ഒരു കൂട്ടം.

10.5.1. Activating the submenu

  • You can access this command from the image menu through Filters → Python-Fu

10.5.2. The Python-Fu Console

പൈത്തൺ-ഫു കൺസോൾ ഒരു “Python shell” (ഇന്ററാക്ടീവ് മോഡിൽ ഒരു പൈത്തണ് ഓടി) ഒരു ഡയലോഗ് വിൻഡോ ആണ്. ഈ കൺസോൾ libgimp ആന്തരിക GIMP ലൈബ്രറി റൂട്ടീനുകളിലൂടെ ഉപയോഗം വരുത്തുവാൻ സജ്ജീകരിച്ചു.

പരസ്പരം പൈത്തൺ കമാൻഡുകൾ പരിശോധിക്കുന്നതിന് python-ഫു കൺസോൾ ഉപയോഗിക്കാം.

കൺസോൾ നിങ്ങൾക്ക് പൈഥൺ കമാൻഡുകൾ ടൈപ്പ് കഴിയുന്ന ഒരു വലിയ സ്ക്രോൾ പ്രധാന ഇൻപുട്ട്, ഔട്ട്പുട്ട് വേണ്ടി വിൻഡോ ഉള്ളത്. നിങ്ങൾ തുടർന്ന് Enter കീ അമർത്തുക പൈഥൺ കമാൻഡ് ടൈപ്പ് ചെയ്യുമ്പോൾ, കമാൻഡ് പൈത്തണ് വെടിവച്ച് ആണ്. കമാൻഡ് ന്റെ ഔട്ട്പുട്ട് തന്നെ കമ്പനിയുടെ മടക്കം മൂല്യം (അതിന്റെ പിശക് സന്ദേശം, എന്തെങ്കിലുമുണ്ടെങ്കിൽ) പ്രധാന വിൻഡോയിൽ ദൃശ്യമാകും.

രക്ഷിക്കും

ഈ കമാൻഡ് പ്രധാന ജാലകത്തിന്റെ ഉള്ളടക്കം സംരക്ഷിക്കാൻ അനുവദിക്കുന്നു, ആ python-ഫു കൺസോൾ ഇൻപുട്ട്, ഔട്ട്പുട്ട് (“>>>” പ്രോംപ്റ്റ് ഉൾപ്പെടെ) ആണ്.

തെളിഞ്ഞ

Wenn ഈ ബട്ടൺ ക്ലിക്ക് പ്രധാന വിൻഡോയുടെ ഉള്ളടക്കം നീക്കംചെയ്യും. നിങ്ങൾ Save കമാൻഡ് ഉപയോഗിച്ച് നീക്കംചെയ്ത ഉള്ളടക്കങ്ങൾ തിരികെ ലഭിക്കില്ല എന്ന് ശ്രദ്ധിക്കുക.

Browse

ക്ലിക്കുചെയ്യുമ്പോൾ, procedure browser വിൻഡോയുടെ താഴെയുള്ള അധിക ബട്ടൺ Apply കൂടെ മാറികിട്ടി.

നടപടി ബ്രൗസറിൽ ഈ Apply ബട്ടൺ അമർത്തുമ്പോൾ, തിരഞ്ഞെടുത്ത നടപടിക്രമം ഒരു കോൾ പൈഥൺ കമാൻഡ് പോലെ കണ്സോള് ജാലകത്തിൽ ഒട്ടിക്കുകയുള്ളൂ:

അപ്ലൈഡ് PDB നടപടിക്രമം

ഉദാ യഥാർത്ഥ മൂല്യങ്ങൾ കൂടെ: ഇപ്പോൾ നിങ്ങൾക്ക് വെറും (“width”, “height”, ഒപ്പം “type” ഇവിടെ) പാരാമീറ്റർ പേരുകൾ മാറ്റിസ്ഥാപിക്കാൻ ഞങ്ങൾക്കുണ്ട്

ചിത്രം = pdb.gimp_image_new (400, 300, ആർജിബി)

അപ്പോൾ കമാൻഡ് നടപ്പിലാക്കുന്നതിനായി Enter അമർത്തുക.

നിങ്ങൾ (വേണ്ടത്!) നടപടി ന്റെ മാനദണ്ഡങ്ങളുടെ decription കണ്ടെത്താനും സ്ഥിരാങ്കങ്ങൾ, ഉദാഹരണത്തിന് “RGB-IMAGE” അല്ലെങ്കിൽ “OVERLAY-MODE” വേണ്ടി ഉപയോഗിക്കാൻ കഴിയും. RGB_IMAGE, OVERLAY_MODE: എന്നാൽ നിങ്ങൾ അടിവരകൾ (“_”) ഉപയോഗിച്ച് ഹൈഫനുകളോ (“-”) മാറ്റിസ്ഥാപിക്കുക കാര്യം ശ്രദ്ധിക്കുക.

സൂചന

പൈത്തൺ-ഫു വെറും PDB (GIMP നടപടിക്രമങ്ങൾ ഡാറ്റാബേസ്) മുതൽ നടപടിക്രമങ്ങൾ വിളിക്കുന്നു പരിമിതപ്പെടുത്താത്തതിൽ. മുകളിലുള്ള ഉദാഹരണം പോലെ ഒരു പുതിയ ഇമേജ് ഒബ്ജക്റ്റ് സൃഷ്ടിക്കാൻ, നിങ്ങൾക്ക് ടൈപ്പ് ചെയ്യാം

              image = gimp.Image(width, height, type)

(“width”, “height”, ഒപ്പം “type” യഥാർത്ഥ മൂല്യങ്ങൾ ഉപയോഗിച്ച്).
