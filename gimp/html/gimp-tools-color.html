5.1. പൊതു അവലോകനം

ചിത്രം 14.138. The Color tools in the Tools menu

“classical” ഉപകരണങ്ങൾ മെനുവിലൂടെ കളർ ഉപകരണങ്ങൾ ആക്സസ്സുചെയ്യുക.


ചിത്രം 14.139. The Color tools in the Colors menu

കളേഴ്സ് മെനു വഴി പ്രവേശനം എളുപ്പം വേഗത്തിൽ ആണ്.


കളർ ഉപകരണങ്ങൾ നിരവധി മാർഗത്തിൽ ഇമേജ് നിറങ്ങൾ കൗശലപൂർവം കഴിയും:

  • നിറം ബാലൻസ് പരിഷ്ക്കരിക്കുക: വിഭാഗം 5.2, “കളർ ബാലൻസ്”

  • ഹ്യൂയേ, കളർ, ചാപല്യം അളവ് ക്രമീകരിക്കുക: ???

  • വിഭാഗം 5.4, “Colourise”: വർണ്ണമിട്ട ഗ്ലാസ് കാണാൻ ഒരു greyscale പ്രതിമയായി തരണം

  • മിഴിവും കോൺട്രാസ്റ്റ് അളവ് ക്രമീകരിക്കുക: ???

  • പിക്സൽ മൂല്യം അനുസരിച്ച് ഒരു ബ്ലാക്ക് ആൻഡ് വൈറ്റ് ചിത്രം മാറാവുന്ന: വിഭാഗം 5.6, “ഉമ്മറം”

  • ഒരു ചാനലിലെ തീവ്രത പരിധി മാറ്റുക: വിഭാഗം 5.7, “ലെവലുകൾ”

  • നിറം, തെളിച്ചം, തീവ്രത അല്ലെങ്കിൽ സുതാര്യതയും സങ്കീർണ്ണമായ വഴിയിൽ മാറ്റുക: വിഭാഗം 5.8, “കർവ്സ്”

  • നിറങ്ങള് കുറയ്ക്കും: വിഭാഗം 5.9, “Posterise”

  • എല്ലാ വർണ്ണങ്ങളും ചാര ഇതേ ഷേഡുകൾ പരിവർത്തനം: ???
