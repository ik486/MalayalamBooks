7.26. Edit Layer Mask

നിങ്ങൾ Layer Mask ഉപമെനുവുണ്ട് ന് Edit Layer Mask ഇനത്തിൽ ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഒരു ചെക്ക് അതിനടുത്തായി ദൃശ്യമാകും, ലെയർ മാസ്ക് നിലവിലെ പാളി സജീവ ഘടകം മാറുന്നു ലെയർ മാസ്ക് അടർന്നു പോയി ഒരു ചുവന്ന അതിർത്തിയിൽ ഡയലോഗ് പ്രദർശിപ്പിച്ചിരിക്കുന്നു. നിങ്ങൾ അത് അൺചെക്ക് ചെയ്യുമ്പോൾ, പാളി സ്വയം സജീവ ഘടകം മാറുന്നു അതു ഒരു വെളുത്ത അതിർത്തിയിൽ ഉപയോഗിച്ച് പ്രദർശിപ്പിക്കും. നിങ്ങൾ പാളികളുടെ സംഭാഷണം അത് ക്ലിക്കുചെയ്ത് കൂടുതൽ ലളിതമായി നം പ്രവർത്തനസജ്ജമാകും.

7.26.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Mask → Edit Layer Mask.

  • നിങ്ങൾ Layer → Mask മെനുവിൽ മെനു എൻട്രി അൺചെക്കുചെയ്ത് അല്ലെങ്കിൽ ലെയറുകൾ ഡയലോഗിൽ പാളി ഘടകം ക്ലിക്കുചെയ്ത് ഈ പ്രവർത്തനം പഴയപടിയാക്കാനുമാകും.
