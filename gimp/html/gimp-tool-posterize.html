5.9. Posterise

ഈ ഉപകരണം ബുദ്ധിപരമായി നിരക്കു അല്ലെങ്കിൽ സജീവ പാളി പിക്സൽ നിറങ്ങൾ ഭാരവും യഥാർത്ഥ ചിത്രത്തിന്റെ സ്വഭാവസവിശേഷതകൾ ഒരു മരീചിക നിലനിർത്തിക്കൊണ്ടുതന്നെ നിറങ്ങള് കുറയ്ക്കും രൂപകല്പന.

5.9.1. ടൂൾ സജീവമാക്കുന്നു

You can get to this tool in several ways:

  • In the image menu through Tools → Color Tools → Posterize or Colors → Posterize.

  • By clicking on the tool icon in Toolbox, if this tool has been installed there. For this, please refer to വിഭാഗം 1.7, “പണിസഞ്ചി”.

5.9.2. ഓപ്ഷനുകൾ

ചിത്രം 14.173. Posterize tool options


Posterize Levels

അമ്പ്-തല ഈ സ്ലൈഡർ ഇൻപുട്ട് ബോക്സുകൾ നിങ്ങൾ ഉപകരണം സജീവ പാളി വിശദീകരിക്കാനുള്ള ഉപയോഗിക്കുന്ന ഓരോ ആർജിബി ചാനലിൽ അളവ് എണ്ണം (2-256) സജ്ജമാക്കാൻ അനുവദിക്കുന്നു. നിറങ്ങൾ എണ്ണം ഈ നിലകൾ സംയോജനമാണ്. 3 എ ലെവൽ 2 3 = 8 നിറങ്ങൾ തരും.

പ്രിവ്യൂ

പ്രിവ്യൂ ചെക്ക്-ബോക്സ് അവർ ഉടനെ കാണാവുന്നതാണ് അങ്ങനെ ചലനാത്മകമായി എല്ലാ മാറ്റങ്ങളും ചെയ്യുന്നു.

5.9.3. Example

ചിത്രം 14.174. “Posterize” ടൂളിനായി ഉദാഹരണം

ചിത്രം 4 നിലകളിലും ഉള്ള പോസ്റ്ററൈസ്. ആമി പുറമേ കറുപ്പും വെളുപ്പും കൗണ്ടിംഗ്, 4 നിലവാരവും 10 നിറങ്ങൾ കാണിക്കുന്നു.
