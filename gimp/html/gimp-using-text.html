3.1. Embellishing ടെക്സ്റ്റ്

ചിത്രം 9.7. ഫാൻസി ടെക്സ്റ്റ്

ലോഗോ സ്ക്രിപ്റ്റുകൾ ഉപയോഗിച്ച് സൃഷ്ടിച്ച നാല് ഫാൻസി ടെക്സ്റ്റ് ഇനങ്ങൾ: “alien neon”, “bovination”, “frosty”, ഒപ്പം “chalk”. സ്ഥിരസ്ഥിതി ക്രമീകരണങ്ങൾ ഫോണ്ട് വലുപ്പം ഒഴികെയുള്ള എല്ലാം കൂടി ഉപയോഗിച്ചു.


വെറും വ്യത്യസ്ത ഫോണ്ടുകൾ അല്ലെങ്കിൽ വ്യത്യസ്ത നിറങ്ങൾ അതു തർജ്ജമ അപ്പുറം ടെക്സ്റ്റ് രൂപം വ്യത്യാസപ്പെടാം ചെയ്യാൻ കഴിയും പലതും ഉണ്ട്. ഒരു നിരക്കു അല്ലെങ്കിൽ പാതയിലേക്ക് ഒരു ടെക്സ്റ്റ് ഇനം പരിവർത്തനം നിങ്ങൾ രൂപരേഖകൾ, പൂരിപ്പിക്കാൻ കഴിയും സ്ട്രോക്ക്, അതിനെ മാറ്റുകയും, അല്ലെങ്കിൽ പൊതുവായി രസകരമായ ഇഫക്റ്റുകൾ ലഭിക്കുന്നതിന് ജിമ്പ് ഉപകരണങ്ങൾ മുഴുവൻ സർവ്വായുധവർഗ്ഗം ബാധകമാണ്. സാധ്യതകൾ ചില ഒരു പ്രകടനം നിലയിൽ File → Create → Logos ചെയ്തത് "ലോഗോ" സ്ക്രിപ്റ്റുകൾ പരീക്ഷിക്കുക. ഈ സ്ക്രിപ്റ്റുകൾ ഓരോ നിങ്ങൾ ചില ടെക്സ്റ്റ് നൽകാൻ അനുവദിക്കുന്നു, തുടർന്ന് ആ പാഠം നിന്നു പണികഴിപ്പിച്ചത് ഒരു ലോഗോ കാണിക്കുന്ന ഒരു പുതിയ ഇമേജ് ഉണ്ടാക്കുന്നു. നിങ്ങൾ ഈ സ്ക്രിപ്റ്റുകൾ ഒന്ന് പരിഷ്ക്കരിക്കാൻ ഇഷ്ടമായി, അല്ലെങ്കിൽ നിങ്ങളുടെ സ്വന്തം ഒരു ലോഗോ സ്ക്രിപ്റ്റ് നിർമ്മിക്കാൻ ആഗ്രഹിക്കുന്നതെങ്കിൽ, Using Script-Fu ആൻഡ് Script-Fu Tutorial വിഭാഗങ്ങൾ ആരംഭിക്കാൻ നിങ്ങളെ സഹായിക്കും. തീർച്ചയായും, സ്ക്രിപ്റ്റ്-ഫു ഇഫക്റ്റുകൾ ഈ എല്ലാത്തരം സൃഷ്ടിക്കാൻ അവരെ ഓട്ടോമേറ്റ് മാത്രം ആവശ്യമില്ല.
