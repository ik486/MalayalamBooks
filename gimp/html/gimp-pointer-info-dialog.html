5.8. Pointer Dialog

ചിത്രം 15.80. Pointer dialog


ഈ ഡയലോഗ്, നിങ്ങൾ പ്രദാനം ഒരു സമാന വിൻഡോയിൽ നിറം മോഡൽ തത്സമയം, മൌസ് പോയിന്റർ സ്ഥാനം, ഒപ്പം കൂർത്തതും പിക്സൽ ചാനൽ മൂല്യങ്ങൾക്ക്.

5.8.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Pointer” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • Windows → Dockable Dialogs → Pointer: ഇമേജ് മെനുവിൽ നിന്ന്.

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും PLACEHOLDER,-100004 ഞെക്കി, Add Tab → Pointer തിരഞ്ഞെടുത്ത്.

5.8.2. “Pointer” ഡയലോഗ് ഓപ്ഷനുകൾ

പിക്സൽ

എക്സ് (തിരശ്ചീനമായി) ൽ കൂർത്ത പിക്സൽ സ്ഥാനം കാണിക്കുന്നു, Y (ലംബമായ) ഏകോപിപ്പിക്കുകയും, ഉത്ഭവം (ക്യാൻവാസ് മുകളിലെ ഇടത് കോണിൽ) മുതൽ പിക്സലിൽ പ്രസ്താവിച്ചു.

യൂണിറ്റുകൾ

ഇഞ്ച് ലെ, ഉത്ഭവം നിന്നുള്ള ദൂരം കാണിക്കുന്നു.

പോയിന്റർ ബൗണ്ടിംഗ് ബോക്സ്

ഈ വിവരം സജീവ ഒരു നിര ഉള്ളപ്പോൾ ആണ്. X, Y ചതുരാകൃതിയിലുള്ള പരിധികൾ തിരഞ്ഞെടുക്കലുകൾ അണ്ഡാകൃതി ആ ചതുരാകൃതിയിലുള്ള ഫ്രെയിം മുകളിലെ ഇടത് കോണിൽ എന്ന കോർഡിനേറ്റുകളാണ്. എച്ച് ആൻഡ് പ ബോക്സ് ഉയരം വീതി ആകുന്നു.

ഈ വിവരങ്ങൾ മറ്റ് തിരഞ്ഞെടുപ്പുകളെ കടക്കുന്നു, എന്നാൽ അവർ കുറവ് പലിശ വ്യക്തിയാണെന്നും ബൗണ്ടിംഗ് ബോക്സ് ദൃശ്യമാകില്ല.

നിരക്കു സംബന്ധിച്ചു ഈ വിവരം പോയിന്റർ നിർദ്ദേശാങ്കങ്ങൾ വ്യത്യാസപ്പെട്ടിരിക്കും അതേസമയം, മറ്റൊരു ഉപകരണം ഉപയോഗിക്കുമ്പോൾ മാറ്റമില്ല.

ചാനൽ മൂല്യങ്ങൾ

തിരഞ്ഞെടുത്ത color model ചാനൽ മൂല്യങ്ങൾ താഴെ കൊടുത്തിരിക്കുന്നു. പുൾഡൌൺ മെനുകളിൽ രണ്ടും വ്യത്യസ്ത നിറം മോഡലുകൾ ഉപയോഗിച്ച് ഒരു പ്രത്യേക പിക്സൽ നിറം മൂല്യങ്ങൾ താരതമ്യം എളുപ്പമാക്കുന്ന അതേ തിരഞ്ഞെടുപ്പുകൾ, അടങ്ങിയിട്ടുണ്ട്. “Hex” ഹെക്സാഡെസിമൽ ലെ പിക്സൽ നിറം HTML Notation ആണ്. പുൾഡൌൺ മെനുകളിൽ തിരഞ്ഞെടുക്കാൻ (Pixel സ്വതവേ) ഇവയാണ്:

പിക്സൽ

RGB ചാനൽ മൂല്യങ്ങൾ. ഈ ചോയ്സ് 0 255 നും നമ്പറുകൾക്ക്, പിക്സൽ Red, Green, Blue ആൻഡ് Alpha മൂല്യങ്ങൾ പ്രദർശിപ്പിക്കുന്നു.

ആർജിബി

RGB ചാനൽ മൂല്യങ്ങൾ. ഈ ചോയ്സ് ശതമാനം പോലെ പിക്സൽ Red, Green, Blue ആൻഡ് Alpha മൂല്യങ്ങൾ പ്രദർശിപ്പിക്കുന്നു. ഇത് പിക്സലിന്റെ നിറം ഹെക്സാഡെസിമൽ മൂല്യം കാണിക്കുന്നു.

HSV

HSV ഘടകങ്ങൾ. ഈ ചോയ്സ് Hue, പദവികൾ, പിക്സൽ, ശതമാനവും ആയി പ്രദർശിപ്പിക്കുന്നു അതുപോലെ Saturation, Value ആൻഡ് Alpha.

CMYK

CMYK ചാനൽ മൂല്യങ്ങൾ. ഈ ചോയ്സ് ശതമാനം പോലെ പിക്സൽ Cyan, Magenta, Yellow, Black ആൻഡ് Alpha മൂല്യങ്ങൾ പ്രദർശിപ്പിക്കുന്നു.

സാമ്പിൾ ലയിപ്പിച്ചു

ഈ ഓപ്ഷൻ (സ്ഥിരം) ചെക്കുചെയ്തിട്ടുണ്ടെങ്കിൽ, ഒരുപറ്റം എല്ലാ ലേയറുകളിൽ നടപ്പാകും. അതു പരിശോധിക്കാതെ ആണെങ്കിൽ, ഒരുപറ്റം സജീവ പാളി ന് നടപ്പാകും.
