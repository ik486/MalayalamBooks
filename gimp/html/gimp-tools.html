1. പണിസഞ്ചി

1.1. അവതാരിക

GIMP വേഗത്തിൽ പോലുള്ള തിരഞ്ഞെടുക്കൽ നടത്തുന്നതിലൂടെ അല്ലെങ്കിൽ പാതയിൽ ഡ്രോയിംഗ് അടിസ്ഥാന വേറേ വേണ്ടി ഒരു സമഗ്രമായ പണിസഞ്ചി നൽകുന്നു. GIMP ന്റെ ഉപകരണബോക്സിലേക്ക് അടങ്ങിയിരിക്കുന്ന അനേകം ഉപകരണങ്ങൾ ഇവിടെ വിശദമായി ചർച്ച ചെയ്യുന്നുണ്ട്.

(, നിങ്ങൾ ജിജ്ഞാസുവാണെങ്കിൽ കേസിൽ GIMP ലെ പോവാം ഒരു "ടൂൾ" ഡിസ്പ്ലേ ആക്സസ് ആവശ്യമാണ് എന്ന് ഒരു ഇമേജിൽ അഭിനയം ഒരു മാർഗമാണ്, ഒന്നുകിൽ നിങ്ങൾ ഡിസ്പ്ലേ അകത്ത് ചുറ്റും പോയിന്റർ നീക്കി എന്തു ചെയ്യണമെന്ന് സൂചിപ്പിക്കുന്നു അനുവദിക്കുന്നതിന്, അല്ലെങ്കിൽ ഇന്ററാക്ടീവ് നിങ്ങൾ നടത്തിയ മാറ്റങ്ങൾ ഫലങ്ങൾ നിങ്ങളെ കാണിക്കാൻ. എന്നാൽ ഒരു കണ്ടതുപോലെ ഒരു ഉപകരണം ചിന്തിക്കുന്നത് ചെയ്യണമെങ്കിൽ, മരം ഒരു കഷണം പോലെ ഒരു ഇമേജ്, അതു ഒരുപക്ഷേ നിങ്ങൾക്ക് ഉപദ്രവം ഒരു വലിയ ചെയ്യില്ല.)

കുറിപ്പു്

ടൂൾബോക്സ് അതിന്റെ ഘടകങ്ങളും ഒരു അവലോകനം വേണ്ടി Main Windows: The Toolbox കാണുക.

GIMP നിങ്ങൾ ചുമതലകൾ ഒരു വലിയ മുറികൾ നടപ്പിലാക്കുന്നതിനുള്ള അനുവദിക്കുന്ന ഉപകരണങ്ങൾ വൈവിധ്യമാർന്ന പാൻജനസിസ് ഉണ്ട്. ഉപകരണങ്ങൾ അഞ്ച് വിഭാഗമായി വീഴുന്ന കരുതിയിരുന്നത് കഴിയും:

  • വ്യക്തമാക്കുകയും അല്ലെങ്കിൽ പരിഷ്ക്കരിക്കുക തുടർന്നുള്ള പ്രവർത്തനങ്ങൾ ബാധിക്കും ആ ചിത്രം പങ്കു Selection tools;

  • Paint tools, ചിത്രത്തിന്റെ ചില ഭാഗത്ത് നിറങ്ങൾ മാറ്റംവരുത്തിയോ ഏത്;

  • ചിത്രം ജ്യാമിതിയെ മാറ്റംവരുത്തിയോ ഏത് Transform tools;

  • ചിത്രം മുഴുവനായും ഉടനീളം നിറങ്ങൾ വിതരണം മാറ്റംവരുത്തിയോ ഏത് Color tools;

  • Other tools മറ്റ് നാല് വിഭാഗങ്ങളായി വീഴും ചെയ്യാൻ ഏത്.

1.2. ടൂൾ ഐക്കണുകൾ

ചിത്രം 14.1. പണിസഞ്ചിയിൽ ടൂൾ ഐക്കണുകൾ


ഏറ്റവും ഉപകരണങ്ങൾ പണിസഞ്ചി ഒരു ഐക്കണിൽ ക്ലിക്കുചെയ്ത് സജീവമാക്കാം. സ്വതവേ, ചില ഉപകരണങ്ങൾ മാത്രമേ മെനുകൾ (അതായത് കളർ ഉപകരണങ്ങൾ ഒന്നുകിൽ Colors മാത്രം അല്ലെങ്കിൽ Tools → Colors ആയി ആക്സസ്സുചെയ്യാനാവുന്ന) വഴി കഴിയുന്നു. ഓരോ ഉപകരണം, വാസ്തവത്തിൽ, Tools മെനുവിൽ നിന്നും സജീവമാക്കാം; കൂടാതെ, ഓരോ ഉപകരണം ആക്സിലറേറ്റർ കീ ഉപയോഗിച്ച് കീബോർഡിൽ നിന്നും സജീവമാക്കാം.

സഹജമായി, GIMP ആദ്യം ഇൻസ്റ്റാൾ സൃഷ്ടിച്ചു ൽ, എല്ലാ ഉപകരണങ്ങൾ പണിസഞ്ചി ഐക്കണുകൾ കാണിക്കാൻ: കളർ ഉപകരണങ്ങൾ ഒഴിവാക്കിയിരിക്കുന്നു. നിങ്ങൾ Edit → Preferences → Toolbox വഴി പണിസഞ്ചി കാണിച്ചിരിക്കുന്ന ഉപകരണങ്ങളുടെ കസ്റ്റമൈസ് ചെയ്യാം. ഇത് ചെയ്യാൻ താൽപ്പര്യപ്പെട്ടേക്കാവുന്ന രണ്ടു കാരണങ്ങൾ ഉണ്ട്: ആദ്യം, നിങ്ങൾ വിരളമായേ ഒരു ഉപകരണം ഉപയോഗിച്ച് എങ്കിൽ ലളിതമാക്കി നിങ്ങൾ ആഗ്രഹിക്കുന്ന ഉപകരണങ്ങൾ കണ്ടെത്താൻ തെറ്റിദ്ധരിപ്പിക്കുന്നതും ഐക്കൺ നീക്കം ആകേണ്ടതിന്നു; രണ്ടാമതായി, കളർ ഉപകരണങ്ങൾ ധാരാളം ഉപയോഗിക്കുകയാണെങ്കിൽ നിങ്ങൾക്ക് ഉള്ളിടത്തോളം എളുപ്പം ലഭ്യമായ അവർക്ക് ഐക്കണുകൾ ഞങ്ങൾക്കുണ്ട് കണ്ടെത്തിയേക്കാം. ഏതെങ്കിലും സാഹചര്യത്തിൽ, പരിഗണിക്കാതെ പണിസഞ്ചി, നിങ്ങൾക്ക് എപ്പോഴും ഉപാധികളൊന്നും ഏത് സമയത്തും ഒരു ചിത്രം മെനുബാർ നിന്ന് Tools മെനു ഉപയോഗിച്ച് ആക്സസ് ചെയ്യാൻ കഴിയും.

കഴ്സർ ആകൃതി (എങ്കിൽ Preferences നിങ്ങളെ Image Windows → Mouse Pointers → Pointer mode → Tool icon സജ്ജമാക്കി) സജീവമാണ് ഉപകരണം സൂചിപ്പിക്കുന്ന ഒന്നു ലേക്ക് അതു ഒരു ഇമേജ് അകത്ത് മാറ്റങ്ങൾ.

1.3.1. കളർ ഏരിയ

ചിത്രം 14.3. പണിസഞ്ചിയിൽ സജീവ നിറങ്ങൾ


കളർ ഏരിയ

ഈ പ്രദേശത്ത് രണ്ടു നിറങ്ങൾ, മുന്നിലും പശ്ചാത്തലത്തിലും, പെയിന്റിംഗ്, നിറയുന്നത് മറ്റു പല ഓപ്പറേഷൻസ് ഉപയോഗിക്കുന്ന അടങ്ങുന്ന ജിമ്പ് അടിസ്ഥാന പാലറ്റ് കാണിക്കുന്നു. നിറം പ്രദർശനങ്ങളുടെ ഒന്നുകിൽ ക്ലിക്കുചെയ്യുന്നത് ഇത് മാറ്റാൻ അനുവദിക്കുന്നു ഒരു കളർ എഡിറ്റർ ഡയലോഗ്, ചെയ്യുന്നു.

സ്വതേ നിറങ്ങൾ

ഈ ചെറിയ ചിഹ്നത്തിൽ ക്ലിക്കുചെയ്തുകൊണ്ട് യഥാക്രമം കറുപ്പും വെളുപ്പും ഫോർഗ്രൌണ്ട്, പശ്ചാത്തല വർണ്ണങ്ങൾ പുനഃസജ്ജമാക്കുന്നു. D കീ അമർത്തിയാൽ അതേ പ്രതീതി ഉണ്ട്.

സ്വാപ്പ് FG / ബി.ജി. നിറങ്ങൾ

രണ്ടു arrowheads ചെറിയ വളഞ്ഞ ലൈനിൽ ക്ളിക്കു മുൻവശം കാരണമാകുന്നു, പശ്ചാത്തല വർണ്ണങ്ങൾ മാറ്റി ലേക്ക്. X കീ അമർത്തിയാൽ അതേ പ്രതീതി ഉണ്ട്.

പണിസഞ്ചി ഈ ഭാഗം നിലവിൽ തിരഞ്ഞെടുത്ത ബ്രഷ്, പാറ്റേൺ ആൻഡ് ഗ്രേഡിയന്റ് കാണിക്കുന്നു. അവരിൽ ഏതെങ്കിലും ക്ലിക്കുചെയ്താൽ ഇത് മാറ്റാൻ അനുവദിക്കുന്ന ഒരു ഡയലോഗ് ചെയ്യുന്നു.

1.4. ടൂൾ ഓപ്ഷനുകൾ

ചിത്രം 14.6. ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗ്

എയർബ്രഷ് ഉപകരണത്തിൻറെ ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗ്.


നിങ്ങൾ സജ്ജീകരിക്കാൻ അധികപേരും ചെയ്യാൻ പോലുള്ള കാര്യങ്ങൾ ഉണ്ടെങ്കിൽ, ഒരു ഉപകരണം സജീവമാക്കുന്നു അതിന്റെ ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗ് പണിസഞ്ചി താഴെ ദൃശ്യമാകാൻ കാരണമാകുന്നു. കാര്യങ്ങൾ സജ്ജീകരിക്കാൻ ഈ വഴി ഇല്ലെങ്കിൽ, നിങ്ങൾ ഒരുപക്ഷേ ഇതായിരിക്കണം: അതു വളരെ പ്രയാസമുള്ള അവരുടെ ഓപ്ഷനുകൾ കൃത്രിമം കഴിയാത്തതിൽ ഇല്ലാതെ ഫലപ്രദമായി ഉപകരണങ്ങൾ ഉപയോഗിക്കുക എന്നതാണ്.

സൂചന

ടൂൾ ഓപ്ഷനുകൾ സഹജമായി ലെ പണിസഞ്ചി കീഴെ ദൃശ്യമാകും. നിങ്ങൾ ഏതെങ്കിലും തരത്തിൽ അതു നഷ്ടപ്പെട്ടാൽ, പണിസഞ്ചി താഴെ ഡോക്ക് പിന്നീട് Windows → Dockable Dialogs → Tool Options ഉപയോഗിച്ച് ഒരു പുതിയ ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗ് സൃഷ്ടിച്ചും വഴി തിരികെ ലഭിക്കും. നിങ്ങൾക്ക് സഹായം ആവശ്യമുണ്ടെങ്കിൽ Dialogs and Docking വിഭാഗം കാണുക.

ഓരോ ഉപകരണം ഓപ്ഷനുകൾ സ്വന്തം കൂട്ടം പ്രത്യേക ഉണ്ട്. നിങ്ങൾ അവരെ മാറ്റാൻ വരെ നിങ്ങൾ അവർക്ക് വേണ്ടി തിരഞ്ഞെടുപ്പുകൾ, സെഷനിൽ മുഴുവൻ സൂക്ഷിക്കണം. സത്യത്തിൽ, പ്രയോഗം ഓപ്ഷനുകൾ സെഷനിൽ നിന്ന് സെഷനിൽ പരിപാലിക്കുന്നത്. സെഷനുകളിലുടനീളം ഉപകരണം ഓപ്ഷനുകളിൽ നിര്ബന്ധം ചിലപ്പോൾ വിളിയ്ക്കാം പൊതുശല്യം കഴിയും: ഒരു ഉപകരണം വളരെ അപരിചിതമായ പെരുമാറുന്ന, നിങ്ങൾ രണ്ടാഴ്ച മുമ്പ്, അസ്വാഭാവികമായ ചില ഓപ്ഷൻ നിങ്ങൾ അവസാനം അത് പ്രവർത്തിച്ചിട്ടുണ്ട് ഉപയോഗിച്ച ഓർക്കുക വരെ നിങ്ങൾ എന്തുകൊണ്ട് മനസിലാക്കാൻ കഴിയുന്നില്ല.

ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗ് ചുവടെ, നാല് ബട്ടണുകൾ ദൃശ്യമാകും:

ഉപാധികൾ സംരക്ഷിക്കുക

ഈ ബട്ടൺ ആ അതുവഴി അവയെ പുനഃസ്ഥാപിക്കാൻ കഴിയും, നിലവിലെ ടൂളിനായി ക്രമീകരണങ്ങൾ സംരക്ഷിക്കാൻ നിങ്ങളെ അനുവദിക്കുന്നു. അതു നിങ്ങൾ പുതിയ പ്രീസെറ്റ് ഒരു പേര് നൽകാൻ അനുവദിക്കുന്നത് ??? ചെയ്യുന്നു. നിങ്ങൾ ഓപ്ഷനുകൾ പുനഃസ്ഥാപിക്കുക, സജീവമായ ഉപകരണം മാത്രം സംരക്ഷിച്ച പ്രീസെറ്റുകൾ കാണിച്ചിരിക്കുന്നു, അതിനാൽ നിങ്ങൾ ഇവിടെ ഒരു പേര് നൽകുന്നതു് ഉപകരണം പേര് ഉൾപ്പെടെ വിഷമിക്കേണ്ട ആവശ്യമില്ല.

ഓപ്ഷനുകൾ പുനഃസ്ഥാപിക്കുക

ഈ ബട്ടൺ സജീവമായ ടൂളിനായി ഓപ്ഷനുകൾ ഒരു മുമ്പ് സംരക്ഷിച്ച പ്രീസെറ്റ് പുനഃസ്ഥാപിക്കാൻ അനുവദിക്കുന്നു. യാതൊരു പ്രീസെറ്റുകൾ എന്നേക്കും സജീവ ടൂളിനായി സംരക്ഷിച്ചു, ബട്ടൺ സംവേദനക്ഷമരഹിത ആയിരിക്കും. അല്ലാത്തപക്ഷം, എല്ലാ സംരക്ഷിച്ച ഓപ്ഷൻ സെറ്റുകളിൽ പേരുകൾ കാണിക്കുന്ന ഒരു മെനു വരുത്തും ക്ളിക്ക്: ഒരു മെനു എൻട്രി ശേഷമേ ആ ക്രമീകരണങ്ങൾ ബാധകമാക്കും.

ഓപ്ഷനുകൾ ഇല്ലാതാക്കുക

ഈ ബട്ടൺ സജീവമായ ടൂളിനായി ഓപ്ഷനുകൾ ഒരു മുമ്പ് സംരക്ഷിച്ച സെറ്റ് ഇല്ലാതാക്കുന്നത് അനുവദിക്കുന്നു. യാതൊരു ഓപ്ഷൻ-സെറ്റുകൾക്ക് എന്നേക്കും സജീവ ടൂളിനായി സംരക്ഷിച്ചു, ബട്ടൺ ലളിതമായി ഉപകരണം പേര് ആവർത്തിക്കുക ചെയ്യും. അല്ലെങ്കിൽ, ക്ളിക്ക് അത് എല്ലാ സംരക്ഷിച്ച പ്രീസെറ്റുകളിൽ പേരുകൾ കാണിക്കുന്ന ഒരു മെനു വരുത്തും: തിരഞ്ഞെടുത്ത പ്രീസെറ്റ് ഇല്ലാതാക്കും.

പഴയഐച്ഛീകങ്ങളിലേക്കു്മാറ്റുക

ഈ ബട്ടൺ സ്ഥിര മൂല്യങ്ങളിൽ സജീവ പ്രയോഗം ഓപ്ഷനുകൾ പുനഃസജ്ജമാക്കുന്നു.

New sliders

Option sliders have changed with GIMP-2.8: it is not visible, but the slider area is now divided into upper and lower parts.

ചിത്രം 14.7. The new sliders of tool options dialogs

The upwards arrow pointer in the top half of the slider area

The horizontal two-way arrow pointer in the lower half of the slider area


  • In the top half of the slider area:  Clicking with the up arrow pointer sets slider to a value that depends on the position of the pointer (no reference, imprecise). Clicking and dragging the up arrow pointer sets the value by large amounts.

  • In the lower half of the slider area:  Clicking with the two-way arrow pointer has no effect. Clicking and dragging the two-way arrow pointer sets the value by small amounts.

Once you have set the value approximately, you can tune it precisely using the two small arrow buttons at the right of the slider.

The value area in the slider area works as a text editor: there, you can edit the value or enter a new value directly.

For some options, you can drag the pointer outside the tool dialog. For example with the size slider, whose maximum value is 10,000, you can drag the mouse pointer up to the right side of your screen.
