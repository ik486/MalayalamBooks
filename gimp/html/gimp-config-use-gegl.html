8.5. Use GEGL

GEGL (സാമാന്യ ഗ്രാഫിക്സ് ലൈബ്രറി) എല്ലാ ജിമ്പ്-3.0 ഉപയോഗിക്കപ്പെടും ഒരു ഗ്രാഫ് അധിഷ്ഠിത ഇമേജ് പ്രോസസ്സിംഗ് ചട്ടക്കൂടാണ്. GEGL വച്ച് ആന്തരിക പ്രോസസ്സിംഗ് ഫ്ലോട്ടിങ് പോയിന്റ് ലീനിയർ വെളിച്ചം RGBA 32bit ചെയ്യപ്പെടുന്നത്. സ്വതവേ ലെഗസി 8bit കോഡ് പാതകൾ ഇപ്പോഴും ഉപയോഗിക്കുന്നു, എന്നാൽ കൗതുകകരമായ ഉപയോക്താവിന് ഈ ഉപാധി ഉപയോഗിച്ച് നിറം പ്രവർത്തനങ്ങൾക്ക് GEGL ഉപയോഗം ഓൺ ചെയ്യാനാകും.

GEGL നിറം പ്രവർത്തനങ്ങൾക്കായുള്ള നു്അനുയോജ്യമാക്കിക്കൊണ്ടിരിയ്ക്കുന്നു പുറമേ, ഒരു പരീക്ഷണാത്മക GEGL Operation tool ഉപകരണങ്ങൾ മെനുവിലെ കണ്ടു ചേർത്തിരിക്കുന്നു. ഇത് ഒരു ചിത്രത്തിലേക്ക് GEGL ഓപ്പറേഷൻസ് അപേക്ഷിക്കുന്ന പ്രാപ്തമാക്കുന്നു അതു ഫലങ്ങൾ ഓൺ-കാൻവാസ് പ്രിവ്യൂകൾ നൽകുന്നു.

മുന്നറിയിപ്പു്

GEGL മുഴുവൻ അപ്ലിക്കേഷൻ മൂടുന്നു വരെ GIMP 8-ബിറ്റ്സ് നിലനിൽക്കുന്നു ദയവായി ശ്രദ്ധിക്കുക.

8.5.1. Activating the option

You can access this option from the image menubar through Colors → Use GEGL. Clicking on this item toggles the use of GEGL.
