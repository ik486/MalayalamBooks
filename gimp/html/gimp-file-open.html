2.4. തുറക്കുക

Open കമാൻഡ് നിങ്ങളുടെ ഹാർഡ് ഡ്രൈവ് അല്ലെങ്കിൽ ഒരു ബാഹ്യ ഇടത്തരം നിന്ന് നിലവിലുള്ള ഇമേജ് ലോഡ് അനുവദിക്കുന്ന ഡയലോഗ് സജീവമാക്കുന്നു. ഫയലുകൾ തുറക്കുമ്പോൾ ബദൽ പലപ്പോഴും കൂടുതൽ സൗകര്യപ്രദമായ, വഴികൾ, കമാൻഡുകൾ (വിഭാഗം 2.5, “Open as Layers” മുതലായവ) കാണുക.

2.4.1. Activate Dialog

  • You can access the Open dialog from an image window through: File → Open.

  • നിങ്ങൾക്ക് കീബോർഡ് കുറുക്കു Ctrl+O ഉപയോഗിച്ച് ഡയലോഗ് തുറക്കാൻ കഴിയും.

2.4.2. ഫയൽ ബ്രൗസിംഗ്

ചിത്രം 16.10. Open Dialog


ഈ ബ്രൗസർ മറ്റ് ബ്രൗസറുകൾ പോലെ കാണപ്പെടുന്നു കുരുതി സ്വയം വിശദീകരിക്കുന്ന ആണ്. ഇത് എങ്കിലും ചില particuliar സവിശേഷതകൾ ഉണ്ട്.

  1. തമ്മിൽ ചേർക്കാനും Location ടെക്സ്റ്റ് ബോക്സ് നീക്കം ടോഗിൾചെയ്യുന്നു Type a file name ബട്ടൺ.

    Ctrl+L കീ കോമ്പിനേഷൻ ഈ ബട്ടൺ അതേ നടപടി ഉണ്ട്.

  2. Location ടെക്സ്റ്റ് ബോക്സിൽ നിങ്ങൾ ഒരു ഇമേജ് ഫയൽ പാത്ത് ടൈപ്പ് ചെയ്യാം. നിങ്ങൾക്ക് ഏതെങ്കിലും പാത ടൈപ്പ് ഇല്ലെങ്കിൽ, തിരഞ്ഞെടുത്ത ഫയൽ നാമം ദൃശ്യമാകും. നിങ്ങൾക്ക് പേരിന്റെ ആദ്യത്തെ അക്ഷരങ്ങൾ ടൈപ്പ് ചെയ്യാം: തീരുമാനം ഓട്ടോ-പൂർത്തിയാക്കി ഈ കത്തുകൾ തുടങ്ങി ഫയൽ പേരുകൾ ഒരു ലിസ്റ്റ് പ്രദർശിപ്പിക്കും ആയിരിക്കും.

    നിങ്ങൾ Search സവിശേഷത ഉപയോഗിച്ച് ഒരു ഫയൽ അല്ലെങ്കിൽ ഡയറക്ടറി തിരയുമ്പോൾ (താഴെ കാണാം ഇനം 4), Search നിങ്ങൾ ലേബൽ മാറ്റങ്ങൾ ഈ ടെക്സ്റ്റ് ബോക്സിൽ പേര് നൽകാം.

  3. നിലവിലെ ഫോൾഡറിൽ പാത്ത് പ്രദർശിപ്പിച്ചിരിക്കുന്നു. നിങ്ങൾ ഒരു ഘടകത്തിലെ ക്ലിക്കുചെയ്ത് ഈ പാതയിലൂടെ നാവിഗേറ്റ് ചെയ്യാം.

  4. Search ഉപയോഗിച്ച്, നിങ്ങൾ ആ ഫയൽ കൃത്യമായ പേര് അറിയില്ല പോലും, ഒരു ഫയൽ (അല്ലെങ്കിൽ ഡയറക്ടറി) നോക്കുക. ഒരു ഫയൽ പേര് അല്ലെങ്കിൽ മുകളിലുള്ള പാഠ ബോക്സിൽ, അമർത്തുക Enter ഒരു ഫയൽ പേര് ഒരു ഭാഗം ടൈപ്പ്, Search ക്ലിക്ക്. അപ്പോൾ കേന്ദ്ര ഫ്രെയിം (7) നിങ്ങൾ ടൈപ്പുചെയ്ത പാഠം ഉൾക്കൊള്ളുന്ന പേരുകൾ ഉപയോഗിച്ച് നിങ്ങളുടെ ഹോം ഡയറക്ടറിയിൽ എല്ലാ ഫയലുകളും ഡയറക്ടറികളും ലിസ്റ്റ് ചെയ്യും. നിർഭാഗ്യവശാൽ ഒരു നിശ്ചിത തരം (10) ഫയലുകളിൽ ഫലങ്ങൾ നിയന്ത്രിക്കാൻ കഴിയില്ല.

    Recently used സ്വയം വിശദീകരിക്കുന്ന ആണ്.

  5. ഇവിടെ നിങ്ങൾ നിങ്ങളുടെ പ്രധാന ഫോൾഡറുകൾ നിങ്ങളുടെ സ്റ്റോർ ഉപകരണങ്ങളിലേക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും.

  6. ഇവിടെ നിങ്ങൾ ബുക്ക്മാർക്കുകൾ ഫോൾഡറുകൾ, Add അല്ലെങ്കിൽ നിങ്ങൾ കേന്ദ്ര പാനൽ ക്ലിക്കുചെയ്തുകൊണ്ട് ഒരു ഫോൾഡർ നേടുന്നു Add to Bookmarks ഓപ്ഷൻ ഉപയോഗിച്ച് ചേർക്കാൻ കഴിയും, കൂടാതെ അവരെ നീക്കം.

  7. തിരഞ്ഞെടുത്ത ഫോൾഡറിൽ ഉളളടക്കം ഇവിടെ പ്രദർശിപ്പിച്ചിരിക്കുന്നു. ഈ പാനലിൽ ഒരു ഫോൾഡർ ഡബിൾ ഇടത് ക്ലിക്കുചെയ്തുകൊണ്ട് നിലവിലുള്ള ഫോൾഡർ മാറ്റുക. ഒറ്റ ഇടത് ക്ലിക്കിൽ ഒരു ഫയൽ തിരഞ്ഞെടുക്കുക. എന്നിട്ട് നിങ്ങൾ Open ബട്ടൺ ക്ലിക്കുചെയ്ത് തിരഞ്ഞെടുത്ത ഫയൽ തുറക്കാൻ കഴിയും. ഇരട്ട ഇടത് ക്ലിക്കിൽ നേരിട്ട് ഫയൽ തുറക്കുന്നു. നിങ്ങൾ ഇമേജ് ഫയലുകൾ മാത്രം തുറക്കാൻ എന്ന് ഓർക്കുക.

    ഒരു ഫോൾഡർ നാമം വലതു ബട്ടൺ ഒരു സന്ദർഭ മെനു തുറക്കുന്നു:

    The folder context menu

  8. തിരഞ്ഞെടുത്ത ചിത്രം Preview വിൻഡോയിൽ പ്രദർശിപ്പിച്ചിരിക്കുന്നു. അതു GIMP, പ്രമാണത്തിന്റെ വലിപ്പം സൃഷ്ടിച്ച ഒരു ചിത്രമാണെങ്കിൽ ചിത്രം ഇമേജ് രചനയാണ് പ്രിവ്യൂ വിൻഡോ ചുവടെ പ്രദർശിപ്പിച്ചിരിക്കുന്നു.

    സൂചന

    നിങ്ങളുടെ ചിത്രം മറ്റൊരു പ്രോഗ്രാം മാറ്റിയിട്ടുണ്ടെങ്കിൽ, അത് പുതുക്കുന്നതിനായി പ്രിവ്യൂ ജാലകത്തിൽ ക്ലിക്ക് ചെയ്യുക.

  9. Add ബട്ടൺ ക്ലിക്കുചെയ്യുന്നതിലൂടെ, ബുക്ക്മാർക്കുകൾ തിരഞ്ഞെടുത്ത ഫോൾഡർ ചേർക്കുക.

    Remove ക്ലിക്കുചെയ്യുന്നതിലൂടെ, പട്ടികയിൽ നിന്നും തിരഞ്ഞെടുത്ത ബുക്ക്മാർക്ക് നീക്കം.

  10. സാധാരണ All images പേരുകൾ പ്രദർശിപ്പിക്കാൻ ഇഷ്ടപ്പെടുന്നത് ചെയ്യും. നിങ്ങൾക്ക് All files തിരഞ്ഞെടുക്കാം. നിങ്ങൾക്ക് ചിത്രത്തിന്റെ ഒരു പ്രത്യേക തരം (gif, jpg, പി.എൻ.ജി. ...) സ്വയം പരിമിതപ്പെടുത്തും.

  11. Select File Type: മിക്ക സമയത്തും നിങ്ങൾ GIMP സ്വയം ഫയൽ തരം നിർണ്ണയിക്കാൻ കഴിയും കാരണം, ഈ ഏതെങ്കിലും ശ്രദ്ധ ചെയ്യരുത് ആവശ്യമില്ല. ഫയലിൽ ഏതാനും അപൂർവ സാഹചര്യങ്ങളിൽ, ഫയൽ എക്സ്റ്റൻഷൻ വേണ്ടാ ആന്തരിക വിവരങ്ങൾ ആരും ഇല്ല GIMP ഫയൽ തരം പറയാൻ മാത്രം. ഇങ്ങനെ സംഭവിക്കുകയാണെങ്കിൽ, നിങ്ങൾ ലിസ്റ്റിൽ നിന്ന് തിരഞ്ഞെടുത്ത് ഇത് സജ്ജമാക്കാം.
