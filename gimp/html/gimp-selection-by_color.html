4.6. By Color

Select By Color കമാൻഡ് “Select By Color” ഉപകരണം, അടിസ്ഥാന ഉപകരണങ്ങൾ ഒന്നാണ് പ്രവേശിക്കുന്നതിനായി ഒരു ഇതര മാർഗമാണ്. കണ്ടെത്താവുന്നതാണ് Select By Color ഈ ഉപകരണം ഉപയോഗിച്ച് കൂടുതൽ വിവരങ്ങൾ.

4.6.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Select → By Color,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Shift+O ഉപയോഗിച്ച്.
