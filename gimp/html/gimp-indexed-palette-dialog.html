2.4. Colormap സംഭാഷണം

ചിത്രം 15.14. 6 നിറങ്ങളിൽ അതിന്റെ Colormap ഡയലോഗ് ഗണമാണോയെന്നത് ചിത്രം


Colormap (സൂചികയിലാക്കിയത് പാലറ്റ് ഒരു മികച്ച പേര് ആണ്) ഡയലോഗ് ഗണമാണോയെന്നത് ചിത്രത്തിന്റെ colormap എഡിറ്റ് അനുവദിക്കുന്നു. (സജീവമായ ചിത്രത്തിന്റെ മോഡ് പകരം ഇൻഡെക്സ് ആർജിബി അല്ലെങ്കിൽ ഗ്രേസ്കെയിൽ ആണെങ്കിൽ, ഡയലോഗ് ശൂന്യമായ ഉപയോഗയോഗ്യമല്ലാതെയായിട്ടുണ്ട്.) ഇതൊരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി Dialogs and Docking ന് വിഭാഗം കാണുക.

2.4.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Colormap” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • ചിത്രം മെനുവിൽ നിന്ന്: Windows → Dockable Dialogs → Colormap;

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Colormap തിരഞ്ഞെടുത്ത്.

Windows മെനുവിൽ, കുറഞ്ഞത് ഒരു ഡയലോഗ് തുറന്നുതന്നെയിരിക്കുന്ന മാത്രമേ ഉണ്ട് detached windows ഒരു ലിസ്റ്റ് ഉണ്ട്. Windows → Colormap: ഈ സാഹചര്യത്തിൽ, ഇമേജ്-മെനുവിൽ നിന്നും “Colormap” ഡയലോഗ് ഉണ്ടാകാം.

2.4.2. Colormaps സൂജികപത്രവും ചിത്രങ്ങൾ

ഗണമാണോയെന്നത് സ്വരൂപത്തിൽ, പകരം നേരിട്ട് ഓരോ നിറവും ചെയ്തുകൊണ്ടിരിക്കുന്നു (ആർജിബി ചെയ്ത് Grayscale ചിത്രങ്ങൾ സംഭവിക്കുന്നില്ല പോലെ), നിറങ്ങൾ പിക്സൽ പരോക്ഷമായ രീതി, ഒരു colormap വിളിച്ചു ലുക്ക് അപ്പ് പട്ടികയിൽ നിർണ്ണയിക്കുന്നത്.

ആ പിക്സൽ വേണ്ടി കാണിച്ചിരിക്കണം ആ നിറം നിർണ്ണയിക്കാൻ, ജിമ്പ് ചിത്രം ന്റെ colormap ലെ സൂചിക തോന്നുന്നു. ഓരോ ഇൻഡെക്സ്ഡ് ചിത്രം സ്വന്തം സ്വകാര്യ colormap ഉണ്ട്. ജിമ് ഒരു colormap എൻട്രികളുടെ പരമാവധി എണ്ണം 256. പരമാവധി വലുപ്പത്തിലുള്ള colormap ഉണ്ടു 0 മുതൽ 255 ഓരോ സൂചിക എത്രവേണമെങ്കിലും ആർജിബി നിറങ്ങളുടെ നിയോഗിക്കുന്നു. അവർ ദൃശ്യമാകും സൂചിക അല്ലെങ്കിൽ ഓർഡർ നിയോഗിച്ചിട്ടുള്ള കഴിയുന്ന നിറങ്ങൾ നിയന്ത്രിയ്ക്കാനുള്ള വ്യവസ്ഥകളൊന്നും ഉണ്ട്: ഏതെങ്കിലും ഇന്ഡക്സ് ഏതെങ്കിലും നിറവും കഴിയും.

ഇത് colormap വർണ്ണങ്ങൾ ഗണമാണോയെന്നത് ചിത്രത്തിനായുള്ള only colors available (നിങ്ങൾ colormap പുതിയ നിറങ്ങൾ ചേർക്കാതിരുന്നാൽ എന്നോ,) ആകുന്നു ഗ്രഹിക്കാൻ പ്രധാനമാണ്. ഈ പല ജിമ്പ് ഓപ്പറേഷൻസ് ഒരു പ്രധാന പ്രഭാവം ഉണ്ട്, ഉദാഹരണത്തിന്: ഒരു പാറ്റേൺ ഫിൽ ലെ ജിമ്പ് സാധാരണയായി colormap കൃത്യമായി ശരിയായ നിറങ്ങൾ കണ്ടെത്താൻ കഴിയില്ല, അങ്ങനെ അത് ലഭ്യമായ അടുത്തുള്ള നിറം ഉപയോഗിച്ച് അവരെ ഏകദേശം ചെയ്യും. ഇത് ചിലപ്പോൾ Quantization അറിയപ്പെടുന്നത്. colormap വളരെ പരിമിതമായ അല്ലെങ്കിൽ മോശമായി തിരഞ്ഞെടുക്കപ്പെട്ടവളാണെങ്കിൽ ഈ എളുപ്പത്തിൽ വളരെ പാവപ്പെട്ട ചിത്രത്തിന്റെ നിലവാരം ഉത്പാദിപ്പിക്കാൻ കഴിയും.

Colormap ഡയലോഗ് ഒന്നുകിൽ പുതിയ എൻട്രികൾ സൃഷ്ടിച്ചുകൊണ്ട്, അല്ലെങ്കിൽ നിലവിലുള്ള എൻട്രികൾ നിറങ്ങൾ മാറ്റിക്കൊണ്ട്, ഒരു ചിത്രത്തിനായി colormap മാറ്റം അനുവദിക്കുന്നു. തന്നിരിക്കുന്ന ഒരു സൂചിക ബന്ധപ്പെട്ട നിറം മാറ്റുകയും, ആ സൂചിക നിയോഗിച്ചിട്ടുള്ള എല്ലാ പിക്സൽ ഒരു വർണ്ണ ഷിഫ്റ്റ് ആയി, ഇമേജ് മുഴുവൻ പ്രതിഫലിക്കും മാറ്റങ്ങൾ കാണും. എൻട്രികൾ അതിന്റെ മുതലായവ ലേക്ക് മുകളിലെ ഇടത് മൂലയിൽ 0, 1 എണ്ണുന്നു

2.4.3. Colormap ഡയലോഗ് ഉപയോഗിച്ച്

ഇവിടെ നിങ്ങൾക്ക് ഈ ഡയലോഗ് ഉപയോഗിച്ച് നടത്താം തന്നേ:

ഒരു കളർ എൻട്രി ക്ലിക്ക്

ഈ പണിസഞ്ചി കളർ ഏരിയ കാണുന്നത് പോലെ നിങ്ങൾ ക്ലിക്ക് നിറം ലേക്ക് ജിമ്പ് ന്റെ പുരോതല നിറം സജ്ജീകരിക്കുന്നു. തത്ഫലമായി ഈ നിറം നിങ്ങൾ ചെയ്യുന്ന അടുത്ത പെയിന്റിങ് ഓപ്പറേഷൻ ഉപയോഗിക്കും.

ഒരു കളർ എൻട്രി ന് Ctrl -click

ഈ പണിസഞ്ചി കളർ ഏരിയ കാണുന്നത് പോലെ നിങ്ങൾ ന് -click Ctrl നിറം ലേക്ക് GIMP ന്റെ പശ്ചാത്തല നിറം സജ്ജീകരിക്കുന്നു.

ഒരു കളർ എൻട്രി ഇരട്ട-ക്ലിക്കുചെയ്യുക

ഇത് നിങ്ങൾ ക്ലിക്ക് നിറം ലേക്ക് ജിമ്പ് ന്റെ പുരോതല നിറം സജ്ജീകരിക്കുന്നു, ഒപ്പം ഒരു പുതിയ നിറം ആ colormap എൻട്രി മാറ്റാൻ അനുവദിക്കുന്ന ഒരു കളർ എഡിറ്റർ ചെയ്യുന്നു.

വർണ്ണ സൂചിക

നിങ്ങൾക്ക് ഇവിടെ അതിന്റെ സൂചിക ടൈപ്പുചെയ്യുമ്പോൾ, അല്ലെങ്കിൽ വലതുവശത്ത് spinbutton ക്ലിക്കുചെയ്ത് നിങ്ങൾക്ക് മറ്റൊരു colormap എൻട്രി തിരഞ്ഞെടുക്കാം.

എച്ച്ടിഎംഎൽ-നൊട്ടേഷൻ

ഈ പ്രദേശത്ത് നിലവിൽ തിരഞ്ഞെടുത്ത colormap എൻട്രി നിയോഗിച്ചിട്ടുള്ള നിറം ഒരു ഹെക്സാ-കോഡ് പ്രാതിനിധ്യം (പോലുള്ള HTML ൽ ഉപയോഗിക്കുന്നു) കാണിക്കുന്നു. നിങ്ങൾക്ക് കളർ ഇവിടെ, പകരം ഒരു കളർ എഡിറ്റർ ഉപയോഗിച്ച്, നിങ്ങൾക്ക് ചെയ്യണമെങ്കിൽ എഡിറ്റ് കഴിയും. HTML notation കാണുക

നിറം എഡിറ്റ്

ഈ ബട്ടൺ (ഡയലോഗ് താഴത്തെ ഇടത് മൂലയിൽ) നിങ്ങൾ നിലവിൽ തിരഞ്ഞെടുത്ത colormap എൻട്രി നിറം മാറ്റുകയും അനുവദിക്കുന്ന ഒരു കളർ എഡിറ്റർ ചെയ്യുന്നു. ഇഫക്ട് അത് ജിമ്പ് ന്റെ മുൻഭാഗത്തെ വർണ്ണം സജ്ജമാക്കുക ഇല്ല അല്ലാതെ എൻട്രി ഡബിൾ ക്ലിക്ക് സമാനമാണ്.

വർണം ചേർക്കുക

ഈ ബട്ടൺ (ഡയലോഗ് താഴത്തെ വലത് കോണിലുള്ള) colormap പുതിയ നിറങ്ങൾ ചേർക്കാൻ അനുവദിക്കുന്നു. നിങ്ങൾ ബട്ടൺ, നിലവിലെ മുൻഭാഗത്തെ വർണ്ണം ക്ലിക്ക് ചെയ്താൽ, പണിസഞ്ചി കാണുന്നത് പോലെ colormap അവസാനത്തിലേക്ക് ചെയ്തപ്പോഴാണ് ചെയ്യും. നിങ്ങൾ Ctrl അമർത്തി ക്ലിക്കുചെയ്യുക പകരം പക്ഷം പണിസഞ്ചി നിന്ന് പശ്ചാത്തല നിറം ചേർക്കും. (Colormap 256 എൻട്രികൾ അടങ്ങിയിരിക്കുന്ന, അത് നിറഞ്ഞിരിക്കുന്നു, കൂടുതൽ ചേർക്കുന്നതിന് ശ്രമിക്കുന്ന ഫലമില്ല ഉണ്ടായിരിക്കും.)

സൂചന

നിങ്ങൾ ഒരു തെറ്റ് ഉണ്ടാക്കുന്നു എങ്കിൽ ആരുടെ colormap നിങ്ങൾ ചിത്രത്തിൽ പോയിന്റർ ഫോക്കസുചെയ്ത് അത് പഴയപടിയാക്കാനുമാകും മാറ്റി, തുടർന്ന് Ctrl+Z പ്രസും അല്ലെങ്കിൽ ഇമേജ് മെനുവിൽ Edit → Undo ശേഷമേ.
കുറിപ്പു്

ഈ ഡയലോഗ് ഗണമാണോയെന്നത് ചിത്രത്തിനായുള്ള colormap പന്തിന്റെ ഏറ്റവും കൂടുതലായി ഉപയോഗിക്കുന്ന രീതികൾ നൽകുന്നു. പോലുള്ള തീവ്രത /, ഹ്യൂയേ / സാച്ചുറേഷൻ, തുടങ്ങിയവ നിറം ഉപകരണങ്ങൾ, ഇൻഡെക്സ്ഡ് ചിത്രങ്ങളും ഓപ്പറേറ്റ് ചെയ്യരുത്. അങ്ങനെ ചെയ്യുന്നത് കുറച്ച് plug- ഇന്നുകൾ, “Normalize”, “VColor Enhance”, ഒപ്പം “Stretch Contrast” ഓപ്പറേഷൻസ് ഉൾപ്പെടെ ഉണ്ട്, അതു പോലെ മറ്റുള്ളവരെ സൃഷ്ടിക്കാൻ സാധ്യമാണ്.
കുറിപ്പു്

നിങ്ങൾ Colormap കാര്യമാണ് ഒരു നിറത്തിൽ ഗണമാണോയെന്നത് ചിത്രം വരയ്ക്കാൻ എങ്കിൽ ജിമ്പ് Colormap ഏറ്റവും സമാനമായ നിറം ഉപയോഗിക്കും.
