5. മെനു പ്രവർത്തനങ്ങൾ കുറുക്കുവഴികൾ സൃഷ്ടിക്കുന്നു

നിരവധി പ്രവർത്തനങ്ങൾ ചിത്രം മെനു വഴി ആക്സസ്സുചെയ്യാനാവുന്ന സ്വതവേയുള്ളകീബോര്ഡ്തുറക്കുക കുറുക്കുവഴി. നിങ്ങൾ ഒരു ഒരുപാട് ഉപയോഗിക്കുന്ന ഒരു കമാൻഡ് ഒരു പുതിയ കുറുക്കുവഴി സൃഷ്ടിക്കാൻ നിലവിലുള്ള ഒരു കുറുക്കുവഴി എഡിറ്റ് കൂടുതൽ അപൂർവ്വമായി, ഒന്നുണ്ടെങ്കിലോ ഇല്ല ആഗ്രഹിക്കുന്നുണ്ടാവും. ഈ ചെയ്യുന്നത് രണ്ട് രീതികൾ ഉണ്ട്.

പ്രക്രിയ 12.1. ഡൈനാമിക് കീബോർഡ് കുറുക്കുവഴികൾ ഉപയോഗിച്ച്

  1. ആദ്യം, Preferences മെനുവിന്റെ Interface ഇനത്തിൽ Use dynamic keyboard shortcuts ഓപ്ഷൻ പരിശോധിച്ചുകൊണ്ട് ഈ ശേഷി സജീവമാക്കുന്നതിന് ഞങ്ങൾക്കുണ്ട്. ഈ ഓപ്ഷൻ സാധാരണയായി ഒരു അനാവശ്യ കുറുക്കുവഴി സൃഷ്ടിക്കുന്നതിൽ നിന്ന് ആകസ്മികമായ കീകളുടെ തടയാൻ ചെക്കുചെയ്തിട്ടില്ലെന്ന്.

  2. നിങ്ങളുടെ കുറുക്കുവഴി സംരക്ഷിക്കപ്പെടുന്ന നിങ്ങള് ചെയ്യുന്നത് സമയത്ത്, പുറമേ Save keyboard shortcuts on exit ഓപ്ഷൻ പരിശോധിക്കുക.

  3. ഒരു കീബോർഡ് കുറുക്കുവഴി സൃഷ്ടിക്കാൻ, മെനുവിൽ ഒരു കമാൻഡ് ൽ മൗസ് പോയിന്റർ സ്ഥാപിക്കുക: അതു പിന്നെ പ്രമുഖമാക്കിക്കാട്ടും. മൗസ് സൂചികയെ അമർത്തുമ്പോൾ കീകൾ സൂക്ഷിക്കുന്നതിനുമുള്ള, മൂന്ന് താക്കോലുകൾ ഒരു കൂട്ടം നീക്കാൻ ടൈപ്പ് സൂക്ഷിക്കുക. ഈ അനുക്രമം കമാൻഡിന്റെ വലതുവശത്ത് ദൃശ്യമാകും കാണും.

  4. ഇത് നിങ്ങളുടെ ഇച്ഛാനുസൃത കുറുക്കുവഴികൾക്കായി Ctrl+Alt+Key അനുക്രമം ഉപയോഗിക്കാൻ നല്ലത്.

ചിത്രം 12.31. കീബോർഡ് കുറുക്കുവഴികൾ ക്രമീകരിക്കുക


പ്രക്രിയ 12.2. കീബോർഡ് കുറുക്കുവഴി എഡിറ്റർ ഉപയോഗിച്ച്

  1. നിങ്ങൾ Preferences മെനുവിന്റെ “Interface” ഇനത്തിൽ Configure keyboard shortcuts ക്ലിക്കുചെയ്ത് എഡിറ്റർ നേടുകയും.

  2. ഈ ഡയലോഗിൽ കാണിച്ചിരിക്കുന്നതുപോലെ, “Action” പ്രദേശത്തെ നിങ്ങൾക്ക് ഒരു കുറുക്കുവഴി സൃഷ്ടിക്കാൻ ആഗ്രഹിക്കുന്ന കമാൻഡ് തിരഞ്ഞെടുക്കാം. അപ്പോൾ നിങ്ങൾ മുകളിൽ ആയി നിങ്ങളുടെ കീ ക്രമം ടൈപ്പ് ചെയ്യുക. തത്ത്വത്തിൽ സ്പെയ്സ് ബാർ കുറുക്കുവഴി ക്ലിയർ ചെയ്യണം. (പ്രായോഗികമായി അതു കോടയിറങ്ങിയ, പക്ഷെ ഇല്ലാതാക്കുന്നില്ല.)

  3. ഈ കുറുക്കുവഴി എഡിറ്റര് കീബോർഡ് ഉപയോഗിച്ച് control the tool parameter settings അനുവദിക്കുന്നു. ഈ ഡയലോഗ് മുകളിൽ നിങ്ങൾ ഉപകരണം പാരാമീറ്ററുകൾ നയിക്കുന്ന ഒരു Context മെനു കണ്ടെത്താൻ കഴിയും. നിങ്ങളുടെ ജോലി എളുപ്പമാക്കുന്നതിന്, ഉപകരണം തരം ചെറിയ ഐക്കണുകൾ ഉപയോഗിച്ച് അടയാളപ്പെടുത്തിയിരിക്കുന്നു.

കുറിപ്പു്

Custom Keyboard shortcuts are stored in one of Gimp's hidden directory (/home/[username]/.gimp-2.8/menurc) under Linux, and C:\Documents and Settings\[Username]\.gimp-2.8\menurc under Windows XP. It is a simple text file that you can transport from one computer to another.
