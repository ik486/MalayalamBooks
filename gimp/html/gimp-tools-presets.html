13. പ്രീസെറ്റുകളിൽ

നിങ്ങൾ പലപ്പോഴും പ്രത്യേക ക്രമീകരണങ്ങൾ ഉപയോഗിച്ച് ഉപകരണങ്ങൾ ഉപയോഗിക്കുകയാണെങ്കിൽ, പ്രീസെറ്റുകൾ നിങ്ങൾക്കായുള്ളതാണ്. ഈ ക്രമീകരണങ്ങൾ ലാഭിക്കാൻ ആഗ്രഹിക്കുന്ന അവരെ തിരികെ ലഭിക്കും.

Paint tools, which are normally in Toolbox, have a preset system that have been much improved with GIMP-2.8. Color tools (except Posterize and Desaturate), which are not normally in Toolbox, have their own preset system.

എല്ലാ ഉപകരണങ്ങൾ ഓപ്ഷനുകൾ ഡയലോഗുകൾ താഴെയുള്ള നാല് ബട്ടണുകൾ, സംരക്ഷിക്കാനായി, ഇല്ലാതാക്കുക പുനഃസ്ഥാപിക്കുക റീസെറ്റ് പ്രീസെറ്റുകൾ അനുവദിക്കുക. PLACEHOLDER,-101607

പെയിന്റ് പ്രീസെറ്റുകൾ വിഭാഗം 5.1, “ടൂൾ പ്രീസെറ്റുകളിൽ സംഭാഷണം” വിവരിച്ചിരിക്കുന്നു.

കളർ ടൂൾ പ്രീസെറ്റുകൾ വിഭാഗം 5.1.1, “Color Tool Presets” വിവരിച്ചിരിക്കുന്നു.
