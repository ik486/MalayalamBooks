3.12. ബഫർ

ചിത്രം 16.19. “Edit” മെനുവിന്റെ “Buffer” ഉപമെനു


ഈ ഉപമെനുവുണ്ട് ലെ കമാൻഡുകൾ named buffers ന് ഓപ്പറേറ്റ്. നിങ്ങൾ സൃഷ്ടിച്ചു ഏതെങ്കിലും പേരുള്ള ബഫറുകളെയും കാണാനും കൈകാര്യം ചെയ്യാനും Buffers dialogue ഉപയോഗിക്കാൻ കഴിയും.

3.12.1. പുതിയഉപമെനു സജീവമാക്കുക

  • നിങ്ങൾ ഇമേജ് മെനുബാർ നിന്ന് Edit → Buffer മുഖേന ഈ ഉപമെനുവുണ്ട് ആക്സസ് ചെയ്യാൻ കഴിയും.

3.12.2. ഉപ-മെനു എൻട്രികൾ

മുറിക്കുക Named

Cut Named കമാൻഡ് സാധാരണ രീതി സജീവ പാളി നിന്ന് നിരക്കു ഉള്ളടക്കങ്ങൾ കുറച്ചു പകരം ആഗോള ക്ലിപ്ബോർഡ് ഉള്ളടക്കങ്ങൾ സംഭരിക്കുന്നതിനുള്ള, അത് നിങ്ങൾ ഒരു പോപ്പ്-അപ്പ് ഡയലോഗ് പേര് ഉപയോഗിക്കാൻ ഒരു പ്രത്യേക ബഫർ അത് സംഭരിക്കുന്നു.

പകർത്തുക Named

Copy Named കമാൻഡ് പകർപ്പുകൾ സാധാരണ രീതി സജീവ പാളി നിന്ന് നിരക്കു ഉള്ളടക്കങ്ങൾ, പക്ഷേ പകരം ആഗോള ക്ലിപ്ബോർഡ് ലെ ഉള്ളടക്കം സംഭരിക്കുന്നതിനുള്ള, അത് നിങ്ങൾ ഒരു പോപ്പ്-അപ്പ് ഡയലോഗ് പേര് ഉപയോഗിക്കാൻ ഒരു പ്രത്യേക ബഫർ അത് സംഭരിക്കുന്നു.

ദൃശ്യം നാമകരണം ചെയ്ത പകർത്തുക

Copy Visible Named കമാൻഡ് പകർപ്പുകൾ സാധാരണപോലെ വഴിയിൽ ദൃശ്യമായ എല്ലാ പാളികളിൽ നിരക്കു ഉള്ളടക്കങ്ങൾ, പകരം ആഗോള ക്ലിപ്ബോർഡ് ലെ ഉള്ളടക്കം സംഭരിക്കുന്നതിന്, അതു നിങ്ങൾ ഒരു പോപ്പ്-അപ്പ് ഡയലോഗ് പേര് ഉപയോഗിക്കാൻ ഒരു പ്രത്യേക ബഫർ അത് സംഭരിക്കുന്നു.

ഒട്ടിക്കുക Named

Paste Named കമാൻഡ് ലളിതമായി Buffers dialogue ചെയ്യുന്നു. ലിസ്റ്റുചെയ്ത ബഫറുകളെയും ഒരെണ്ണം തിരഞ്ഞെടുത്ത്, താഴെ ബട്ടണുകളിൽ ഒന്ന് അമർത്തുന്നതിലൂടെ, Paste Buffer, Paste Buffer Into, അല്ലെങ്കിൽ Paste Buffer as New ഒന്നുകില്.
