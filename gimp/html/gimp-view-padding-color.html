5.18. Padding Color

ചിത്രം 16.51. Contents of the “Padding Color” submenu


നിങ്ങൾ Padding Color കമാൻഡ് ഉപയോഗിച്ച് ഇമേജ് ചുറ്റുമുള്ള ക്യാൻവാസ് നിറം മാറ്റാൻ കഴിയും. ക്യാൻവാസ് ചിത്രം സ്ഥിതിചെയ്യുന്നത് ഉപരിതലം. അത് ചിത്രത്തിന്റെ വിൻഡോയിൽ ഇമേജ് ചുറ്റും ഒരു ഫ്രെയിം പോലെ കാണപ്പെടുന്നു. പാഡിംഗോട് നിറം ഇമേജ് ഏതെങ്കിലും പ്രഭാവം ഇല്ല ഇത്, സ്വകാര്യ മുൻഗണന വെറും ഒരു കാര്യം. ഈ വർണ്ണം നൽകുക പ്രയോഗം നിറം സമാനമല്ലാത്ത ദയവായി ശ്രദ്ധിക്കുക.

5.18.1. Activating the submenu

  • You can access this submenu from the image menubar through View → Padding Color.

5.18.2. “Padding Color” Options

From Theme

Preferences Theme നിർവചിച്ച തീം നിറം ഉപയോഗിക്കുന്നു.

Light/Dark Check Color

Preferences Display നിർവചിക്കുന്നത് ഏത് സുതാര്യത പ്രതിനിധീകരിച്ച് ചെക്ക് ഉപയോഗിക്കപ്പെടുന്നു.

Select Custom Color

നിങ്ങൾ ഉപയോഗിക്കാൻ ഒരു നിറം തിരഞ്ഞെടുക്കൂ അനുവദിക്കുന്നതിന് കളർ സെലക്ടർ വിൻഡോ തുറക്കുന്നു.

As in Preferences

Image Window Appearance തിരഞ്ഞെടുത്ത നിറം ഉപയോഗിക്കുന്നു.
