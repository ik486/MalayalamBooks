6.11. കാൻവാസ് വലിപ്പം

“canvas” ചിത്രം ദൃശ്യമായ ആണ്. സ്വതവേ ക്യാൻവാസ് വലിപ്പം പാളികൾ വലിപ്പം അവധികാലത്തിനായുള്ള. Canvas Size കമാൻഡ് വലുതാക്കുന്നു അല്ലെങ്കിൽ ക്യാൻവാസ് വലിപ്പം കുറയ്ക്കാൻ അനുവദിക്കുന്നു. നിങ്ങൾ, നിങ്ങൾ ആഗ്രഹിക്കുന്ന പക്ഷം പാളികൾ വലിപ്പം പരിഷ്കരിക്കാനും കഴിയും. നിങ്ങൾ ക്യാൻവാസ് വിശാലമാക്കുമ്പോൾ, ഇമേജിൽ ഉളളടക്കം ചുറ്റും ഇടം. നിങ്ങൾ അത് കുറയ്ക്കാൻ ചെയ്യുമ്പോൾ, ദൃശ്യമായ വലുപ്പം എന്നാൽ പാളികൾ ഇപ്പോഴും ക്യാൻവാസ് അതിർത്തിയിൽ അപ്പുറം ദീർഘിപ്പിക്കും.

നിങ്ങൾ ക്യാൻവാസ് വലിപ്പം കുറയ്ക്കാനും, പുതിയ ക്യാൻവാസ് പ്രിവ്യൂ ഒരു നേർത്ത നെഗറ്റീവ് അതിർത്തിയിൽ ചുറ്റും ലഭ്യമാകുന്നു. മൗസ് സൂചികയെ ഓടിക്കൊണ്ടിരുന്ന ക്രോസ്; ഈ ഫ്രെയിം നേരെ ഇമേജ് നീക്കാൻ ക്ലിക്ക് ചെയ്ത് ഡ്രാഗ്.

6.11.2. Description of the “Canvas Size” dialog

ചിത്രം 16.60. The “Canvas Size” dialog


കാൻവാസ് വലിപ്പം

Width; Height

നിങ്ങൾ Width ക്യാൻവാസ് എന്ന Height സജ്ജമാക്കാൻ കഴിയും. സ്ഥിര യൂണിറ്റുകൾ ഉദാ പിക്സൽ പക്ഷേ വ്യത്യസ്ത യൂണിറ്റുകൾ തിരഞ്ഞെടുക്കാം, ശതമാനം, നിലവിലെ അളവുകളിലേക്ക് ആപേക്ഷിക പുതിയ അളവുകൾ സജ്ജീകരിക്കണമെങ്കിൽ. വീതിയും ഉയരവും വലത് ചെയിൻ തകർത്തു ചെയ്തിട്ടില്ലെങ്കിൽ, വീതിയും ഉയരവും പരസ്പരം ഒരേ താരതമ്യേന വലിപ്പമുള്ള പ്രമാണിച്ചു. ആ മറ്റൊരാളും ഒരു ഇതേ തുക മാറുമ്പോൾ, നിങ്ങൾ മൂല്യങ്ങളിൽ ഒന്ന് മാറ്റാൻ എങ്കിൽ ആണ്. നിങ്ങൾ അത് ക്ലിക്കുചെയ്ത് ചെയിൻ ലംഘിക്കുകയും, പ്രത്യേകമായി വീതിയും ഉയരവും സജ്ജമാക്കാൻ കഴിയും.

എന്തുതന്നെ നിങ്ങൾ ഉപയോഗിക്കുന്ന യൂണിറ്റുകൾ, പിക്സൽ വലുപ്പം, നിലവിലെ ചിത്രം കുറിച്ചുള്ള വിവരങ്ങൾ എപ്പോഴും Width ആൻഡ് Height നിലങ്ങളും ചുവടെ പ്രദർശിപ്പിച്ചിരിക്കുന്നു. നിങ്ങൾ Canvas Size ഡയലോഗിൽ റെസലൂഷൻ മാറ്റാൻ കഴിയില്ല; അത് ചെയ്യാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, Print Size dialog ഉപയോഗിക്കുക.

ഓഫ്സെറ്റ്

Offset മൂല്യങ്ങൾ കാൻവാസിൽ ചിത്രം (ചിത്രം, സജീവ പാളി) സ്ഥാപിക്കുക ഉപയോഗിക്കുന്നു. വലുപ്പമുള്ള ഡയലോഗ് ജാലകത്തിന്റെ പ്രിവ്യൂ ക്യാൻവാസ് ഉള്ളടക്കങ്ങൾ കാണാൻ കഴിയും. ക്യാൻവാസ് ചിത്രം ചെറുതാണ് ചെയ്യുമ്പോൾ, പ്രിവ്യൂ വിൻഡോ നേർത്ത നെഗറ്റീവ് അതിർത്തിയിൽ ഒരു ഫ്രെയിം അത് കാണിക്കുന്നു.

എക്സ്; പിന്നെ

X ആൻഡ് Y ക്യാൻവാസ് മുകളിലെ ഇടത് കോണിൽ ആപേക്ഷിക ചിത്രം മുകളിലെ ഇടത് കോണിൽ കോർഡിനേറ്ററുകൾ വ്യക്തമാക്കുക. ക്യാൻവാസ് ചിത്രം ചെറുതാണ് അവർ നെഗറ്റീവ്. PLACEHOLDER,-103828: വ്യത്യസ്ത വഴികൾ (തീർച്ചയായും, നിർദ്ദേശാങ്കങ്ങൾ കാൻവാസ് അതിരുകൾ കവിയരുത്) ഇമേജ് പതിക്കാൻ കഴിയും

പാളികൾ

ജിമ്പ്-2.4 പതിപ്പ് മുമ്പ്, “Canvas Size” പാളി വലുപ്പം യാതൊരു സ്വാധീനവും. അതു മാറ്റാൻ, നിങ്ങൾ Layer Boundary Size കമാൻഡ് ഉപയോഗിക്കാൻ ഉണ്ടായിരുന്നു. “Layers” ഐച്ഛികം ഇപ്പോൾ എങ്ങനെ ഒരുപക്ഷേ, പാളികൾ വലുപ്പം ചെയ്യും വ്യക്തമാക്കാൻ നിങ്ങളെ അനുവദിക്കുന്നു. ഡ്രോപ്പ്-ഡൌൺ ലിസ്റ്റ് നിരവധി സാധ്യതകൾ പ്രദാനം:

ചിത്രം 16.61. The Resize layers list


  • None: സ്ഥിര ഓപ്ഷൻ. ഇല്ല പാളി വലുപ്പം, മാത്രമല്ല ക്യാൻവാസ് ആണ്.

  • All Layers: എല്ലാ പാളികളും ക്യാൻവാസ് വലിപ്പം വലുപ്പം ചെയ്യുന്നു.

  • Image-sized layers: മാത്രമേ ഇമേജ് അതേ വലിപ്പമുള്ള പാളികൾ ക്യാൻവാസ് വലുപ്പം വലുപ്പത്തിലാണെന്ന്.

  • All visible layers: മാത്രം കാണാവുന്ന പാളികൾ ഒരു ഐക്കൺ അടയാളപ്പെടുത്തിയിട്ടുള്ള, ലെയർ ഡയലോഗിൽ, ക്യാൻവാസ് വലുപ്പം വലുപ്പത്തിലാണെന്ന്.

  • All linked layers: ലെയർ ഡയലോഗ് ഒരു അടയാളപ്പെടുത്തിയിട്ടുള്ള മാത്രം ലിങ്ക്ഡ് പാളികൾ, ക്യാൻവാസ് വലുപ്പം വലുപ്പത്തിലാണെന്ന്.

കേന്ദ്രം

Center ബട്ടൺ നിങ്ങൾ കാൻവാസിൽ ചിത്രം മധ്യഭാഗത്താക്കാൻ അനുവദിക്കുന്നു. നിങ്ങൾ കേന്ദ്രം ബട്ടൺ ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഓഫ്സെറ്റ് മൂല്യങ്ങൾ സ്വയം കണക്കാക്കി ടെക്സ്റ്റ് ബോക്സുകൾ പ്രദർശിപ്പിക്കുന്നു.

കുറിപ്പു്

നിങ്ങൾ Resize ബട്ടണില് ക്ലിക്ക് ചെയ്യുമ്പോൾ, ക്യാൻവാസ് വലുപ്പം, പക്ഷേ പിക്സൽ വിവരങ്ങൾ ചിത്രത്തിന്റെ ഡ്രോയിംഗ് സ്കെയിൽ മാറ്റമില്ല ആകുന്നു.

നിങ്ങൾ അതിന്റെ വലിപ്പം മാറ്റി മുമ്പ് ചിത്രത്തിന്റെ പാളികൾ ക്യാൻവാസ് അതിരുകളെ കഴിയാത്തതു, അത് വ്യാപ്തിയിൽ ചേർത്തതാണ് ആ ക്യാൻവാസ് ഭാഗത്തുനിന്നു പാളികളില്ല. അതുകൊണ്ടു, ക്യാൻവാസ് ഈ ഭാഗം സുതാര്യമായ ഒരു ശൂരനാടിന് പാറ്റേൺ ഉപയോഗിച്ച് പ്രദർശിപ്പിക്കും, അതു ഉടനടി പെയിന്റിംഗിന് ലഭ്യമല്ല. നിങ്ങൾ flatten നിങ്ങൾ കൃത്യമായി ക്യാൻവാസ് പാകമാകുന്ന ഒരൊറ്റ പാളി ഒരു ചിത്രം ലഭിക്കും അല്ലെങ്കിൽ നിങ്ങൾ മറ്റെന്തെങ്കിലും പാളികൾ മാറ്റുന്നതിൽ ഇല്ലാതെ സജീവ പാളി വലുപ്പം Layer to Image Size കമാൻഡ് ഉപയോഗിക്കാം സാഹചര്യത്തിൽ ചിത്രം, ഒന്നുകില്. നിങ്ങൾക്ക് ഒരു പുതിയ ലയർ ഉണ്ടാക്കുക നിങ്ങൾ ആഗ്രഹിക്കുന്ന പശ്ചാത്തലവും പൂരിപ്പിക്കാൻ കഴിയും. അതിലൂടെ, നിങ്ങൾ ഒരു ഡിജിറ്റൽ “passe-partout” (ഗ്ലാസ് ഒരു തരത്തിലുള്ള ഒരു ഫോട്ടോഗ്രാഫിൽ വഴുതുന്നു വേണ്ടി ഒരു നീക്കം തിരികെ മൌണ്ട്) സൃഷ്ടിക്കുക.

നാം ഒരേ വലിപ്പം ഒരു സ്ഥിര ക്യാൻവാസ് നിർവചിക്കുന്ന 100x100 പിക്സൽ ഒരു പച്ച പശ്ചാത്തല പാളി ആരംഭിച്ചു. പിന്നീട് ഞങ്ങൾ ഒരു പുതിയ ചുവന്ന പാളി 80x80 പിക്സലുകൾ ചേർത്തു. സജീവ പാളി പരിധികൾ ഒരു കറുപ്പും മഞ്ഞയും അതിനെക്കുറിച്ച് ഉപയോഗിച്ച് അടയാളപ്പെടുത്തിയിരിക്കുന്നു. ചുവന്ന പാളി പൂർണ്ണമായും ക്യാൻവാസ് പൂരിപ്പിക്കുക ഇല്ല: ആളൊഴിഞ്ഞ ഭാഗം സുതാര്യമാണ്. പണിസഞ്ചിയിൽ പശ്ചാത്തല നിറം മഞ്ഞ ആണ്.
