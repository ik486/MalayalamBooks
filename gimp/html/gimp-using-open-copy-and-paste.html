3.7. പകര്ത്തി ഒട്ടിക്കുക

ക്ലിപ്പ്ബോർഡിൽ നിന്ന് ഒരു പുതിയ ഇമേജ് സൃഷ്ടിക്കാൻ File → Create → From Clipboard ഉപയോഗിക്കുക; മറ്റൊരുവിധത്തിൽ, നിങ്ങൾ Edit → Past as → New image ഉപയോഗിക്കാം. പല പ്രയോഗങ്ങളും പിന്നീട് കയറി GIMP ഒട്ടിക്കാൻ കഴിയുന്ന ക്ലിപ്പ്ബോർഡിലേക്ക് ഒരു ചിത്രം പകർത്തുന്നത് പിന്തുണയ്ക്കുന്നില്ല. പല ഓപ്പറേറ്റിങ് സിസ്റ്റങ്ങൾ ക്ലിപ്പ്ബോർഡിലേക്ക് സ്ക്രീനുകളും പകർത്തുന്നതിൽ പിന്തുണയ്ക്കുന്നില്ല. Print Screen സാധാരണ പകർപ്പുകൾ ക്ലിപ്പ്ബോർഡിലേക്ക് സ്ക്രീൻ, ഒപ്പം ALT+Print Screen അതല്ലല്ലോ സജീവ ജാലകം. പ്രിന്റ് സ്ക്രീൻ സാർവ്വത്രികമായി പിന്തുണയ്ക്കുന്നില്ല, നിങ്ങളുടെ ഓപ്പറേറ്റിംഗ് സിസ്റ്റം ക്ലിപ്പ്ബോർഡിലേക്ക് ഒരു ചിത്രം പകർത്തി കരുതി, GIMP ബോർഡിൽ നിന്നുള്ള ചിത്രം ഉപയോഗിക്കാം ഇതിനർത്ഥമില്ല. നിങ്ങളുടെ ഏറ്റവും മികച്ച പന്തയം അതു ശ്രമിക്കുക അത് പ്രവർത്തിക്കുന്നുണ്ടോയെന്ന് നോക്കാനാണ്.
