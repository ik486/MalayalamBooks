6.27. ഗ്രിഡ് ക്രമീകരിക്കുക

Configure Grid കമാൻഡ് നിങ്ങൾ അതിൽ പ്രവർത്തിക്കുകയാണ് സമയത്ത്, നിങ്ങൾ നിങ്ങളുടെ ചിത്രം മേൽ പ്രദർശിപ്പിക്കുകയും കഴിയും ഗ്രിഡ് സജ്ജമാക്കാൻ അനുവദിക്കുന്നു. GIMP മാത്രം കാര്ട്ടീഷ്യന്പ്ലോട്ട് ഗ്രിഡുകൾ നൽകുന്നു. നിങ്ങൾ സ്വതന്ത്രമായി തിരശ്ചീന, ലംബ ഗ്രിഡ് ലൈനുകൾ വേണ്ടി, ചിത്രം ഉത്ഭവം നിന്ന് ഗ്രിഡ് ലൈനുകൾ നിറവും സ്പേസിംഗ്, ഓഫ്സെറ്റുകളുണ്ടു് തിരഞ്ഞെടുക്കാം. നിങ്ങൾ അഞ്ചു വ്യത്യസ്ത ഗ്രിഡ് ശൈലികളുടെ തിരഞ്ഞെടുക്കാം.

6.27.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Image → Configure Grid.

6.27.2. “Configure Grid” ഡയലോഗിന്റെ വിവരണം

ചിത്രം 16.84. The “Configure grid” dialog


കാഴ്ച

Configure Image Grid ഡയലോഗിൽ, നിങ്ങൾ ഇമേജ് ഗ്രിഡ് ഓൺ ചെയ്യുമ്പോൾ ഏത് കാണിച്ചിരിക്കുന്നു ഗ്രിഡ് സജ്ജമാക്കാൻ കഴിയും.

Line style
കവലകളും (ഡോട്ടുകൾ)

ഈ രീതിയിൽ, കുറഞ്ഞത് പാതകൾ, ഗ്രിഡ് ലൈനുകൾ ഓരോ ജംഗ്ഷനിൽ ഒരു ലളിതമായ ഡോട്ട് കാണിക്കുന്നു.

കവലകളും (crosshairs)

ഈ രീതിയിൽ, സ്വതവേ, ഗ്രിഡ് ലൈനുകൾ ഓരോ ജംഗ്ഷനിൽ ഒരു പ്ലസ് ആകൃതിയിലുള്ള അടയാളസൂചി കാണിക്കുന്നു.

ഒന്നിടവിട്ട

ഈ രീതിയിൽ ഗ്രിഡിന്റെ മുന്ഭാഗത്തെ തകർത്തുകളഞ്ഞു വരികൾ കാണിക്കുന്നു. വരികൾ ഒരുമിച്ചു വളരെ അടുത്താണ് എങ്കിൽ, ഗ്രിഡ് മനോഹരമായിരിക്കും ചെയ്യില്ല.

ഇരട്ട ഡാഷ്ഡ്

This style shows dashed lines, where the foreground and background colors of the grid alternate.

സോളിഡ്

This style shows solid grid lines in the foreground color of the grid.

Foreground and Background colors

ഗ്രിഡ് ഒരു പുതിയ നിറം തിരഞ്ഞെടുക്കുന്നതിന് നിറം പാർക്കയില്ല ക്ലിക്കുചെയ്യുക.

അകലം

Width and Height

നിങ്ങൾ ഗ്രിഡിന്റെ സെൽ വലിപ്പം, അളക്കാനുള്ള യൂണിറ്റ് തിരഞ്ഞെടുക്കാം.

ഓഫ്സെറ്റ്

Width and Height

നിങ്ങൾ ആദ്യം സെല്ലിന്റെ ഓഫ്സെറ്റ് സജ്ജമാക്കാൻ കഴിയും. കോർഡിനേറ്റ് വംശജനായ ചിത്രം മുകളിലെ ഇടത് കോണിൽ ആണ്. സ്വതവേ, ഗ്രിഡ് കോർഡിനേറ്റ് ഉത്ഭവം തുടങ്ങുന്നത്, (0,0).
