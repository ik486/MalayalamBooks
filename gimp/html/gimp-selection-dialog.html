4.8. തിരഞ്ഞെടുക്കൽ എഡിറ്റർ

Selection Editor കമാൻഡ് “Selection Editor” ഡയലോഗ് ജാലകം പ്രദർശിപ്പിക്കുന്നു. ഈ ഡയലോഗ് വിൻഡോ നിലവിലെ ചിത്രം സജീവ നിരക്കു പ്രദർശിപ്പിക്കുന്നു നിങ്ങൾ നിരക്കു കമാൻഡുകൾ എളുപ്പത്തിൽ ആക്സസ് നൽകുന്നു. അതു ശരിക്കും നേരിട്ട് എഡിറ്റിംഗ് തിരഞ്ഞെടുപ്പുകളെ ഉദ്ദേശിച്ചുള്ളതല്ല, എന്നാൽ നിങ്ങൾ ഒരു നിര പ്രവർത്തിച്ചുകൊണ്ടിരിക്കുന്നു എങ്കിൽ അതു ഹാൻഡി നിരക്കു ഒന്നിച്ച് കല്പിക്കുന്നു ഞങ്ങൾക്കുണ്ട്, അതു കൽപന മരത്തിൽ കമാൻഡുകൾ തിരയാൻ അധികം ബട്ടണിൽ ക്ലിക്ക് എളുപ്പം ശേഷം മെനു ബാർ. “Selection Editor” പുറമേ “Select to Path” കമാൻഡ് ചില വിപുലമായ ഓപ്ഷനുകൾ പ്രദാനം ചെയ്യുന്നു.

4.8.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Select → Selection Editor.

4.8.2.  Description of the “Selection Editor” dialog window

ചിത്രം 16.27. The “Selection Editor” dialog window


The Buttons

“Selection Editor” ഡയലോഗ് ജാലകം എളുപ്പത്തിൽ നിരക്കു കമാൻഡുകൾ ആക്സസ് ഉപയോഗിക്കാനും കഴിയുന്ന നിരവധി ബട്ടണുണ്ടെങ്കില്:

  • Select All ബട്ടൺ .

  • Select None ബട്ടൺ .

  • Select Invert ബട്ടൺ .

  • Save to Channel ബട്ടൺ .

  • To Path ബട്ടൺ . നിങ്ങൾ ഈ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുമ്പോൾ Shift കീ ഹോൾഡ് എങ്കിൽ “Advanced Settings” ഡയലോഗ് പ്രദർശിപ്പിച്ചിരിക്കുന്നു. ഈ ഓപ്ഷനുകൾ കുറിച്ച് വിവരങ്ങൾക്കായി അടുത്ത വിഭാഗം കാണുക.

  • Stroke Selection ബട്ടൺ .

The display window

ഡിസ്പ്ലേ വിൻഡോയിൽ, ചിത്രത്തിന്റെ തിരഞ്ഞെടുത്ത പ്രദേശങ്ങൾ വെളുത്ത, തിരഞ്ഞെടുത്ത നോൺ പ്രദേശങ്ങൾ കറുത്ത ആകുന്നു, ഭാഗികമായും തിരഞ്ഞെടുത്ത പ്രദേശങ്ങൾ ചാര ഷേഡുകൾ ഇരിക്കുന്നു. ഈ ജാലകത്തിൽ ക്ലിക്കുചെയ്യുന്നത് Select by Color പോലെ പ്രവർത്തിക്കുന്നു. ചുവടെയുള്ള ഉദാഹരണം കാണുക.

ചിത്രം 16.28. “Selection Editor” ഡിസ്പ്ലേ ജാലകത്തിൽ അമര്ത്തുന്നതിനു ഉദാഹരണം

“Select By Color” ലേക്കുള്ള “Selection Editor” ഡിസ്പ്ലേ വിൻഡോയിൽ ക്ലിക്കുചെയ്യുന്നത്. “Select By Color” ഇമേജ് വിൻഡോയിൽ ഉപയോഗിക്കുമ്പോൾ ഈ കണക്കുകൾ പോലെ “Selection Editor” ജാലകത്തിന്റെ രൂപം കാണിക്കാൻ കഴിഞ്ഞില്ല ശ്രദ്ധിക്കുക.


4.8.3.  The “Selection to Path Advanced Settings” dialog

ചിത്രം 16.29. The “Advanced Settings” dialog window


“Selection to Path Advanced Settings” ഡയലോഗ്, നിങ്ങൾ Shift ക്ലിക്ക് ചെയ്ത് നേടുകയും Selection to Path ബട്ടൺ, നിങ്ങൾ ഒരു സ്ലൈഡർ ബാർ അല്ലെങ്കിൽ ടെക്സ്റ്റ് ബോക്സ് കൊണ്ട് സജ്ജമാക്കാൻ കഴിയും ഭൂരിഭാഗം നിരവധി ഓപ്ഷനുകൾ അടങ്ങിയിരിക്കുന്നു. ഒറ്റ ചെക്ക് ബോക്സ് ഉണ്ട്. ഈ ഓപ്ഷനുകൾ കൂടുതലും നൂതന ഉപയോക്താക്കൾക്ക് ഉപയോഗിക്കുന്നുണ്ട്. അവർ:

  • Align Threshold: രണ്ടു റോഡിൻറെ ഈ മൂല്യം അടുത്ത് അവ തുല്യമോ നടത്തിയില്ല.

  • Corner Always Threshold: ഒരു പോയിന്റ് നിർവചിക്കുന്ന കോൺ അതിന്റെ മുൻഗാമികൾ ആൻഡ് പ്രതിനിധികളാക്കുകയും ഈ ചെറുതാണ് എങ്കിൽ, അത് ഒരു കോണിൽ, ഒരു ചെറിയ കോണിൽ ഒരു ബിന്ദുവിൽ Corner Surround പിക്സൽ ഉള്ളിൽ പോലും ആണ്.

  • Corner Surround: ഒരു പോയിന്റ് ഒരു കോണിൽ ഇല്ലയോ എന്ന് തീരുമാനിക്കുന്നതിനുള്ള ശ്രദ്ധിക്കേണ്ട കാര്യങ്ങൾ എണ്ണം.

  • Corner Threshold: ഒരു പോയിന്റ്, അതിന്റെ മുൻഗാമികൾ, അതിന്റെ പിൻഗാമികളുടെയും ഈ ചെറുതാണ് ഒരു കോണിൽ നിർവചിക്കാനുള്ള, അത് ഒരു കോണിൽ ആണ്.

  • Error Threshold: പിശക് തുക ഏത് ഒരു തരപ്പെട്ടവരുമാണവർ spline PLACEHOLDER,-104258 അസ്വീകാര്യമാണ്. ഏതെങ്കിലും പിക്സൽ കൂടുതൽ അകലെ തരപ്പെട്ടവരുമാണവർ കർവ് നിന്ന് അതിനെക്കാൾ ആണെങ്കിൽ, അൽഗോരിതം വീണ്ടും ശ്രമിക്കുന്നു.

  • Filter Alternative Surround: സമീപത്തെ പോയിന്റ് രണ്ടാം നമ്പർ ഫിൽട്ടർ പരിഗണിക്കേണ്ട.

  • Filter Epsilon: Filter Surround നിർമ്മിക്കുന്ന സദിശങ്ങളെയും Filter Alternative Surround പോയിന്റ് തമ്മിലുള്ള കോണുകളിൽ Filter Alternative Surround നിന്ന് ഒന്ന് ഉപയോഗിക്കുക ഈ കൂടുതൽ ഹരണഫലം.

  • Filter Iteration Count: തവണ എണ്ണം യഥാർത്ഥ ഡാറ്റ പോയിന്റുകൾ സ്മൂത്. 50 അല്ലെങ്കിൽ അങ്ങനെ ലേക്ക്, നാടകീയമായി ഈ നമ്പർ വർദ്ധനവ് രൂപകങ്ങളുടെ മെച്ചപ്പെട്ട ഫലങ്ങൾ ഉണ്ടാക്കാൻ കഴിയും. എന്നാൽ കോണിലും ആയിരിക്കും “should” ഏതെങ്കിലും പോയിന്റ് കണ്ടെത്തിയില്ല എങ്കിൽ, കർവ് ആ ഘട്ടത്തിൽ ചുറ്റും കാട്ടു പോകുന്നു.

  • Filter Percent: പുതിയ പോയിന്റ് ഹാജരാക്കണം പഴയ പോയിന്റ് ഉപയോഗിക്കുക പ്ലസ് ഈ തവണ അയൽക്കാർ.

  • Filter Secondary Surround: Filter Surround പോയിന്റ് ഒരു വര നിഷ്കർഷിക്കുന്നു പരിഗണിക്കാവുന്ന സമീപത്തെ പോയിന്റ് എണ്ണം.

  • Filter Surround: ഫിൽട്ടർ പരിഗണിക്കേണ്ട സമീപത്തെ പോയിന്റ് എണ്ണം.

  • Keep Knees: ഈ ചെക്ക് ബോക്സ് ഔട്ട്ലൈൻ കണ്ടെത്തുന്നതിൽ ശേഷം “knee” പോയിന്റ് നീക്കം വേണ്ടയോ എന്ന് പറയുന്നു.

  • Line Reversion Threshold: ഒരു spline ഈ മൂല്യം ഒരു വര ചേർച്ചയിലാകുന്നു, അത് ഒരു വര, അത് തിരികെ ഒരു കർവുകളിൽ മാറ്റം വന്നാൽ പോലും തുടരുന്നു. ഈ പുനസ്ഥാപിക്കാനാകുന്നതിന് അവിടങ്ങളിൽ കർവുകൾ കൂടുതൽ സാധ്യത ഉണ്ടാക്കുവാൻ കർവ് നീളം സ്ക്വയർ വഴി വെയ്റ്റഡ് ആണ്.

  • Line Threshold: അതു ഒരു വര മാറുന്നു മുമ്പ് ഒരു spline അതിന്റെ റോഡിൻറെ നിർണ്ണയിക്കുന്നത് ലൈനിൽ നിന്ന് ഭിന്നത എത്ര പിക്സൽ (ശരാശരി).

  • Reparametrize Improvement: reparameterization ഇത്രയധികം ശതമാനം ഫിറ്റ് മെച്ചപ്പെടുത്താൻ ചെയ്യുന്നില്ലെങ്കിൽ, അൽഗോരിതം അത് ചെയ്യുന്നത് നിർത്തുന്നു.

  • Reparametrize Threshold: പിശക് തുക ഇപ്പോൾ അവരുടെ reparameterize ലേക്ക് കഥയായി. ഈ അൽഗോരിതം ഒരു ഒരൊറ്റ spline ഉപയോഗിച്ച് ഒരു “O” പുറത്തുള്ള രൂപരേഖ fit ശ്രമിക്കുന്നു ഉദാഹരണമായി, സംഭവിക്കുന്നു. പ്രാരംഭ ഫിറ്റ് അതു മെച്ചപ്പെടുത്താൻ ന്യൂട്ടൻ-Raphson ആവർത്തന മതി നന്നല്ല. അത് അൽഗോരിതം ഏതെങ്കിലും കോണിലും കണ്ടില്ല സാഹചര്യങ്ങളിൽ കണ്ടെത്താനാകുന്നില്ല നല്ലത് എന്ന് ഞാൻ വരാം.

  • Subdivide Search: അകലെ ഏറ്റവും മോശം പോയിന്റ് മുതൽ കർവ് ശതമാനം വിഭജിച്ചിരിക്കുന്നു ലേക്ക് ഉത്തമമായ ഒരു സ്ഥലം നോക്കി.

  • Subdivide Surround: തന്നിരിക്കുന്ന പോയിന്റ് വിഭജിച്ചിരിക്കുന്നു ലേക്ക് ഉത്തമമായ ഒരു സ്ഥലം തീരുമാനിക്കുമ്പോൾ ശ്രദ്ധിക്കേണ്ട കാര്യങ്ങൾ എണ്ണം.

  • Subdivide Threshold: എത്ര പിക്സൽ ഒരു പോയിന്റ് ഒരു വര നിന്ന് ഭിന്നത ഇപ്പോഴും വിഭജിച്ചിരിക്കുന്നു ഒരു മെച്ചപ്പെട്ട സ്ഥലം പരിഗണിക്കാം.

  • Tangent Surround: ആ ഘട്ടത്തിൽ ടാൻജെന്റ് ലേക്ക് ഏകദേശ കമ്പ്യൂട്ടിങ് ഒരു പോയിന്റ് ഇരുവശങ്ങളിലും നോക്കണം പോയിന്റ് എണ്ണം.
