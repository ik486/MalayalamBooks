2.8. മുൻവശം തിരഞ്ഞെടുക്കുക

ചിത്രം 14.30. The “Foreground Select” tool in the Toolbox


ഈ ഉപകരണം നിങ്ങളെ സജീവ പാളി നിന്ന് അല്ലെങ്കിൽ ഒരു നിര നിന്ന് മുൻഭാഗത്തെ എക്സ്ട്രാക്റ്റ് അനുവദിക്കുന്നു. ഇത് SIOX രീതി (സിമ്പിൾ ഇന്ററാക്ടീവ് ഒബ്ജക്റ്റ് വേർതിരിക്കൽ) അടിസ്ഥാനമാക്കിയുള്ളതാണ്. നിങ്ങൾ ??? അതിന്റെ വെബ് പേജ് സന്ദർശിക്കാം.

2.8.1. Directions for use

ഈ ഉപകരണം ഉപയോഗിച്ച് ഒരു നിര സൃഷ്ടിയും ഘട്ടങ്ങൾ ഒരു പ്രണയവും പ്രവർത്തിക്കുന്നു:

  1. Roughly select the foreground നിങ്ങൾ പുറത്തെടുക്കുകയും ആഗ്രഹിക്കുന്നു. നിങ്ങൾ ഈ ഉപകരണം തെരഞ്ഞെടുക്കുമ്പോൾ, മൌസ് പോയിന്റർ lasso എന്ന ഐക്കൺ പോകുന്നു. ഇത് യഥാർത്ഥത്തിൽ അവ്യക്തമായ തിരഞ്ഞെടുക്കുക പ്രയോഗം പോലെ പ്രവർത്തിക്കുന്നു. പശ്ചാത്തല നിന്ന് കഴിയുന്നത്ര ചെറിയ തിരഞ്ഞെടുക്കുക.

    ഉടൻ മൌസ് ബട്ടൺ റിലീസ്, ചിത്രത്തിന്റെ നോൺ തിരഞ്ഞെടുത്ത ഭാഗം ഇരുണ്ട നീല മാസ്ക് മൂടിയിരിക്കുന്നു. എങ്കിൽ നിരക്കു അടച്ചിട്ടില്ല, അതിന്റെ അറ്റത്ത് ഒരു വര ഓട്ടോമാറ്റിക്കായി ഒരുമിച്ചു ചെയ്യും. മൗസ് സൂചികയെ അടുത്ത ഘട്ടത്തിലേക്കു വേണ്ടി പെയിന്റ്-ബ്രഷ് ഐക്കൺ ഇപ്പോൾ പോകുന്നു.

    ചിത്രം 14.31. The foreground is roughly selected


  2. Draw a line through the foreground: ആരുടെ വലിപ്പം ഓപ്ഷനുകൾ മാറ്റാൻ സാധിക്കും അന്തസ്സായിത്തന്നെയാണു, ഉപയോഗിച്ച് ഉൽപ്പാദിപ്പിക്കുന്നതിന് സൂക്ഷിക്കും ഏത് നിറങ്ങൾ കടന്നുചെല്ലുന്ന തിരഞ്ഞെടുത്ത മുൻഭാഗത്തെ ഒരു വരിയായി ആകർഷിക്കും. ലൈൻ വരയ്ക്കാനുപയോഗിയ്ക്കേണ്ട നിറം യാതൊരു പ്രാധാന്യമുള്ള; മുൻഭാഗത്തെ അതേ നിറം ഉപയോഗിച്ച് നല്ലത്. പശ്ചാത്തല പിക്സൽ മഷി സൂക്ഷിച്ചുകൊള്ളുവീൻ.

    ചിത്രം 14.32. The line drawn on the foreground

    ഈ ഉദാഹരണത്തിൽ, അതു ലൈൻ പൂവ് മഞ്ഞ capitulum അടിക്കുമ്പോൾ പ്രധാനമാണ്.


  3. മൌസ് ബട്ടൺ റിലീസ് ചെയ്യുമ്പോൾ, എല്ലാ-തിരഞ്ഞെടുത്ത പ്രദേശങ്ങൾ അന്ധകാരത്തിൽ:

    ചിത്രം 14.33. The area which will be selected


  4. നിങ്ങൾക്ക് ഇപ്പോഴും പിടികൂടാനുള്ള നിരക്കു ലഭിക്കാൻ Enter അമർത്തുക:

    ചിത്രം 14.34. Foreground is selected


കുറിപ്പു്

നിങ്ങൾ Enter അമർത്തുക വരെ നിങ്ങൾക്ക് Ctrl+Z വഴി വേണ്ടാ Select → None വഴി ഈ നിരക്കു പഴയപടിയാക്കാൻ കഴിയില്ല, തട്ടിപ്പുകാർക്ക് ചരിത്രം ബന്ധപ്പെട്ട അല്ല. ഈ നിരക്കു ഇല്ലാതാക്കുന്നതിന്, നിങ്ങൾ മറ്റൊരു ഉപകരണം തിരഞ്ഞെടുക്കുക വേണം.

2.8.2. ടൂൾ സജീവമാക്കുന്നു

നിങ്ങൾക്ക് രണ്ട് തരത്തിൽ ഫോർഗ്രൌണ്ട് തിരഞ്ഞെടുക്കുക ഉപകരണം സജീവമാക്കാം:

  • by clicking on the tool icon in the Toolbox,

  • through Tools → Selection Tools → Foreground Select in the image menu.

  • This tool has no shortcut, but you can set one using Edit → Preferences → Interface → Configure Keyboard Shortcuts → Tools → Foreground Select

2.8.3. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

Ctrl,

Ctrl കീ അമർത്തിയാൽ, നിങ്ങൾ മുന്നിലും പശ്ചാത്തലത്തിലും നിരക്കു പെയിന്റിംഗ് അസൗകര്യവും കഴിയും.

2.8.4. ഓപ്ഷനുകൾ

ചിത്രം 14.35. “Foreground Select” tool options


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

മോഡ്; ആന്റിഎലിയാസിംഗ്; തൂവല് അറ്റങ്ങൾ
കുറിപ്പു്

ഈ എല്ലാ ഉപകരണങ്ങൾ സാധാരണമാണ് ഉപാധികൾക്ക് സഹായത്തിന് Selection Tools കാണുക. ഈ ഉപകരണം മാത്രമായുള്ള ഓപ്ഷനുകൾ മാത്രം ഇവിടെ വിശദീകരിച്ചു ചെയ്യുന്നു.
ഐലന്റ്

ഈ ഓപ്ഷൻ പ്രാപ്തമാക്കിയാൽ, തലോടാൻ ഐലന്റ് മാത്രമേ പ്രദേശം തിരഞ്ഞെടുക്കും. അല്ലാത്തപക്ഷം ഒരേ നിറങ്ങൾ എല്ലാ പ്രദേശങ്ങൾ തിരഞ്ഞെടുക്കും.

ചിത്രം 14.36. “Contiguous” option effect

രണ്ട് ഒരേ നിറം മേഖലകളിൽ വേർതിരിച്ചു. ഇടതുവശത്ത്, മാത്രം ഇടത് പ്രദേശം അടയാളപ്പെടുത്തി.

ചിതറി ഓപ്ഷൻ പരിശോധിയ്ക്കുന്നു: ചായം ലൈൻ അടുത്ത് മാത്രമേ പ്രദേശം തിരഞ്ഞെടുത്തു.

ചിതറി ഉപാധി ചെക്കുചെയ്തിട്ടില്ല: ഇരുവരും പ്രദേശങ്ങൾ അവർ ചിതറി ആണെങ്കിലും, തിരഞ്ഞെടുത്തു.


Interactive refinement

ഇവിടെ നിങ്ങളുടെ തിരഞ്ഞെടുപ്പിനെ കൂടുതൽ കൃത്യമായി പ്രവർത്തിക്കാൻ ചില ഉപാധികൾ:

മാർക്ക് മുൻഭാഗത്തെ

സ്ഥിര ഓപ്ഷൻ. പണിസഞ്ചി ഫോർഗ്രൗണ്ടിലെ നിറം വരയ്ക്കാൻ ഉപയോഗിക്കുന്നു. ചായം ലൈൻ വഴി മൂടി നിറങ്ങൾ ഉൽപ്പാദിപ്പിക്കുന്നതിന് ഉപയോഗിക്കും.

അടയാളപ്പെടുത്തുക പശ്ചാത്തലം

നിങ്ങൾ Ctrl കീ അമർത്തിയാൽ ഈ ഓപ്ഷൻ ആക്സസ് ഒന്നുകിൽ ഇനി ഒരല്പം റേഡിയോ ബട്ടൺ ക്ളിക്ക്, വഴി കഴിയും. മൗസ് സൂചികയെ ഒരു ചെറിയ എറേസർ ഐക്കൺ പോകുന്നു. നിറം പണിസഞ്ചി പശ്ചാത്തലത്തിൽ നിറം. “erased” പിക്സൽ ലഭ്യമാക്കിയ ചെയ്യില്ല അതേ നിറം ഏത് തിരഞ്ഞെടുപ്പിന്റെ പിക്സലുകൾ.

Small brush / Large brush

ഈ സ്ലൈഡർ നിങ്ങളെ ലൈൻ വരയ്ക്കാൻ ഉപയോഗിക്കുന്ന ബ്രഷ് സൈസ് പൊരുത്തപ്പെടാൻ അനുവദിക്കുന്നു. ഒരു ചെറിയ ബ്രഷ് നേര്ത്ത വിശദാംശങ്ങൾ അനുയോജ്യമാകും.
