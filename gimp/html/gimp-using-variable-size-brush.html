9. വ്യത്യസ്ത ബ്രഷ് സൈസ്

GIMP -2,4 മുതൽ എല്ലാ നയപ്രഖ്യാപന ഒരു വേരിയബിൾ വലിപ്പം ഞങ്ങൾക്കുണ്ട്.

9.1. എങ്ങനെ ഒരു ബ്രഷ് ഉയരം വ്യത്യാസപ്പെടാൻ

നിങ്ങൾ മൂന്നു തരത്തിലാണ് ബ്രഷ് സൈസ് വ്യത്യസ്ത ലഭിക്കും:

  1. ബ്രഷ് ഉപയോഗിക്കുന്ന ടൂൾ Scale സ്ലൈഡർ ഉപയോഗിക്കുന്നു. പെൻസിൽ, അന്തസ്സായിത്തന്നെയാണു, എറേസർ, എയർബ്രഷ്, ക്ലോൺ, സൌഖ്യമാക്കേണമേ കാഴ്ചപ്പാട് ക്ലോൺ, ഇഫക്റ്റ് / ഷാർപ്പെൻചെയ്യുക, ഡോഡ്ജ് / ഉപകരണങ്ങൾ ബ്രഷ് സൈസ് വ്യത്യാസപ്പെടാം ഒരു സ്ലൈഡർ ഞങ്ങൾക്കുണ്ട് എരിഞ്ഞു.

    ചിത്രം 7.20. സ്കെയിൽ സ്ലൈഡർ


  2. മൌസ് ചക്രം പ്രോഗ്രാമിങ്:

    1. ജിമ് പ്രധാന ജാലകത്തിൽ, Edit → Preferences ക്ലിക്ക് ചെയ്യുക.

    2. പുതിയ വിൻഡോയുടെ ഇടത് കോളത്തിൽ, Input Devices → Input Controllers തിരഞ്ഞെടുക്കുക.

    3. Available Controllers ആൻഡ് Active Controllers: നിങ്ങൾക്ക് രണ്ട് നിരകൾ കൂടെ Additional Input Controllers കാണാൻ കഴിയും.

      കോളം Active Controllers ൽ Main Mouse Wheel ബട്ടൺ ഇരട്ട ക്ലിക്ക് ചെയ്യുക.

    4. Configure Input Controller: അപ്പോൾ നിങ്ങൾക്ക് പുതിയ വിൻഡോ കാണാം.

      ഇടത് കോളത്തിൽ Event ൽ Scroll Up ക്ലിക്ക് ഇത് നേടുക എടുത്തുകാട്ടിയ ലേക്ക്.

    5. Edit ബട്ടൺ (പട്ടികയുടെ താഴെ മധ്യഭാഗത്ത്) ക്ലിക്ക് ചെയ്യുക.

    6. ജാലക Select Controller Event Action കാണാൻ കഴിയും.

      ഡ്രോപ്പ്-ഡൌൺ Tools ഇനം അതിന്റെ ഇടത്തും ചെറിയ ചതുരത്തിൽ ക്ലിക്ക് ചെയ്തുകൊണ്ട്.

    7. ഇടത് കോളത്തിൽ Action ൽ OK ബട്ടൺ ക്ലിക്ക്, ഹൈലൈറ്റുചെയ്യാൻ അതിൽ Increase Brush Scale ക്ലിക്ക് ചെയ്യുക.

    8. ഇപ്പോൾ Scroll Up മുന്നിൽ ഡിസ്പ്ലേ tools-paint-brush-scale-increase ആണ്.

    9. ജനല് അടക്കുക.

    10. അതേ രീതി, Decrease Brush Scale കൂടെ പ്രോഗ്രാം Scroll Down കൂടെ.

    11. Preferences പ്രധാന ജാലകത്തിന്റെ OK ബട്ടൺ ക്ലിക്ക് മറക്കരുത്.

    ഈ അല്പമെങ്കിലും നീണ്ട വിശദീകരണങ്ങൾ ശേഷം, നിങ്ങൾക്ക് വലിപ്പം ബ്രഷ് വ്യത്യാസപ്പെടാം നിങ്ങളുടെ മൗസ് വീൽ ഉപയോഗിക്കാൻ കഴിയും. ഉദാഹരണത്തിന്, “Circle” ബ്രഷ് ഉപയോഗിച്ച് പെൻസിൽ ടൂൾ തിരഞ്ഞെടുക്കുക. രണ്ടു ഭാഗത്തേക്കും, നിങ്ങൾ “Circle” കുറയുക അല്ലെങ്കിൽ നീട്ടിക്കൊണ്ട് കാണാനാകും, ഇമേജ് വിൻഡോയിൽ പോയിന്റർ സജ്ജമാക്കുക, മൌസ് ചക്രം ഉപയോഗിക്കുമ്പോൾ.

  3. നിങ്ങൾ കീബോർഡിന്റെ “Up” ആൻഡ് “Down” ആരോ കീ പ്രോഗ്രാമിംഗ്.

    രീതി മൗസ് ചക്രം സമാനമാണ്. മാത്രം വ്യത്യാസങ്ങൾ ഇവയാണ്:

    • കോളം Active Controllers ൽ ഡബിൾ ക്ലിക്ക് Main Keyboard.

    • കോളം Event ൽ രണ്ടാം കീ ആദ്യ കീ Cursor Up, ഒപ്പം Cursor Down ക്ലിക്ക് ചെയ്യുക.

    • പിന്നെ, രണ്ടു കീകൾ ഉപയോഗിക്കുക (മുകളിലേക്കുള്ള അമ്പടയാളം Down ആരോ) നിങ്ങൾ മൌസ് ചക്രം കൊണ്ട് പോലെ ഫലം തുല്യമായിരിക്കണം.
