2.9. Save as

The Save as command displays the “Save Image” dialog. Since GIMP-2.8, the file is automatically saved in the XCF format and you can't save in another file format (for this, you have to export the file). The Save as dialog allows you to save with another name and/or to another folder.

2.9.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through File → Save as,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Shift+Ctrl+S ഉപയോഗിച്ച്.

2.9.2. The “Save Image” dialog

ചിത്രം 16.12. The “Save Image” dialog


  1. ഇടത് പാനൽ രണ്ടു ഭാഗങ്ങളായി തിരിച്ചിരിക്കുന്നു. മുകളിലെ ഭാഗം നിങ്ങളുടെ പ്രധാന ഡയറക്ടറികൾ നിങ്ങളുടെ സംഭരണ ​​ഡിവൈസുകൾ ലിസ്റ്റുചെയ്യുന്നു; ഈ ലിസ്റ്റിൽ പരിഷ്കരിക്കാൻ കഴിയുകയില്ല. താഴത്തെ ഭാഗം നിങ്ങളുടെ ബുക്ക്മാർക്കുകൾ ലിസ്റ്റുചെയ്യുന്നു; ചേർക്കാനോ നീക്കം ചെയ്യാം bookmarks. ഒരു ബുക്ക്മാർക്ക് ചേർക്കാൻ മധ്യത്തിൽ പാനലിൽ ഒരു ഡയറക്ടറി അല്ലെങ്കിൽ ഒരു ഫയൽ തിരഞ്ഞെടുത്ത് ഇടത് പാനലിൽ ചുവടെയുള്ള Add ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക. നിങ്ങൾക്ക്, സന്ദർഭ മെനുവിൽ Add to bookmarks കമാൻഡ് ഉപയോഗിക്കാം നിങ്ങൾ ശരിയായ മൌസ് ബട്ടൺ ഉപയോഗിച്ച്് ലഭിക്കുന്നതെന്ന്. നിങ്ങൾ Remove ബട്ടൺ അതു ക്ലിക്കു തിരഞ്ഞെടുത്ത് ഒരു ബുക്ക്മാർക്ക് ഇല്ലാതാക്കാൻ കഴിയും.

  2. മിഡിൽ പാനൽ നിലവിലെ ഡയറക്ടറിയിൽ ഫയലുകൾ പട്ടിക ലഭ്യമാക്കി. ഈ പാനലിൽ ഒരു ഡയറക്ടറി ഇരട്ട ഇടത്-ക്ലിക്കുചെയ്തുകൊണ്ട് നിലവിലുള്ള ഡയറക്ടറി മാറ്റുക. ഒറ്റ ഇടത് ക്ലിക്കിൽ ഒരു ഫയൽ തിരഞ്ഞെടുക്കുക. നിങ്ങൾക്ക് അതിനുശേഷം ഫയൽ Save ബട്ടൺ ക്ളിക്ക് തിരഞ്ഞെടുത്ത സംരക്ഷിക്കാനാകും. ഇരട്ട ഇടത് ക്ലിക്കിൽ നേരിട്ട് ഫയൽ സംരക്ഷിക്കുന്നു ശ്രദ്ധിക്കുക.

    നിങ്ങൾ Show Hidden Files കമാൻഡ് ആക്സസ് മിഡിൽ പാനലിൽ വലത് ക്ലിക്ക് ചെയ്യാം.

  3. The selected image is displayed in the Preview window. File size, resolution and image composition are displayed below the preview window.

    നിങ്ങളുടെ ചിത്രം മറ്റൊരു പ്രോഗ്രാം മാറ്റിയിട്ടുണ്ടെങ്കിൽ, അത് പുതുക്കുന്നതിനായി പ്രിവ്യൂ ക്ലിക്ക് ചെയ്യുക.

  4. Enter the filename of the new image file here.

    കുറിപ്പു്

    ചിത്രം ഇതിനകം തന്നെ സംരക്ഷിച്ചു എങ്കിൽ GIMP നിങ്ങൾക്ക് ഒരേ ഫയൽനാമം സൂചിപ്പിക്കുന്നത്. നിങ്ങൾ Save ക്ലിക്ക്, ഫയൽ എഴുതപ്പെട്ടു.

  5. മിഡിൽ പാനൽ മുകളിൽ, നിലവിലെ ഡയറക്ടറിയുടെ പാത്ത് പ്രദർശിപ്പിച്ചിരിക്കുന്നു. നിങ്ങൾ ബട്ടണുകൾ ഒന്നിലെ ക്ലിക്കുചെയ്ത് ഈ പാതയിലൂടെ നാവിഗേറ്റ് ചെയ്യാം.

  6. നിങ്ങൾ ഇതുവരെ നിലവിലില്ലാത്ത ഒരു ഫോൾഡറിലേക്ക് ചിത്രം സംരക്ഷിക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ നിങ്ങൾക്ക് Create Folder ക്ലിക്കുചെയ്ത് നിർദ്ദേശങ്ങൾ താഴെ ഉണ്ടാക്കുന്നു.

  7. ഈ ബട്ടൺ സ്വതവേ All Images കാണിക്കുന്നു. ഇത് എല്ലാ ചിത്രങ്ങളും ഏതൊരു അവരുടെ ഫയൽ തരം മധ്യത്തിൽ പാനൽ, ദൃശ്യമാകും എന്നാണ്. ഈ ലിസ്റ്റ് വികസ്വര, നിങ്ങൾ ഫയൽ ഒരേയൊരു തരം കാണിക്കാൻ തിരഞ്ഞെടുക്കാം.

  8. At Select File Type, you can select a compressed format for your XCF file.
