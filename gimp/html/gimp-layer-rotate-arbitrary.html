7.48. Arbitrary Rotation

Arbitrary Rotation കമാൻഡ് പറഞ്ഞിരിക്കുന്ന ഒരു കോണിൽ ഒരു പാളി കറങ്ങുന്നു. ഇത് Rotate tool നിയന്ത്രിക്കാനുതകുന്ന ഒരു ഇതര മാർഗമാണ്. കൂടുതൽ വിവരങ്ങൾക്ക് ആ ഉപകരണം കുറിച്ച് വിഭാഗം കാണുക.

7.48.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Transform → Arbitrary Rotation,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Shift+R ഉപയോഗിച്ച്.
