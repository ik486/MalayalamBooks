7.14. Select Top Layer

Select Top Layer കമാൻഡ് സ്റ്റാക്ക് ലെ മുകളിലെ പാളി ചിത്രത്തിനായി സജീവ പാളി ചെയ്യുന്നു ലെയറുകൾക്കുമായുള്ള ഡയലോഗിൽ അതു എടുത്തുകാണിക്കുന്നു. സജീവ പാളി ഇതിനകം സ്റ്റാക്കിലുള്ള മുകളിലെ പാളി, ഈ മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്.

കുറിപ്പു്

ഒരു സാധാരണ വിൻഡോസ്-രീതിയിൽ ഇംഗ്ലീഷ് കീബോർഡിലെ ശ്രദ്ധിക്കുക, സ്വതവേയുള്ള കീബോർഡ് കുറുക്കുവഴി Home അക്കങ്ങള് അറിയണ്ടേ കീ ലേക്ക്, എന്നാൽ സംഖ്യാ കീപാഡ് ഇടതു ആറു കീകൾ സംഘത്തിലെ മറ്റു Home കീ സൂചിപ്പിക്കുന്നത്.

7.14.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Stack → Select Top Layer,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Home ഉപയോഗിച്ച്.

അല്ലെങ്കിൽ നിങ്ങൾക്ക് ലളിതമായി ലയർ ഡയലോഗിൽ പാളി നാമത്തിൽ ക്ലിക്ക് ചെയ്യുക.
