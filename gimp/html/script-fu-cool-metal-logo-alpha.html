16.12. കൂൾ ലോഹ

16.12.1. പൊതു അവലോകനം

ചിത്രം 17.398. “Cool Metal” ഫിൽട്ടർ ഉദാഹരണം

“Cool Metal” ഫിൽട്ടർ

“Cool Metal” ലോഗോ


ഈ ഫിൽട്ടർ മിറർ നിലത്തു ഒരു പ്രതിഫലനം, ഒരു രസകരമായ ഡ്രോപ്പ് ഷാഡോ ലോഹ പോലുള്ള ഒരു പ്രഭാവം സൃഷ്ടിക്കുന്നു.

അരിപ്പ മുകളിൽ കാണുന്ന പോലെ ഒരു ടെക്സ്റ്റ് നിന്നുമുള്ള ലോഗോ സൃഷ്ടിക്കുന്ന “Cool Metal” സ്ക്രിപ്റ്റ് (ചിത്രം വിൻഡോയിൽ File → Create → Logos → Cool Metal), നിന്നാണ്.

16.12.2. ഫിൽട്ടർ സജീവമാക്കുക

ഈ ഫിൽട്ടർ Filters → Alpha to Logo → Cool Metal കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ കാണപ്പെടുന്നത്.

16.12.3. ഓപ്ഷനുകൾ

ചിത്രം 17.399. “Cool Metal” ഓപ്ഷനുകൾ


പ്രഭാവം വലുപ്പം (പിക്സൽ)

ഈ യഥാർത്ഥത്തിൽ “Cool Metal” സ്ക്രിപ്റ്റ്-ഫു സ്ക്രിപ്റ്റ് ഫോണ്ട് വലിപ്പം തീരുമാനം. ചില ആന്തരിക മൂല്യങ്ങൾ, ഭദ്രമാക്കുന്ന ബ്ലറിങ്, കൊത്തുപണി, ഒപ്പം റിപ്പിൾ പാറ്റേണുകൾ സൃഷ്ടിക്കുന്നത് ഈ വലിപ്പം ബന്ധപ്പെട്ട് സജ്ജമാക്കും.

പശ്ചാത്തല നിറം

ഫിൽറ്റർ ചേർത്തു പശ്ചാത്തല പാളി നിറം. നിങ്ങൾക്ക് കളർ Swatch ബട്ടൺ ക്ലിക്ക് ചെയ്യുമ്പോൾ, നിറം തിരഞ്ഞെടുത്ത ഡയലോഗ് മാറികിട്ടി.

ഗ്രേഡിയന്റ്

തണുത്ത ലോഹം സൃഷ്ടിക്കാൻ സ്ഥിര ഗ്രേഡിയന്റ് “Horizon 1” ആണ്. ഗ്രേഡിയന്റ് ബട്ടൺ ക്ലിക്കുചെയ്യുന്നത് മറ്റേതെങ്കിലും വ്യത്യാസത്തിനു തിരഞ്ഞെടുക്കുക കഴിയുന്ന ലളിതമായ gradient dialog, തുറക്കും.

ഗ്രേഡിയന്റ് റിവേഴ്സ്

സ്വതവേ, തിരഞ്ഞെടുത്ത ഗ്രേഡിയന്റ് മുകളിൽ നിന്നും താഴേക്ക് പ്രയോഗിക്കും. ഈ ഓപ്ഷൻ എപ്പോഴാണ് ചെക്കുചെയ്തിട്ടുണ്ടെന്നും, വിപരീത ചെയ്യും.

16.12.4. ഫിൽറ്റർ വിശദാംശങ്ങൾ

ഫിൽട്ടർ ഇഫക്റ്റുകൾ ചിലരെങ്കിലും ചുരുക്കത്തിൽ വിവരിച്ച ചെയ്യണം: ഫിൽട്ടർ നിങ്ങൾ സ്വയം ഈ ഇഫക്റ്റുകൾ പുനർനിർമ്മാണം എങ്ങനെ, പ്രതിഫലനവും ഈ ആശംസകള് നിഴൽ സൃഷ്ടിക്കുന്നു എങ്ങനെ, അല്ല. വാസ്തവത്തിൽ, മാത്രം ഹാട്രിക് ഉപയോഗിക്കാൻ ഉപകരണം അറിയുന്നത് ...

പ്രതിഫലനം ഉണ്ടാക്കുന്നു

പിന്നെ, ആൽഫാ ഒരു ഗ്രേഡിയന്റ് നിറഞ്ഞു ചെയ്തു കരുതുന്നു:

  1. ഉദാഹരണത്തിന് Copy ആൻഡ് Paste പുതിയ പാളി പ്രദേശത്തെ വേണ്ടി, മിറർ ആഗ്രഹിക്കുന്ന പ്രദേശം ഉൾക്കൊള്ളുന്ന ഒരു പുതിയ ലെയർ സൃഷ്ടിക്കുക.

  2. പ്രതിഫലനം ലുക്ക് കൂടുതൽ സ്വാഭാവിക വരുത്തുന്നതിനായി, പാളി ഇറങ്ങി scale (ഫിൽട്ടർ യഥാർത്ഥ ഉയരം 85% വരെ മാറ്റുന്നു). ഈ ഉദാ ചെയ്യാൻ കഴിയും Scale Layer കമാൻഡ് അല്ലെങ്കിൽ Scale Tool ഉപയോഗിച്ച്.

  3. പിന്നെ ലംബമായി പാളി move അതു ഇടിച്ചു flip.

  4. ഇപ്പോൾ add a layer mask ഒരു gradient (ഉദാഹരണത്തിന് കറുത്ത വൈറ്റ് അല്ലെങ്കിൽ ചാര വേണ്ടി) കൊണ്ട് ലെയർ മാസ്ക് നിറഞ്ഞു, തീർച്ചയായും, apply the layer mask.

നിഴൽ ഉണ്ടാക്കുന്നു

ആൽഫ കറുത്ത കൂടെ അപ്പോൾ Duplicate Layer ആൻഡ് Alpha to Selection വഴി ഉദാഹരണത്തിന് നിറയ്ക്കുക

  1. ഉദാ അകലുകയും, തലം സ്ലാന്റ് Perspective ഉപകരണം ഉപയോഗിച്ച്

  2. , തലം ഒരു Gaussian Blur ബാധകമാണ്.
