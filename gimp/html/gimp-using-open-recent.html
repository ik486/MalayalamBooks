3.3. സമീപകാലത്തെ തുറക്കുക

GIMP ലെ അടുത്തിടെ തുറന്നു ഒരു ഇമേജ് തുറക്കുന്നതിനായി എളുപ്പമാർഗ്ഗം, File → Open Recent ആയിരിക്കും ഉപയോഗിക്കുന്നത്. ഇത് അവരെ പുറമെ ഐക്കണുകൾ ഉപയോഗിച്ച് കൂടുതലും സമീപകാലത്ത് തുറന്ന ചിത്രങ്ങൾ സ്ക്രോൾ ലിസ്റ്റ് പ്രദർശിപ്പിക്കുന്നു. തിരഞ്ഞെടുത്ത് ആവശ്യമുള്ള ഇമേജ് തുറക്കുക.
