5.8. കർവ്സ്

കർവ്സ് ഉപകരണം സജീവ പാളി അല്ലെങ്കിൽ ഒരു നിര നിറം, തെളിച്ചം, തീവ്രത അല്ലെങ്കിൽ സുതാര്യത മാറ്റുന്നതിൽ ഏറ്റവും പരിഷ്കൃതമായ ഉപകരണമാണ്. ലെവലുകൾ ഉപകരണം നിങ്ങളെ ഷാഡോകളിലും ഹൈലൈറ്റുകളിലും ജോലിക്ക് അനുവദിക്കുന്നു അതേസമയം, കർവ്സ് ഉപകരണം നിങ്ങളെ ഏതെങ്കിലും ടോണൽ ശ്രേണി ജോലിക്ക് അനുവദിക്കുന്നു. ഇത് ആർജിബി ചിത്രങ്ങൾ പ്രവർത്തിക്കുന്നു.

  • By clicking on the tool icon in Toolbox, if this tool has been installed there. For this, please refer to വിഭാഗം 1.7, “പണിസഞ്ചി”.

ചാനൽ

There are five options:

വില

നിങ്ങൾ സംയുക്ത ചിത്രം അവരെ കാണാം കർവ് മൂല്യം, പിക്സൽ അതായത് തെളിച്ചം പ്രതിനിധീകരിക്കുന്നു.

റെഡ്; ഗ്രീൻ; ബ്ലൂ

കർവ് മൂന്ന് ആർജിബി ചാനലുകൾ ഓരോ നിറം അളവ് പ്രതിനിധീകരിക്കുന്നു. ഇവിടെ, dark നിറം little എന്നാണ്. Light നിറം a lot എന്നാണ്.

ആൽഫ

കർവ് പിക്സൽ അതാര്യത പ്രതിനിധീകരിക്കുന്നു. Dark very transparent എന്നാണ്. Light very opaque എന്നാണ്. നിങ്ങളുടെ ചിത്രം അല്ലെങ്കിൽ സജീവ പാളി പ്രാപ്തമാക്കും ഈ ഓപ്ഷനായി ആൽഫാ ചാനൽ ഉണ്ടായിരിക്കണം.

Reset Channel

This button deletes all changes made to the selected channel and returns to default values.

ലീനിയർ ആൻഡ് ലോഗരിതമിക് ബട്ടണുകൾ

ഈ ബട്ടണുകൾ ലീനിയർ അല്ലെങ്കിൽ ഹിസ്റ്റോഗ്രാം എന്ന െപരുമാറാതിരിക്കുകൈ തരം തിരഞ്ഞെടുക്കാൻ അനുവദിക്കും. നിങ്ങൾക്ക് ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗിൽ അതേ ഓപ്ഷനുകൾ ഉപയോഗിക്കാം. ഈ ആമി സ്വതവേ പ്രദർശിപ്പിക്കില്ല ചാരനിറത്തിലായിത്തീരുന്നു.

മെയിൻ എഡിറ്റിംഗ് ഏരിയ

  • The horizontal gradient: അതു ഇൻപുട്ട് ടോണൽ സ്കെയിൽ പ്രതിനിധീകരിക്കുന്നു. ഇത് വളരെ നിന്ന് 0 255 (കറുത്ത) (വെള്ള), ഷാഡോസ് നിന്നുള്ള ഹൈലൈറ്റുകൾ വരെയാണ്. നിങ്ങൾ കർവ് ക്രമീകരിക്കാൻ, അത് രണ്ട് ഭാഗങ്ങളായി കയറി വേറ്തിരിക്കുന്നു; മുകളിലെ ഭാഗം തുടർന്ന് പാളി അല്ലെങ്കിൽ തിരഞ്ഞെടുപ്പിന്റെ tonal balance പ്രതിനിധീകരിക്കുന്നു.

  • The vertical gradient: അതു ലക്ഷ്യസ്ഥാനം, ഔട്ട്പുട്ട് ടോണൽ സ്കെയിൽ പ്രതിനിധീകരിക്കുന്നു. ഇത് ഷാഡോസ് നിന്ന് ഹൈലൈറ്റുചെയ്തവയിലേക്ക്, നിന്ന് 0 255 (കറുത്ത) (വെള്ള) വരെയാണ്.

  • The chart: കർവ് ഒരു ഗ്രിഡ് പാത്രമായി ചുവടെ ഇടത് മൂലയിൽ നിന്നും മുകളിൽ വലത് കോണിലുള്ള പോകുന്നു ആണ്. പോയിന്റർ X / Y സ്ഥാനം ഗ്രിഡ് മുകളിൽ ഇടത് ഭാഗത്ത് സ്ഥിരമായി പ്രദർശിപ്പിച്ചിരിക്കുന്നു. സ്വതവേ, ഈ കർവ് ഓരോ ഇൻപുട്ട് തലത്തിൽ ഒരേ ouput ടോൺ യോജിക്കുന്നുവോ കാരണം, നേരായ. ജിമ്പ് സ്വയം ഒരു ആങ്കർ കറുത്ത (0), വൈറ്റ് (255) വേണ്ടി, വഞ്ചിപ്പിക്കുന്നവയിൽ രണ്ടറ്റവും മതിയാവില്ല.

    നിങ്ങൾ കർവ് ക്ലിക്ക്, ഒരു പുതിയ anchor സൃഷ്ടിക്കപ്പെടുന്നു. മൗസ് സൂചികയെ ഒരു നങ്കൂരം അടിക്കുമ്പോൾ, അത് ഒരു ചെറിയ കൈ രൂപത്തിൽ. നിങ്ങൾ ക്ലിക്ക്-ആൻഡ്-വലിച്ചിടാനാകും കർവ് പേറി ആങ്കർ. നിങ്ങൾ കർവ് പുറത്ത് ക്ലിക്ക് എങ്കിൽ ഒരു ആങ്കർ പുറമേ സൃഷ്ടിച്ചത് കര്വ് സ്വയം അത് ഉൾപ്പെടുന്നു.

    Unactive നങ്കൂരം കറുത്തും ഇരിക്കുന്നു. സജീവ ആങ്കർ വെളുത്ത. താങ്കൾക്ക് ക്ലിക്കുചെയ്ത് ഒരു നങ്കൂരം പ്രവർത്തനസജ്ജമാകും. നിങ്ങളുടെ കീബോർഡ് ഇടതും വലതും ആരോ കീകൾ ഉപയോഗിച്ച് ആങ്കർ സജീവമാക്കൽ സ്വാപ്പ് കഴിയും. നിങ്ങൾ Up, Down ആരോ കീകൾ ഉപയോഗിച്ച് ലംബമായി ആങ്കർ നീക്കാൻ കഴിയൂ. ഇത് നിങ്ങൾ ട്യൂൺ ആങ്കർ സ്ഥാനം പിഴ അനുവദിക്കുന്നു. Shift താഴേയ്ക്കാക്കുമ്പോൾ നിങ്ങൾ 15 പിക്സൽ ഇന്ക്രിമെന്റ് അതിനെ നീക്കാൻ അനുവദിക്കുന്നു.

    രണ്ട് നങ്കൂരം ഏത് പാളി ഒരു ടോണൽ റേഞ്ച് സൂചിപ്പിയ്ക്കുന്ന ഒരു curve segment റോഡിലാണ്. നിങ്ങൾ ക്ലിക്ക്-ആൻഡ്-വലിച്ചു കഴിയും ഈ സെഗ്മെന്റ് (ഇത് ഒരു പുതിയ ആങ്കർ തയ്യാറാക്കുന്നു). തീർച്ചയായും, അവസാനം അവതാരകർ അപ്പുറം ഡ്രാഗ് കഴിയില്ല.

    എല്ലാ നങ്കൂരം (പുറമെ രണ്ടറ്റവും നിന്ന്) ഇല്ലാതാക്കണമെന്ന്, Reset Channel ബട്ടൺ ക്ലിക്ക് ചെയ്യുക. ഒരേയൊരു ആങ്കർ ഇല്ലാതാക്കാൻ, തിരശ്ചീന അക്ഷത്തിൽ സമീപ ആങ്കർ പോകാനുള്ള.

    അതേസമയം, ക്യാൻവാസിൽ, മൌസ് പോയിന്റർ കണ്ണ്-dropper രൂപത്തിൽ ഉണ്ട്. നിങ്ങൾ ഒരു പിക്സൽ ക്ലിക്ക് എങ്കിൽ, ഒരു ലംബമായ രേഖ ചാർട്ട്, തിരഞ്ഞെടുത്ത ചാനലിനെ ഈ പിക്സൽ ഉറവിടം മൂല്യത്തിന് പരിസരത്തിൽ ദൃശ്യമാകുന്നു. നിങ്ങൾ -click Shift എങ്കിൽ നിങ്ങൾ തിരഞ്ഞെടുത്ത ചാനൽ ഒരു ആങ്കർ സൃഷ്ടിക്കുക. നിങ്ങൾ -click Ctrl, നിങ്ങൾക്ക് ഒരു നങ്കൂരം എല്ലാ ചാനലുകളിലും ഒരുപക്ഷേ ആൽഫാ ചാനൽ ഉൾപ്പെടെ സൃഷ്ടിക്കുക. നിങ്ങൾക്ക് Shift ചെയ്യാനാകും - വലിച്ചിടുക Ctrl -drag: ഈ ലംബരേഖ നീക്കും, മൗസ് അവശേഷിക്കുന്നു ബട്ടൺ പുറത്തുവിടുകയും സമയത്ത് ആങ്കർ കാണിക്കും.

    തിരഞ്ഞെടുത്ത ചാനലിനെ സജീവ അല്ലെങ്കിൽ പാളി നിരക്കു ഹിസ്റ്റോഗ്രാം ചാർട്ടിൽ ഗ്രേ ഔട്ട് പ്രതിനിധീകരിക്കുന്നു. വെറും ഒരു റഫറൻസ് തുടർന്ന്.

Curve type
സുഗമമായ

ഇത് ഡീഫോൾട്ട് മോഡ്. ഇക്കൊല്ലം അത് മിനുസമുള്ള ഒരു വരിയിൽ കർവ് തരം നിർബ്ബന്ധിക്കുന്നു. ഇത് ഇനിപ്പറയുന്ന കൂടുതൽ റിയലിസ്റ്റിക് റെൻഡർ നൽകുന്നു.

Free Hand

ഈ മോഡിൽ ഉപയോഗിച്ച്, നിങ്ങൾ വീണ്ടും Curve Type ബട്ടൺ ഉപയോഗിച്ച്് സ്മൂത് കഴിയുന്ന തകർന്ന ലൈൻ വരയ്ക്കാൻ കഴിയും.

പ്രിവ്യൂ ബട്ടൺ പുതിയ തലത്തിലേക്ക് ക്രമീകരണങ്ങൾ ഉടനെ കാണാൻ കഴിയുന്ന ചലനാത്മകമായി അങ്ങനെ തലങ്ങളിൽ എല്ലാ മാറ്റങ്ങളും ചെയ്യുന്നു.

Tool Options dialog

സാമ്പിൾ ശരാശരി

ഈ സ്ലൈഡർ നിറം-സാഹസമാണ് പ്രദേശത്തെ “radius” സജ്ജമാക്കുന്നു. ഈ പ്രദേശം നിങ്ങൾ ഒരു പിക്സൽ ന് ക്ലിക്കിൽ നിലനിർത്താൻ ഒരു കൂടുതലോ കുറവോ വിശാലമായിരിക്കുന്നു ചതുരശ്ര ദൃശ്യമാകുന്നു. ഇവിടെ, കണ്ണ്-dropper ഒരു പിക്സൽ കണ്ടെത്താൻ ഉപയോഗിക്കുന്നു: ആരം = 1 മികച്ച തോന്നുന്നു.

5.8.3. Using the “Curves” tool

5.8.3.1. ചുരുക്കം അടിസ്ഥാന രൂപങ്ങൾ

നാം കർവ് ന് ആൻക്കറുകളും സെഗ്മെന്റുകൾ സൃഷ്ടിക്കുക, ഞങ്ങൾ കർവ് രൂപീകരണത്തിലും നീക്കുക. ഈ കർവ് “output” ടൺ ലേക്ക് സജീവ അല്ലെങ്കിൽ പാളി തിരഞ്ഞെടുപ്പിന്റെ “input” ടൺ മാപ്പുചെയ്യുന്നു.

എങ്ങനെ കർവ്സ് ഉപകരണം പ്രവൃത്തികൾ

ഇടത് താഴ്ന്ന അവസാന പോയിന്റ് വലതുവശത്ത് ലേക്ക് മുകളിലെ അവസാന പോയിന്റ് നീക്കുന്നത് തോത് ഉപകരണം ഇടത് കറുത്ത സ്ലൈഡർ വലതുവശത്ത് വൈറ്റ് സ്ലൈഡർ ചലിക്കുന്ന തന്നെയാണ് മൂല്യം വൈറ്റ് പോയിൻറ് കൂടുതൽ ആണ് എല്ലാ പിക്സൽ ( കർവ് പരന്ന ഭാഗം) തിരഞ്ഞെടുത്ത ചാനലിനെ പ്രകാരം നിറമുള്ള / കൂടുതൽ ഇരുണ്ട) വെളുത്ത (കൂടുതൽ നിർമ്മിച്ചിരിക്കുന്നത്. ആരുടെ മൂല്യം കറുത്ത പോയിന്റ് (താഴ്ന്ന നിവർത്തിയ കർവ്) കുറവാണ് എല്ലാ പിക്സലുകൾ കറുത്ത (കറുത്ത / പൂർണ്ണമായും സുതാര്യമായ) നിർമ്മിച്ചിരിക്കുന്നത്. ഉയർച്ചയുണ്ടായി എന്നു വഞ്ചിപ്പിക്കുന്നവയിൽ പോയിന്റ് പൊരുത്തപ്പെടുന്ന പിക്സൽ ലൈറ്റർ നിർമ്മിച്ചിരിക്കുന്നത്. താഴേക്ക് നീക്കി എന്നു വഞ്ചിപ്പിക്കുന്നവയിൽ പോയിന്റ് പൊരുത്തപ്പെടുന്ന പിക്സൽ ഇരുണ്ട (പച്ച അമ്പു) നിർമ്മിച്ചിരിക്കുന്നത്. ഇത്തരം എല്ലാ പിക്സൽ മുഴുവൻ ഔട്ട്പുട്ട് ടോണൽ ശ്രേണി വ്യാപിപ്പിക്കും.

ആമി കറുത്ത (0) നിന്ന് വൈറ്റ് (255) ലേക്ക്, മൂല്യങ്ങളുടെ വിപുലീകരണം കാണിക്കുന്നു: വിപരീതമായി വർദ്ധിപ്പിക്കുന്നു. മൂല്യം ചാനൽ തിരഞ്ഞെടുത്ത എന്നതിനാല്, മാറ്റങ്ങൾ എല്ലാ നിറം ചാനലുകളും നിറങ്ങൾ വർധന ബാധിക്കും.
