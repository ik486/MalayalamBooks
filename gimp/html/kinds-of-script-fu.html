2.4. സ്ക്രിപ്റ്റ്-Fus വിവിധ തരത്തിലുള്ള

സ്ക്രിപ്റ്റ്-Fus രണ്ട് തരം ഉണ്ട്:

ഹ്രസ്വമായ സ്ക്രിപ്റ്റ്-Fus

നിങ്ങൾ ഇമേജ് മെനുബാർ ലെ File → Create → Type of Script കീഴിൽ ഒറ്റയ്ക്കുള്ള വകഭേദങ്ങളും കണ്ടെത്തും (ചിത്രം കാണുക).

ചിത്രം 13.1. വിഭാഗം അനുസരിച്ച് സ്ക്രിപ്റ്റ്-Fus


ചിത്രം-ആശ്രിത സ്ക്രിപ്റ്റ്-Fus

മെനുകൾ പുനഃസംഘടിപ്പിച്ചു ചെയ്തു. ഒരു പുതിയ നിറങ്ങൾ-മെനു ലഭ്യമാകുന്നു. ഒരുമിച്ച് ഗ്രൂപ്പുകൾ ചെഞ്ചായം ക്രമീകരിക്കാൻ ആ മാതൃക ഉപകരണങ്ങൾ, സാച്ചുറേഷൻ, ചാപല്യം ..., ഫിൽട്ടറുകൾ ... മുതലായവയ്ക്ക്, നിറങ്ങൾ പ്രവർത്തിക്കുന്ന എല്ലാ സ്ക്രിപ്റ്റുകൾ. അരിപ്പകൾ-മെനു സ്ക്രിപ്റ്റ്-ഫു-മെനു ഒറ്റ അരിപ്പകൾ-മെനുവിൽ ലയിക്കുന്നു ഒപ്പം അത് പുതിയ വിഭാഗങ്ങളിൽ പ്രകാരം സംഘടിപ്പിക്കുന്നത്. ചിത്രം-ആശ്രിത പ്ലഗിന്നുകളായും സ്ക്രിപ്റ്റ്-Fus ഇപ്പോൾ image- മെനുകളിൽ പകര്ന്ന് ചെയ്യുന്നു. ഉദാഹരണത്തിന്, ആൽഫ അരിപ്പ നിറം നിറങ്ങൾ-മെനുവിൽ ആണ്. തുടക്കത്തിൽ, അതു disconcerting, എന്നാൽ നിങ്ങൾ അതു കൂടുതൽ ലോജിക്കൽ എന്നതിനാൽ ഈ ഉപയോഗിക്കും പൂർത്തിയാക്കുന്നതിന്.

താഴെക്കാണുന്ന കണക്കുകൾ ഇമേജ്-അവയെ മെനുവിൽ കണ്ടെത്താം.
