6.10. ഗില്ലറ്റിൻ

Guillotine കമാൻഡ് കഷണങ്ങൾ അപ്പ് നിലവിലെ ചിത്രം, ചിത്രം ഗൈഡുകൾ അടിസ്ഥാനമാക്കി. ഓരോ ഗൈഡ് സഹിതം ചിത്രം ഒരു ഗില്ലറ്റിൻ (പേപ്പർ മുറിക്കുന്ന) ഉപയോഗിച്ചുള്ള ഒരു ഓഫീസിൽ പ്രമാണങ്ങൾ മുറിക്കൽ സമാനമായ മുറിവു കഷണങ്ങളായി നിന്നു പുതിയ ഇമേജുകൾ ഉണ്ടാക്കുന്നു. ഗൈഡുകൾ കൂടുതൽ വിവരങ്ങൾക്ക്, വിഭാഗം 2.2, “വഴികാട്ടി” കാണുക.

6.10.1. Activate the Command

  • You can access this command from the image menubar through Image → Transform → Guillotine.
