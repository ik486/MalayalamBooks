3.5. ഫയൽ മാനേജർ

നിങ്ങൾ GIMP ഒരു ഇമേജ് ഫയൽ തരം ബന്ധപ്പെട്ട എങ്കിൽ, ഒന്നുകിൽ നിങ്ങൾ GIMP അല്ലെങ്കിൽ പിന്നീട് ഇൻസ്റ്റാൾ ചെയ്താൽ നിങ്ങൾ ഫയൽ ഒരു ഫയൽ മാനേജർ ഉപയോഗിച്ച് (ലിനക്സ് ൽ നോട്ടിലസോ കോൺക്വെറർ, അല്ലെങ്കിൽ വിൻഡോകൾ വിൻഡോസ് എക്സ്പ്ലോറർ പോലെ) നാവിഗേറ്റുചെയ്യാം, നിങ്ങൾക്ക് ഒരിക്കൽ ഫയൽ ക്ലിക്ക് double-, അത് കണ്ടെത്തി. ശരിയായി കോൺഫിഗർ, ചിത്രം GIMP തുറക്കും.
