14.2. ImageMap

വെബ് സെൻസിറ്റീവ് ചിത്രങ്ങൾ പതിവായി നിർവചിക്കപ്പെട്ട പ്രദേശങ്ങൾ മിനാരങ്ങൾ പ്രാപ്തമാക്കി ചില ഇഫക്റ്റുകൾ നേടുകയും ഉപയോഗിക്കുന്നു. വ്യക്തമായും ഏറ്റവും ഉപയോഗിച്ച പ്രഭാവം സെൻസിറ്റീവ് പ്രദേശങ്ങൾ ഒന്ന് ക്ലിക്കുചെയ്യപ്പെടുമ്പോൾ മറ്റൊരു വെബ് പേജ് ഒരു ഡൈനമിക് ലിങ്ക് ആണ്. ഈ “filter” ഒരു ചിത്രം എളുപ്പത്തിൽ സെൻസിറ്റീവ് പ്രദേശങ്ങൾ രൂപകൽപ്പന അനുവദിക്കുന്നു. വെബ് സൈറ്റ് ഡിസൈൻ സോഫ്റ്റ്വെയറുകൾ ഒരു സാധാരണ ചടങ്ങിൽ ഈ ഞങ്ങൾക്കുണ്ട്. GIMP നിങ്ങൾ സമാന രീതിയിൽ ചെയ്യാൻ കഴിയും.

14.2.1. പൊതു അവലോകനം

ഈ പ്ലഗ്-ഇൻ നിങ്ങളെ ഗ്രാഫിക് രൂപകൽപ്പനയും ഫ്രണ്ട്ലി നിങ്ങളുടെ പ്രദർശിപ്പിച്ച ചിത്രം മേൽ delimit ആഗ്രഹിക്കുന്ന എല്ലാ പ്രദേശങ്ങൾ അനുവദിക്കുന്നു. നിങ്ങളുടെ പേജ് HTML കോഡ് ന്യായമോ സ്ഥലത്തു ലയിപ്പിക്കും ചെയ്തിരിയ്ക്കണം HTML ടാഗുകൾ പ്രസക്തമായ ഭാഗം നേടുക. നിങ്ങൾ വളരെ ഈ പ്രദേശങ്ങളിൽ ലിങ്കുചെയ്തിരിക്കുന്ന ചില പ്രവർത്തനങ്ങൾ നിർവചിക്കാനാകും.

ഈ പൂർണ്ണമായും ഇവിടെ വിവരിച്ച ചെയ്യപ്പെടാത്ത (ഈ ഫംഗ്ഷൻ വാഗ്ദാനം വെബ് പേജ് പോലെ ഏകദേശം പ്രവർത്തിക്കുന്നു) ഒരു സങ്കീർണ്ണമായ ഉപകരണമാണ്. എന്നിരുന്നാലും നാം ഇവിടെ ഏറ്റവും നിലവിലെ handlings ചില വിശദീകരിക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ. നിങ്ങൾ ആഗ്രഹിക്കുന്നുണ്ടെങ്കിൽ, എന്ന [GROKKING02] കൂടെ ജിമില Grokking ഒരു പൂര്ണ്ണ വിവരണങ്ങൾ കണ്ടെത്താം.

14.2.2. ഫിൽട്ടർ സജീവമാക്കുക

ഒരു ഇമേജ് വിൻഡോയിൽ നിന്നും, നിങ്ങൾ Filters → Web → ImageMap മുഖേന ഈ ഫിൽട്ടർ കണ്ടെത്താനാകും

വിൻഡോ ഒരു ചെറിയ ഒന്നാണ്, നിങ്ങൾ അതു മഹത്വപ്പെടുത്തുകയും കഴിയും. പ്രധാന ഉപയോഗപ്രദമായ പ്രദേശങ്ങൾ ഇവയാണ്:

  • പൂർണ്ണമായും ഇടതുവശത്ത് ലംബമായി ഒറ്റ മേഖല ഉള്ള എഡിറ്റുചെയ്യാൻ, ഒടുവിൽ ഒരുത്തൻ ഒരു തിരഞ്ഞെടുത്ത മേഖല മായ്ക്കാൻ, വിവിധ ആകാരം പ്രദേശങ്ങൾ ജനറേറ്റ് ഉപകരണങ്ങൾ വിളിച്ചതിന് പ്രദർശിപ്പിച്ച ഐക്കണുകൾ, ചൂണ്ടുന്ന ഒന്നു മൂന്ന്; നിങ്ങൾ മാപ്പിംഗ് മെനു ഉപയോഗിച്ച് ഈ ഫങ്ഷനുകളെ വിളിക്കാൻ കഴിയും,

  • ശരിയായ നിങ്ങളെ പ്രസക്തമായ ഉപകരണങ്ങൾ ഉപയോഗിച്ച് നിങ്ങൾക്ക് ആവശ്യമുള്ള എല്ലാ രൂപങ്ങൾ പ്രദേശങ്ങൾ ആകർഷിക്കും കഴിയുന്ന തൊഴിലാളി മേഖലയാണ്,

  • വലതുഭാഗത്ത് ഒരു ഐക്കൺ ലംബമായ സെറ്റ് പ്രദർശിപ്പിച്ചിരിക്കുന്നു; അതിന്റെ ഉപയോഗം വ്യക്തം എന്നാൽ ഒരു സഹായവും പോപ്പ്-അപ്പ് നിങ്ങൾ ഓരോ ചടങ്ങിൽ കുറിച്ചുള്ള ചില വിവരങ്ങൾ നൽകുന്നു,

  • ഒടുവിൽ, പോലും വലതുഭാഗത്ത് സൃഷ്ടിച്ച പ്രദേശങ്ങൾ ഒരു പ്രോപ്പർട്ടി ലിസ്റ്റ് പോലെ ഒരു ഡിസ്പ്ലേ മേഖലയാണ്. ലിസ്റ്റ് ഒന്ന് ഇനത്തിന്റെ ഒരു ക്ലിക്കിൽ, തൊഴിലാളി പ്രദേശത്ത് ഇതേ ആകാരം സ്വയം തെരഞ്ഞെടുക്കുന്നു

മാപ്പിംഗ്

നിങ്ങൾ എഴുതാറില്ല കൂടുതൽ എളുപ്പത്തിൽ തൊഴിലാളി പ്രദേശത്തെ ഇടതുഭാഗത്തായി ഐക്കണുകൾ എന്നിവയിൽ ക്ലിക്കുചെയ്ത് നിരക്കു ഉപകരണങ്ങൾ ആക്സസ് ചെയ്യാൻ കഴിയും ശേഷം, ഈ മെനു ഉപയോഗിക്കും.

അമ്പടയാളം

ഇവിടെ അമ്പ് നീക്കുക ഉപകരണം പ്രതിനിധീകരിക്കുന്നു. സജീവമാക്കി ഉപകരണം തിരഞ്ഞെടുത്തു, നിങ്ങൾക്ക് തിരഞ്ഞെടുത്ത് നീക്കാനാകും ചിത്രം ഒരു പ്രദേശം.

ഒരു പോളിഗണിലെ ഉപയോഗിച്ച്, ചുവന്ന പോയിന്റ് ഒന്ന് നീക്കാൻ ആരോ ഉപയോഗിക്കാൻ കഴിയും. നിരവധി കൂടെ വേറെ വാഗ്ദാനം ചെയ്യുന്ന പുതിയ പോയിന്റ് ചേർക്കാൻ സാധ്യത ഒരു പോപ്പ്-അപ്പ് മെനു തുറക്കുന്നതിന് രണ്ടു ചുവന്ന പോയിന്റ് തമ്മിലുള്ള ഒരു സെഗ്മെന്റ് ക്ലിക്ക് കൈവാക്കിന്റെ. നിങ്ങൾ ഒരു ചുവന്ന പോയിന്റ് റൈറ്റ് ക്ലിക്ക്, നിങ്ങൾക്ക് അത് നീക്കം.

ചതുരം; സർക്കിൾ; പോളിഗൺ

ഈ ഉപകരണങ്ങൾ നിങ്ങളെ വിവിധ ആകാരം പ്രദേശങ്ങൾ സൃഷ്ടിക്കാനും:, ചിത്രം ക്ലിക്ക് പോയിന്റർ നീക്കുക, വീണ്ടും ക്ലിക്കുചെയ്യുക.

Instead of creating geometrical shapes to select the active areas you may use an array of rectangles, each representing an active area, by clicking on the “Create guides”. In the menu popping up you set the width and height of the rectangles, the space between them, the number of rows and columns, and the upper and left startpoint for the array. All measures are in pixels. If you are not satisfied with the result you may adjust each rectangle by moving the red squares as usual.

14.2.3.2. ടൂൾ ബാർ

ഇവിടെ ഏറ്റവും എൻട്രികൾ ഇതിനകം വിവരിച്ച ചില പ്രവർത്തനങ്ങൾക്കായി വെറും കുറുക്കുവഴികൾ. ഒഴിവാക്കലുകൾ:

മുന്നിലേക്ക് നീക്കുക; തിരികെ അയയ്ക്കുക

ഇവിടെ നിങ്ങൾ ഏരിയ ലിസ്റ്റ് ചുവടെ (“Move to Front”) അല്ലെങ്കിൽ മുകളിൽ ഒരു ഏരിയ എൻട്രി (“Send to Back”) നീക്കാൻ കഴിയും.

14.2.3.3. ജോലി ഏരിയ

ചിത്രം 17.348. ജോലി ഏരിയ


imagemap ജാലകത്തിന്റെ പ്രധാന പരിസരത്ത് ഇടതു വശത്ത്, നിങ്ങൾ പ്രസക്തമായ ഉപകരണങ്ങൾ ഉപയോഗിച്ച് നിങ്ങൾക്ക് ആവശ്യമുള്ള എല്ലാ രൂപങ്ങൾ പ്രദേശങ്ങൾ ആകർഷിക്കും കഴിയുന്ന അധ്വാനിക്കുന്ന പ്രദേശം കണ്ടെത്തും.

തൊഴിലാളി പ്രദേശത്ത് പുറമെ ലംബമായി പ്രദർശിപ്പിച്ച ഐക്കണുകൾ, ചൂണ്ടുന്ന ഒന്നു വിവിധ ആകാരം പ്രദേശങ്ങൾ ജനറേറ്റ് ഉപകരണങ്ങൾ വിളിക്കുന്നു മൂന്നു ഒറ്റ മേഖല ഉള്ള എഡിറ്റുചെയ്യാൻ, ഒടുവിൽ ഒരുത്തൻ ഒരു തിരഞ്ഞെടുത്ത മേഖല മായ്ക്കാൻ ഉണ്ട്; നിങ്ങൾ വളരെ മാപ്പിംഗ് മെനു ഉപയോഗിച്ച് ഈ ഫങ്ഷനുകളെ വിളിക്കാം.

ശ്രദ്ധിക്കുക

പ്രദേശങ്ങൾ ഓവർലാപ് കാര്യം ശ്രദ്ധിക്കുക.
14.2.3.4. നിരക്കു ഏരിയ

വലത് സൃഷ്ടിച്ച പ്രദേശങ്ങൾ ഒരു പ്രോപ്പർട്ടി ലിസ്റ്റ് പോലെ ഒരു ഡിസ്പ്ലേ മേഖലയാണ്. ലിസ്റ്റ് ഒന്ന് ഇനത്തിന്റെ ഒരു ക്ലിക്കിൽ സ്വയം തൊഴിലാളി പ്രദേശത്ത് ഇതേ ആകൃതിയും തിരഞ്ഞെടുക്കുക, പിന്നെ നിങ്ങൾ വരുത്താനും സാധിക്കുന്നതാണ്.

Beside the display is an icon vertical set; its use is obvious but a help pop-up gives you some information about each function.

നിർഭാഗ്യവശാൽ, മുകളിലേക്കും താഴേക്കും ലിസ്റ്റ് എൻട്രി നീക്കുന്നതിനായി അമ്പ് ചിഹ്നങ്ങൾ ഇവിടെ പ്രവർത്തിക്കുന്നില്ല. പക്ഷേ തീർച്ചയായും നിങ്ങൾ ശ്രദ്ധാപൂർവ്വം കവിഞ്ഞു പ്രദേശങ്ങൾ സൃഷ്ടിക്കാൻ ഒഴിവാക്കണം, അതിനാൽ നിങ്ങൾ ഒട്ടും ഈ ഫംഗ്ഷനുകളും ഉപയോഗിക്കരുത്.
