6.23. New Guide

New Guide കമാൻഡ് ചിത്രം ഒരു ഗൈഡ് ചേർക്കുന്നു.

സൂചന

നിങ്ങൾക്ക് വഴി ചിത്രത്തിലേക്ക് കേവലം ഇമേജ് ഭരണാധികാരികൾ നിന്ന് വഴികാട്ടികളേ ക്ലിക്കുചെയ്ത് ചെയ്തോ നിങ്ങൾക്ക് ആഗ്രഹിക്കുന്ന അവരെ പൊസിഷനിങ് വഴി, കൂടുതൽ വേഗത്തിൽ ചേർക്കാൻ കഴിയും, പക്ഷേ കുറവ് കൃത്യമായി.

6.23.1. Activate the Command

You can access this command from the image menubar through Image → Guides → New Guide

6.23.2. “New Guide” Options

നിങ്ങൾ New Guide തിരഞ്ഞെടുക്കുക, ഒരു ഡയലോഗ് തുറക്കുന്നു, നിങ്ങൾ Direction ആൻഡ് Position, പിക്സലിൽ, പുതിയ ഗൈഡ് കൂടുതൽ കൃത്യമായി click- ആൻഡ്-വലിച്ചു ഉപയോഗിച്ച് അധികം സജ്ജമാക്കാൻ അനുവദിക്കുന്ന.

ചിത്രം 16.82. The “New Guide” Dialog


സംവിധാനം

ഡ്രോപ്-ഡൌണ് ലിസ്റ്റ് ഉപയോഗിച്ച്, ഗൈഡിന്റെ Direction തിരഞ്ഞെടുക്കാം Horizontal അല്ലെങ്കിൽ Vertical ഒന്നുകിൽ.

സ്ഥാനം

Position ഉറവിട കോർഡിനേറ്റ് ക്യാൻവാസ് മുകളിലെ ഇടത് കോണിൽ ആണ്.
