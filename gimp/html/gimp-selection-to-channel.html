4.17. Save to Channel

Save to Channel കമാൻഡ് ഒരു ചാനലിലേക്ക് ആയി നിരക്കു സംരക്ഷിക്കുന്നു. ചാനൽ തുടർന്ന് ഒരു ചാനൽ തിരഞ്ഞെടുപ്പ് മാസ്ക് ഉപയോഗിക്കാൻ കഴിയും. നിങ്ങൾ Channel Dialog വിഭാഗത്തിൽ കണ്ടെത്താനാകും അവരെ കൂടുതൽ വിവരങ്ങൾ.

എത്ര വിഭാഗം 16, “ലോഗോ ഫിൽട്ടറുകൾ ആൽഫ” ആമുഖത്തിൽ ഈ കമാൻഡ് ഉപയോഗിക്കാൻ ഒരു ഉദാഹരണം കണ്ടെത്തും. ഇത് അങ്ങനെ ഈ തിരഞ്ഞെടുത്തത് ലോഗോ അരിപ്പ ഒരു ആൽഫ ഉപയോഗപ്പെടുത്തുക ഒരു ആല്ഫാ ചാനലിലേക്ക് ഒരു നിര പരിവർത്തനം കാണിക്കുന്നു.

4.17.1. Activate the Command

  • You can access this command from the image menubar through Select → Save to Channel.

  • നിങ്ങൾക്ക് Selection Editor നിന്ന് അത് ആക്സസ് ചെയ്യാൻ കഴിയും.
