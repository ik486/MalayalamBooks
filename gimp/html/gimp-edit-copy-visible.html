3.8. ദൃശ്യം പകർത്തുക

Copy Visible കമാൻഡ് Copy കമാൻഡ് സമാനമാണ്. എന്നിരുന്നാലും, നിലവിലെ പാളി ഉളളടക്കം പകർത്താനും ഇല്ല; അതു കാണാവുന്ന പാളികൾ (അല്ലെങ്കിൽ ദൃശ്യമായ പാളികൾ നിരക്കു), അതായത് ഒരു “eye” അടയാളപ്പെടുത്തിയിട്ടുള്ള ഞാൻ കാണിച്ചുതരാം പകർപ്പുകൾ ഉള്ളടക്കങ്ങൾ.

കുറിപ്പു്

ഇമേജ് ഡാറ്റ ക്ലിപ്ബോർഡ് ഇട്ടു ചെയ്യുമ്പോൾ പാളികൾ കുറിച്ചുള്ള വിവരങ്ങൾ നഷ്ടമാകുന്നു കാര്യം ശ്രദ്ധിക്കുക. നിങ്ങൾ പിന്നീട് ക്ലിപ്ബോർഡിലെ ഉള്ളടക്കം ഒട്ടിക്കുക ചെയ്യുമ്പോൾ മാത്രമേ പാളി, സകല അടയാളപ്പെടുത്തിയ പാളികൾ കൂടിച്ചേരൽ ആണ് അവിടെ.

3.8.1. കമാൻഡ് സജീവമാക്കുന്നു

  • നിങ്ങൾ ഇമേജ് മെനുബാർ നിന്ന് Edit → Copy Visible മുഖേന ഈ കമാൻഡ് ആക്സസ് ചെയ്യാൻ കഴിയും.
