12.3. Context Help

Context Help കമാൻഡ് മൗസ് പോയിന്റർ സന്ദർഭ-സംവേദി ചെയ്യുന്നു ഒരു “?” അതിന്റെ ആകൃതി മാറ്റുന്നു. നിങ്ങൾക്ക് ഒരു ജാലകം ക്ലിക്ക് അതു ലഭ്യമാണ് എങ്കിൽ ഡയലോഗ് മെനു എൻട്രി ആൻഡ് GIMP ദൃശ്യങ്ങളിലും അതിനെക്കുറിച്ച് സഹായിക്കും. നിങ്ങൾക്ക് സമയത്ത് മൗസ് പോയിന്റർ നിങ്ങളെ പറ്റി സഹായം ആഗ്രഹിക്കുന്ന വസ്തു മീതെ F1 കീ അമർത്തി ഏത് സമയത്തും സന്ദർഭ സഹായവും ആക്സസ് ചെയ്യാൻ കഴിയും.

12.3.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menu through Help → Context Help

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Shift+F1 ഉപയോഗിച്ച്.
