2.6. വർണ്ണ മാർഗം തിരഞ്ഞെടുക്കുക

ചിത്രം 14.24. Select by Color tool icon in the Toolbox


കളർ ടൂൾ അനുസരിച്ച് തിരഞ്ഞെടുക്കുക നിറം സമാനത അടിസ്ഥാനമാക്കി ഒരു ചിത്രത്തിന്റെ പ്രദേശങ്ങൾ തിരഞ്ഞെടുക്കുക രൂപകല്പന. ഇത് അവ്യക്തമായ തിരഞ്ഞെടുക്കുക ഉപകരണം (“Magic Wand”) പോലെ ഒരു പ്രവർത്തിക്കുന്നു. അവരെ തമ്മിലുള്ള പ്രധാന വ്യത്യാസം മാന്ത്രികനായാണ് വലിയ വിടവുകൾ അടങ്ങുന്ന പാതകൾ വഴി ആരംഭ പോയിന്റിൽ കണക്റ്റുചെയ്തിരിക്കുന്ന എല്ലാ ഭാഗങ്ങൾ contiguous പ്രദേശങ്ങളും തെരഞ്ഞെടുക്കുന്നു എന്നതാണ്; കളർ ടൂൾ അനുസരിച്ച് തിരഞ്ഞെടുക്കുക സമയത്ത് പരിഗണിക്കാതെ അവർ എവിടെയാണെന്ന്, നിങ്ങൾക്ക് ക്ലിക്ക് പിക്സൽ ലേക്കുള്ള നിറം ന്യൂറൽ സമാനമായ എല്ലാ പിക്സലുകൾ തെരഞ്ഞെടുക്കുന്നു.

2.6.1. Activating the tool

You can access the Select by Color Tool in different ways:

  • From the image menu bar Tools → Selection Tools → By Color Select,

  • by clicking on the tool icon in the ToolBox,

  • കീബോർഡ് കുറുക്കുവഴി Shift +O ഉപയോഗിച്ച്.

2.6.2. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

നിറം പ്രയോഗം തിരഞ്ഞെടുത്ത ഏതെങ്കിലും പ്രത്യേക കീ മോഡിഫയറുകൾ, അതേ വിധത്തിൽ എല്ലാ നിരക്കു ഉപകരണങ്ങൾ ബാധിക്കുന്ന മാത്രമല്ല ഇല്ല. ഈ കൂടെ സഹായത്തിനായി Selection Tools കാണുക.

2.6.3. കൈകാര്യം ഉപകരണം

ചിത്രം 14.25.  Using Select by Color tool: selected pixels are not only contiguous


ഫസി ഉപകരണം ഉപയോഗിച്ച് നിലയിൽ നിരക്കു ഉടൻ ക്ലിക്ക് ആരംഭിച്ച് റഫറൻസ് ആദ്യ ക്ലിക്കുചെയ്ത പിക്സലാണെന്നും. നിങ്ങൾ ക്ലിക്കുചെയ്ത് വലിച്ചിടുക, നിങ്ങൾക്ക് ഫസി ഉപകരണം ഉപയോഗിച്ച് അതേ വഴിയിൽ ദൂരമാണു മാറ്റാവുന്നതാണ്.

നിരക്കു നീക്കാൻ Moving selections കാണുക.

2.6.4. ഓപ്ഷനുകൾ

ചിത്രം 14.26. Tool Options for the Select by Color tool


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

കുറിപ്പു്

ഈ എല്ലാ ഉപകരണങ്ങൾ സാധാരണമാണ് ഉപാധികൾക്ക് സഹായത്തിന് Selection Tools കാണുക. ഈ ഉപകരണം മാത്രമായുള്ള ഓപ്ഷനുകൾ മാത്രം ഇവിടെ വിശദീകരിച്ചു ചെയ്യുന്നു.
