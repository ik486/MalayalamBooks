5.6. ഉമ്മറം

ത്രെഷോൾഡ് പ്രയോഗം നിലവിലെ പാളി വെള്ള പിക്സലുകൾ മൂല്യം ദൂരമാണു റേഞ്ചും, കറുത്ത പിക്സൽ ഉമ്മരപ്പടി മൂല്യം പരിധിക്കുപുറത്താണ് കൂടെ പിക്സൽ പ്രതിനിധാനം ചിത്രത്തിന്റെ പിക്സലുകൾ പ്രതിനിധാനം എവിടെ ഒരു ബ്ലാക്ക് ആൻഡ് വൈറ്റ് ചിത്രം, കയറി നിരക്കു അവസ്ഥയാണ്.

നിങ്ങൾ ഒരു ബ്ലാക്ക് ആൻഡ് വൈറ്റ് ചിത്രം (ഉദാഹരണത്തിന് ഒരു സ്കാൻ വാചകം) മെച്ചപ്പെടുത്തുന്നതിനായി അല്ലെങ്കിൽ നിരക്കു മാസ്കുകൾ സൃഷ്ടിക്കാൻ ഇത് ഉപയോഗിക്കാൻ കഴിയും.

കുറിപ്പു്

ഈ ഉപകരണം ഒരു ബ്ലാക്ക് ആൻഡ് വൈറ്റ് ചിത്രം സൃഷ്ടിക്കുന്നു പോലെ, യഥാർത്ഥ ചിത്രത്തിന്റെ വിരുദ്ധ-അലിയാസിങ് മുങ്ങി. ഈ ഒരു പ്രശ്നം ഉയർത്തുന്നുണ്ട് എങ്കിൽ പകരം Levels ടൂൾ ഉപയോഗിക്കുക.

5.6.1. ടൂൾ സജീവമാക്കുന്നു

ഉപകരണം സജീവമാക്കുന്നതിന് വിവിധ സാധ്യതകളും ഉണ്ട്:

  • You can access this tool from the image menu through Tools → Color Tools → Threshold,,

  • or through Colors → Threshold,

  • or by clicking on the icon in Toolbox if this tool has been installed in it. For this, please refer to വിഭാഗം 1.7, “പണിസഞ്ചി”.

ത്രെഷോൾഡ് പ്രയോഗം സജീവ അല്ലെങ്കിൽ പാളി നിരക്കു തീവ്രത മൂല്യം ഒരു ദൃശ്യ ഗ്രാഫ് ഒരു ഹിസ്റ്റോഗ്രാം ലഭ്യമാക്കുന്നു. നിങ്ങൾ ഇൻപുട്ട് ബോക്സുകൾ ക്ലിക്കുചെയ്യുന്നതിലൂടെയോ ബട്ടൺ 1 ഉപയോഗിച്ച്, ഗ്രാഫ് ന് വലിച്ചോ ഒന്നുകിൽ ദൂരമാണു പരിധി സജ്ജമാക്കാൻ കഴിയും. നിങ്ങൾക്കു തമ്മിൽ തീവ്രതയോടെ ഒരു പശ്ചാത്തലത്തിൽ നിന്ന് ചില തീവ്രതയോടെ ചിത്രം ഒരു ഭാഗം തിരഞ്ഞെടുക്കുക അനുവദിക്കുന്നു. ശ്രേണിക്കുള്ളിൽ പിക്സൽ വെളുത്ത ആയിരിക്കും, മറ്റുള്ളവർ കറുത്തു. നിങ്ങൾ കറുത്ത പശ്ചാത്തലത്തിൽ വെള്ള ആഗ്രഹിക്കുന്ന നിരക്കു നേടുകയും ശ്രേണി ക്രമീകരിക്കുക.

പ്രിവ്യൂ

മാറ്റങ്ങൾ തീവ്രത നില വരെ പോൾ പ്രിവ്യൂ ടോഗിൾ സജീവ അല്ലെങ്കിൽ പാളി തിരഞ്ഞെടുപ്പിന്റെ ഡൈനാമിക് അപ്ഡേറ്റുചെയ്യുന്നവ അനുവദിക്കുന്നു.

5.6.3. ഒരു തിരഞ്ഞെടുപ്പ് മാസ്ക് സൃഷ്ടിക്കാൻ തുടക്കവും പെട്ടെന്ന് മാസ്ക് ഉപയോഗിക്കൽ

അത് എല്ലായ്പ്പോഴും അങ്ങനെ, എന്നാൽ നിങ്ങൾ ഒരു ചിത്രത്തിൽ നിന്ന് പുറത്തെടുക്കുകയും ചെയ്യണമെന്ന് ഒരു മൂലകം നന്നായി പശ്ചാത്തലത്തിൽ നേരെ നിൽക്കാൻ കഴിയും. ഈ സാഹചര്യത്തിൽ, പരിധി ടൂൾ ഉപയോഗിക്കാൻ കഴിയും മൊത്തമായി ഈ മൂലകത്തിന്റെ തിരഞ്ഞെടുക്കുന്നതിന്. ജിമ്പ് Grokking ഒരു ചാനൽ മാസ്ക് അടിസ്ഥാനമാക്കി ഒരു വിവരിച്ചിരുന്നു, ഇപ്പോൾ Quick mask ഉപയോഗിച്ച് എളുപ്പം.

  1. ആദ്യം Decompose അരിപ്പ ഉപയോഗിച്ച് അതിന്റെ ആർജിബി ആൻഡ് HSV ഘടകങ്ങൾ നിങ്ങളെ ഇമേജ് decomposing ആരംഭിക്കുക. ഒരു പുതിയ ചാര-സ്കെയിൽ ചിത്രം തയ്യാറാക്കി, ഘടകങ്ങളും ലെയർ ഡയലോഗിൽ പാളികൾ ലഭ്യമാകുന്നു. ഈ പാളികൾ ഒരു ലഘുചിത്രം വരും എന്നാൽ അത് ഒരു എളുപ്പ പഠനത്തിൽ തീരെ ചെറുതാണ്. നിങ്ങൾ, തീർച്ചയായും, ഡയലോഗ് മെനു (ചെറിയ ത്രികോണ ബട്ടൺ) ഉപയോഗിച്ച് ഈ പ്രിവ്യൂവിന്റെ വലിപ്പം വർദ്ധിപ്പിക്കാൻ കഴിയും, എന്നാൽ “eyes ” കളിച്ച് വിഘടിപ്പിക്കുന്നു സ്വരൂപത്തിൽ പിടികൂടാനുള്ള പാളി പ്രദർശിപ്പിക്കാൻ ലളിതമാണ്. മൂലകം മികച്ച isolates ആ പാളി തിരഞ്ഞെടുക്കുക.

    ചിത്രം 14.156. യഥാർത്ഥ ചിത്രം, വിഘടിപ്പിക്കുന്നു ചിത്രം അതിന്റെ ലെയർ ഡയലോഗ്


  2. വിളിക്കുക വിഘടിപ്പിക്കുന്നു ഇമേജിൽ നിന്നും പരിധി ഉപകരണം. കറുത്ത കഴ്സർ, മികച്ച നിങ്ങളെ ഒരിക്കൽ ആഗ്രഹിക്കുന്ന മൂലകം വേർതിരിച്ചെടുത്തത് ഫിറ്റ് ദൂരമാണു നീക്കിക്കൊണ്ട്. ഇത് തികഞ്ഞ ചെയ്യില്ല: ഞങ്ങൾ സൃഷ്ടിക്കാൻ പോകുന്നു നിരക്കു മുഖംമൂടി ധരിച്ച ഫലം പരിപോഷിപ്പിക്കുകയും.

    മുന്നറിയിപ്പു്

    അതു തുറന്നിരിക്കുമ്പോൾ, മറ്റൊരു പാളി ലേക്ക് മാറ്റാൻ കഴിയില്ല: നിങ്ങൾ ശരിയായ പാളി തിരഞ്ഞെടുത്ത നിങ്ങൾ പരിധി പ്രയോഗം വിളിക്കുമ്പോൾ ഉറപ്പാക്കുക.

    ചിത്രം 14.157. ദൂരമാണു ഫിറ്റ് ശേഷം തിരഞ്ഞെടുത്ത ലെയർ

    ഞങ്ങളുടെ പുഷ്പം മികച്ച ബാഹ്യരേഖാ ലഭിച്ചു. ഞങ്ങൾ നീക്കം ചെയ്യണം പല ചുവന്ന വസ്തുക്കൾ ഉണ്ട്.


  3. തിരഞ്ഞെടുത്ത ലെയർ പ്രദർശിപ്പിക്കുന്നത് ചിത്രം സജീവമാണ് Ctrl+C കൂടെ ക്ലിപ്പ്ബോർഡിലേക്ക് പകർത്തുന്നതിന് ഉറപ്പാക്കുക.

  4. ഇപ്പോൾ യഥാർത്ഥ ചിത്രത്തിന്റെ സജീവമാക്കുന്നത്. ചിത്രം വിൻഡോയുടെ താഴെ ഇടത് കോണിൽ Quick Mask ബട്ടണിൽ ക്ലിക്ക്: ചിത്രം ഒരു ചുവന്ന (സ്ഥിരം) അർദ്ധസുതാര്യ മാസ്ക് മൂടിയിരിക്കുന്നു നേരിടുമ്പോഴാണ്. ഈ ചുവന്ന നിറം വളരെ ചുവന്ന ഞങ്ങളുടെ ചിത്രം നന്നായി കാര്യങ്ങളുമായി ഇല്ല, ചാനൽ സംഭാഷണം പോകുക “Quick mask” ചാനൽ സജീവമാക്കുന്നതിനും Edit Channel Attributes ഉപയോഗിച്ച് ഈ നിറം മാറ്റുക. യഥാർത്ഥ ചിത്രം മടങ്ങുക. മുമ്പ് പകർത്തി പാളി ഒട്ടിക്കാൻ Ctrl+V അമർത്തുക.

    ചിത്രം 14.158. The mask


  5. വോയ്ല. നിങ്ങളുടെ നിരക്കു മാസ്ക് തയ്യാറാണ്: നിങ്ങൾ പോലെ സാധാരണയായി നിരക്കു മെച്ചപ്പെടുത്താൻ കഴിയും. നിരക്കു തയ്യാറാണ് അവസ്ഥയിൽ അതിന്റെ ബട്ടണിൽ വീണ്ടും ഉപയോഗിച്ച്് പെട്ടെന്ന് മാസ്ക് അപ്രാപ്തമാക്കുകയോ: നിങ്ങൾ നിരക്കു ചുറ്റും നടക്കുന്നു ഉറുമ്പുകൾ കാണും.

    ചിത്രം 14.159. The result


    നാം സൂം, lasso വലിയ അനാവശ്യ പ്രദേശങ്ങൾ, പെൻസിൽ നീക്കം ഒരു പിക്സൽ തലത്തിൽ പ്രവർത്തിക്കാൻ കറുത്ത പെയിന്റ് തിരഞ്ഞെടുത്ത പ്രദേശങ്ങൾ നീക്കം വെളുത്ത പെയിന്റ് പ്രത്യേകിച്ച് ബ്രൈൻ വേണ്ടി, തിരഞ്ഞെടുത്ത പ്രദേശങ്ങൾ ചേർക്കാൻ ഉപയോഗിച്ച (ഹാർഡ് പരിധികൾ ലഭിക്കാൻ).
