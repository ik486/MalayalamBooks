3.6. പെന്സില്

ചിത്രം 14.69. Pencil tool


പെൻസിൽ ടൂൾ ഒരു ഹാർഡ് വായ്ത്തലയാൽ ഫ്രീ ഹാൻഡ് ലൈനുകൾ വരയ്ക്കുക ഉപയോഗിക്കുന്നു. പെൻസിൽ ആൻഡ് അന്തസ്സായിത്തന്നെയാണു സമാനമായ ഉപകരണങ്ങളാണ്. രണ്ടു ഉപകരണങ്ങൾ തമ്മിലുള്ള പ്രധാന വ്യത്യാസം ഇരു ബ്രഷ് ഒരേ തരത്തിലുള്ള ഉപയോഗിക്കാൻ ആണെങ്കിലും, പെൻസിൽ ടൂൾ പോലും വളരെ ഫസി ബ്രഷ് ഉപയോഗിച്ച്, വ്യക്തമല്ലാത്ത വേണമങ്കിലും ഉത്പാദിപ്പിക്കാനുള്ള എന്നതാണ്. അതു പോലും സാമാജ്യത്വ അലിയാസിങ് ചെയ്യാൻ ഇല്ല.

എന്തുകൊണ്ടാണ് നിങ്ങൾ അത്തരം ഒരു ക്രൂഡ് ഉപകരണം പ്രവർത്തിക്കാൻ ആഗ്രഹിക്കുക? ഒരുപക്ഷേ ഏറ്റവും പ്രധാനപ്പെട്ട ഉപയോഗം പോലെ ഒരു ഉയർന്ന സൂം നിലയിൽ ഓപ്പറേറ്റ് ഓരോ പിക്സൽ ശരിയായ നേടുകയും ചെയ്യേണ്ടിവരുന്ന ഐക്കണുകൾ, വളരെ ചെറിയ ചിത്രങ്ങൾ, കൂടെ പ്രവർത്തിക്കുമ്പോൾ ആണ്. പെൻസിൽ ഉപകരണം ഉപയോഗിച്ച്, ബ്രഷ് ഔട്ട്ലൈൻ ഉള്ളിൽ ഓരോ പിക്സൽ നീ ആഗ്രഹിക്കുന്ന കൃത്യമായി മാറ്റമുണ്ടാകില്ല ഉറപ്പുണ്ടായിരിക്കാൻ കഴിയും.

സൂചന

നിങ്ങൾ പെൻസിൽ (അല്ലെങ്കിൽ മറ്റ് നിരവധി പെയിന്റ് ഉപകരണങ്ങൾ ഏതെങ്കിലും) കൂടെ രേഖാഖണ്ഡം വരയ്ക്കണമെങ്കിൽ, ആരംഭ ഘട്ടത്തിൽ ക്ലിക്ക് Shift പിടിക്കുക, അവസാനിക്കുകയും പോയിന്റിലും ക്ലിക്കുചെയ്യുക.

3.6.1. ടൂൾ സജീവമാക്കുന്നു

  • The Pencil Tool can be called from the image-menu: Tools → Paint Tools → Pencil

  • The Tool can also be called by clicking the tool icon:

  • or by clicking on the N keyboard shortcut.

3.6.2. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

Ctrl,

ഈ കീ ഒരു Color Picker പെൻസിൽ മാറ്റുന്നു.

മാറ്റം

ഈ കീ വര മോഡിലേക്ക് പെൻസിൽ ടൂൾ മതിയാവില്ല. ഹോൾഡിംഗ് Shift Button 1 ക്ലിക്ക് ചെയ്യുമ്പോൾ ഒരു വര സൃഷ്ടിക്കും. തുടർച്ചയായ ക്ലിക്കുകൾ കഴിഞ്ഞ വരിയുടെ അവസാനം നിന്നും ആ നേരായ ഡ്രോയിംഗ് തുടരും.

3.6.3. ഓപ്ഷനുകൾ

ചിത്രം 14.70. “Pencil” Tool options


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

Mode; Opacity; Brush; Dynamics; Dynamics Options; Apply Jitter; Smooth Stroke; Incremental
പല അല്ലെങ്കിൽ എല്ലാ പെയിന്റ് ഉപകരണങ്ങൾ ബാധകമായ ഉപകരണം ഓപ്ഷനുകൾ ഒരു വിവരണം വേണ്ടി Common Paint Tool Options കാണുക.
