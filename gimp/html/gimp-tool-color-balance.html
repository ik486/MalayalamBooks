5.2. കളർ ബാലൻസ്

നിറം ബാലൻസ് പ്രയോഗം സജീവ നിരക്കു ലെയർ നിറം ബാലൻസ് മാറ്റം. മാറ്റങ്ങൾ വല്യമ്മ അല്ല. ഈ ഉപകരണം ഡിജിറ്റൽ ഫോട്ടോകളിൽ പ്രബലമായ നിറങ്ങൾ തിരുത്താൻ അനുയോജ്യമായിട്ടുള്ളത്.

5.2.1. ടൂൾ സജീവമാക്കുന്നു

You can get to the Color balance tool in several ways :

  • ചിത്രം-മെനുവിൽ വഴി: Tools → Color Tools → Color Balance, അല്ലെങ്കിൽ Colors → Color Balance,

  • by clicking the tool icon: in Toolbox, provided that you have installed color tools in Toolbox. For this, please refer to വിഭാഗം 1.7, “പണിസഞ്ചി”.

5.2.2. ഓപ്ഷനുകൾ

ചിത്രം 14.140. Color Balance options


Presets

നിങ്ങൾ Add settings to favourites ബട്ടൺ ക്ലിക്കുചെയ്ത് നിങ്ങളുടെ ചിത്രം നിറം ക്രമീകരണങ്ങൾ സംരക്ഷിക്കാൻ കഴിയും

The button opens a menu:

ചിത്രം 14.141. Preset Menu


ഏത് നിങ്ങൾ Import Settings from File അല്ലെങ്കിൽ Export Settings to File അനുവദിക്കുന്നു, നിങ്ങൾ നിയന്ത്രിക്കുന്ന ക്രമീകരണങ്ങൾ സംരക്ഷിക്കുക ഡയലോഗ് ആക്സസ് നൽകുന്നു:

ചിത്രം 14.142. Manage saved Settings Dialog


Select range to adjust

ഈ ഓപ്ഷനുകളിൽ ഒന്ന് തിരഞ്ഞെടുക്കുന്നത് ഷാഡോസ് വേണ്ടി സ്ലൈഡറുകൾ അല്ലെങ്കിൽ ഇൻപുട്ട് ബോക്സുകൾ മാറ്റി ഏത് നിറങ്ങൾ ശ്രേണി (ഇരുട്ടിലും പിക്സൽ) ചുരുക്കുകയാണ്, മിഡ്ടോണുകൾ (ഇടത്തരം പിക്സൽ) ഹൈലൈറ്റുകളും (തിളക്കത്തോടെ പിക്സലുകൾ).

Adjust color levels

നിരക്കികള് മൂന്നു ആർജിബി നിറങ്ങൾ അവരുടെ തീർക്കുന്ന നിറങ്ങൾ (CMY) ശ്രേണി. പൂജ്യം സ്ഥാനം യഥാർത്ഥ സ്വരൂപത്തിൽ പിക്സൽ ഇപ്പോഴത്തെ മൂല്യം തുല്യമാണ്. നിങ്ങൾക്ക് Red അല്ലെങ്കിൽ സിയാൻ, ഗ്രീൻ അല്ലെങ്കിൽ മജന്ത, ബ്ലൂ മഞ്ഞ നേരെ പിക്സൽ നിറം മാറ്റാൻ കഴിയും.

Reset Range

ഈ ബട്ടൺ ശ്രേണിയുടെ നിറം അളവ് തിരികെ പൂജ്യം സ്ഥാനം (യഥാർത്ഥ മൂല്യങ്ങൾ) സജ്ജമാക്കുന്നു.

Preserve Luminosity

ഈ ഉപാധി സജീവ അല്ലെങ്കിൽ പാളി നിരക്കു തെളിച്ചം പരിപാലിക്കുന്ന ഉറപ്പാക്കാം. തിളക്കമുള്ള പിക്സൽ മൂല്യം മാറ്റിയില്ല.

പ്രിവ്യൂ

പ്രിവ്യൂ ചെക്ക്ബോക്സ് ചലനാത്മക ചിത്രം അപ്ഡേറ്റുചെയ്യുന്നതിൽ ടോഗിൾ ചെയ്യുന്നു. ഈ ഓപ്ഷൻ ൽ ആണെങ്കിൽ, ആർജിബി തലങ്ങളിൽ വരുത്തുന്ന മാറ്റം ഉടൻ സജീവ നിരക്കു അല്ലെങ്കിൽ പാളി ന് കാണാം.
