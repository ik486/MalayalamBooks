8.35. Filter Pack

8.35.1. പൊതു അവലോകനം

ചിത്രം 16.189. Example for the “Filter Pack” filter

Original image

ഫില്റ്റര് “FilterPack” (കൂടുതൽ ബ്ലൂ, കൂടുതൽ സാച്ചുറേഷൻ) പ്രയോഗിച്ചു


ഈ ഉപകരണം നിങ്ങളെ ചിത്രം കൈകാര്യം ലേക്കുള്ള ഏകീകൃത ഫിൽറ്ററുകൾ ശേഖരം പ്രദാനം ചെയ്യുന്നു. ഗതി, ഒരേ ഫങ്ഷനുകളെ പ്രത്യേക ഫിൽട്ടറുകൾ നടത്താൻ കഴിയൂ, എന്നാൽ ഇവിടെ ഒരു രസകരമായ അവബോധജന്യവും അവലോകനം ഞങ്ങൾക്കുണ്ട്.

8.35.2. Activate the filter

You can find this filter in the image window menu under Colors → Filter Pack.

8.35.3. ഓപ്ഷനുകൾ

ചിത്രം 16.190. All the options for filter “Filter Pack”

  • Selection in context: ചിത്രം ഉള്ളിൽ നിരക്കു.

വിൻഡോസ്

നിങ്ങൾ തമ്മിലുള്ള തിരഞ്ഞെടുക്കാം:

  • Hue മൂന്ന് പ്രാഥമിക നിറങ്ങൾ ആർജിബി നിറങ്ങളുടെ മോഡൽ മൂന്നു തീർക്കുന്ന നിറങ്ങൾ ഓരോ ഒറ്റ പ്രിവ്യൂ ചെയ്യുന്നു. ഒരു നിറം തുടർച്ചയായി ക്ലിക്കുചെയ്യുന്നതിലൂടെ, ഈ നിറം ബാധിതമായ ശ്രേണിയിലേക്ക്, Roughness തക്കവണ്ണം ചേർക്കുക. നിറം കുറയ്ക്കേണ്ട, തീർക്കുന്ന നിറം സമ്മുഖ നിറം ക്ലിക്ക് ചെയ്യുക.

    ചിത്രം 16.191.  Hue option of the “Filterpack” filter


  • Saturation: കൂടുതലോ കുറവോ സാച്ചുറേഷൻ വേണ്ടി മൂന്ന് പ്രിവ്യൂകൾ.

    ചിത്രം 16.192.  The saturation option of the “Filterpack” filter


  • Value: കൂടുതലോ കുറവോ പ്രതല വേണ്ടി മൂന്ന് പ്രിവ്യൂകൾ.

    ചിത്രം 16.193.  Value option of the “Filterpack” filter


  • Advanced: പിന്നീട് വികസിപ്പിച്ച.

Affected range

നിങ്ങൾ പ്രവർത്തിക്കാൻ ആഗ്രഹിക്കുന്ന മിഴിവ് ഏത് സജ്ജമാക്കാൻ നിങ്ങളെ അനുവദിക്കുന്നു.

  • Shadows: ഇരുണ്ട ടൺ.

  • മിഡ്ടോണുകൾ

  • Highlights: മിനുക്കിയ ടൺ

Select pixels by

HSV ചാനൽ എന്താണ് തിരഞ്ഞെടുത്ത പരിധിയിൽ ബാധിക്കും നിർണ്ണയിക്കുന്നു. നിങ്ങൾ തമ്മിലുള്ള തിരഞ്ഞെടുക്കാം:

  • ഹ്യൂയേ

  • സാച്ചുറേഷൻ

  • വില

Roughness

ഒരു ഹ്രസ്വ ഘട്ടം അല്ലെങ്കിൽ ഒരു വലിയ ഒറ്റ (- 1 0) എടുക്കൽ: നിങ്ങൾ ഒരു ജാലകം ക്ലിക്ക് ചെയ്യുമ്പോൾ ചിത്രം മാറ്റാൻ എങ്ങനെ ഈ സ്ലൈഡർ സെറ്റുകൾ.

വിപുലമായ ഓപ്ഷനുകൾ

ചിത്രം 16.194.  Advanced options of the “Filterpack” filter


ഈ വിപുലമായ ഓപ്ഷനുകൾ നിങ്ങൾക്ക് ചിത്രത്തിലേക്ക് പ്രിവ്യൂ വലിപ്പം ബാധകമാകുന്ന മാറ്റങ്ങൾ സംബന്ധിച്ചുള്ള കൂടുതൽ കൃത്യമായി പ്രവർത്തിക്കുന്നത്.

പ്രിവ്യൂ വലിപ്പം

പ്രിവ്യൂ സംബന്ധിച്ചുള്ള ഒരു സൂം അരിച്ചെടുക്കുക. സാധാരണ വലുപ്പം 80 ആണ്.

സൂചന

പ്രിവ്യൂ വലിപ്പം ഓപ്ഷൻ വകവയ്ക്കാതെ ഈ വലിപ്പം പലപ്പോഴും വളരെ ചെറുതാണ്. നിങ്ങൾ ഒരു ഫോട്ടോയിലെ മുഖം ഉദാഹരണത്തിന്, ഒരു റെസിഡന്റ് നിരക്കു പ്രവർത്തിക്കുന്നു വഴി ഈ നഷ്ടപരിഹാരം കഴിയും. തുടർന്ന്, ചിത്രത്തിൻറെ മറ്റു ഭാഗത്ത് പ്രവർത്തിക്കാൻ നിരക്കു വിപരീത.
Affected range

ഇവിടെ നിങ്ങൾ ഫിൽട്ടർ ബാധിക്കുമെന്ന് ടോൺ പരിധി സജ്ജമാക്കാൻ കഴിയും.

ഈ ജാലകത്തിൽ കർവ് ചിത്രം പ്രയോഗിച്ചു മാറ്റങ്ങൾ പ്രാധാന്യം പ്രതിനിധീകരിക്കുന്നു. ഷാഡോസ്, Midtons അല്ലെങ്കിൽ ഉയർത്തിക്കാട്ടുന്നു: ഈ വഞ്ചിപ്പിക്കുന്നവയിൽ വശം നിങ്ങൾ തിരഞ്ഞെടുത്ത ബാധിക്കപ്പെട്ട റേഞ്ച് ആശ്രയിച്ചിരിക്കുന്നു. നിങ്ങൾ അരിപ്പ പ്രധാന വിൻഡോയിൽ Roughness സ്ലൈഡർ ഉപയോഗിച്ച് കർവ് കഴിവുമുള്ളവർ സജ്ജമാക്കാൻ കഴിയും.

ലഭ്യമായ നിയന്ത്രണങ്ങൾ (സ്ലൈഡർ ആൻഡ് ത്രികോണങ്ങൾ) ഉപയോഗിക്കുന്നതിലൂടെ, കൃത്യമായും പ്രവർത്തനം കർവ് രൂപത്തിൽ സജ്ജമാക്കാൻ കഴിയും.
