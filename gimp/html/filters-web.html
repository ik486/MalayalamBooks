14. വെബ് ഫിൽട്ടറുകൾ

14.1. അവതാരിക

ചിത്രം 17.342. വെബ് ഫിൽട്ടറുകൾ മെനു


ഈ ഫിൽട്ടറുകൾ കൂടുതലും വെബ് സൈറ്റുകളിലും പരാമർശിച്ചു ചിത്രങ്ങൾ ഉപയോഗിച്ചിരിക്കുന്നത്. ഫിൽട്ടർ ImageMap ചിത്രം അല്ലങ്കിൽ “hot spots” ചേർക്കാൻ ഉപയോഗിക്കുന്നു. ഫിൽട്ടർ Semi-Flatten ആൽഫാ ചാനൽ ഇല്ലാത്ത ചിത്രം ഫോർമാറ്റുകളിൽ സെമി സുതാര്യത പകർത്താനായി ഉപയോഗിക്കുന്നു. Slice അരിപ്പ സെൻസിറ്റീവ് ഇമേജുകളുടെ HTML പട്ടികകൾ സൃഷ്ടിക്കുന്നു.
