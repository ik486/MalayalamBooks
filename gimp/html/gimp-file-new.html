2.2. പുതിയ

New Image ഡയലോഗ് ഉപയോഗിച്ച്, നിങ്ങൾ ഒരു പുതിയ ശൂന്യമായ ഇമേജ് സൃഷ്ടിക്കുകയും അതിന്റെ സജ്ജമാക്കാൻ കഴിയും. ചിത്രം ഒരു പുതിയ ഇമേജ് വിൻഡോയിൽ കൊടുത്തിരിക്കുന്നു. ഒരേ സമയം നിങ്ങളുടെ സ്ക്രീനിൽ ഒന്നിലധികം ഇമേജ് ഉണ്ടായിരിക്കാം.

2.2.1. Activate the command

  • You can access the command in the Image menu through: File → New,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Ctrl+N ഉപയോഗിച്ച്.

2.2.2. അടിസ്ഥാന ഓപ്ഷനുകൾ

ചിത്രം 16.4. The “New Image” dialog


Template

ചിത്രം 16.5. The “Template” dialog


പകരം എല്ലാ മൂല്യങ്ങളും നൽകുന്നതിലൂടെ, നിങ്ങൾ ചില പറവാനുണ്ടു സാധാരണയായി ഉപയോഗപ്രദമായ ഇമേജ് തരം പ്രതിനിധാനം ചെയ്ത ഫലകങ്ങൾ ഒരു മെനുവിൽ നിന്ന് നിങ്ങളുടെ ചിത്രത്തിനായുള്ള നിർവചിച്ച മൂല്യങ്ങൾ, തിരഞ്ഞെടുക്കാം. ടെംപ്ലേറ്റുകൾ നിങ്ങൾ ഇടയ്ക്കിടെ ഉപയോഗിക്കുന്ന ഒരു പ്രത്യേക ചിത്രം ആകാരം ഉപയോഗിച്ച് അത് ലിസ്റ്റിൽ ലഭ്യമാകുന്നില്ല എങ്കിൽ, നിങ്ങൾ Templates ഡയലോഗ് ഉപയോഗിച്ച് ഒരു പുതിയ ടെംപ്ലേറ്റ് സൃഷ്ടിക്കാൻ കഴിയും വലിപ്പം, ചിത്രം, അഭിപ്രായങ്ങൾ മുതലായവ വേണ്ടി മൂല്യങ്ങൾ സജ്ജമാക്കാൻ.

ചിത്രം വലിപ്പം

ഇവിടെ നിങ്ങൾ Width പുതിയ ചിത്രത്തിന്റെ Height നിർത്തി. സ്ഥിര യൂണിറ്റുകൾ പിക്സൽ, എങ്കിലും നിങ്ങൾ .ഹാളിനൊരുവശത്തായി മെനു ഉപയോഗിച്ച്, നിങ്ങൾക്ക് ഇഷ്ടമാണെങ്കിൽ മറ്റൊരു യൂണിറ്റ് തിരഞ്ഞെടുക്കാം. നിങ്ങൾ ചെയ്യേണ്ടത്, ലഭിക്കുന്ന പിക്സൽ സൈസ് X, Y റെസലൂഷൻ (നിങ്ങൾ Advanced Options മാറ്റില്ല കഴിയും) നിർണ്ണയിക്കുന്നത് ശ്രദ്ധിക്കുക View മെനുവിൽ “Dot for Dot” ക്രമീകരിച്ച്.

ഇമേജ് തുറക്കുക എങ്കിൽ, “New” ഇമേജ് നിങ്ങളെ നിശ്ചയിച്ചിരിക്കുന്നു സ്ഥിര വലുപ്പം കൂടെ ശൂന്യമായ ഇമേജ് വിൻഡോയിൽ തുറന്നു നോക്കും. മറ്റൊരു തുറക്കുമ്പോൾ (അല്ലെങ്കിൽ എത്തിയിട്ടില്ല) നിങ്ങൾ “New” ചിത്രം തുറക്കാൻ എങ്കിൽ, അത് മറ്റൊരു വിൻഡോയിൽ, ആദ്യ ചിത്രം പോലെ ഒരേ വലിപ്പം കൊണ്ട് തുറന്നു നോക്കും.

കുറിപ്പു്

ഒരു ചിത്രത്തിന്റെ ഓരോ പിക്സൽ മെമ്മറി സംഭരിച്ചിരിക്കുന്ന ഓർമിക്കുക. നിങ്ങൾ ഒരു ഉയർന്ന പിക്സൽ സാന്ദ്രത വലിയ ഫയലുകൾ ഉണ്ടാക്കി, GIMP ഇമേജിൽ ബാധകമാണ് ഓരോ ചടങ്ങിൽ കാലവും മെമ്മറിയും ആവശ്യമാണ്.
ഛായാചിത്രം / ലാൻഡ്സ്കേപ്പ് ബട്ടണുകൾ

അർദ്ധകായ സ്കേപ്പ് മോഡ് തമ്മിൽ ടോഗിൾ രണ്ട് ബട്ടണുകൾ ഉണ്ട്. ഇവ യഥാർത്ഥത്തിൽ ചെയ്യാൻ വീതിയും ഉയരവും മൂല്യങ്ങളെ കൈമാറാൻ ആണ്. (വീതിയും ഉയരവും ഒരേ എങ്കിൽ ഈ ബട്ടണുകൾ സജീവമാക്കി അല്ല.) X, Y റെസലൂഷൻ ഒരേ (നിങ്ങൾ വിപുലമായ ഓപ്ഷനുകൾ ഇരുത്തുന്ന കഴിയും) അല്ല എങ്കിൽ, ഈ മൂല്യങ്ങൾ പുറമേ മാറ്റിക്കളഞ്ഞു ചെയ്യുന്നു. ഡയലോഗിന്റെ വലതുവശത്ത്, ഇമേജ് സൈസ്, സ്ക്രീൻ റെസലൂഷൻ നിറം സ്പേസ് പ്രദർശിപ്പിക്കുന്നു.

2.2.3. വിപുലമായ ഓപ്ഷനുകൾ

ചിത്രം 16.6. New Image dialog (Advanced Options)


Advanced Options കൂടുതൽ വിപുലമായ GIMP ഉപയോക്താക്കളെ താത്പര്യമുള്ള കൂടുതലും ആകുന്നു. ഡയലോഗ് വിൻഡോ താഴത്തെ വിളുമ്പിൽ ചെറിയ ചതുരത്തിൽ ക്ലിക്കുചെയ്ത് ഈ ഓപ്ഷനുകൾ പ്രദർശിപ്പിക്കാൻ കഴിയും.

X and Y resolution

X resolution ആൻഡ് Y resolution മേഖലകളിലെ മൂല്യങ്ങൾ പ്രിന്റിങ് പ്രധാനമായും വിവരിച്ചുതരാം അവർ പിക്സലിൽ ചിത്രത്തിന്റെ വലുപ്പം ബാധിക്കുന്നില്ല, പക്ഷേ അച്ചടിക്കാൻ പാകത്തിൽ അവർ അതിന്റെ ശാരീരിക വലിപ്പം നിർണ്ണയിക്കാം. പിക്സൽ പോലുള്ള മില്ലിമീറ്റർ അല്ലെങ്കിൽ ഇഞ്ച് മറ്റ് അളക്കൽ യൂണിറ്റുകൾ, പരിഭാഷപ്പെടുത്തിയിട്ടുണ്ട് എങ്ങനെയാണെന്നും X, Y നൽകുന്ന മൂല്യങ്ങൾ നിർണ്ണയിക്കാൻ കഴിയും.

സൂചന

നിങ്ങൾ ശരിയായ അളവുകൾ സ്ക്രീനിൽ ചിത്രം പ്രദർശിപ്പിക്കാൻ ആഗ്രഹിക്കുന്നുണ്ടെങ്കിൽ, സൂം ഘടകം തിരഞ്ഞെടുക്കുക View → Dot for Dot സെറ്റ് 100% അതിന്റെ യഥാർഥ സ്ക്രീൻ വലുപ്പത്തിൽ ഇമേജ് കാണാൻ. GIMP ഇൻസ്റ്റാൾ സ്ക്രീൻ വലിപ്പം കാലിബ്രേഷൻ സാധാരണയായി നടക്കുന്ന, എന്നാൽ ചിത്രം ശരിയായ വലുപ്പത്തിൽ പ്രദർശിപ്പിക്കുകയോ ചെയ്യുന്നില്ല എങ്കിൽ നിങ്ങൾ GIMP സ്ക്രീനിന് പാരാമീറ്ററുകൾ ക്രമീകരിക്കാൻ ഉണ്ടായേക്കാം. നിങ്ങൾ Preferences ഡയലോഗിൽ ചെയ്യാൻ കഴിയും.
Colorspace

നിങ്ങൾ ഒരു RGB ചിത്രം അല്ലെങ്കിൽ ഗ്രേസ്കെയിലാക്കി ഇമേജ് ഒന്നുകിൽ പോലെ വിവിധ color modes പുതിയ ഇമേജ് തയ്യാറാക്കാം.

RGB color

ചിത്രം നിങ്ങളുടെ മോണിറ്റർ അല്ലെങ്കിൽ നിങ്ങളുടെ ടെലിവിഷൻ സ്ക്രീനിൽ ഉപയോഗിക്കുന്ന ഒന്നാണ് റെഡ്, ഗ്രീൻ, നീല നിറം സിസ്റ്റം, സൃഷ്ടിക്കുന്നതാണ്.

Greyscale

ചിത്രം ചാര വിവിധ ഷേഡുകൾ കൂടെ കറുപ്പിലും വെളുപ്പിലും സൃഷ്ടിക്കപ്പെട്ടിരിക്കുന്നത്. ലിപിയിലോ നിങ്ങളുടെ കലാപരമായ താൽപ്പര്യങ്ങൾ നിന്നുള്ള ചിത്രം ഈ തരം ചില പ്ലഗ്-ഇന്നുകൾ ആവശ്യമാണ്. എന്നിരുന്നാലും GIMP നിങ്ങൾ change an RGB image into grayscale, നിങ്ങള്ക്ക് ആഗ്രഹിക്കുന്നു എങ്കിൽ അനുവദിക്കുന്നു.

ഈ മെനു നേരിട്ട് ഗണമാണോയെന്നത് ചിത്രം സൃഷ്ടിക്കാൻ കഴിയില്ല, പക്ഷേ തീർച്ചയായും നിങ്ങൾക്ക് എപ്പോഴും സൃഷ്ടിച്ചുകഴിയുമ്പോൾ ശേഷം ഇൻഡെക്സ്ഡ് മോഡിലേക്ക് ചിത്രം പരിവർത്തനം ചെയ്യാനാകും. ആ ചെയ്യാൻ, Image → Mode → Indexed കമാൻഡ് ഉപയോഗിക്കുക.

നിറയ്ക്കുക

ഇവിടെ നിങ്ങൾ നിങ്ങളുടെ പുതിയ ചിത്രം ഉപയോഗിക്കുന്ന പശ്ചാത്തല നിറം വ്യക്തമാക്കുക. ഇത് വളരെ പിന്നീട് ഒരു ചിത്രത്തിന്റെ പശ്ചാത്തലം മാറ്റാൻ തീർച്ചയായും സാധ്യമാണ്. കണ്ടെത്താവുന്നതാണ് Layer dialog ആ ചെയ്യുന്നത് കൂടുതൽ വിവരങ്ങൾ.

നിരവധി തിരഞ്ഞെടുക്കാൻ ഉണ്ട്:

  • പണിസഞ്ചി കാണിച്ചിരിക്കുന്ന നിലവിലെ Foreground color ഉപയോഗിച്ച് ഇമേജ്, നിറയ്ക്കുക.

    “New Image” ഡയലോഗ് ജാലകം തുറന്നിരിക്കുന്ന സമയത്ത് നിങ്ങൾ മുൻഭാഗത്തെ നിറം മാറ്റാൻ ശ്രദ്ധിക്കുക.

  • പണിസഞ്ചി കാണിച്ചിരിക്കുന്ന നിലവിലെ Background color ഉപയോഗിച്ച് ഇമേജ്, നിറയ്ക്കുക. (പശ്ചാത്തലത്തിൽ നിറം ഡയലോഗ് വിൻഡോ തുറന്നിരിക്കുന്ന സമയത്ത്, വളരെ മാറ്റാൻ കഴിയും.)

  • White ഉപയോഗിച്ച് ഇമേജ് നിറയ്ക്കുക.

  • Transparency ഉപയോഗിച്ച് ഇമേജ് നിറയ്ക്കുക. നിങ്ങൾ ഈ ഓപ്ഷൻ തിരഞ്ഞെടുക്കുകയാണെങ്കിൽ, ചിത്രം ഒരു alpha channel ഉപയോഗിച്ച് തയ്യാറാക്കി, പശ്ചാത്തല സുതാര്യമാണ്. ചിത്രത്തിന്റെ സുതാര്യമായ ഭാഗങ്ങൾ പിന്നീട് സുതാര്യത സൂചിപ്പിക്കാൻ ഒരു ശൂരനാടിന് പാറ്റേൺ പ്രദർശിപ്പിച്ചിരിക്കുന്നു.

അഭിപ്രായം

നിങ്ങൾക്ക് ഇവിടെ ഒരു വിവരണാത്മക അഭിപ്രായം എഴുതാം. ടെക്സ്റ്റ് ഒരു parasite ചിത്രം ചേർത്തിരിക്കുന്നു, ചില ഫയൽ ഫോർമാറ്റുകൾ (PNG, JPEG, gif) ചിത്രം ഉപയോഗിച്ച് സംരക്ഷിച്ചു.

കുറിപ്പു്

നിങ്ങൾ Image Properties ഡയലോഗിൽ കാണാനും എഡിറ്റ് ചെയ്യാനും ഈ അഭിപ്രായം.
