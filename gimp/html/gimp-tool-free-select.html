2.4. Free Selection (Lasso)

ചിത്രം 14.17. Free Selection icon in the Toolbox


സൗജന്യ തിരഞ്ഞെടുക്കൽ ഉപകരണം അല്ലെങ്കിൽ lasso, നിങ്ങൾ അത് (ടാബ്ലെറ്റ് നേരെ അതു അമർത്തുന്നത് അല്ലെങ്കിൽ, സ്റ്റൈലസ് വേണ്ടി,) ഇടത് മൗസ് ബട്ടൺ അമർത്തിപ്പിടിച്ച്, ഫ്രീ കൈ ഡ്രോയിംഗ് പോയിന്റർ കൂടെ ഒരു നിര സൃഷ്ടിക്കാൻ അനുവദിക്കുന്നു. മൌസ് ബട്ടൺ റിലീസ് ചെയ്യുമ്പോൾ, നിരക്കു ഒരു വര കൊണ്ട് ആരംഭ ലൊക്കേഷനിലേക്ക് നിലവിലെ പോയിന്റർ ലൊക്കേഷൻ ബന്ധിപ്പിച്ചുകൊണ്ട് അടച്ചു. നിങ്ങൾ ചിത്ര പ്രദർശന വായ്ത്തലയാൽ പുറത്ത് പോയി നിങ്ങൾക്കു ചെയ്യണമെങ്കിൽ ലെ തിരിച്ചെത്താം. Lasso പലപ്പോഴും “roughing in” ഒരു നിര ഉപയോഗിക്കാൻ ഒരു നല്ല ഉപകരണമാണ്; അതു കൃത്യമായ നിർവചനം അങ്ങനെ നന്നല്ല. പരിചയമുള്ള ഉപയോക്താക്കൾ അതു lasso എന്ന ഉപകരണം ഉപയോഗിച്ച് തുടങ്ങും പലപ്പോഴും ഉള്ളിടത്തോളം എന്നു കണ്ടെത്താൻ, എന്നാൽ വിശദമായി പണിക്ക് QuickMask മോഡിലേക്ക്.

തിരഞ്ഞെടുക്കലുകൾ വിവരങ്ങൾക്ക് അവർ എങ്ങനെ ജിമ്പ് Selections കാണാൻ ഉപയോഗിക്കുന്നു. എല്ലാ നിരക്കു ഉപകരണങ്ങൾ പൊതുവായുള്ള സവിശേഷതകൾ വിവരങ്ങൾക്കായി Selection Tools കാണുന്നു.

കുറിപ്പു്

സൗജന്യ തിരഞ്ഞെടുക്കൽ പ്രയോഗം മൌസ് ഉപയോഗിച്ച് അധികം ടാബ്ലറ്റ് കൂടെ ഉപയോഗിക്കാൻ വളരെ എളുപ്പമാണ്.

ഒരു പുതിയ സാധ്യത ജിമ്പ്-2.6 കൂടെ വന്നു; ബഹുഭുജചിഹ്നം നിരക്കു. പകരം ക്ലിക്ക്-ആൻഡ്-വലിച്ചിടുമ്പോൾ ഒരു സ്വതന്ത്ര കൈ നിരക്കു ചെല്ലാനും മാത്രമേ ക്ലിക്ക് ചെയ്യാം. ഇത് ഒരു ആങ്കർ പോയിന്റ് സൃഷ്ടിക്കുന്നു. അപ്പോൾ മൌസ് പോയിന്റർ ചലിക്കുന്ന നിങ്ങൾ ഇതിനായി വീണ്ടും ക്ലിക്ക് പോലെ (മൗസ് പോയിന്റർ ചലിക്കുന്ന ക്രൂശിൽ കൊണ്ട് വരുന്നു) പോകാം എന്ന് ഒരു പുതിയ ആങ്കർ പോയിന്റ് കൊണ്ട് ഒരു ലൈൻ അടുപ്പിക്കുന്നു. വീണ്ടും ക്ലിക്കുചെയ്യുന്നത് ഈ പോയിന്റ് ആൻക്കറുകളും ഒരു സെഗ്മെന്റ് സൃഷ്ടിക്കുന്നു. 15 ° കോണുകളിൽ ചലിക്കുന്ന മൌസ് പോയിന്റർ contrains നീക്കുന്നതിനിടെ Ctrl കീബോർഡ് കീ അമർത്തി.

So, you can mix free hand segments and polygonal segments.

2.4.1. Activating the tool

You can access the Lasso Tool in different ways:

  • From the image menu bar Tools → Selection Tools → Free Select,

  • by clicking on the tool icon in the ToolBox,

  • by using the keyboard shortcut F.

2.4.2. Key modifiers

സൗജന്യ തിരഞ്ഞെടുക്കുക ഉപകരണം ഏതെങ്കിലും പ്രത്യേക കീ മോഡിഫയറുകൾ, അതേ വിധത്തിൽ എല്ലാ നിരക്കു ഉപകരണങ്ങൾ ബാധിക്കുന്ന മാത്രമല്ല ഇല്ല. ഈ കൂടെ സഹായത്തിനായി Selection Tools കാണുക.
