4.2. ചിത്രങ്ങൾ സംഭാഷണം

ചിത്രം 15.66. ചിത്രങ്ങൾ ഡയലോഗ്


“Images” സംഭാഷണം നിങ്ങളുടെ സ്ക്രീനിൽ തുറക്കുക ചിത്രങ്ങൾ കാണിക്കുന്നു; ഓരോ ലഘുചിത്രത്തോടുകൂടിയ പ്രതിനിധീകരിക്കുന്നു. നിങ്ങൾ സ്ക്രീനിൽ പല കവിഞ്ഞു ചിത്രങ്ങൾ എത്തുമ്പോൾ ഈ ഡയലോഗ് ഉപയോഗപ്രദമാണ്: ഇപ്രകാരം നിങ്ങളോടു മുൻഭാഗത്തെ ആഗ്രഹിച്ചു ചിത്രം ഉണ്ടാകാം.

4.2.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Images” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • ചിത്രം മെനുവിൽ നിന്ന്: Windows → Dockable Dialogs → Images;

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Images തിരഞ്ഞെടുത്ത്.

Windows മെനുവിൽ, കുറഞ്ഞത് ഒരു ഡയലോഗ് തുറന്നുതന്നെയിരിക്കുന്ന മാത്രമേ ഉണ്ട് detached windows ഒരു ലിസ്റ്റ് ഉണ്ട്. Windows → Images: ഈ സാഹചര്യത്തിൽ, ഇമേജ്-മെനുവിൽ നിന്നും “Images” ഡയലോഗ് ഉണ്ടാകാം.

4.2.2. ചിത്രങ്ങൾ ഡയലോഗ് ഉപയോഗിച്ച്

മൾട്ടി വിൻഡോ മോഡിൽ, ഡയലോഗ് മുകളിൽ, “Show Image Selection” ഓപ്ഷൻ ടാബ് മെനു ചെക്കുചെയ്തിട്ടുണ്ടെന്നതും എങ്കിൽ ഇമേജുകൾ തുറക്കുക ഒരു തുള്ളി-ലിസ്റ്റ് ലഭ്യമാകുന്നു.

കേന്ദ്രത്തിൽ, ഇമേജുകൾ തുറക്കുക, ഒരു ലിസ്റ്റ് അല്ലെങ്കിൽ ഒരു ഗ്രിഡ് ആയി, തിരഞ്ഞെടുത്ത മോഡ് തക്കവണ്ണം ദൃശ്യമാകും. നിലവിലെ ചിത്രം ലിസ്റ്റ് മോഡ്, ഗ്രിഡ് മോഡിൽ അനുസരിച്ചുള്ള ലെ ഹൈലൈറ്റുചെയ്തിരിക്കുന്നു. ഒരു ചിത്രം പേര് ഒരു ഇരട്ട ക്ലിക്കിൽ, നിങ്ങളുടെ സ്ക്രീനിന്റെ റോസറ്ററും ഈ ചിത്രം ഉയർത്തുന്നു. അങ്ങനെ ഡയലോഗിന്റെ ബട്ടണുകൾ അതു പ്രവർത്തിക്കാൻ കഴിയും ഈ ചിത്രം തിരഞ്ഞെടുക്കുക ഒരു ലളിതമായ ക്ലിക്കിലൂടെ.

ഗ്രിഡ്, ലിസ്റ്റ് മോഡുകൾ, പ്രിവ്യൂ വലുപ്പം

“Images” ഡയലോഗ് ടാബ് മെനുവിൽ View as Grid ആൻഡ് View as List തമ്മിലുള്ള തിരഞ്ഞെടുക്കാം. ഗ്രിഡ് മോഡിൽ, ചിത്രങ്ങൾ ചതുരാകൃതിയിലുള്ള അണിനിരന്നു നിരത്തിയിരിക്കുന്നത്. പട്ടിക മോഡിൽ അവർ ലംബമായി, ഓരോ വരി ചിത്രത്തിന്റെ ഉള്ളടക്കങ്ങളുടെ ഒരു ലഘുചിത്രം അതിന്റെ പേര്, അതിന്റെ പിക്സൽ തലങ്ങളും കാണിക്കുന്ന കൂടിയ ചെയ്യുന്നു.

സൂചന

Ctrl+F ഒരു തിരയൽ ഫീൽഡ് തുറക്കുന്നു. പട്ടിക ആയി കാണുക; ഗ്രിഡായി കാണുക കാണുക

ഡയലോഗ് ന്റെ ടാബ് മെനുവിലെ “Preview Size” സബ്മെനു ഉപയോഗിച്ച് ഡയലോഗിൽ ഇമേജ് പ്രിവ്യൂകളുടെ വലുപ്പം മാറ്റാൻ കഴിയും.

ബട്ടണുകൾ

ഡയലോഗ് താഴെയുള്ള മൂന്ന് ബട്ടണുകൾ നിങ്ങൾ തിരഞ്ഞെടുത്ത ചിത്രം ന് ഓപ്പറേറ്റ് അനുവദിക്കുന്നു. “Show button bar” ടാബ് ഡയലോഗ് ചെക്കുചെയ്തിട്ടുണ്ടെന്നതും ഈ ബട്ടണുകൾ കൂടിയിരിക്കുന്നു. ഡയലോഗ് റൈറ്റ് ക്ലിക്ക് ചെയ്ത് പോപ്പ് മെനു വഴി ഒരേ കമാൻഡുകൾ ലഭിക്കും.

ഈ ചിത്രം ദൃശ്യമാക്കുന്നു ഉയർത്തലും

തിരഞ്ഞെടുത്ത ചിത്രം നിങ്ങളുടെ സ്ക്രീനിന്റെ മുൻഭാഗത്തെ ദൃശ്യമാകുന്നു. ഈ ചിത്രം മറ്റൊരു കാഴ്ച ഉണ്ടെങ്കിൽ, ഈ വീക്ഷണവും ഉയര്ന്നതാണ് യഥാർത്ഥ പിന്നിൽ തുടരുന്നു ആണ്. പോപ്പ്-അപ്പ് മെനുവിൽ അതേ ഓപ്ഷൻ, റൈറ്റ് ക്ലിക്ക് വഴി നേടുകയും എന്നു “Raise views ” വിളിക്കുന്നു

ഈ ചിത്രം ഒരു പുതിയ ഡിസ്പ്ലേ സൃഷ്ടിക്കുക

തിരഞ്ഞെടുത്ത ചിത്രത്തിന്റെ ഇമേജ് വിൻഡോ (ചിത്രത്തിലെ) തനിപ്പകർപ്പുകളൊന്നുമില്ല.

ഇല്ലാതാക്കുക

ഈ കമാൻഡ് മാത്രമേ ജാലകം ഇല്ലാതെ കയറ്റിയ ഒരു ചിത്രം പ്രവർത്തിക്കുന്നു. ചിത്രങ്ങൾ പുതിയ വിൻഡോ കമാൻഡ് തുറക്കാൻ കഴിയും എങ്കിലും ഇമേജ് ഇതിനകം ഒരു പ്രിമിറ്റീവ് നടപടിക്രമം കമാൻഡ് (അത്തരം gimp-image-new, file-png-load മുതലായവ പോലെ) വഴി വിൻഡോ ഇല്ലാതെ ലോഡ് ചെയ്തിട്ടുണ്ടെങ്കിൽ അത് വിൻഡോസ് കഴിഞ്ഞ അടച്ചിട്ടു പോലും ടിപ്പറിലുണ്ടായിരുന്ന കഴിയില്ല . അപ്പോൾ അത് അടയ്ക്കാൻ ഈ കമാൻഡ് ഉപയോഗിക്കുക.
