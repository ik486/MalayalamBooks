2.13. Export As...

Export As... allows you to save your image in a format other than XCF.

കുറിപ്പു്

Please refer to വിഭാഗം 1, “ഫയലുകൾ” for information about exporting in different file formats.

2.13.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through File → Export As...,

  • or by using the keyboard shortcut Shift+Ctrl+E.
