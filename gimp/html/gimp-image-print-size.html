6.14. പ്രിന്റ് വലിപ്പം

നിങ്ങൾ dimensions of a printed image അതിന്റെ resolution മാറ്റാൻ Print Size ഡയലോഗ് ഉപയോഗിക്കാൻ കഴിയും. ഈ കമാൻഡ് ഇമേജിൽ പിക്സലുകളുടെ എണ്ണം മാറ്റാൻ ഇല്ല അത് ചിത്രത്തിന്റെ റീസാമ്പിൾ ഇല്ല. (അത് റീസാമ്പിൾ ഒരു ചിത്രത്തിന്റെ വലുപ്പം മാറ്റണമെങ്കിൽ, Scale Image കമാൻഡ് ഉപയോഗിക്കുക.)

6.14.1. Activating the Dialog

  • You can access this dialog from the image menubar through Image → Print Size.

6.14.2. Options in the “Print Size” Dialog

ചിത്രം 16.67. The “Print Size” dialog


The output resolution determines the number of pixels used per unit length for the printed image. Do not confuse the output resolution with the printer's resolution, which is a printer feature and expressed in dpi (dots per inch); several dots are used to print a pixel.

ഡയലോഗ് പ്രദർശിപ്പിക്കുമ്പോൾ, പെട്ടികളിൽ കാണിച്ചിരിക്കുന്ന ചിത്രം യഥാർത്ഥ ചിത്രത്തിന്റെ ചിത്രം ആണ്. നിങ്ങൾക്ക് ഔട്ട്പുട്ട് ചിത്രം വർദ്ധിപ്പിക്കാൻ എങ്കിൽ അച്ചടിച്ച പേജ് കൂടുതൽ പിക്സൽ നീളം യൂണിറ്റ് ഉപയോഗിക്കുന്ന ശേഷം, ചെറിയ ആയിരിക്കും. അതുപോലെ, ഒരേ കാരണത്താൽ, ഇമേജ് വലിപ്പംമാറ്റിക്കൊണ്ടിരിക്കുന്നജാലകങ്ങളിലുള്ളവകാണിയ്ക്കുക ചിത്രം മാറ്റം.

അച്ചടിച്ച പേജിന്റെ മൂര്ച്ച വർദ്ധിപ്പിക്കാൻ ചിത്രം ഫലങ്ങളിൽ വർദ്ധിക്കുന്നത്. യാതൊരു പിക്സൽ (ആരും ഇമേജ് വിവരങ്ങൾ) നീക്കം സ്ഥിതിക്ക് ഇത് അത് ഞെട്ടിപ്പോകും ഇമേജ് വലുപ്പം കുറച്ചുകൊണ്ട് നിന്ന് തികച്ചും വ്യത്യസ്തമാണ്.

Width; Height

ടെക്സ്റ്റ് ബോക്സുകൾ ഉപയോഗിച്ച് അച്ചടി വീതിയും ഉയരവും സജ്ജമാക്കാൻ കഴിയും. ഡ്രോപ്പ്ഡൗൺ ലിസ്റ്റിൽ നിന്ന് ഈ മൂല്യങ്ങൾ വേണ്ടി യൂണിറ്റുകൾ തിരഞ്ഞെടുക്കാം.

ഉടൻ വീതി അല്ലെങ്കിൽ ഉയരം മാറ്റാൻ പോലെ എക്സ് ഒപ്പം / അല്ലെങ്കിൽ Y നൽകുന്ന മൂല്യങ്ങൾ സ്വയം അതിനനുസരിച്ച് മാറ്റുക. രണ്ടു റെസലൂഷൻ മൂല്യങ്ങൾ ലിങ്ക്ഡ് നിലനില്ക്കുന്നു എങ്കിൽ ചിത്രത്തിന്റെ ഉയരം വീതിയുടെ ബന്ധവും സ്വയം പരിപാലിക്കുന്നത്. നിങ്ങൾ സ്വതന്ത്രമായി പരസ്പരം ഈ മൂല്യങ്ങൾ സജ്ജമാക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, കേവലം ലിങ്ക് തകർക്കാൻ ചെയിൻ ചിഹ്നം ക്ലിക്ക് ചെയ്യുക.

X resolution; Y resolution

നിങ്ങൾ എന്നു അതിൽ പിക്സലുകളുടെ എണ്ണം ചിത്രത്തിന്റെ ഫിസിക്കൽ വലിപ്പം, നിന്ന് അച്ചടിച്ച വീതിയും ഉയരവും കണക്കുകൂട്ടാൻ ഉപയോഗിക്കുന്ന ചിത്രം സജ്ജമാക്കാൻ കഴിയും.

ഈ ചിത്രം മൂല്യങ്ങൾ മാറ്റാൻ ടെക്സ്റ്റ് ബോക്സുകൾ ഉപയോഗിക്കുക. അവർ തങ്ങളുടെ ബന്ധം നിരന്തരമായ നിലനിർത്താൻ ലിങ്കുചെയ്യാനാകൂ. രണ്ട് ബോക്സുകൾ തമ്മിലുള്ള അടച്ച ചെയിൻ ചിഹ്നം മൂല്യങ്ങൾ ഒരുമിച്ചു പോകുന്നു സൂചിപ്പിക്കുന്നു. ചെയിന് ചിഹ്നം ക്ലിക്കുചെയ്ത് കണ്ണി പൊട്ടി, നിങ്ങൾക്ക് സ്വതന്ത്രമായി പരസ്പരം മൂല്യങ്ങൾ സജ്ജമാക്കാൻ കഴിയും.
