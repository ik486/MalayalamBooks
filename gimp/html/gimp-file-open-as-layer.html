2.5. Open as Layers

Open Image as layers ഡയലോഗ് Open Image ഡയലോഗ് സമാനമാണ്. തിരഞ്ഞെടുത്ത ഫയൽ പാളികൾ സ്റ്റാക്ക് മുൻനിര പാളികൾ പോലെ നിലവിലെ ചിത്രം ചേർത്തു.

2.5.1. Activate Command

  • You can access this command from the image menubar through File → Open as layers,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Ctrl+Alt+O ഉപയോഗിച്ച്.
