7.49. ഓഫ്സെറ്റ്

Offset കമാൻഡ് സജീവ പാളി content മാറ്റുകയും ചെയ്യുന്നു. എന്തും പാളി അതിർത്തി വലുപ്പം പുറത്ത് മാറ്റി. ഈ കമാൻഡ് പാളി അത് എങ്ങനെ വീശുന്ന ശൂന്യമായ നിലയിലാണെങ്കിൽ ആ സ്ഥലം എത്ര മാറ്റണമെന്ന് വ്യക്തമാക്കാൻ അനുവദിക്കുന്ന ഒരു ഡയലോഗ് പ്രദർശിപ്പിക്കുന്നു.

7.49.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Transform → Offset,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Shift+Ctrl+O ഉപയോഗിച്ച്.

7.49.2. Using the “Offset” Command

ചിത്രം 16.114. The “Offset” dialog


ഓഫ്സെറ്റ്
എക്സ്; പിന്നെ

ഈ രണ്ടു മൂല്യങ്ങൾ ഉപയോഗിച്ച്, പാളി ഉള്ളടക്കങ്ങളും തിരശ്ചീനമായി (എക്സ്) ൽ മാറ്റണം എത്ര ദൂരം വ്യക്തമാക്കുന്ന ലംബമായും (y) ദിശകൾ. ടെക്സ്റ്റ് ബോക്സുകളിൽ ഓഫ്സെറ്റുകളിലാണു് നൽകാൻ കഴിയും. പോസിറ്റീവ് മൂല്യങ്ങൾ ശരിയായ കീഴോട്ടു ലെയർ നീക്കുക. യൂണിറ്റ് പിക്സൽ, എന്നാൽ ഡ്രോപ്-ഡൗൺ മെനുവിൽ അളവിന്റെ മറ്റൊരു യൂണിറ്റ് തിരഞ്ഞെടുക്കാം. “%” ഒരു യൂണിറ്റ് ചിലപ്പോൾ ഉപയോഗപ്പെടുന്നു.

Offset by x/2, y/2

ഈ ബട്ടൺ ഉപയോഗിച്ച്, സ്വയം അങ്ങനെ ഉള്ളടക്കങ്ങൾ കൃത്യമായി അർദ്ധ വീതി ചിത്രത്തിന്റെ പകുതി ഉയരം വഴി മാറ്റി ചെയ്യുന്നു X ആൻഡ് Y ഓഫ്സെറ്റുകളുണ്ടു് സജ്ജമാക്കാൻ കഴിയും.

എഡ്ജ് പെരുമാറ്റം

പാളി ഉളളടക്കം മാറ്റി ചെയ്യുമ്പോൾ നിങ്ങൾ ശൂന്യമായ അവശേഷിക്കുന്നു പ്രദേശങ്ങളെ ലേക്ക് മൂന്നു തരത്തിൽ നൽകാം:

  • Wrap around: പാളി ഒരു വശത്ത് ഒഴിഞ്ഞ ഇടം മറ്റെ വശത്തു നിന്നു മാറ്റി ഏത് പാളി ഭാഗമായി നിറഞ്ഞു, അതിനാൽ ഉള്ളടക്കം ആരും നഷ്ടമാകുന്നു.

  • Fill with background color: ഒഴിഞ്ഞ സ്പേസ് പണിസഞ്ചി നിറം ഏരിയയിൽ കാണിക്കുന്നു ഏത് പശ്ചാത്തല നിറം, നിറയും.

  • Make transparent: ഒഴിഞ്ഞ സ്പേസ് സുതാര്യമായ ഉണ്ടാക്കിയ. പാളി ഒരു ആല്ഫാ ചാനൽ ഇല്ലെങ്കിൽ, ഈ നിര (ഗ്രേ ഔട്ട്) ലഭ്യമല്ല.
