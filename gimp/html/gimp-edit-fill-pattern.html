3. “Edit” മെനു  

3.16. ശ്രേണി നിറക്കുക

Fill with Pattern കമാൻഡ് പണിസഞ്ചി ഓഫ് ബ്രഷ് / പാറ്റേൺ / ഗ്രേഡിയന്റ് ഏരിയയിൽ കാണിക്കുന്നു പാറ്റേൺ ഉപയോഗിച്ച് ഇമേജ് ൻറെ നിരക്കു നിറയുന്നു. (പാറ്റേൺ പുറമേ മെനു എൻട്രി ഇടതു കാണിച്ചിരിക്കുന്നു.) ചിത്രത്തിന്റെ ചില പ്രദേശങ്ങൾ ഭാഗികമായി മാത്രം തിരഞ്ഞെടുത്ത (ഉദാഹരണത്തിന്, നിരക്കു ഭദ്രമാക്കുന്ന ഫലമായി) എങ്കിൽ, അവർ തിരഞ്ഞെടുത്ത എത്ര അനുപാതത്തിൽ നിറയും.

നിങ്ങൾ Pattern dialogue ഉപയോഗിച്ച് മറ്റൊരു പാറ്റേൺ തിരഞ്ഞെടുക്കാം.

കുറിപ്പു്

ചിത്രം യാതൊരു സെലക്ഷനുണ്ടെന്ന് എങ്കിൽ മുഴുവൻ സജീവ പാളി നിറഞ്ഞു ദയവായി ശ്രദ്ധിക്കുക.

3.16.1. കമാൻഡ് സജീവമാക്കുക

  • നിങ്ങൾ ഇമേജ് മെനുബാർ നിന്ന് Edit → Fill with Pattern മുഖേന ഈ കമാൻഡ് ആക്സസ് ചെയ്യാൻ കഴിയും,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Ctrl+; ഉപയോഗിച്ച്.
