3.11. ഒട്ടിക്കുക

ഈ കമാൻഡ് ക്ലിപ്ബോർഡിലെ ഉള്ളടക്കം pastes. തീർച്ചയായും, “Copy” കമാൻഡ് മുമ്പിൽ ക്ലിപ്ബോർഡും എന്തോ ഉണ്ട് അങ്ങനെ ഉപയോഗിക്കണം. , PLACEHOLDER,-103992 അല്ലെങ്കിൽ നിങ്ങൾ മറന്നു എന്തെങ്കിലും ഉണ്ടെങ്കിൽ അതു ഒട്ടിക്കുകയുള്ളൂ: മറ്റെല്ലാം ഒരു മുന്നറിയിപ്പ് ആവശ്യപ്പെടും! ക്ലിപ്ബോര്ഡ്കാലിയാണു് യാതൊരു വഴിയും ഇല്ല.

3.11.1. പുതിയ ഇമേജ് ഒട്ടിക്കുക

Paste As New Image കമാൻഡ് ഒരു പുതിയ ഇമേജ് ഉണ്ടാക്കുന്നു അതിൽ ക്ലിപ്ബോർഡ് നിന്നും ചിത്രം ഡാറ്റ pastes. ഡാറ്റ രൂപമുള്ള ചതുരാകൃതിയിലുള്ള ചതുര അല്ല എങ്കിൽ, നിരക്കു പുറത്ത് ഏതെങ്കിലും പ്രദേശങ്ങളും സുതാര്യമായ അവശേഷിക്കുന്നു (ഒരു ആല്ഫാ ചാനൽ യാന്ത്രികമായി സൃഷ്ടിക്കുന്നു). തീർച്ചയായും, നിങ്ങൾ നിരക്കു അതേ അളവുകൾ ഒരു ഇമേജ് നേടുകയും അതുവഴി താങ്കൾക്ക് ഈ കമാൻഡ് ഉപയോഗിക്കുക മുമ്പ് നിങ്ങളുടെ നിരക്കു പകർത്താനും ഞങ്ങൾക്കുണ്ട്.

ഈ കമാൻഡ് File → Create → From Clipboard കമാൻഡ് പോലെ തന്നെ നടപടി ഉണ്ട്.

3.11.1.1. കമാൻഡ് സജീവമാക്കുക

  • നിങ്ങൾ ഇമേജ് മെനുബാർ നിന്ന് Edit → Paste as → New Image മുഖേന ഈ കമാൻഡ് ആക്സസ് ചെയ്യാൻ കഴിയും.

3.11.2. പുതിയ ലെയർ ഒട്ടിക്കുക

Paste As New Layer കമാൻഡ് സജീവ ചിത്രം ഒരു പുതിയ പാളി സൃഷ്ടിക്കുകയും അതിലേക്കു ക്ലിപ്ബോർഡ് നിന്നും ചിത്രം ഡാറ്റ pastes. ഡാറ്റ രൂപമുള്ള ചതുരാകൃതിയിലുള്ള ചതുര എങ്കിൽ, ക്യാൻവാസ് അറ്റം വരെ നീട്ടാൻ ഏതെങ്കിലും പ്രദേശങ്ങളും സുതാര്യമായ അവശേഷിക്കുന്നു (ആൽഫാ ചാനൽ യാന്ത്രികമായി സൃഷ്ടിക്കുന്നു). തീർച്ചയായും, ഈ കമാൻഡ് ഉപയോഗിക്കുക മുമ്പ് നിങ്ങളുടെ തിരഞ്ഞെടുപ്പ് പകർത്തുക ഞങ്ങൾക്കുണ്ട്.

3.11.2.1. കമാൻഡ് സജീവമാക്കുക

  • നിങ്ങൾ ഇമേജ് മെനുബാർ നിന്ന് Edit → Paste as → New Layer മുഖേന ഈ കമാൻഡ് ആക്സസ് ചെയ്യാൻ കഴിയും.

3.11.3. ന്യൂ ബ്രഷ് ഒട്ടിക്കുക

ഈ കമാൻഡ് പുതിയ ബ്രഷ് പേര് നൽകിയിരിക്കുന്നു ഒരു ഡയലോഗ് വിൻഡോ തുറക്കുന്നു. ബ്രഷ് Brushes dialogue ലഭ്യമാകുന്നു.

3.11.3.1. ഓപ്ഷനുകൾ

ചിത്രം 16.17. “New Brush” ഡയലോഗ്


ബ്രഷ് പേര്

Brush name അതു “Brushes” ചർച്ചയ്ക്കു ഇരിക്കും പേരാണ്.

ഫയലിന്റെ പേര്

പുതിയ ബ്രഷ് നിങ്ങളുടെ സ്വകാര്യ brushes ഫോൾഡറിൽ File name (വിപുലീകരണം .gbr കൂടെ) സംരക്ഷിച്ചു.

അകലം

Spacing: ബ്രഷ് ഒരു വരി ചാടിച്ചു, അത് യഥാർത്ഥത്തിൽ സ്റ്റാമ്പുകൾ ബ്രഷ് ഐക്കൺ ആവർത്തിച്ച്. ബ്രഷ് സ്റ്റാമ്പുകൾ വളരെ അടുത്താണ്, നിങ്ങൾക്ക് ഒരു സോളിഡ് വരിയുടെ തോന്നലും.

3.11.3.2. കമാൻഡ് സജീവമാക്കുക

  • നിങ്ങൾ ഇമേജ് മെനുബാർ നിന്ന് Edit → Paste as → New Brush മുഖേന ഈ കമാൻഡ് ആക്സസ് ചെയ്യാൻ കഴിയും.

3.11.4. പുതിയ മാതൃക ഒട്ടിക്കുക

ഈ കമാൻഡ് നിങ്ങളുടെ പുതിയ മാതൃക പേര് നൽകാൻ അനുവദിക്കുന്ന ഡയലോഗ് വിൻഡോ തുറക്കുന്നു. പാറ്റേൺ Patterns dialogue ലഭ്യമാകുന്നു.

3.11.4.1. ഓപ്ഷനുകൾ

ചിത്രം 16.18. “New Pattern” ഡയലോഗ്


ശ്രേണി പേര്

Pattern name അതു ശ്രേണി ചർച്ചയ്ക്കു ഇരിക്കും പേരാണ്.

ഫയലിന്റെ പേര്

പുതിയ മാതൃക നിങ്ങളുടെ സ്വകാര്യ patterns ഫോൾഡറിൽ File name (വിപുലീകരണം .pat കൂടെ) സംരക്ഷിച്ചു.

3.11.4.2. കമാൻഡ് സജീവമാക്കുക

  • നിങ്ങൾ ഇമേജ് മെനുബാർ നിന്ന് Edit → Paste as → New Pattern മുഖേന ഈ കമാൻഡ് ആക്സസ് ചെയ്യാൻ കഴിയും.
