6.24. New Guide (by Percent)

New Guide (by Percent) കമാൻഡ് ചിത്രം ഒരു ഗൈഡ് ചേർക്കുന്നു. ഗൈഡ് സ്ഥാനം കാൻവാസ് ഉയരവും വീതി ശതമാനമായി വ്യക്തമാക്കുന്നത്.

സൂചന

നിങ്ങൾക്ക് ഈ ഇമേജ് ഭരണാധികാരികൾ നിന്ന് വഴികാട്ടികളേ ക്ലിക്കുചെയ്ത് ചെയ്തോ നിങ്ങൾക്ക് ആഗ്രഹിക്കുന്ന അവരെ പൊസിഷനിങ് വഴി കൂടുതൽ വേഗത്തിൽ ചിത്രത്തിലേക്ക് ഗൈഡുകൾ ചേർക്കാൻ കഴിയും. നിങ്ങൾ ക്ലിക്കിൽ-ആൻഡ്-വലിച്ചു കോരുവാൻ വഴികാട്ടി എന്നിരുന്നാലും, പോലെ തലമുറകള്ക്ക് കമാൻഡ് ഉപയോഗിച്ച് പിടിക്കുകയും പോലെ സ്ഥാനങ്ങളിലാണ്.

6.24.1. Activate the Command

You can access this command from the image menubar through Image → Guides → New Guide (by Percent).

6.24.2. “New Guide (by Percent)” Options

ഈ മെനു ഇനം തിരഞ്ഞെടുക്കുക, ഒരു ഡയലോഗ് തുറക്കുന്നു, പുതിയ ഗൈഡിന്റെ ശതമാനം നിങ്ങളെ Direction ആൻഡ് Position, സജ്ജമാക്കാൻ അനുവദിക്കുന്ന.

ചിത്രം 16.83. The “New Guide (by Percent)” Dialog


സംവിധാനം

ഡ്രോപ്-ഡൌണ് ലിസ്റ്റ് ഉപയോഗിച്ച്, ഗൈഡിന്റെ Direction തിരഞ്ഞെടുക്കാം Horizontal അല്ലെങ്കിൽ Vertical ഒന്നുകിൽ.

സ്ഥാനം

പുതിയ ഗൈഡിന്റെ Position തിരഞ്ഞെടുക്കാം. കോർഡിനേറ്റ് വംശജനായ കാൻവാസ് മുകളിലെ ഇടത് മൂലയിൽ ആണ്.
