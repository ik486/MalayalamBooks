3.3. പാറ്റേണുകൾ സംഭാഷണം
3. ചിത്രം-ഉള്ളടക്കം ബന്ധപ്പെട്ട ഡയലോഗുകൾ  

3.3. പാറ്റേണുകൾ സംഭാഷണം

GIMP, ഒരു pattern വശങ്ങളിലായി പകർപ്പുകൾ സ്ഥാപിച്ച് പ്രദേശങ്ങൾ നിറയ്ക്കാൻ ഉപയോഗിക്കുന്ന ഒരു ചെറിയ ചിത്രം. പാറ്റേണുകൾ അവർ എങ്ങനെ സൃഷ്ടിക്കുകയും ഉപയോഗിക്കാൻ കഴിയും അടിസ്ഥാന വിവരങ്ങൾക്ക് Patterns വിഭാഗം കാണുക.

നിങ്ങൾ Bucket Fill ആൻഡ് Clone ഉപകരണങ്ങളും Fill with pattern കമാൻഡ് അവരെ ഉപയോഗിക്കാം.

“Patterns” ഡയലോഗ് ലിസ്റ്റ് അല്ലെങ്കില് ഗ്രിഡ് കാഴ്ചയിൽ അതിൽ ക്ലിക്കുചെയ്ത് ഒരു പാറ്റേൺ തിരഞ്ഞെടുക്കുന്നതിനായി ഉപയോഗിച്ച: തിരഞ്ഞെടുത്ത പാറ്റേൺ തുടർന്ന് പണിസഞ്ചി ഓഫ് ബ്രഷ് / പാറ്റേൺ / ഗ്രേഡിയന്റ് പ്രദേശത്ത് കാണിക്കും. കുറച്ച് ഡസൻ കൂടുതലോ കുറവോ ക്രമരഹിതമായി തിരഞ്ഞെടുത്ത പാറ്റേണുകൾ ജിമ്പ് വിതരണം ചെയ്തിരിക്കുന്നു, എളുപ്പത്തിൽ നിങ്ങളുടെ സ്വന്തം പുതിയ പാറ്റേണുകൾ ചേർക്കാൻ കഴിയും.

3.3.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Patterns” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • പണിസഞ്ചി നിന്നും, ബ്രഷ് / പാറ്റേൺ / ഗ്രേഡിയന്റ് പ്രദേശത്ത് പാറ്റേൺ ചിഹ്നത്തിൽ ക്ലിക്കുചെയ്തുകൊണ്ട്.

  • ചിത്രം മെനുവിൽ നിന്ന്: Windows → Dockable Dialogs → Patterns;

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Patterns തിരഞ്ഞെടുത്ത്.

  • Clone tool ആൻഡ് Bucket Fill tool ഓഫ് ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗിൽ നിന്നും പാറ്റേൺ സോഴ്സ് ബട്ടൺ മാകാം, നിങ്ങൾ വേഗത്തിൽ പട്ടികയിൽ നിന്നും ഒരു പാറ്റേൺ തിരഞ്ഞെടുക്കുന്നതിനായി അനുവദിക്കുന്ന സമാനമായ പ്രവർത്തനപരതയോടുകൂടിയ പോപ്പ്-അപ്പ് നേടുക; നിങ്ങൾ ബക്കറ്റ് ന് clic എങ്കിൽ പോപ്പ്-അപ്പ് വലത് താഴെ ബട്ടൺ ഇന്നത്തെ നിറയ്ക്കുക, നിങ്ങൾ യഥാർത്ഥ പാറ്റേൺ ഡയലോഗ് തുറക്കും. നിങ്ങളുടെ മുൻഗണനകളും അനുസരിച്ച് പോപ്പ്-അപ്പ് തിരഞ്ഞെടുത്ത ഒരു പാറ്റേൺ മാത്രമേ നിലവിൽ സജീവ പ്രയോഗം, മറ്റ് പെയിന്റ് ഉപകരണങ്ങൾ ബാധകമാകാമെന്ന, ശ്രദ്ധിക്കുക. കൂടുതൽ വിവരങ്ങൾക്ക് Tool Option Preferences വിഭാഗം കാണുക.

3.3.2. പാറ്റേൺ ഡയലോഗ് ഉപയോഗിച്ച്

ഗ്രിഡ് / പട്ടിക മോഡുകൾ

ടാബ് മെനുവിൽ View as Grid ആൻഡ് View as List തമ്മിലുള്ള തിരഞ്ഞെടുക്കാം. ഗ്രിഡ് മോഡിൽ, പാറ്റേണുകൾ പുറത്തു ചതുരാകൃതിയിലുള്ള അണിനിരന്നു, എളുപ്പമാക്കുന്നു പല കണ്ടു നിന്നെ തിരയുന്ന കണ്ടെത്താനായിട്ടില്ല making കിടക്കുന്നു. പട്ടിക മോഡിൽ, പാറ്റേണുകൾ അപ്പ് ഒരു ലിസ്റ്റിൽ, അവർക്ക് പുറമെ പേരുകൾ കൂടിയ ചെയ്യുന്നു.

സൂചന

ഒരു പാറ്റേൺ യഥാർത്ഥ വലിപ്പം ഇൻഡിപെൻഡൻറ് എല്ലാ പാറ്റേണുകൾ ഡയലോഗിൽ ഒരേ വലിപ്പം കാണിച്ചിരിക്കുന്നു. നിങ്ങൾ ലിസ്റ്റിൽ ഡയലോഗ് അല്ലെങ്കിൽ ഗ്രിഡ് കാണുക എന്നത് ഒരു വിഷയമല്ല - അങ്ങനെ വലിയ പാറ്റേണുകൾ ഈ നിങ്ങൾക്ക് എല്ലാ ഡയലോഗിൽ മാതൃകയുടെ ഒരു ഭാഗം മാത്രമേ കാണാൻ എന്നാണ്. പൂർണ്ണ പംക്തി നിങ്ങൾ കേവലം ഒരു രണ്ടാം പാറ്റേൺ and hold the mouse button ക്ലിക്ക് ചെയ്യുക.
കുറിപ്പു്

ടാബ് മെനുവിൽ, ഓപ്ഷൻ Preview Size നിങ്ങളുടെ ശബ്ദമോ പാറ്റേൺ പ്രിവ്യൂകളുടെ വലിപ്പം ചെയ്യൽ അനുവദിക്കുന്നു.

ചിത്രം 15.37. പാറ്റേണുകൾ ഡയലോഗ്

പട്ടിക കാഴ്ച

ഗ്രിഡ് മോഡ്


പാറ്റേണുകൾ ഡയലോഗ് (ഗ്രിഡ് മോഡ്) ഉപയോഗിക്കുന്നു

മുകളിൽ നിലവിൽ തിരഞ്ഞെടുത്ത പാറ്റേണുകളുടെ പേര് കാണുന്നു, പിക്സലിൽ അതിന്റെ തലങ്ങളും.

കേന്ദ്രത്തിൽ അനുസരിച്ചുള്ള നിലവിൽ തിരഞ്ഞെടുത്ത ഒറ്റ, ലഭ്യമായ എല്ലാ പാറ്റേണുകളുടെ ഒരു ഗ്രിഡ് കാഴ്ച ലഭ്യമാകുന്നു. അവരിൽ ഒന്നിൽ ക്ളിക്കു ജിമ്പ് നിലവിലെ പാറ്റേൺ അത് സജ്ജമാക്കുന്നു, അതു പണിസഞ്ചി ഓഫ് ബ്രഷ് / പാറ്റേൺ / ഗ്രേഡിയന്റ് പ്രത്യക്ഷപ്പെടുന്ന കാരണമാകുന്നു.

പാറ്റേണുകൾ ഡയലോഗ് (പട്ടിക കാഴ്ച) ഉപയോഗിക്കുന്നു

ഈ കാഴ്ചയിൽ, പകരം ഒരു ഗ്രിഡിന്റെ നിങ്ങൾ പാറ്റേണുകളുടെ ഒരു ലിസ്റ്റ് ഓരോ പേര് വ്യാപ്തി ലേബൽ കാണുന്നു. ലിസ്റ്റിലെ ഒരു വരി ക്ലിക്കുചെയ്യുന്നത് ഗ്രിഡിലെ കാഴ്ചയിൽ ചെയ്യുന്നതു പോലെ, ജിമ്പ് നിലവിലെ പാറ്റേൺ ആ പാറ്റേൺ സജ്ജമാക്കുന്നു.

നിങ്ങൾ ഒരു പാറ്റേൺ നാമത്തിൽ double-click, നിങ്ങൾ പേര് എഡിറ്റ് കഴിയും. നിങ്ങൾ നിങ്ങൾതന്നെ ജിമ്പ് വിതരണം എന്ന് പശുക്കൾ ചേർത്ത പാറ്റേണുകൾ പേരുമാറ്റാൻ അനുവദിച്ചിരിക്കുന്ന ശ്രദ്ധിക്കുക. നിങ്ങൾ മാറ്റാനുള്ള അനുമതി ചെയ്യാത്ത ഒരു പേര് എഡിറ്റുചെയ്തിട്ടുണ്ടെങ്കിൽ ഉടൻ മടങ്ങാൻ അല്ലെങ്കിൽ മറ്റൊരു നിയന്ത്രണം നീക്കും തട്ടുന്നതിന് പോലെ, പേര് വീണ്ടും അതിന്റെ മുൻ മൂല്യം പഴയപടിയാകും.

ലിസ്റ്റ് കാഴ്ചയിൽ മറ്റെല്ലാം ഗ്രിഡിലെ കാഴ്ചയിൽ ചെയ്യുന്നതു അതേ പ്രവർത്തിക്കുന്നു.

പാറ്റേൺ ഇല്ലാതാക്കുക

ഈ ബട്ടൺ അമർത്തിയാൽ പട്ടികയിൽ നിന്നും പാറ്റേൺ നീക്കംചെയ്യുന്നു അതു പ്രതിനിധീകരിച്ച് ഫയൽ ഡിസ്കിൽ നിന്നും ഇല്ലാതാക്കി കാരണമാകുന്ന. നിങ്ങൾ ജിമ്പ് വിതരണം, സിസ്റ്റം patterns ഡയറക്ടറിയിൽ ഇൻസ്റ്റോൾ മാതൃകൾ ഏതെങ്കിലും നീക്കം ശ്രദ്ധിക്കുക; മാത്രമേ താങ്കൾ എഴുതുന്ന അനുമതിയുള്ള ഫോൾഡറുകൾ ലേക്ക് നിങ്ങൾ ചേർത്ത പാറ്റേണുകൾ നീക്കംചെയ്യാൻ കഴിയും.

പുതുക്കുക പാറ്റേണുകൾ

ഈ ബട്ടൺ അമർത്തിയാൽ ലിസ്റ്റ് ഏതെങ്കിലും പുതുതായി കണ്ടെത്തിയ പാറ്റേണുകൾ ചേർത്ത്, നിങ്ങളുടെ പാറ്റേൺ തിരയൽ പാതയിൽ ഫോൾഡറുകൾ ഐഎസ്ഒയില് ലേക്കുള്ള GIMP കാരണമാകുന്നു. ഒരു ഫോൾഡർ പുതിയ പാറ്റേണുകൾ ചേർക്കുക, GIMP പുനരാരംഭിക്കുന്നതിന് ഇല്ലാതെ അവ ലഭ്യമാക്കുക ചെയ്യണമെങ്കിൽ ഈ ബട്ടൺ ഉപയോഗപ്പെടുന്നു.

ഇമേജ് ആയി തുറക്കുക പാറ്റേൺ

ഈ ബട്ടൺ ക്ലിക്ക്, നിലവിലെ പാറ്റേൺ ഒരു പുതിയ ഇമേജ് വിൻഡോയിൽ തുറന്നു നോക്കും. അതുപോലെ, നിങ്ങൾ തിരുത്താം. എന്നാൽ നിങ്ങൾ ഒരു പുതിയ പേര്, .pat അതു രക്ഷിക്കാൻ ശ്രമിക്കുക, ഈ ഇമേജ് ഫയൽ “root” കാരണം ഒരു “Denied permission ” പ്രശ്നം .പുറമെയുള്ള ചെയ്യും. എന്നാൽ ഈ സാധ്യത കുറവ് സംരക്ഷിത, വിൻഡോസ് കീഴിലാണ്.

3.3.3. ടാഗ്ചെയ്യൽ

You can use tags to reorganize the patterns display. See വിഭാഗം 3.6, “ടാഗ്ചെയ്യൽ”.

3.3.4. ശ്രേണി സന്ദർഭ മെനു

നിങ്ങൾ “Patterns” ഡയലോഗ് റൈറ്റ്-ക്ലിക്കുചെയ്തുകൊണ്ട് നേടുക. ഈ മെനുവിൽ കല്പനകൾ ക്ലിപ്പ്ബോർഡിലേക്ക് കയറി മാതൃക വഴിയും പകർത്തുക അനുവദിക്കുന്ന Copy Location ഒഴികെ ബട്ടണുകൾ വിവരിച്ചിരിക്കുന്നു.

3.3.5. ക്ലിപ്ബോർഡ് പാറ്റേൺ

നിങ്ങൾ പകർത്തി അല്ലെങ്കിൽ മുറിക്കുക കമാൻഡ് ഉപയോഗിക്കുമ്പോൾ, ഒരു പകർപ്പ് പാറ്റേണുകൾ ഡയലോഗ് മുകളിലെ ഇടത് മൂലയിൽ ഒരു പുതിയ മാതൃക ദൃശ്യമാകുന്നു. വീണ്ടും പകർത്തുക (അല്ലെങ്കിൽ മുറിക്കുക) കമാൻഡ് ഉപയോഗിക്കാം വരെ ഈ ബ്രഷ് തുടരുകയും ചെയ്യും. നിങ്ങൾ ജിമ്പ് അടയ്ക്കുമ്പോൾ ഇത് അപ്രത്യക്ഷമാകും.
