2.5. അവ്യക്തമായ നിരക്കു (മാന്ത്രികനായാണ്)

ചിത്രം 14.21. Magic Wand tool icon in the Toolbox


അവ്യക്തമായ തിരഞ്ഞെടുക്കുക (മാന്ത്രികനായാണ്) ടള നിറം സമാനത അടിസ്ഥാനമാക്കി നിലവിലെ അല്ലെങ്കിൽ പാളി ചിത്രത്തിന്റെ പ്രദേശങ്ങൾ തിരഞ്ഞെടുക്കുക രൂപകല്പന.

ഈ ഉപകരണം ഉപയോഗിക്കുമ്പോൾ തുടക്കം തിരഞ്ഞെടുക്കുവാനും വളരെ പ്രധാനമാണ്. നിങ്ങൾ തെറ്റായ സ്പോട്ട് തിരഞ്ഞെടുക്കുകയാണെങ്കിൽ, നീ എന്തു ചെയ്യണമെന്ന് നിന്ന് വളരെ വ്യത്യസ്തമായി എന്തെങ്കിലും, അല്ലെങ്കിൽ സമ്മുഖ ലഭിക്കും.

തല്ലാണോ മൂർച്ചയുള്ള അരികുകളുള്ള വസ്തുക്കൾ തിരഞ്ഞെടുത്ത് ഒരു നല്ല ഉപകരണമാണ്. സാര് പലപ്പോഴും അതു ഒരു ഉപയോഗിച്ച് ആരംഭിക്കുന്നതിന് അങ്ങനെ ഉപയോഗിക്കാൻ രസകരമാണ്. നിങ്ങൾ ഒരുപക്ഷേ, നിങ്ങൾ അതു കൂടുതൽ കൂടുതൽ ഉപയോഗിക്കുമ്പോൾ കൂടുതൽ നിരാശരായി നിങ്ങൾ എന്താണ് വേണ്ടതെന്ന് കൃത്യമായി തിരഞ്ഞെടുക്കാനുള്ള പ്രയാസത്തോടെ ഇനി ആരും കുറവ് മാറും എന്നിരുന്നാലും കണ്ടെത്തും. കൂടുതൽ ഉപയോക്താക്കളാണ് Path ആൻഡ് Color Select ഉപകരണങ്ങൾ പലപ്പോഴും കൂടുതൽ കാര്യക്ഷമമായി ആകുന്നു, കുറവ് തല്ലാണോ ഉപയോഗിക്കുന്ന കണ്ടെത്താൻ. എങ്കിലും, അത് ഒരു കോണ്ടൂർ ഉള്ളിൽ ഒരു പ്രദേശത്തെ, അല്ലെങ്കിൽ അപൂർണ തിരഞ്ഞെടുക്കലുകളെ അപ്പ് തൊട്ടതു ഉപയോഗപ്രദമാണ്. ഇത് പലപ്പോഴും ഒരു സോളിഡ്-നിറമുള്ള (അല്ലെങ്കിൽ ഏതാണ്ട് solid- നിറമുള്ള) പശ്ചാത്തല ഏരിയയിൽ തിരഞ്ഞെടുത്ത് വളരെ നന്നായി പ്രവർത്തിക്കുന്നു.

Threshold ഓപ്ഷൻ അനുസരിച്ച് അത് ചെറിയ വിടവുകൾ ചാടി കഴിവുള്ള: തിരഞ്ഞെടുത്ത പ്രദേശത്ത് കേന്ദ്രത്തിൽ നിന്നും പുറമെയുള്ള വലുതാവുമ്പോൾ അതു മാത്രമല്ല പരസ്പരം സ്പർശിക്കുന്ന പിക്സൽ വ്യാപിപ്പിക്കുന്നതിന് കാര്യം ശ്രദ്ധിക്കുക. / കുറയുന്നു ത്രെഷോൾഡ് വർദ്ധിപ്പിക്കാൻ, വ്യക്തമല്ലാത്ത തിരഞ്ഞെടുക്കൽ ഉപയോഗം സമയത്ത്, ആദ്യ ബട്ടൺ-അമർത്തുക ശേഷം പോയിന്റർ വലിച്ചോ താഴേ (അല്ലെങ്കിൽ വലതുവശത്ത്) അല്ലെങ്കിൽ മേലോട്ടു (ഇടത്തോട്ടോ).

2.5.1. Activating the tool

You can access the Magic Wand Tool in different ways:

  • From the image menu bar Tools → Selection Tools → Fuzzy Select,

  • by clicking on the tool icon in the ToolBox,

  • by using the keyboard shortcut U.

ചിത്രം 14.22. മാന്ത്രികനായാണ് ടൂൾ ഉപയോഗിച്ച്: തിരഞ്ഞെടുത്ത പിക്സലുകൾ ഐലന്റ് ആകുന്നു


ആരുടെ നിറങ്ങൾ ആരംഭ പിക്സൽ സമാനമായ ഐലന്റ് പിക്സലുകൾ തിരഞ്ഞെടുക്കുന്നത് ചിത്രം ഒരു സ്ഥാനത്താണ് ക്ലിക്കുചെയ്യുമ്പോൾ അത് തിരഞ്ഞെടുത്ത് ആരംഭിക്കുന്നു, വെള്ളം, വെള്ളപ്പൊക്കം താഴ്ന്ന പ്രദേശങ്ങളിൽ പോലെ പുറത്തേക്ക് ആനുപാതികമായി,. നിങ്ങൾ താഴോട്ട് അല്ലെങ്കിൽ വലതുവശത്ത് മൗസ് ഇഴച്ച് സാമ്യം ഉമ്മരപ്പടി നിയന്ത്രിക്കാനാകും: അകന്ന നിങ്ങൾ അത് ഡ്രാഗ് നിങ്ങൾ തിരഞ്ഞെടുത്ത മേഖല നേടുകയും വലുതാക്കി. നിങ്ങൾ .ഒമ്പതംഗങ്ങളില് ഇടത്തോട്ടോ വലിച്ചിടുന്നതിലൂടെയോ നിരക്കു കുറയ്ക്കാൻ കഴിയും.

ചിത്രം 14.23. Tool Options for the Magic Wand tool

ഈ എല്ലാ ഉപകരണങ്ങൾ സാധാരണമാണ് ഉപാധികൾക്ക് സഹായത്തിന് Selection Tools കാണുക. ഈ ഉപകരണം മാത്രമായുള്ള ഓപ്ഷനുകൾ മാത്രം ഇവിടെ വിശദീകരിച്ചു ചെയ്യുന്നു.

ഈ ഓപ്ഷൻ മാന്ത്രികനായാണ് പൂർണമായും സുതാര്യമായ പ്രദേശങ്ങളിൽ തിരഞ്ഞെടുക്കുക കഴിവ് നൽകുന്നു. എങ്കിൽ ഈ ഓപ്ഷൻ ചെക്കുചെയ്തിട്ടില്ലെങ്കിൽ സുതാര്യമായ പ്രദേശങ്ങൾ നിരക്കു ഉൾപ്പെടുത്തിയിട്ടുണ്ട് ഒരിക്കലും.

സാമ്പിൾ ലയിപ്പിച്ചു

നിങ്ങളുടെ ചിത്രത്തിന്റെ നിരവധി പാളികൾ എത്തുമ്പോൾ ഈ ഓപ്ഷൻ പ്രസക്തമായ വർധിക്കുകയും സജീവ പാളി ഒന്നുകിൽ അർദ്ധസുതാര്യം അല്ലെങ്കിൽ സാധാരണ മറ്റൊരു പാളി മോഡ് സജ്ജമാക്കിയിരിക്കുന്നു. ഈ സാഹചര്യത്തിൽ ആണെങ്കിൽ, പാളിയായി ഇപ്പോഴത്തെ നിറങ്ങൾ സംയുക്ത സ്വരൂപത്തിൽ നിറങ്ങൾ നിന്ന് വ്യത്യസ്തമായിരിക്കും. “Sample Merged” ഓപ്ഷൻ പരിശോധിക്കാതെ എങ്കിൽ, തല്ലാണോ മാത്രം നിറം ലേക്കുള്ള സജീവ പാളി അത് ഒരു നിര സൃഷ്ടിക്കുമ്പോൾ പ്രതികരിക്കും. അതു ചെക്കുചെയ്തിട്ടുണ്ടെങ്കിൽ അത് ദൃശ്യമായ എല്ലാ പാളികൾ സംയുക്ത നിറം .ശക്തവും. കൂടുതൽ വിവരങ്ങൾക്ക്, ഗ്ലോസ്സറി Sample Merged കാണും.

ഉമ്മറം

തട്ടിച്ചുനോക്കുമ്പോൾ, വലിയ കാരണമാകുന്നു നിരക്കു: ഈ സ്ലൈഡർ നിമിഷം നിങ്ങൾ വലിച്ചുകൊണ്ട് മുമ്പ്, പ്രാരംഭ പോയിന്റിൽ പോയിന്റർ ക്ലിക്ക് ന് തിരഞ്ഞെടുക്കും നിറങ്ങൾ പരിധി നിർണയിക്കുന്നത്. ആദ്യ ബട്ടൺ അമർത്തുക ശേഷം താഴേ വലത്തോട്ടോ സൂചിക വലിച്ചോ നിരക്കു വലിപ്പം വർദ്ധിപ്പിക്കും; മേലോട്ടു വലിച്ചിട്ടുകൊണ്ടോ ഇടത്തേക്ക് കുറയും. ഇപ്രകാരം, നിങ്ങൾ പരിഗണിക്കാതെ പരിധി ക്രമീകരണം സാധ്യതകൾ ഒരേ വെച്ചിരിക്കുന്നു; എന്തു വ്യത്യാസപ്പെട്ടിരിക്കുന്നു നിങ്ങൾ ആഗ്രഹിക്കുന്ന ഫലം ലഭിക്കാൻ ചെയ്യേണ്ടത് വലിച്ചോ അളവാണ്.
