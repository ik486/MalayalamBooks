7.16. Raise Layer

Raise Layer കമാൻഡ് പാളി സ്റ്റാക്ക് സജീവ പാളി ഒരു സ്ഥാനം രേഖപ്പെടുത്തുന്നു. സജീവ പാളി മുകളിൽ ഇതിനകം അല്ലെങ്കിൽ ഒരേയൊരു പാളി ഉണ്ടെങ്കിൽ, ഈ മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്. എങ്കിൽ സജീവ പാളി സ്റ്റാക്ക് ചുവടെ അത് ഒരു ആല്ഫാ ചാനൽ ഇല്ലെങ്കിൽ, അതിനെ ഒരു ആൽഫ ചാനൽ ചേർക്കുക വരെ ഉയിർത്തെഴുന്നേൽക്കയും കഴിയില്ല.

7.16.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Stack → Raise Layer,

  • അല്ലെങ്കിൽ ലെയറുകൾ ഡയലോഗ് താഴെയുള്ള മുകളിലേക്കുള്ള അമ്പടയാളം ഐക്കണിൽ ക്ലിക്കുചെയ്ത്.
