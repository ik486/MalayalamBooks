3.14. Blur/Sharpen

ചിത്രം 14.93. Blur/Sharpen tool icon in the Toolbox


ഇഫക്റ്റ് / മൂര്ച്ചകൂട്ടല്പ്രഭാവംടോഗിള്ചെയ്യുക പ്രയോഗം പ്രാദേശികമായി മങ്ങിക്കുന്നതോ നിങ്ങളുടെ ചിത്രം മൂർച്ച നിലവിലെ ബ്രഷ് ഉപയോഗിക്കുന്നു. നിങ്ങളുടെ ചിത്രത്തിൻറെ ഏതെങ്കിലും ഗ്രൂപ്പിലെ വളരെയധികം നിലകൊള്ളുന്നു, നിങ്ങൾ അത് ഹാർഡ്നെസ്സ് ആഗ്രഹിക്കുന്നുവെങ്കിൽ അത് കൊണ്ട് ബ്ലറിംഗോ ഉപയോഗപ്പെടും. നിങ്ങൾ ഒരു മുഴുവൻ പാളി, ഒന്നോ ഒരു വലിയ ഭാഗം ബ്ലർ ആഗ്രഹിക്കുന്നുണ്ടെങ്കിൽ, നിങ്ങൾ ഒരുപക്ഷേ Blur Filters ഒന്ന് ഉപയോഗിച്ച് നല്ലത് ചെയ്യും. ഒരു brushstroke ദിശയിൽ ഫലമില്ല: നിങ്ങൾ ദിശയിലുമുള്ള മങ്ങിക്കലിന് ആഗ്രഹിക്കുന്നുവെങ്കിൽ, ഉപയോഗിക്കാൻ സ്മഡ്ജ് ടൂൾ.

“Sharpen” മോഡിൽ, ഉപകരണം എവിടെ ബ്രഷ് പ്രയോഗിക്കുന്നു തീവ്രത വർദ്ധിച്ചുവരുന്ന പ്രവർത്തിക്കുന്നു. ഈ കുറച്ചു പ്രയോജനപ്പെടുന്നത്, പക്ഷേ over- അപ്ലിക്കേഷൻ മുഴക്കം താനേ വന്നുകൊള്ളും. Enhancement Filters ചില പ്രത്യേകിച്ച് Unsharp Mask, ഒരു പാളി മേഖലകളിലെ മൂർച്ച വളരെ ക്ലീനറും ജോലി ചെയ്യാൻ.

സൂചന

നിങ്ങൾ ക്ലോൺ ഉപകരണം ഉപയോഗിച്ച് ഒരു വഴക്കങ്ങൾ ഷാർപ്പെനിംഗോ ബ്രഷ് സൃഷ്ടിക്കാൻ കഴിയും. ഇത് ചെയ്യുന്നതിന്, നിങ്ങൾ പ്രവർത്തിക്കുന്ന ഒരു ഷാർപ്പെനിംഗോ ഫിൽട്ടർ പ്രവർത്തിപ്പിക്കാൻ ആഗ്രഹമുണ്ടോ പാളി പകർപ്പിൽ, അൺഷാർപ്പ് മാസ്ക് പോലുള്ള തനിപ്പകർപ്പാക്കുന്നത് ആരംഭിക്കുക. അപ്പോൾ ക്ലോൺ ഉപകരണം സജീവമാക്കുന്നതിന്, അതിന്റെ ടൂൾ ഓപ്ഷനുകൾ ലെ “Registered” ലേക്ക് “Image source” ആൻഡ് അലൈന്മെന്റ് ലേക്ക് ഉറവിടം സജ്ജമാക്കി. പോലുള്ള സ്രോതസ്സ് ചിത്രം ഉണ്ടാക്കുവാൻ പകർപ്പിൽ അപ്പോൾ 10. Ctrl -click, കഴിയാൻ മൂല്യം ചുമർചിത്രത്തിന്റെ സജ്ജമാക്കുക. ഇപ്പോൾ യഥാർത്ഥ പാളി ന് പെയിന്റ്, നിങ്ങൾ ഒരുമിച്ചു, ബ്രഷ് എവിടെ പ്രയോഗിക്കപ്പെടുന്നതിനെത്തുടർന്ന് unsharpened പതിപ്പുമായി മൂർച്ചകൂട്ടി പതിപ്പ് ഇളക്കുക, ചെയ്യും.

രണ്ടും മങ്ങിക്കലും വർദ്ധനവനുസരിച്ച് പണി മൂർച്ച: വിസ്തൃതിയിൽ ആവർത്തിച്ച് ബ്രഷ് കൊണ്ട് പരസ്പരം അധിക പാസ് ഉപയോഗിച്ച് പ്രാബല്യത്തിൽ വർദ്ധിപ്പിക്കും. നിരക്ക് നിയന്ത്രണം താങ്കൾ കൈവരിക്കും എങ്ങനെ വേഗത്തിൽ നിർണ്ണയിക്കാൻ അനുവദിക്കുന്നു. അതാര്യത നിയന്ത്രണം എന്നിരുന്നാലും, പരിഗണിക്കാതെ തീർത്തതു എത്ര പാസുകൾ, ഒരു ഒറ്റ brushstroke നിർമ്മിക്കുന്ന കഴിയും ബ്ലറിംഗ് അളവ് പരിമിതപ്പെടുത്താൻ ഉപയോഗിക്കാൻ കഴിയും.

3.14.1. ടൂൾ സജീവമാക്കുന്നു

ഉപകരണം സജീവമാക്കുന്നതിന് വിവിധ സാധ്യതകളും ഉണ്ട്:

  • From the image-menu: Tools → Paint tools → Blur/Sharpen.

  • The Tool can also be called by clicking the tool icon: in the Toolbox.

  • കീബോർഡ് കുറുക്കുവഴി Shift+U ഉപയോഗിക്കുന്നതിലൂടെ.

3.14.2. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

എല്ലാ പെയിന്റ് പ്രയോഗങ്ങളിൽ അതേ പ്രതീതി കീ മോഡിഫയറുകൾക്കുള്ളകീബോർഡ് ഒരു വിവരണം വേണ്ടി Paint Tools' Common Features കാണുക.

Ctrl,

ഇഫക്റ്റ്, മോഡുകൾ മൂർച്ച തമ്മിലുള്ള Ctrl കീ ടോഗിൾ താഴേയ്ക്കാക്കുമ്പോൾ; അതു ടൂൾ ഓപ്ഷനുകൾ കാണിച്ചിരിക്കുന്ന ക്രമീകരണം ദിശമാറുന്നതുമായ.

3.14.3. ഓപ്ഷനുകൾ

ചിത്രം 14.94. Tool Options for the Blur/Sharpen tool


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

Opacity; Brush; Dynamics; Dynamics Options; Apply Jitter; Hard Edges
പല അല്ലെങ്കിൽ എല്ലാ പെയിന്റ് ഉപകരണങ്ങൾ ബാധകമായ ഉപകരണം ഓപ്ഷനുകൾ ഒരു വിവരണം വേണ്ടി Common Paint Tool Options കാണുക.
Convolve Type

Blur മോഡ് അതുവഴി brushstroke ഏരിയയ്ക്ക് ഉള്ളിൽ പിക്സൽ സാമ്യം വർധിപ്പിക്കുകയും അയൽ പിക്സലുകൾ ഉള്ള ശക്തരോ ചെയ്യേണ്ട ബ്രഷ് ബാധിച്ച ഓരോ പിക്സൽ കാരണമാകുന്നു. Sharpen മോഡ് ഓരോ പിക്സൽ മുമ്പ് ആയിരുന്നു അതിൻറെ കൂട്ടുകാരെ കൂടുതൽ വ്യത്യസ്ത ആകാൻ കാരണമായ: അതു brushstroke പ്രദേശത്ത് അകത്ത് തീവ്രത വർദ്ധിപ്പിക്കുന്നു. വളരെയധികം മൂര്ച്ചകൂട്ടല്പ്രഭാവംടോഗിള്ചെയ്യുക ഒരു വൃത്തികെട്ട flocculation വശം അവസാനിക്കുന്നത്. നിങ്ങൾ ഇവിടെ തിരഞ്ഞെടുക്കുന്ന സജ്ജീകരിക്കുന്നത് എന്തുമാകട്ടെ, Ctrl കീ പിടിച്ച് മടങ്ങാം ഞൊടിയിടയിൽ.

“Convolve” മാട്രിക്സ് ഉപയോഗിച്ച് ഒരു ഗണിത രീതി സൂചിപ്പിക്കുന്നു.
