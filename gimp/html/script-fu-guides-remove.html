6.26. Remove all guides

Remove all Guides കമാൻഡ് ഇമേജിൽ നിന്നും എല്ലാ ഗൈഡുകൾ നീക്കം. ക്ലിക്കുചെയ്യുന്നത്-ആൻഡ്-വലിച്ചിടുമ്പോൾ ഒരു പ്രമാണി ലേക്കായി ഒന്നോ രണ്ടോ വഴികാട്ടികളേ അവരെ നീക്കം ദ്രുതഗതിയിലുള്ള വഴി. നിരവധി ഗൈഡുകൾ സ്ഥാനങ്ങളിലാണ് എങ്കിൽ ഈ കമാൻഡ് ഉപയോഗപ്പെടുന്നു.

6.26.1. Activate the Command

You can access this command from the image menubar through Image → Guides → Remove all guides.
