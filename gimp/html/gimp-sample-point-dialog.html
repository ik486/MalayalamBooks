5.7. സാമ്പിൾ പോയിന്റുകൾ സംഭാഷണം
5. പലവക. ഡയലോഗുകൾ  

5.7. സാമ്പിൾ പോയിന്റുകൾ സംഭാഷണം

Color Picker ഒരു പിക്സൽ കുറിച്ച് നിറം വിവരങ്ങൾ പ്രദർശിപ്പിക്കാൻ കഴിയും സമയത്ത്, “Sample Points” ഡയലോഗ് ഒരേ സമയം, സജീവ പാളി അല്ലെങ്കിൽ ചിത്രം നാലു പിക്സൽ ഡാറ്റ പ്രദർശിപ്പിക്കാൻ കഴിയും. മറ്റൊരു പ്രധാന വ്യത്യാസം ഇമേജിൽ പ്രവർത്തിക്കുന്നു പോലെ ഇവർ പോയിന്റുകൾ മൂല്യങ്ങൾ തത്സമയം മാറ്റി എന്നതാണ്.

5.7.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Sample Points” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • Windows → Dockable Dialogs → Sample Points: ഇമേജ് മെനുവിൽ നിന്ന്.

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും PLACEHOLDER,-100693 ഞെക്കി, Add Tab → Sample Points തിരഞ്ഞെടുത്ത്.

5.7.2. സാമ്പിൾ പോയിന്റുകൾ ഉപയോഗിക്കുന്നു

ഒരു സാമ്പിൾ പോയിന്റ് സൃഷ്ടിക്കുന്നതിന്, Ctrl ഇമേജ് വിൻഡോയുടെ രണ്ടു അളവു നിയമങ്ങൾ ഒന്ന് -click മൌസ് പോയിന്റർ വലിച്ചിടുക. രണ്ട് ലംബമായി വഴികാട്ടികൾ ദൃശ്യമാകും. ഇരുവരും ഗൈഡുകൾ കൂട്ടിമുട്ടുന്ന എവിടെ സാമ്പിൾ ബിന്ദുവാണ്. നിങ്ങൾ താഴെ ഇടത് മൂലയിൽ ഇമേജ് ജാലകത്തിന്റെ വിവരങ്ങൾ ബാറിലെ അതിന്റെ നിർദ്ദേശാങ്കങ്ങൾ കാണാൻ കഴിയും. മൌസ് ബട്ടൺ റിലീസ്.

നിങ്ങൾ, Ctrl നേടുകയും + reticle ക്ലിക്ക്-ആൻഡ്-വലിച്ചോ ചട്ടം നിന്ന്.

സ്വതവേ, ഈ സാമ്പിൾ പോയിന്റ് ഒരു ചുറ്റും അടയാളം ഒരു ഓർഡർ നമ്പർ വരുന്നു. നിങ്ങൾ View മെനുവിലെ Show Sample Points ഓപ്ഷൻ അൺചെക്കുചെയ്ത് ഈ മാർക്ക് റദ്ദാക്കാം.

നിങ്ങൾ ഒരു സാമ്പിൾ പോയിന്റ് സൃഷ്ടിക്കുമ്പോൾ “Sample Points” ഡയലോഗ് സ്വയം തുറക്കും. ഈ കേസ് അല്ല; കായികമായി തുറക്കാൻ ഞങ്ങൾക്കുണ്ട്.

നിങ്ങൾക്ക് ഒരു സാമ്പിൾ പോയിന്റ്, ക്ലിക്ക്-ആൻഡ്-വലിച്ചിടുമ്പോൾ ഒരു ഭരണം അതു വഴി, നിങ്ങളെ ഗൈഡുകൾ കൂടെ ചെയ്യുന്നതുപോലെ ഇല്ലാതാക്കാനാകും. ഓർഡർ നമ്പറുകൾ യാന്ത്രികമായി ഡയലോഗ് വിൻഡോയിൽ വീണ്ടും ക്രമീകരിച്ചിരിക്കുന്നത്; ഏറ്റവും പുതിയ ഒരു റാങ്കിന് മുകളിലേയ്ക്ക് നീക്കി ചെയ്യുന്നു.

സ്വതവേ, ഒരുപറ്റം എല്ലാ ലേയറുകളിൽ നടപ്പാകും. നിങ്ങൾ സജീവ പാളി ന് രുചിച്ച് ആഗ്രഹിക്കുന്നുണ്ടെങ്കിൽ, ടാബ് മെനുവിലെ Sample merged ഓപ്ഷൻ അൺചെക്ക്:

ചിത്രം 15.78. “Sample Point” മെനു


5.7.3. “Sample Points” ഡയലോഗ് വിവരണം

ചിത്രം 15.79. സാമ്പിൾ പോയിന്റുകൾ ഡയലോഗ്


നാല് സാമ്പിൾ പോയിന്റ് കുറിച്ചുള്ള വിവരങ്ങൾ ഈ ജാലകത്തിൽ പ്രദർശിപ്പിച്ചിരിക്കുന്നു. നിങ്ങൾ നിലവിലുള്ള ഒപ്പം ദൃശ്യമാക്കില്ലെങ്കിലും ഏത്, കൂടുതൽ സൃഷ്ടിക്കാൻ കഴിയും. കാണിക്കുന്നതിന്, നിങ്ങൾ പ്രദർശിപ്പിച്ച പോയിന്റ് ഇല്ലാതാക്കാൻ ഞങ്ങൾക്കുണ്ട്.

സാമ്പിളുകൾ പോയിന്റ് നിറം ഒരു Swatch ബോക്സിൽ പ്രദർശിപ്പിക്കുന്നു.

ഡ്രോപ്പ്-ഡൌൺ ലിസ്റ്റിൽ തമ്മിൽ തിരഞ്ഞെടുക്കാം:

പിക്സൽ

ഈ ചോയ്സ് 0 255 നും നമ്പറുകൾക്ക്, പിക്സൽ Red, Green, Blue ആൻഡ് Alpha മൂല്യങ്ങൾ പ്രദർശിപ്പിക്കുന്നു.

ആർജിബി

ഈ ചോയ്സ് ശതമാനം പോലെ പിക്സൽ Red, Green, Blue ആൻഡ് Alpha മൂല്യങ്ങൾ പ്രദർശിപ്പിക്കുന്നു. ഇത് പിക്സലിന്റെ നിറം ഹെക്സാഡെസിമൽ മൂല്യം കാണിക്കുന്നു.

HSV

ഈ ചോയ്സ് Hue, പദവികൾ, പിക്സൽ, ശതമാനവും ആയി പ്രദർശിപ്പിക്കുന്നു അതുപോലെ Saturation, Value ആൻഡ് Alpha.

CMYK

ഈ ചോയ്സ് ശതമാനം പോലെ പിക്സൽ Cyan, Magenta, Yellow, Black ആൻഡ് Alpha മൂല്യങ്ങൾ പ്രദർശിപ്പിക്കുന്നു.

ഡാറ്റ നിറം മോഡൽ ഓരോ ചാനലിനായി വിതരണം. ആൽഫാ ചിത്രം ആൽഫാ ചാനൽ താങ്ങി മാത്രമേ അതില്ല.

Hexa മാത്രം ആർജിബി മോഡ് കാണാം. ആ HTML Notation എന്ന ഹെക്സാഡെസിമൽ കോഡ്.
