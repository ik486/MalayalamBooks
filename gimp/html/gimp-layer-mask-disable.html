7.27. Disable Layer Mask

ഉടൻ ഒരു പാളി മാസ്ക് സൃഷ്ടിക്കാൻ പോലെ അത് ചിത്രത്തിന്റെ പ്രവർത്തിക്കുന്നു. Disable Layer Mask കമാൻഡ് ഈ പ്രവർത്തനം താൽക്കാലികമായി അനുവദിക്കുന്നു. മെനു എൻട്രി ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഒരു ചെക്ക് അടുത്ത അതു ലെയറുകൾക്കുമായുള്ള സംഭാഷണം ചുവപ്പാകുകയും ലെ ലെയർ മാസ്ക് ലഘുചിത്ര അതിരായ പ്രദർശിപ്പിച്ചിരിക്കുന്നു.

7.27.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Mask → Disable Layer Mask,

  • അല്ലെങ്കിൽ ലെയറുകൾ ഡയലോഗിൽ ലെയർ മാസ്ക് ലഘുചിത്ര ന് Ctrl കീ (ചില സിസ്റ്റങ്ങളിൽ Ctrl+Alt) സിംഗിൾ ക്ലിക്ക് ഇറക്കി പിടിച്ച്.

  • നിങ്ങൾ ലെയർ മാസ്ക് ലഘുചിത്ര വീണ്ടും Layer → Mask മെനുവിലെ മെനു എൻട്രി അൺചെക്കുചെയ്ത് അല്ലെങ്കിൽ Ctrl -clicking (അല്ലെങ്കിൽ Ctrl+Alt -clicking) വഴി ഈ പ്രവർത്തനം പഴയപടിയാക്കാനുമാകും.
