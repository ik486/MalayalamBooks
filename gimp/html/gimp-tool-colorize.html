5.4. Colourise

വർണ്ണാഭമാക്കുക പ്രയോഗം സജീവ അല്ലെങ്കിൽ പാളി നിരക്കു വർണ്ണമിട്ട ഗ്ലാസ് കാണാൻ ഒരു greyscale പ്രതിമയായി റെൻഡർ. നിങ്ങളുടെ ചിത്രവും ഒരു “Sepia” പ്രഭാവം നൽകാൻ അത് ഉപയോഗിക്കാൻ കഴിയും. ഹ്യൂയേ, സാച്ചുറേഷൻ വേണ്ടി കളർ മോഡൽ കാണുക, പ്രതല.

5.4.1. Activating tool

You can get to the Colorize tool in two ways:

  • In the image-menu through: Tools → Color Tools → Colorize or Colors → Colorize,

  • or by clicking the tool icon: in Toolbox, provided that you have installed color tools in Toolbox. For this, please refer to വിഭാഗം 1.7, “പണിസഞ്ചി”.

5.4.2. ഓപ്ഷനുകൾ

ചിത്രം 14.147. Colorize options


Presets

ചിത്രം 14.148. Preset Menu


ഏത് നിങ്ങൾ Import Settings from File അല്ലെങ്കിൽ Export Settings to File അനുവദിക്കുന്നു, നിങ്ങൾ നിയന്ത്രിക്കുന്ന ക്രമീകരണങ്ങൾ സംരക്ഷിക്കുക ഡയലോഗ് ആക്സസ് നൽകുന്നു:

ചിത്രം 14.149. Manage saved Settings Dialog


വർണ്ണ തിരഞ്ഞെടുക്കുക

  • Hue: (- 360 0) സ്ലൈഡർ സാംഖിക ടെക്സ്റ്റ് ബോക്സ് നിങ്ങൾ HSV നിറം സർക്കിളിൽ ഇവൻമാരാണ് തിരഞ്ഞെടുക്കാൻ അനുവാദം.

  • Saturation: 100 0: സ്ലൈഡർ ഇൻപുട്ട് ബോക്സ് നിങ്ങൾ ഒരു സാച്ചുറേഷൻ തിരഞ്ഞെടുക്കുക അനുവദിക്കുന്നു.

  • Lightness: 100 (പ്രകാശം) -100 (ഇരുണ്ട): സ്ലൈഡർ ടെക്സ്റ്റ് ബോക്സില് നിങ്ങൾക്ക് ഒരു മൂല്യം തിരഞ്ഞെടുക്കുക അനുവദിക്കുന്നു.

പ്രിവ്യൂ
