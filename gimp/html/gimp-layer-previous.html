7.12. Select Previous Layer

Select Previous Layer കമാൻഡ് പാളി വെറും പാളി സ്റ്റാക്ക് സജീവ പാളി മുകളിൽ തെരഞ്ഞെടുക്കുന്നു. കമാൻഡ് ലെയറുകൾ ഡയലോഗിൽ പാളി എടുത്തുകാണിക്കുന്നു പുതിയ സജീവ പാളി ചെയ്യുന്നു. സജീവ പാളി സ്റ്റാക്ക് മുകളിൽ ഇതിനകം ഉണ്ടെങ്കിൽ ഈ മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്.

കുറിപ്പു്

ഒരു സാധാരണ വിൻഡോസ്-രീതിയിൽ ഇംഗ്ലീഷ് കീബോർഡിലെ ശ്രദ്ധിക്കുക, സ്ഥിര കുറുക്കുവഴി Page_Up അക്കങ്ങള് അറിയണ്ടേ കീ ലേക്ക്, എന്നാൽ സംഖ്യാ കീപാഡ് ഇടതു ആറു കീകൾ സംഘത്തിലെ മറ്റു Page_Up കീ സൂചിപ്പിക്കുന്നത്.
സൂചന

നിങ്ങൾ പതിവായി പെയിന്റിംഗിന് മറ്റൊരു പാളി ന്, നിങ്ങൾ കൂടുതൽ Ctrl കീ പിടിച്ച് ലഭിക്കുന്നതെന്ന്, നിങ്ങൾ നിറം-പിക്കർ ഉപകരണം ഉപയോഗിക്കുമ്പോൾ അവ പ്രത്യേകിച്ച് ഉപയോഗിക്കാൻ പാളി നിന്ന് നിറങ്ങൾ തിരഞ്ഞെടുക്കുക എങ്കിൽ Select Previous Layer ആൻഡ് Select Next Layer കീബോർഡ് കുറുക്കുവഴികൾ വളരെ ഉപകാരപ്രദമായ ആയിരിക്കാം പെയിന്റിംഗ് പ്രയോഗങ്ങളുടെ.

7.12.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Stack → Select Previous Layer,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Up ഉപയോഗിച്ച്.

അല്ലെങ്കിൽ നിങ്ങൾക്ക് ലളിതമായി ലയർ ഡയലോഗിൽ പാളി നാമത്തിൽ ക്ലിക്ക് ചെയ്യുക.
