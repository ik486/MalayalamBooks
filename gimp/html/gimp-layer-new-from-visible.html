7.4. New From Visible

This command merges the visible layers into a new layer at the top of the layer stack.

ലക്ഷ്യം കൂടുതൽ ഫലം കൃത്രിമം, എന്നാൽ ഈ സാഹചര്യം സൃഷ്ടിച്ച ഘട്ടങ്ങൾ പ്രമാണിച്ചു. ഉദാഹരണം: തിരഞ്ഞെടുത്ത് നിങ്ങളുടെ മൾട്ടിലെയർ ചിത്രത്തിന്റെ ചില പ്രദേശങ്ങൾ ബ്ലർ ആഗ്രഹിക്കുന്നു. നിങ്ങൾ കാണുന്നത് നിങ്ങൾക്ക് നിന്ന് ഒരു പുതിയ ലയർ ഉണ്ടാക്കുക അതു ബ്ലർ തുടർന്ന് നിങ്ങളുടെ യഥാർത്ഥ സൃഷ്ടി കാണിക്കാൻ ആഗ്രഹിക്കുന്ന ഭാഗങ്ങൾ മായ്ച്ചുകളയാൻ ഒരു സാധാരണ സംഭവം.

7.4.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → New From Visible.
