3.4. ഘനമായ സംഭാഷണം

ചിത്രം 15.39. സ്ക്രീൻഷോട്ട് ഘനമായ ഡയലോഗ് കീയെക്കാൾ


ഒരു ലീനിയർ സ്കെയിലിൽ ക്രമീകരിച്ചിരിക്കുന്നത് ഒരു കൂട്ടം വർണങ്ങൾ - - Blend tool മറ്റു നിരവധി പ്രവർത്തനവുമായി ഉപയോഗത്തിന് “Gradients” ഡയലോഗ് ഒരു ഗ്രേഡിയന്റ് തിരഞ്ഞെടുക്കുന്നതിന് ഉപയോഗിക്കുന്ന ഒരു ഗ്രേഡിയന്റ് പാലറ്റ് പ്രദാനം ചെയ്യുന്നു. നിങ്ങൾ ഘനമായ കൃത്രിമം പല ഫംഗ്ഷനുകളും ആക്സസ് നൽകുന്നു. താങ്കൾക്ക് പട്ടികയിൽ ക്ലിക്ക് ചെയ്തു് ഒരു ഗ്രേഡിയന്റ് തെരഞ്ഞെടുക്കാം: അതു പിന്നെ പണിസഞ്ചി ഓഫ് ബ്രഷ് / പാറ്റേൺ / ഗ്രേഡിയന്റ് പ്രദേശത്ത് കാണിക്കും. കുറച്ച് ഡസൻ ആശംസകള് ഘനമായ ജിമ്പ് പ്രീ-ഇൻസ്റ്റാൾ വരും. നിങ്ങൾക്ക് Gradient Editor ഉപയോഗിച്ച് കൂടുതൽ സൃഷ്ടിക്കാൻ കഴിയും. ഘനമായ അവർ എങ്ങനെ ജിമില ഉപയോഗിക്കുന്നു കുറിച്ച് പൊതുവിവരങ്ങൾ Gradients വിഭാഗത്തിൽ കണ്ടെത്താനാകും.

ആദ്യ അഞ്ചു ഘനമായ പ്രത്യേക; അവർ വിവിധ വഴികളിലൂടെ ഉപകരണബോക്സിലേക്ക് ഫോർഗ്രൗണ്ടിലെ, പശ്ചാത്തല വർണ്ണങ്ങൾ തമ്മിലുള്ള വ്യത്യാസത്തിനു പുനർനിർമ്മാണം.

  • FG to BG (Hardedge): മാത്രമേ കറുപ്പും വെളുപ്പും മൂർച്ചയുള്ള പരിധിയില്ലാതെ.

  • FG to BG (HSV clock-wise/counter-clockwise Hue): മുന്നിലും പശ്ചാത്തലത്തിലും നിറം, ഘടികാരദിശയിൽ അല്ലെങ്കിൽ അപ്രദക്ഷിണ തമ്മിലുള്ള നിറം സർക്കിളിൽ ഒട്ടൊക്കെ അതിശയത്തോടെ.

  • FG to BG (RGB): സ്ഥിര ഗ്രേഡിയന്റ്, മുൻവശത്തും ആർജിബി മോഡിൽ പണിസഞ്ചി പശ്ചാത്തല നിറങ്ങൾ, തമ്മിലുള്ള.

  • FG to Transparent: മാത്രം സുതാര്യത പൂർത്തിയാക്കാൻ പൂർണ്ണമായ അതാര്യതയിലോ നിന്ന് ഒറ്റ നിറം (പുരോതലനിറം) ഉപയോഗിക്കുന്നു. നിങ്ങൾ പതുക്കെ സവാളും കൊളാഷുകൾ അല്ലെങ്കിൽ മൂടൽമഞ്ഞ് ഇഫക്റ്റുകൾ പ്രവർത്തിക്കാൻ വരുമ്പോൾ ഈ വ്യത്യാസത്തിനു വളരെ ഉപയോഗപ്പെടുന്നു.

3.4.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Gradients” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • ഒരു ഇമേജ് മെനുവിൽ നിന്ന്: Windows → Dockable Dialogs → Gradients;

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Gradients തിരഞ്ഞെടുത്ത്,

  • പണിസഞ്ചി നിന്ന്, ബ്രഷ് / പാറ്റേൺ / ഗ്രേഡിയന്റ് പ്രദേശത്ത് നിലവിലെ ഗ്രേഡിയന്റ് മാകാം,

  • Ctrl+G കുറുക്കുവഴി ഉപയോഗിച്ച് ഇമേജിൽ നിന്നും.

Windows മെനുവിൽ, കുറഞ്ഞത് ഒരു ഡയലോഗ് തുറന്നുതന്നെയിരിക്കുന്ന മാത്രമേ ഉണ്ട് detached windows ഒരു ലിസ്റ്റ് ഉണ്ട്. Windows → Gradients: ഈ സാഹചര്യത്തിൽ, ഇമേജ്-മെനുവിൽ നിന്നും “Gradients” ഡയലോഗ് ഉണ്ടാകാം.

3.4.2. “Gradients” ഡയലോഗ് ഉപയോഗിച്ച്

ഡയലോഗ് ഏറ്റവും അടിസ്ഥാന ഏറ്റവും സാധാരണയായി ഉപയോഗിക്കുന്ന, ഓപ്പറേഷൻ അതു പിന്നീട് ഒരു ഗ്രേഡിയന്റ് ഒത്തിണക്കി ഏതെങ്കിലും ഓപ്പറേഷൻ വഴി ഉപയോഗിക്കുന്ന ജിമ്പ് നിലവിലെ ഗ്രേഡിയന്റ്, നടത്തുമ്പോൾ, സ്ക്രോൾ പട്ടികയിൽ ഘനമായ ഒന്ന് ക്ലിക്ക് ഉചിതമാവുക.

നിങ്ങൾ ഒരു ഗ്രേഡിയന്റ് ന് double-click, നിങ്ങൾ അതിന്റെ പേര് എഡിറ്റുചെയ്യാൻ കഴിയും, അവിടെ ഗ്രേഡിയന്റ് എഡിറ്റർ തുറക്കുക. കുറിപ്പ് നിങ്ങൾ മാത്രമല്ല ജിമ്പ് പ്രീ-ഇൻസ്റ്റാൾ വരുന്ന മാത്രമല്ല നിങ്ങൾ സ്വയം നിർമ്മിച്ച ഘനമായ പേരുകൾ മാറ്റാൻ അനുവദിച്ചിട്ടുണ്ടോ എന്ന്. നിങ്ങൾ ഒരു പ്രീ-ഇൻസ്റ്റാൾ വ്യത്യാസത്തിനു പേരുമാറ്റാൻ ശ്രമിച്ചാൽ, പേര് എഡിറ്റ് കഴിയും, എന്നാൽ ഉടൻ മടങ്ങാൻ അല്ലെങ്കിൽ മറ്റെവിടെയെങ്കിലും ക്ലിക്ക് തട്ടുന്നതിന് പോലെ, പേര് അതിന്റെ യഥാർത്ഥ മൂല്യം പഴയപടിയാകും. അതു നിങ്ങൾ വിഭവങ്ങൾ നിങ്ങൾക്ക് കൂടുതൽ ജിമ്പ് പ്രീ-ഇൻസ്റ്റാളുകളുടെ മാറ്റം വരുത്താൻ കഴിയില്ല സാധാരണ ആധിപത്യവും: നയപ്രഖ്യാപന, പാറ്റേണുകൾ, ഘനമായ, തുടങ്ങിയവ; നിങ്ങൾ സ്വയം സൃഷ്ടിക്കുന്ന മാത്രമാണ്.

ഗ്രിഡ് / പട്ടിക മോഡുകൾ

ടാബ് മെനുവിൽ View as Grid ആൻഡ് View as List തമ്മിലുള്ള തിരഞ്ഞെടുക്കാം. ഗ്രിഡ് മോഡിൽ, ഘനമായ ചതുരാകൃതിയിലുള്ള അണിനിരന്നു നിരത്തിയിരിക്കുന്നത്. ഈ വഴി കണ്ടപ്പോൾ അവർ വളരെ കണ്ണഞ്ചിപ്പിക്കുന്ന നോക്കൂ, അതു വളരെ എളുപ്പമാണ് കാരണം അയൽ പശുക്കൾ നിന്നും ഇടപെടലുകൾക്ക്, നിങ്ങൾ ആഗ്രഹിക്കുന്ന തിരഞ്ഞെടുക്കുക എന്നതാണ്. പട്ടിക മോഡ്, കൂടുതൽ ഈടുറ്റതും സ്വതവേ, ഘനമായ അപ്പ് ലംബമായി, ഓരോ വരി പേര് കാണിക്കുന്ന കൂടിയ ചെയ്യുന്നു.

ടാബ് മെനുവിൽ, ഓപ്ഷൻ Preview Size നിങ്ങളുടെ ശബ്ദമോ വ്യത്യാസത്തിനു പ്രിവ്യൂകളുടെ വലിപ്പം ചെയ്യൽ അനുവദിക്കുന്നു.

ഡയലോഗ് ചുവടെയുള്ള ബട്ടണുകൾ പല വഴികളിൽ ഘനമായ ന് ഓപ്പറേറ്റ് അനുവദിക്കുന്നു:

ഗ്രേഡിയന്റ് എഡിറ്റ്

ഈ ബട്ടൺ Gradient Editor സജീവമാക്കുന്നു.

പുതിയ ഗ്രേഡിയന്റ്

ഒരു സാധാരണ ഗ്രേസ്കെയിലാക്കി ആയി സമാരംഭിച്ചിട്ടില്ല പുതിയ ഗ്രേഡിയന്റ്, സൃഷ്ടിക്കുന്നു അങ്ങനെ നിങ്ങൾ അത് ഭേദഗതി ഗ്രേഡിയന്റ് എഡിറ്റർ സജീവമാക്കുന്നു. നിങ്ങൾ സൃഷ്ടിക്കുന്ന ഘനമായ സ്വയം ഏത് നിന്ന് ജിമ്പ് ആരംഭിക്കുമ്പോൾ അവർ സ്വയം ലഭ്യമാകുന്നില്ല നിങ്ങളുടെ സ്വകാര്യ ജിമ്പ് ഡയറക്ടറിയുടെ gradients ഫോൾഡറിൽ, സംരക്ഷിച്ചു. (നിങ്ങൾ മുൻഗണനകൾ ഡയലോഗ് ഉപയോഗിച്ച്, ഈ ഫോൾഡർ മാറ്റാം അല്ലെങ്കിൽ പുതിയവ ചേർക്കുക.)

തനിപ്പകർപ്പ് ഗ്രേഡിയന്റ്

ഈ നിലവിൽ തിരഞ്ഞെടുത്ത ഗ്രേഡിയന്റ് ഒരു പകർപ്പ് സൃഷ്ടിക്കുന്നു. നിങ്ങൾ യഥാർത്ഥ എഡിറ്റ് ചെയ്യാൻ സാധിക്കില്ല എങ്കിൽ പോലും പകർപ്പ് എഡിറ്റുചെയ്യുക കഴിയും.

ഗ്രേഡിയന്റ് ഇല്ലാതാക്കുക

നിങ്ങൾ അങ്ങനെ ചെയ്യാൻ അനുമതിയില്ല അതാണ് ഗ്രേഡിയന്റ് എല്ലാ അംശമോ നീക്കം. അതു എന്തെങ്കിലും ചെയ്യുന്നതിനുമുമ്പ് സ്ഥിരീകരണം ആവശ്യപ്പെടുന്നു.

പുതുക്കുക ഘനമായ

ഈ ഡയലോഗ് അധികം ചിലപ്പോൾ മറ്റു നിങ്ങളുടെ സ്വകാര്യ gradients ഫോൾഡറിലേക്ക് ഘനമായ ചേർക്കാൻ, ഈ ബട്ടൺ പുതിയ എൻട്രികൾ ലഭ്യമാകും അങ്ങനെ റീലോഡ് ലിസ്റ്റ് കാരണമാകുന്നു.

ഈ ബട്ടണുകൾ നടത്തുന്ന എന്നിവയുടെ, ഡയലോഗ് പോപ്പ്-അപ്പ് മെനുവിൽ നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയൂ ടാബ് മെനു ക്ലിക്കുചെയ്തുകൊണ്ട് ഗ്രേഡിയന്റ് പട്ടികയിൽ എവിടെയെങ്കിലും, അല്ലെങ്കിൽ Gradient Menu വഴി സജീവമാക്കാം:

ചിത്രം 15.42. ഗ്രേഡിയന്റ് എഡിറ്റർ പോപ്പ്-അപ്പ് മെനു


ഒന്നുകിൽ നിങ്ങൾ ഗ്രേഡിയന്റ് ഡിസ്പ്ലേ റൈറ്റ് ക്ലിക്ക് ചെയ്ത്, അല്ലെങ്കിൽ ഡയലോഗ് ന്റെ ടാബ് മെനു ഏറ്റവും മുകളിലായുള്ള തിരഞ്ഞെടുത്തുകൊണ്ട് ഗ്രേഡിയന്റ് എഡിറ്റർ മെനു ആക്സസ് ചെയ്യാൻ കഴിയും. മെനു നിങ്ങൾ എൻഡ്പോയിന്റ് ന്റെ നിറം (ഓരോ വിഭാഗത്തിനായി ഇടത്, വലത് എഡ്ജ് നിറങ്ങൾ സജ്ജമാക്കാൻ), നിറങ്ങൾ കുഴയ്ക്കുക, ഒരു കളർ മോഡൽ തിരുത്തുക വിഭജനങ്ങൾ തിരഞ്ഞെടുക്കുക എഡിറ്റ് അനുവദിക്കുന്നു. ഈ എഡിറ്റർ കസ്റ്റം ഘനമായ അല്ലെങ്കിൽ ഒരു സിസ്റ്റം ചെരിവുമാനം ഒരു പകർപ്പ് മാത്രം പ്രവർത്തിക്കുന്നു.

റീസെന്റർ വിഭജനത്തിന്റെ ലേഖനമാണ്

ഈ ഓപ്ഷൻ അയൽ കറുത്ത ത്രികോണങ്ങൾ തമ്മിൽ ഒരു പോയിന്റ് ഉടലെടുത്തത് തിരഞ്ഞെടുത്ത ശ്രേണിയിലെ ഓരോ വിഭാഗത്തിനായി വെള്ള ത്രികോണം നീക്കുന്നു.

സെഗ്മെന്റിലെ വീണ്ടും വിതരണം ഹാൻഡിലുകളും

തെരഞ്ഞെടുത്ത ശ്രേണിയിലെ കറുപ്പും വെളുപ്പും ത്രികോണങ്ങളെ മാറ്റി കാരണമാകുന്ന അങ്ങനെ അടുത്ത ഒന്നിൽ നിന്നും ദൂരത്തിലുള്ള എല്ലാം തുല്യരാണ്.

യോജിപ്പിച്ച് നിറങ്ങൾ

ഈ ഉപാധികൾ ഒന്നിൽ കൂടുതൽ സെഗ്മെന്റ് തിരഞ്ഞെടുത്തു മാത്രമേ ലഭ്യമാകൂ.

ലയിപ്പിക്കുക എൻഡ്പോയിന്റ് 'നിറങ്ങൾ

ഓരോ വിഭാഗത്തിൽ നിന്നും അടുത്തതിലേക്ക് കൈമാറ്റങ്ങൾലളിതമാക്കാനും ആണ് ഈ ഓപ്ഷൻ, ശ്രേണിയിലെ ഇന്റീരിയർ റോഡിൻറെ ചെയ്തത് നിറങ്ങൾ ശരാശരി കാരണമാകുന്ന.

എൻഡ്പോയിന്റ് 'അതാര്യത ലയിപ്പിക്കുക

ഈ ഓപ്ഷൻ ഒരേ കാര്യം മുൻ ഐച്ഛികമായി, പക്ഷേ പകരം അതാര്യതയോടെ നിറം ചെയ്യുന്നു.

ശ്രദ്ധിക്കുക

ഗ്രേഡിയന്റ് എഡിറ്റർ ഉള്ളിൽ ലഭ്യമല്ല “undo” ഉണ്ട്, അതിനാൽ ശ്രദ്ധിക്കുക!
3.4.3.4. ഗ്രേഡിയന്റ് എഡിറ്റർ ഉദാഹരണത്തിന് ഉപയോഗിക്കുന്നു

ഇത്തരം എല്ലാ ഓപ്ഷനുകളും അല്പമെങ്കിലും .അപ്പസ്തോലനടപടികൾ കഴിയും. ഇവിടെ ആശയങ്ങൾ ക്ലിയർ ഒരു ഉദാഹരണം:

  1. ഗ്രേഡിയന്റ് ഡയലോഗ് തുറക്കുക. ക്ലിക്ക് New Gradient . ഗ്രേഡിയന്റ് എഡിറ്റർ തുറന്നു വെളുത്ത കറുത്ത നിന്നും ഒരു ഗ്രേഡിയന്റ് പ്രതിഫലിപ്പിക്കുന്നു.

    ചിത്രം 15.47. പുതിയ ഗ്രേഡിയന്റ്


  2. വലത് ഈ പുതിയ ഗ്രേഡിയന്റ് ക്ലിക്ക് ചെയ്ത് Split Segment Uniformly ക്ലിക്ക് ചെയ്യുക. നിങ്ങൾ ആഗ്രഹിക്കുന്ന സെഗ്മെന്റുകളുടെ എണ്ണം പരിഹരിക്കുക.

    ചിത്രം 15.48. മൂന്നു മേഖലകളെ ഗ്രേഡിയന്റ്

    ഓരോ രണ്ട് കറുത്ത ത്രിരാഷ്ട്ര സ്ലൈഡറുകൾ ഉപയോഗിച്ച് പരിമിതമാണ്. അത് സജീവമാക്കുന്നതിന് ഒരു സെഗ്മെന്റ് ക്ലിക്കുചെയ്യുക. Shift കീ അമർത്തിയാൽ, കുറേ ഐലന്റ് വിഭജനങ്ങൾ തിരഞ്ഞെടുക്കുക കഴിയും.


  3. നിങ്ങൾ ഗ്രേഡിയന്റ് വലത്-ക്ലിക്കുചെയ്ത് നേടുന്നു സന്ദർഭ മെനുവിൽ, തിരഞ്ഞെടുത്ത സെഗ്മെന്റ് അല്ലെങ്കിൽ സെഗ്മെന്റ് ഗ്രൂപ്പിനായി Left Endpoint Color ആൻഡ് Right Endpoint Color സജ്ജമാക്കി.

    ചിത്രം 15.49. ആദ്യം സെഗ്മെന്റ് നിറത്തിലുള്ള

    റെഡ് ഇടത് എൻഡ്പോയിന്റ് വലത്തെ enpoint വേണ്ടി മഞ്ഞ വേണ്ടി തിരഞ്ഞെടുത്തു.


  4. മറ്റ് മേഖലകളെ ഒരേ വഴിയിൽ പോകുക. അപ്പോൾ വിവിധ ഇഫക്റ്റുകൾ നേടാൻ Blending functions for segment ഉപയോഗിക്കുക.
