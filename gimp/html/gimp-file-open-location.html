2.6. സ്ഥാനം തുറക്കുക

Open Location ഡയലോഗ് ഒരു നെറ്റ്വർക്ക് ലൊക്കേഷൻ, ഒരു യുആർഐ വ്യക്തമാക്കിയിട്ടുള്ള GIMP പിന്തുണയ്ക്കുന്ന ഏത് ഫോർമാറ്റിലും നിന്ന് ഒരു ചിത്രം ലോഡ് അനുവദിക്കുന്നു.

2.6.1. Activate Command

  • You can access this command from the Toolbox menubar or the image menubar through File → Open Location….

2.6.2. Description of the dialog window

ചിത്രം 16.11. The “Open Location” dialog window


ഇമേജുകൾ തുറക്കാൻ ഏറ്റവും സാധാരണ പദ്ധതികൾ ഇവയാണ്:

file://

to open an image from a local drive

എന്തായാലും ഇവിടെ ഒരു പരിപൂർണമായ അല്ലെങ്കിൽ ബന്ധപ്പെട്ട പാത ഫയൽനാമം വെയ്ക്കുവാൻ “file://” പ്രിഫിക്സും ഇമേജുകൾ തുറക്കുക ജപം.

ആപേക്ഷികമായ പാതകൾ സ്ഥിര ബേസ് ഡയറക്ടറി നിങ്ങളുടെ ഓപ്പറേറ്റിംഗ് സിസ്റ്റം ആശ്രയിച്ചിരിക്കുന്നു. ഇത് മാക് ഒഎസ് എക്സ് Windows, /Users/<username>/ സാധാരണഗതിയിൽ /home/<username>/ ലിനക്സ്, C:\Documents and Settings\<username> \My Documents\My Images\ ഓൺ ആണ്

FTP: //

to open an image from a ftp server

http: //

to load an image from a website

സൂചന

നിങ്ങൾ ഒരു ഇന്റർനെറ്റ് സൈറ്റ് സന്ദർശിക്കുന്ന, ഒരു ഇമേജ് റൈറ്റ് ക്ലിക്ക് ചെയ്ത് ഡ്രോപ്പ്-ഡൗൺ മെനുവിലെ “Copy link address” തിരഞ്ഞെടുക്കാം. അപ്പോൾ ജിമ് തുറക്കുന്നതിന് “Open Location” ഡയലോഗിൽ ഒട്ടിക്കുക.

ഈ കമാൻഡ് വെബ് സൈറ്റുകളിൽ നിന്നുള്ള ചിത്രങ്ങൾ പിടിച്ചെടുക്കാൻ അതു വളരെ എളുപ്പമാക്കുന്നു പോലും: Please respect the copyright! Images, even if published on the Internet are not always free to be used for you.
