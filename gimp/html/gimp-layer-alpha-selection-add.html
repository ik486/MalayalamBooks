7.39. Add Alpha channel to Selection

Add to Selection കമാൻഡ് ആൽഫാ ചാനൽ നിന്നുള്ള നിലവിലെ പാളി ഒരു നിര സൃഷ്ടിക്കുന്നു. അതാര്യമായസാധാരണഡയലോഗ്പുറകുവശം പിക്സൽ പൂർണ്ണമായി തിരഞ്ഞെടുത്തു, സുതാര്യമായ പിക്സൽ തിരഞ്ഞെടുത്തത് ആകുന്നു, അർദ്ധസുതാര്യ പിക്സൽ ഭാഗികമായും തിരഞ്ഞെടുത്തു. ഈ തിരഞ്ഞെടുപ്പ് നിലവിലുള്ള നിരക്കു ലേക്ക് added ആണ്. ആൽഫാ ചാനലിന് സ്വയം മാറ്റിയില്ല.

പ്രവർത്തനങ്ങളുടെ ഈ ഗ്രൂപ്പിൽ മറ്റ് ആജ്ഞകൾ പകരം സജീവ പാളി നിർമ്മിക്കുന്നത് നിരക്കു നിലവിലുള്ള തെരഞ്ഞെടുപ്പ് ചേർത്തിട്ട് ആ ഒഴികെ സമാനമായ അവർ ഒന്നുകിൽ പൂർണ്ണമായും നിരക്കു ആൽഫാ നിരക്കു നിന്നും ലഭ്യമാക്കുന്ന ഒരു നിര കൂടെ പകരം നിന്ന് ആൽഫാ നിരക്കു കുറയ്ക്കേണ്ട നിരക്കു നിലവിലുള്ള, രണ്ടു യെ ഒരു നിര സൃഷ്ടിക്കുക.

7.39.1. Activate the Command

  • You can access this command from the image menubar through Layer → Transparency → Add to Selection.

7.39.2. Example

ചിത്രം 16.104. Applying “Add to Selection”

സജീവ പാളി നോൺ-സുതാര്യമായ പിക്സൽ നിലവിലുള്ള നിരക്കു ചേർത്തു.
