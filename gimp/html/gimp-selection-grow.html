4.12. Grow

Grow കമാൻഡ് നിലവിലെ ചിത്രം ഒരു നിര വലിപ്പം വർദ്ധിപ്പിക്കുന്നു. ഒരു നിരക്കു വലിപ്പം കുറയുന്നു Shrink കമാൻഡ്, ഒരു സമാനമായ രീതിയിൽ പ്രവർത്തിക്കുന്നു.

4.12.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Select → Grow.

4.12.2. Description of the “Grow Selection” dialog

ചിത്രം 16.32. The “Grow Selection” dialog window


തിരഞ്ഞെടുപ്പ് വളർത്തുക

ടെക്സ്റ്റ് ബോക്സിൽ നിരക്കു വർദ്ധിപ്പിക്കാൻ പ്രകാരമുള്ള തുക നൽകാൻ കഴിയും. അളവിന്റെ യൂണിറ്റ് പിക്സൽ, എന്നാൽ ഡ്രോപ്-ഡൗൺ മെനു ഉപയോഗിച്ച് മറ്റൊരു യൂണിറ്റ് തിരഞ്ഞെടുക്കാം.

4.12.3. ചതുര തിരഞ്ഞെടുക്കലുകൾ ഇല്യ

നിങ്ങൾ ഒരു ചതുരാകൃതിയിലുള്ള സെലക്ഷനും ച്ചെലർക്ക്, ലഭിക്കുന്ന നിരക്കു കോണിലും വൃത്താകാരത്തിലാണ് ചെയ്തു. ഇതിനുള്ള കാരണം താഴെയുള്ള ഇമേജ് കാണിച്ചിരിക്കുന്നു:

ചിത്രം 16.33. വൃത്താകാരം കോണിലും എന്തുകൊണ്ട് വളരുന്ന ചതുരാകൃതിയിലുള്ള സെലക്ഷനും ഫലങ്ങൾ


നിങ്ങൾ വൃത്താകാരം കോണിലും താൽപ്പര്യമില്ലെങ്കിൽ, നിങ്ങൾ ഒരു 0% വ്യാസാർദ്ധം Rounded Rectangle കമാൻഡ് ഉപയോഗിക്കാം.
