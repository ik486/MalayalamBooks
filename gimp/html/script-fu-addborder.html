11. അലങ്കാര ഫിൽട്ടറുകൾ  

11.3. ബോർഡർ ചേർക്കുക

11.3.1. പൊതു അവലോകനം

ചിത്രം 17.218. “Add Border” ഫിൽട്ടർ ഉദാഹരണം

യഥാർത്ഥ ചിത്രം

ബോർഡർ ചേർത്തു


ഈ ഫിൽട്ടർ വെറും പേര് പറയുന്നത് ചെയ്യുന്നു അത് ചിത്രത്തിന്റെ ഒരു ബോർഡർ ചേർക്കുന്നു. നിങ്ങൾ അതിർ കനം അതുപോലെ നിറം വ്യക്തമാക്കാൻ കഴിയും. അതിർത്തിയിൽ നാലു ഭാഗത്തും വിവിധ തണലുകളിൽ നിറത്തിലുള്ള, അതിനാൽ ഇമേജ് പ്രദേശം ഉയിർത്തെഴുന്നേൽക്കയും ദൃശ്യമാകും.

ചിത്രം വക്കുകളുടെ വലിപ്പം വലുതായ ചെയ്യും അതു മാത്രമേ ചായം ചെയ്യില്ല.

11.3.2. ഫിൽട്ടർ സജീവമാക്കുക

ഈ ഫിൽട്ടർ Filters → Decor → Add Border കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ കാണപ്പെടുന്നത്.

11.3.3. ഓപ്ഷനുകൾ

ചിത്രം 17.219. “Add Border” ഓപ്ഷനുകൾ


ബോർഡർ എക്സ് വലുപ്പം, ബോർഡർ വൈ വലുപ്പം

ഇവിടെ നിങ്ങൾ പിക്സലിൽ, ചേർത്തു അതിർത്തിയിൽ കനം തിരഞ്ഞെടുക്കാം. എക്സ് വലുപ്പം (ഒരു ഇടത് വലത്), Y വലുപ്പം (മുകളിൽ താഴെ) വ്യത്യസ്തമായിരിക്കും. പരമാവധി 250 പിക്സൽ ആണ്.

ബോർഡർ വർണ്ണം

ഈ ബട്ടൺ ക്ലിക്കുചെയ്താൽ ഒരു “average” അതിർത്തിയിൽ നിറം (താഴെ കാണുന്ന, Delta value on color) തിരഞ്ഞെടുക്കാൻ അനുവദിക്കുന്ന നിറം സെലക്ടർ ഡയലോഗിന് നൽകുന്നു.

നിറം ഡെൽറ്റ മൂല്യം

ഈ ഓപ്ഷൻ അതിർത്തി വശങ്ങളും വ്യത്യസ്ത തണലുകളിൽ നിറത്തിലുള്ള നൽകുന്നു അതുവഴി ചിത്രം ഉയിർത്തെഴുന്നേൽക്കയും ദൃശ്യമാകാൻ ചെയ്യുന്നു. താഴെ അതത് അതിർത്തി വശത്തു യഥാർത്ഥ നിറം “average” Border color നിന്ന് നിറത്തിൽ ഘടകം, ചുവന്ന പച്ച, നീല, <placeholder-1></placeholder-1> വേണ്ടി കണക്കുകൂട്ടുന്നത് (തത്ഫലമായുണ്ടാകുന്ന അകൗണ്ട് 0 0 സജ്ജമാക്കി മൂല്യങ്ങൾ, 255 ശ്രേഷ്ഠ 255 സജ്ജമാക്കി മൂല്യങ്ങൾ):

  • ടോപ്പ് തണൽ = ബോർഡർ വർണ്ണം + ഡെൽറ്റ

  • വലത് തണൽ = ബോർഡർ വർണ്ണം - ഡെൽറ്റ

  • താഴെയുള്ള തണൽ = ബോർഡർ വർണ്ണം - ഡെൽറ്റ

  • ഇടത് തണൽ = ബോർഡർ വർണ്ണം + ഡെൽറ്റ

ചിത്രം 17.220. ഡെൽറ്റ ഉദാഹരണങ്ങൾ

“Add Border” അരിപ്പ 75, 125, 175, 225, തുടർന്ന്, Delta value 25 പ്രയോഗിച്ചു.


ഉദാഹരണം: default നിറം നീല (38,31,207) ആണ്, സ്ഥിര ഡെൽറ്റാ 25. ആണ് അങ്ങനെ അതിരുകളിൽ തണലുകൾ: മുകളിൽ: (38,31,207) + (25,25,25) = (63,56,232), വലത്: (38,31,207) + (-13, -13, -13) = (25,18,194) മുതലായവ
