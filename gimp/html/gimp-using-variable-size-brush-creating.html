9.2. വേഗത്തിൽ ഒരു ബ്രഷ് തയ്യാറാക്കുന്നു

എളുപ്പത്തിൽ ഒരു പുതിയ ബ്രഷ് സൃഷ്ടിക്കുന്നതിന് രണ്ട് രീതികൾ:

  1. ആദ്യം, “superfast” രീതി. തിരഞ്ഞെടുത്ത ഉപകരണം, നിങ്ങൾ പെൻസിൽ, എയർബ്രഷ് പോലുള്ള ഒരു ഉപകരണം ഉപയോഗിക്കുന്നതിന് അതിൽനിന്ന് ഒരു ബ്രഷ് നടത്തുക ഒരു ഇമേജ് വിസ്തൃതി ... ചതുരാകൃതിയിലുള്ള (അല്ലെങ്കിൽ ദീർഘവൃത്താകൃതിയിലുള്ള) അതു തിരഞ്ഞെടുക്കുക തുടർന്ന് ഈ നിരക്കു ഒരു Copy ചെയ്യാൻ നിങ്ങൾ ഉടനെ കാണാനാകും ബ്രഷ് സംഭാഷണം ഒന്നാം സ്ഥാനവും ഈ പകർപ്പ്, അതിന്റെ പേര് “Clipboard” ആണ്. ഇത് ഉടനെ ഉപയോഗിക്കാൻ കഴിയില്ല.

    ചിത്രം 7.21. തിരഞ്ഞെടുക്കൽ പകർത്തിയതിനുശേഷം ഒരു ബ്രഷ് മാറുന്നു


  2. രണ്ടാമത്തെ രീതി ഏറെ പ്രധാനപ്പെട്ട ആണ്.

    ഒരു വീതിയും 35 പിക്സൽ വികസിത ഓപ്ഷനുകളിൽ Gray Level ആൻഡ് Fill with: white ഒരു Color Space നീളവും ഉദാഹരണത്തിന് File → New കൂടെ ചെയ്ക.

    അതു സനര്ഷിക്കുക ഒരു കറുത്ത പെൻസിൽ അതിനെ വരയ്ക്കാനുള്ള ഈ പുതിയ ചിത്രത്തിന്റെ സൂം.

    Save it with a .gbr extension in the directory /home/name_of_user/.gimp-2.8/brushes/.

    Brushes ഡയലോഗ് വിൻഡോയിൽ, ബട്ടണ് Refresh brushes ക്ലിക്ക് ചെയ്യുക.

    നിങ്ങളുടെ അത്ഭുതമായി ബ്രഷ് മറ്റ് നയപ്രഖ്യാപന സ്വര്ഗം ഇവിടെ ലഭ്യമാകുന്നു. നിങ്ങൾ വീണ്ടും ജിമ്പ് ആരംഭിക്കാതെ ഉടനെ ഉപയോഗിക്കാൻ കഴിയും.

    ചിത്രം 7.22. ഒരു ബ്രഷ് സൃഷ്ടിക്കുന്നതിനുള്ള ഘട്ടങ്ങൾ

    ചിത്രം വരയ്ക്കുക, ബ്രഷ് സംരക്ഷിക്കുക

    നയപ്രഖ്യാപന പുതുക്കുക

    ബ്രഷ് ഉപയോഗിക്കുക
