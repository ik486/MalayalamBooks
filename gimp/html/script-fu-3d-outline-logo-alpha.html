16.2.1. പൊതു അവലോകനം

ചിത്രം 17.369. “3D Outline” ഫിൽട്ടർ ഉദാഹരണം

“3D Outline” ഫിൽട്ടർ

“3D Outline” ലോഗോ


ഈ ഫിൽട്ടർ (മുകളിൽ കാണുക) ഔട്ട്ലൈൻ ടെക്സ്റ്റ് ഒരു ഡ്രോപ്പ് ഷാഡോ ഒരു ലോഗോ സൃഷ്ടിക്കുന്ന “3D Outline” സ്ക്രിപ്റ്റ് (ചിത്രം വിൻഡോയിൽ File → Create → Logos → 3D Outline), നിന്നാണ്.

ഫിൽട്ടർ ഒരു പാറ്റേൺ ഉപയോഗിച്ച് സജീവ പാളി (ആൽഫാ ചാനലിൽ നിന്ന് നിർണ്ണയിക്കുന്നത്) നോൺ-സുതാര്യമായ പ്രദേശങ്ങൾ നൽകിയിട്ടുണ്ട് ഒരു ഡ്രോപ്പ് ഷാഡോ ചേർക്കുന്നു. ഇവിടെ നാം നോൺ-സുതാര്യമായ പിക്സൽ നിർവചിക്കപ്പെട്ട സജീവ പാളി ഈ പ്രദേശങ്ങളിൽ റഫർ alpha പദം ഉപയോഗിക്കും.

ഫിൽട്ടർ ആൽഫ ന്റെ ഔട്ട്ലൈൻ ലഭിക്കാൻ Sobel edge detect ഉപയോഗിക്കുന്നുണ്ട്. അങ്ങനെ ഒരു ലളിതമായ ആൽഫ, ഉദാഹരണത്തിന് ഒരു മായ്ച്ചു ദീർഘചതുരം നിരക്കു, നിങ്ങൾ അതിർത്തി ലഭിക്കും. എന്നാൽ നിങ്ങൾ ഒരു സംഭവം (സാധാരണ സംഭവം മറക്കരുത്) ഉപയോഗിക്കുമ്പോൾ, താഴെ ഉദാഹരണം പോലെ, എഡ്ജ് ഡിറ്റക്ടർ കൂടുതൽ വേണമങ്കിലും അതുവഴി അരിപ്പ പ്രാബല്യത്തിൽ അതും അരികുകളിൽ ബാധകമാകും കണ്ടെത്തും.

ചിത്രം 17.370. ഉദാഹരണം multicolored ലെയർ മാസ്ക് അടിസ്ഥാനമാക്കി

... ഈ ലഭിക്കും.


മുന്നറിയിപ്പു്

ചിത്രം എല്ലായ്പ്പോഴും സജീവ പാളി വലിപ്പത്തിന് വലുപ്പം മാറ്റും.

16.2.2. ഫിൽട്ടർ സജീവമാക്കുക

ഈ ഫിൽട്ടർ Filters → Alpha to Logo → 3D Outline കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ കാണപ്പെടുന്നത്.

സജീവ പാളി ഒരു ആല്ഫാ ചാനൽ ഉണ്ട് എങ്കിൽ അരിപ്പ മാത്രമേ പ്രവർത്തിക്കൂ. അല്ലാത്തപക്ഷം, മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്.

16.2.3. ഓപ്ഷനുകൾ

ചിത്രം 17.371. “3D Outline” ഓപ്ഷനുകൾ


മാതൃക

താങ്കൾക്ക് മാറ്റാനും കഴിയും നിലവിൽ തിരഞ്ഞെടുത്ത പാറ്റേൺ. നിങ്ങൾ പാറ്റേൺ ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഒരു റെസിഡന്റ് പ്രിവ്യൂ പോപ്പ് ചെയ്യും. Browse... ബട്ടൺ അമർത്തിയാൽ നിങ്ങൾക്ക് മറ്റൊരു പാറ്റേൺ തിരഞ്ഞെടുക്കുക കഴിയുന്ന ഒരു ഡയലോഗ് തുറക്കുന്നു.

ബാഹ്യരേഖയുടെ മങ്ങൽ ആരം

ഈ ആരം ആൽഫ blur ലേക്കുള്ള edge detector പാറ്റേൺ നിറഞ്ഞ് പ്രദേശം തിരഞ്ഞെടുക്കും മുമ്പ് ഉപയോഗിക്കുന്നു. അത് വിശാലമായ പക്ഷേ പൂശി മാതൃകയിൽ എന്തുകൊണ്ട് ഒരു ഉയർന്ന മൂല്യം ഫലങ്ങള് തുടർന്ന്:

Bumpmap (ആൽഫാ പാളി) മങ്ങൽ ആരം

ഒരു 3D പ്രഭാവം സൃഷ്ടിക്കാൻ ഉപയോഗിച്ച Bump Map സജീവ പാളി (ആൽഫാ പാളി) പ്രയോഗിച്ചു edge detect അരിപ്പ ആണ്. അതു പാറ്റേൺ പാളി എംബോസ്സ് ഉപയോഗിക്കുന്നു മുമ്പ്, മറ്റൊരു Gaussian blur വ്യക്തമാക്കിയ വ്യാസാർദ്ധം പ്രയോഗിക്കും. അങ്ങനെ ഒരു ഉയർന്ന മൂല്യം 3D പ്രഭാവം കുറയ്ക്കും.

സ്വതേ bumpmap ക്രമീകരണങ്ങൾ

ചെക്കുചെയ്തിട്ടുണ്ടെങ്കിൽ (ഈ സ്വതവേ) കരീനയെ മാപ്പ് പ്ലഗ്-ഇൻ അതിന്റെ സ്ഥിരസ്ഥിതി ഓപ്ഷനുകൾ ഉപയോഗിച്ച് പ്രയോഗിക്കും. അല്ലാത്തപക്ഷം, അരിപ്പ പ്രവർത്തിക്കുന്ന സമയത്ത് Bump Map ഡയലോഗ് ജാലകം പോപപ്പ്; നിങ്ങൾ വ്യത്യസ്ത ഓപ്ഷനുകൾ തിരഞ്ഞെടുക്കാം. നിങ്ങൾ Cancel ബട്ടണ് വിൻഡോ അടയ്ക്കുമ്പോൾ, യാതൊരു കരീനയെ മാപ്പ് പ്രയോഗിക്കും, ശ്രദ്ധിക്കുക.

ഷാഡോ X ഓഫ്സെറ്റ്; ഷാഡോ Y ഓഫ്സെറ്റ്

ഈ നിഴൽ പാളി എന്നെ ശരിയായ (എക്സ്) താഴേക്കുമുള്ള (y) നീക്കി ചെയ്യും പിക്സൽ തുക. അപ്പോൾ പാളി ഇമേജ് വലുപ്പം .ബട്ടണില്ലാത്ത ചെയ്യും. യാതൊരു യഥാർത്ഥ പശ്ചാത്തല പാളി ഇല്ല ശ്രദ്ധിക്കുക നിഴൽ ചലിക്കുന്ന അതിന്റെ യഥാർത്ഥ സ്ഥലം ക്ലിയർ:

ചിത്രം 17.374. ഷാഡോ ഉദാഹരണം ഓഫ്സെറ്റ്

X ഓഫ്സെറ്റ്: 50; Y ഓഫ്സെറ്റ്: 20
