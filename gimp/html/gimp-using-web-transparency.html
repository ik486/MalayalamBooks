2.3. സുതാര്യതയോടെ സംരക്ഷിക്കുന്നു ചിത്രങ്ങൾ

ലളിതമായ ബൈനറി സുതാര്യതയും ആൽഫാ സുതാര്യത: ഗ്രാഫിക് ഫയൽ ഫോർമാറ്റുകൾ ഉപയോഗിക്കുന്ന സുതാര്യം പ്രദേശങ്ങൾ പിന്തുണക്കുന്ന രണ്ടു വ്യത്യസ്ത സമീപനങ്ങളുടെ ഉണ്ട്. ലളിത ബൈനറി സുതാര്യത GIF ഫോർമാറ്റിൽ പിന്തുണയ്ക്കുന്നു; ഇൻഡെക്സ്ഡ് വർണ്ണ പാലറ്റ് നിന്ന് ഒറ്റ കളർ സുതാര്യമായ നിറം അടയാളപ്പെടുത്തി. ആൽഫാ സുതാര്യത PNG ഫോർമാറ്റിൽ പിന്തുണയ്ക്കുന്നു; സുതാര്യ വിവരം ഒരു പ്രത്യേക ചാനൽ, Alpha channel ശേഖരിക്കും.

കുറിപ്പു്

പിഎൻജി അധിക സവിശേഷതകൾ (ഉദാ, ആൽഫാ സുതാര്യത) കൂടെ ജിഫ് എല്ലാ സവിശേഷതകളും പിന്തുണയ്ക്കുന്നു കാരണം GIF ഫോർമാറ്റിൽ വിരളമായി ഉപയോഗിക്കുന്നു. എന്നാൽ, GIF, ഇപ്പോഴും ആനിമേഷനുകൾ ഉപയോഗിക്കുന്നു.

പ്രക്രിയ 6.1. സുതാര്യമായ മേഖലകൾ (ആൽഫാ സുതാര്യത) ഉപയോഗിച്ചുള്ള ഒരു ചിത്രം സൃഷ്ടിക്കുന്നു

  1. ഒന്നാമത് ഞങ്ങൾ ഒരേ ചിത്രം മുൻ ട്യൂട്ടോറിയലുകൾ എന്നപോലെ ജിമ്പ് ഭാഗ്യചിഹ്നം ഉപയോഗിക്കും Wilber.

    ചിത്രം 6.9. Wilber ചിത്രം RGBA മോഡിൽ തുറക്കാൻ


  2. To export an image with alpha transparency, you must have an alpha channel. To check if the image has an alpha channel, go to the channel dialog and verify that an entry for “Alpha” exists, besides Red, Green and Blue. If this is not the case, add a new alpha channel from the layers menu; Layer+Transparency → Add Alpha Channel.

  3. യഥാർത്ഥ എക്സ്.സി.എഫ് ഫയൽ നിങ്ങളെ നീക്കം കഴിയുന്ന പശ്ചാത്തല പാളികൾ അടങ്ങിയിരിക്കുന്നു. GIMP ഘനമായ സൃഷ്ടിക്കുന്നത് പിന്തുണയ്ക്കുന്ന സാധാരണ ഫിൽട്ടറുകൾ ഉപയോഗിച്ച് വരുന്നു; Filters+Light and Shadow ചുവടെ നോക്കുക. മാത്രമേ നിങ്ങൾക്ക് നിങ്ങളുടെ ഭാവന പരിമിതപ്പെടുത്തിയിരിക്കുന്നു. ആൽഫാ സുതാര്യത ശേഷികൾ തെളിയിക്കാനുള്ള, Wilber ചുറ്റും പശ്ചാത്തലത്തിൽ ഒരു സോഫ്റ്റ് തിളക്കം കാണിക്കില്ല.

  4. After you're done with your image, you can export it in PNG format.
