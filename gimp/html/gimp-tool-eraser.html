3.8. തുടച്ചുമാറ്റുന്നവന്
3. പെയിന്റ് ഉപകരണങ്ങൾ  

3.8. തുടച്ചുമാറ്റുന്നവന്

ചിത്രം 14.73. Eraser tool icon in the Toolbox


എറേസർ നിലവിലെ പാളി നിന്ന് ഈ പാളി ഒരു നിര നിന്ന് നിറം പ്രദേശങ്ങൾ നീക്കം ഉപയോഗിക്കുന്നു. ഇറേസർ, പിന്നെ മായ്ക്കുന്ന പശ്ചാത്തല നിറം, നിറം ഏരിയ പ്രദർശിപ്പിച്ച പോലെ കാണിക്കും സുതാര്യത (അത് ഒരു ആല്ഫാ ചാനൽ കിട്ടാത്ത പക്ഷം ഒരു നിര മാസ്ക് ചാനൽ, ഒരു പാളി മാസ്ക്, അല്ലെങ്കിൽ പശ്ചാത്തല പാളി) പിന്തുണയ്ക്കാത്ത എന്തോ ഉപയോഗിക്കുന്നു എങ്കിൽ പണിസഞ്ചി (ഒരു മാസ്ക് കാര്യത്തിൽ, നിരക്കു പരിഷ്ക്കരിക്കും). അല്ലാത്തപക്ഷം, മായ്ക്കുന്നു പ്രയോഗം ഓപ്ഷനുകൾ വേണ്ടി ക്രമീകരണങ്ങൾ അടിസ്ഥാനമാക്കി, ഭാഗിക അല്ലെങ്കിൽ പൂർണ്ണ ഒന്നുകിൽ സുതാര്യത താനേ വന്നുകൊള്ളും. നിങ്ങൾ വിഭാഗം 7.33, “ആൽഫ ചാനൽ ചേർക്കുക” ഒരു പാളി ഒരു ആൽഫ ചാനൽ എങ്ങനെ ചേർക്കാൻ കൂടുതല് പഠിക്കുന്നതിനു കഴിയും.

ചിത്രം 14.74. Eraser and Alpha channel

പശ്ചാത്തല വർണ്ണം വൈറ്റ് ആണ്. ചിത്രം ആൽഫ ചാനൽ ഉണ്ട്. ഇറേസർ (അതാര്യത 100%) ബി.ജി നിറം കാണിക്കുന്നു.

ചിത്രം ആൽഫാ ചാനൽ ഉണ്ട്. ഇറേസർ സുതാര്യത കാണിക്കുന്നു.


നിങ്ങൾ അവരുടെ മുൻ ഉള്ളടക്കങ്ങളുടെ പിന്നിലുള്ളത് പോറൽ, പൂർണ്ണമായും പിക്സൽ ചില ഗ്രൂപ്പ് മായ്ക്കാൻ വേണമെങ്കിൽ, ഉപകരണം ഓപ്ഷനുകൾ ലെ "ദൃഢമായ അരിക്" ബോക്സ് പരിശോധിക്കണം. അല്ലാത്തപക്ഷം, ഉപ-പിക്സൽ ബ്രഷ് പ്ലെയ്സ്മെന്റ് ഭാഗിക ഇറേഷർ ബ്രഷ്-സ്ട്രോക്ക് അറ്റങ്ങളിൽ, ഹാറ്ഡ് ഇരുവായ്ത്തലയുള്ള ബ്രഷ് ഉപയോഗിക്കാൻ പോലും കാരണമാകും.

സൂചന

നിങ്ങൾ ഒരു ടാബ്ലെറ്റ് ഉപയോഗിച്ച് ജിമ്പ് ഉപയോഗിക്കുകയാണെങ്കിൽ, അത് ഒരു മായ്ച്ചു ആയി സ്റ്റൈലസ് വിപരീത അവസാനം പെരുമാറണം സൗകര്യപ്രദമായ കണ്ടെത്താം. ഈ പ്രവർത്തനത്തിലാക്കാൻ, നിങ്ങൾ ചെയ്യേണ്ടത് എല്ലാ പണിസഞ്ചിയിൽ എറേസർ ഉപകരണം റിവേഴ്സ് അവസാനം ക്ലിക്ക് ആണ്. സ്റ്റൈലസ് ഓരോ അവസാനം ഒരു പ്രത്യേക ഇൻപുട്ട് ഉപകരണമായി ചികിത്സ കാരണം, ഓരോ ഇൻപുട്ട് ഡിവൈസ് അതിന്റേതായ വ്യത്യസ്ത പ്രയോഗം നിയമനം ഉണ്ട്, റിവേഴ്സ് അവസാനം തുടർന്ന് എത്രത്തോളം നിങ്ങൾക്ക് അതു കൊണ്ട് മറ്റൊരു ഉപകരണം തിരഞ്ഞെടുക്കുക പോലെ ഒരു മായ്ച്ചു പ്രവർത്തിക്കാൻ തുടരും.

3.8.1. Activating the tool

You can activate this tool in several ways:

  • From the image menu through Tools → Paint Tools → Eraser;

  • from the Toolbox by clicking on the tool icon ;

  • അല്ലെങ്കിൽ കുറുക്കുവഴി Shift+E ഉപയോഗിച്ച് കീബോർഡ് നിന്ന്.

3.8.2. Key modifiers

എല്ലാ പെയിന്റ് പ്രയോഗങ്ങളിൽ അതേ പ്രതീതി കീ മോഡിഫയറുകൾക്കുള്ളകീബോർഡ് ഒരു വിവരണം വേണ്ടി വിഭാഗം 3.1, “സാധാരണ സവിശേഷതകൾ” കാണുക.

Ctrl,

ഇറേസർ വേണ്ടി Ctrl കീ അമർത്തിപ്പിടിക്കുക അതു “color picker” മോഡിലേക്ക്, അതു ക്ലിക്കുചെയ്യപ്പെടുകയും ഏതെങ്കിലും പിക്സൽ നിറം തെരെഞ്ഞെടുക്കുവാൻ അങ്ങനെ ഇടുന്നു. മറ്റ് ബ്രഷ് ഉപകരണങ്ങൾ പക്ഷേ, എറേസർ പുരോതല നിറം background നിറം പകരം സജ്ജമാക്കുന്നു. സുതാര്യത പിന്തുണയ്ക്കാത്ത ഡ്രോയബിൾ ന് മായ്ക്കുന്നത് നിലവിലെ പശ്ചാത്തല നിറം മായ്ച്ചു പ്രദേശങ്ങൾ പകരമാണ് ഇത് കൂടുതൽ ഉപയോഗപ്പെടുന്നു.

സകലതും

താഴേക്ക് ടൂൾ ഓപ്ഷനുകൾ വിഭാഗത്തിൽ താഴെ പറഞ്ഞിരിക്കുന്ന പോലെ, “anti-erase” മോഡിലേക്ക് Alt കീ സ്വിച്ചുകൾ കൈവശമുള്ള അതു എറേസർ വേണ്ടി. ചില സിസ്റ്റങ്ങളിൽ, Alt കീ ജാലകപാലകനുള്ള കെണിയിലകപ്പെടുകയായിരുന്നു ശ്രദ്ധിക്കുക. നിങ്ങൾക്ക് ഇത് സംഭവിക്കുകയാണെങ്കിൽ, പകരമായി Alt+Shift ഉപയോഗിക്കാൻ കഴിഞ്ഞേക്കും.

3.8.3. ടൂൾ ഓപ്ഷനുകൾ

ചിത്രം 14.75. Tool Options for the Eraser tool


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

Brush; Size; Brush Dynamics; Dynamic Options; Apply Jitter; Incremental
പല അല്ലെങ്കിൽ എല്ലാ പെയിന്റ് ഉപകരണങ്ങൾ ബാധകമായ ഉപകരണം ഓപ്ഷനുകൾ ഒരു വിവരണം വേണ്ടി Common Paint Tool Options കാണുക.
അതാര്യത

The Opacity slider, in spite of its name, in this tool determines the “strength” of the tool. Thus, when you erase on a layer with an alpha channel, the higher the opacity you use, the more transparency you get!

Hard Edge

ഈ ഓപ്ഷൻ ബ്രഷ്-സ്ട്രോക്ക് അറ്റങ്ങളിൽ ഭാഗിക ഇറേഷർ ഗുണവുമില്ല. മുകളിൽ കാണുന്ന.

Anti Erase

മായ്കൽ ഉപകരണം വിരുദ്ധ മായ്കൽ ഓപ്ഷൻ അവർ പൂർണ്ണമായും സുതാര്യമായ പോലും, അൺ-മായ്ക്കാൻ കഴിയും ഒരു ചിത്രത്തിന്റെ പ്രദേശങ്ങൾ. ഈ സവിശേഷത ഒരു ആല്ഫാ ചാനലുമായി പാളികൾ ഉപയോഗിച്ചുകൊണ്ടിരിക്കുന്നു മാത്രമേ പ്രവർത്തിക്കൂ. ടൂൾ ഓപ്ഷനുകൾ ലെ ചെക്ക്-ബട്ടൺ പുറമെ, അത് ഞൊടിയിടയിൽ (Alt കീ Alt+Shift ഇറക്കി പിടിച്ച്, വിൻഡോ മാനേജർ കുടുങ്ങി എങ്കിൽ അല്ലെങ്കിൽ) Alt കീ അമർത്തിപ്പിടിച്ച് സജീവമാക്കാം.

കുറിപ്പു്

വിരുദ്ധ മായ്ക്കുന്നു സാധ്യത മനസിലാക്കുന്നതിനും, മായ്ക്കണമെന്നില്ലെങ്കിൽ (അല്ലെങ്കിൽ അതുപോലെ, cutting) മാത്രം ആൽഫാ ചാനൽ, ചിത്രത്തിലെ ഡാറ്റ അടങ്ങിയിരിക്കുന്ന ആർജിബി ചാനലുകൾ ബാധിക്കുന്നു തിരിച്ചറിയണം. ഫലം പൂർണ്ണമായും സുതാര്യമായ പോലും, ആർജിബി ഡാറ്റ ഇപ്പോഴും അവിടെ ആണ് നിങ്ങൾക്ക് അത് കാണാൻ കഴിയില്ല. അങ്ങനെ ഒരിക്കൽ കൂടി ആർജിബി ഡാറ്റയും കാണാനാകും സാമാജ്യത്വ മായ്ക്കുന്നു ആൽഫാ മൂല്യം വർദ്ധിപ്പിക്കുന്നു.
സൂചന

നിങ്ങൾ ഒരു ഫ്ലോട്ടിംഗ് നിരക്കു ആകൃതി മാറ്റാൻ എറേസർ ഉപയോഗിക്കാം. മായ്ച്ചാണ്, നിങ്ങൾ നിരക്കു അറ്റങ്ങൾ ട്രിം കഴിയും.
