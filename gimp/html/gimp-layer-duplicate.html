7.5. പാളി ഡ്യൂപ്ലിക്കേറ്റ്

Duplicate Layer കമാൻഡ് പുതിയ തലം സജീവ പാളി ഏതാണ്ട് സമമാണ്, ഒരു പകർപ്പ് ആണ് ചിത്രത്തിലേക്ക് ചേർക്കുന്നു. പുതിയ പാളി ഒറിജിനൽ പാളി പേര് അതേ, എന്നാൽ അതു അനുബന്ധമായി “ copy” ഉണ്ടു.

നിങ്ങൾ ഒരു ആല്ഫാ ചാനൽ ഇല്ലാത്ത ഒരു പശ്ചാത്തല പാളി ഡ്യൂപ്ലിക്കേറ്റ്, പുതിയ പാളി ഒറ്റ വകയിരുത്തുന്നു. കൂടാതെ, സജീവ പാളി ഘടിപ്പിച്ചിരിക്കുന്ന ഏതെങ്കിലും “parasites” ഉണ്ടെങ്കിൽ അവർ കോപ്പിയാണ് അല്ല. (വചനം “parasites” മനസ്സിലാക്കാൻ നിങ്ങളെ ചെറിയ, അസുഖകരമായ ജീവികൾ പരിമിതപ്പെടുത്തിയിരിക്കുന്നു, കഴിഞ്ഞ വാചകം അവഗണിക്കുക.)

7.5.1. Activate the Command

  • You can access this command from the image menubar through Layer → Duplicate Layer, or from the local pop-up menu that you get by right-clicking on the Layer Dialog.

  • കൂടാതെ, Layer Dialog ന്, നിങ്ങൾ അതു സന്ദർഭം പോപ്പ്-അപ്പ് മെനുവിലെ Duplicate വഴി ആക്സസ്, അല്ലെങ്കിൽ ഈ ഡയലോഗ് അടിയിൽ PLACEHOLDER,-104597 ഐക്കൺ ബട്ടൺ ക്ളിക്ക് കഴിയും.
