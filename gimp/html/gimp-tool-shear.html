4.7. അറുക്കുക

ചിത്രം 14.126. The Shear tool in Toolbox


ശെയാർ ഉപകരണം ഒരു ഇമേജ് ഒരു ഭാഗത്ത് ഒരു പാളി, ഒരു നിര അല്ലെങ്കിൽ ഒരു ദിശയിൽ വിപരീത ദിശയിൽ മറ്റ് ഭാഗം ഒരു മാര്ഗം മാറ്റണമെന്ന് ഉപയോഗിക്കുന്നു. ഉദാഹരണത്തിന്, ഒരു തിരശ്ചീന കത്രിക്കുന്ന ഇടത് അവകാശം താഴ്ന്ന ഭാഗം വരെ മുകളിലെ ഭാഗം shift ചെയ്യും. ഒരു ദീർഘചതുരം വജ്രം മാറുന്നു. ഇത് ഒരു ഭ്രമണം അല്ല: ചിത്രം വികലമായിരിക്കുന്നു. തിരഞ്ഞെടുത്ത് ശേഷം ഈ ടൂൾ ഉപയോഗിക്കുന്നതിന്, ചിത്രം അല്ലെങ്കിൽ നിരക്കു ക്ലിക്ക്: ഒരു ഗ്രിഡ് സാധ്യതയുണ്ട് surperimposed കത്രിക്കുന്ന വിവരം ഡയലോഗ് തുറന്നു ആണ്. ചിത്രത്തിൽ മൌസ് പോയിന്റർ ഇഴച്ച് ഇമേജ് പോയിന്റർ ലേക്ക് നിർദേശങ്ങൾ തക്കവണ്ണം തിരശ്ചീനമായി അല്ലെങ്കിൽ ലംബമായി വളച്ചൊടിക്കാൻ. നിങ്ങൾ സംതൃപ്തിയുണ്ട് ചെയ്യുമ്പോൾ, ഉറപ്പിക്കുന്നതിനായാണ് വിവരം ഡയലോഗിൽ Shear ബട്ടൺ ക്ലിക്ക് ചെയ്യുക.

ചിത്രം 14.127. Shear example


കുറിപ്പു്

ഒരേ സമയത്ത് രണ്ട് വഴികൾ രോമം കഴിയില്ല, അന്ത്യം രണ്ടുതവണ അറുക്കുക ഉപകരണം ഉപയോഗിക്കാൻ ഞങ്ങൾക്കുണ്ട്.

4.7.1. ടൂൾ സജീവമാക്കുന്നു

You can access the Shear Tool in different ways:

  • from the image menu bar Tools → Transform Tools → Shear,

  • by clicking on the tool icon: in Toolbox,

  • by using the Shift+S key combination.

Shear magnitude X
