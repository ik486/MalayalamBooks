16. ലോഗോ ഫിൽട്ടറുകൾ ആൽഫ  

16.10. ക്രോം

16.10.1. പൊതു അവലോകനം

ചിത്രം 17.393. “Chrome” ഫിൽട്ടർ ഉദാഹരണം

പ്രയോഗിച്ചു “Chrome” ഫിൽട്ടർ

“Chrome” ലോഗോ


സ്ക്രിപ്റ്റ് രചയിതാവ് തക്കവണ്ണം - - ഒരു “simplistic, but cool, chromed logo” സൃഷ്ടിക്കുന്നു (മുകളിൽ കാണുക) ഈ ഫിൽട്ടർ “Chrome” ലോഗോ സ്ക്രിപ്റ്റ് (File → Create → Logos → Chrome), ഏത് കഥകളിയുടെ.

ഫിൽട്ടർ, ആൽഫാ ഈ ലളിതമായ ക്രോം പ്രഭാവം ചേർക്കുന്ന നോൺ-സുതാര്യമായ പിക്സൽ (ഒരു “selection by visibility” ഇതിനെ കരുതുക) നിർവ്വചിച്ച സജീവ പാളി പ്രദേശമാണ്. ഫിൽട്ടർ പ്രഭാവം എപ്പോഴും ആൽഫ മൂല്യങ്ങൾ പ്രകാരം പ്രയോഗിക്കും.

ഫിൽട്ടർ നേർത്ത പ്രദേശങ്ങൾ പ്രയോഗിച്ചു ചെയ്യുമ്പോൾ പ്രത്യക്ഷമായും പ്രാബല്യത്തിൽ മാത്രമേ “cool” തോന്നുന്നു. വൈഡ് രൂപങ്ങൾ നിങ്ങൾ Offset മൂല്യം വർദ്ധിപ്പിക്കാൻ ശ്രമിക്കാം; ഉദാഹരണങ്ങൾ below കാണും.

മുന്നറിയിപ്പു്

ചിത്രം എല്ലായ്പ്പോഴും സജീവ പാളി വലിപ്പത്തിന് വലുപ്പം മാറ്റും.

16.10.2. ഫിൽട്ടർ സജീവമാക്കുക

ഈ ഫിൽട്ടർ Filters → Alpha to Logo → Chrome കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ കാണപ്പെടുന്നത്.

16.10.3. ഓപ്ഷനുകൾ

ചിത്രം 17.394. “Chrome” ഓപ്ഷനുകൾ


ഓഫ്സെറ്റ് (2 പിക്സലുകൾ *)

ക്രോം ഇഫക്റ്റ് സൃഷ്ടിക്കുന്നതിൽ ഡ്രോപ്പ് നിഴൽ നല്കുന്നതിനു വേണ്ടി ഈ ഓപ്ഷൻ ഉപയോഗിക്കുന്നു:

ചിത്രം 17.395. “Offset” ഉദാഹരണങ്ങൾ

25 ഓഫ്സെറ്റ്

50 ഓഫ്സെറ്റ്

75 ഓഫ്സെറ്റ്


ഫിൽട്ടർ ആൽഫാ ആകൃതിയിൽ ഒരു തുള്ളി നിഴൽ സൃഷ്ടിക്കുന്നു. ഈ നിഴൽ ആൽഫ ബന്ധപ്പെട്ട് വ്യക്തമാക്കിയിട്ടുള്ള ഓഫ്സെറ്റ് തക്കവണ്ണം നീക്കപ്പെടും: ഓഫ്സെറ്റ് വലതുവശത്ത് 30% ഇറങ്ങി ഓഫ്സെറ്റ് 40% വഴി. ഇത് ഓഫ്സെറ്റ് മൂല്യത്തിന്റെ 50% feathered ചെയ്യും.

ക്രോം പ്രഭാവം ചില താൽക്കാലിക പാളികൾ ഉപയോഗിച്ച് നേടിയെടുക്കാം ചെയ്യും. ഈ പാളികൾ ഒരേ തുക (40% വ്യക്തമാക്കിയ ഓഫ്സെറ്റ് 30%) വഴി നീക്കി മാത്രമല്ല ഓഫ്സെറ്റ് 50% തൂവലുകളും ചെയ്യുന്നു. അങ്ങനെ ആൽഫ രൂപം വളരെ ഓഫ്സെറ്റ് മൂല്യം നിർണ്ണയിക്കുന്നത്.

പശ്ചാത്തല നിറം

ഈ കളർ ഫിൽറ്റർ സൃഷ്ടിച്ച പശ്ചാത്തല പാളി നിറയ്ക്കാൻ ഉപയോഗിക്കുന്നു. ഇത് ലൈറ്റ് ഗ്രേ സ്ഥിര. നിങ്ങൾക്ക് കളർ ബട്ടൺ ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഒരു കളർ സെലക്ടർ വേറെ ഏതെങ്കിലും നിറം തിരഞ്ഞെടുക്കാം കഴിയുന്ന മാറികിട്ടി.
