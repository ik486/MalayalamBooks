3.6. വാചക ബോക്സ് സ്ക്രിപ്റ്റ് വികസിപ്പിക്കുന്നു
3. ഒരു സ്ക്രിപ്റ്റ്-ഫു ട്യൂട്ടോറിയൽ  

3.6. വാചക ബോക്സ് സ്ക്രിപ്റ്റ് വികസിപ്പിക്കുന്നു

3.6.1. ശരിയായി പഴയപടിയാക്കൽ കൈകാര്യം

ഒരു സ്ക്രിപ്റ്റ് സൃഷ്ടിക്കുമ്പോൾ, നിങ്ങളുടെ ഉപയോക്താക്കൾ അവരുടെ പ്രവർത്തനങ്ങൾ ഇല്ലാതാക്കാന് കഴിവ് നൽകാൻ ആഗ്രഹിക്കുന്ന അവർ ഒരു തെറ്റ് ചെയ്യണം. ഈ എളുപ്പത്തിൽ ചിത്രം കൈകാര്യം കോഡ് ചുറ്റും ഫംഗ്ഷനുകളും gimp-undo-push-group-start ആൻഡ് gimp-undo-push-group-end വിളിച്ച് തീർന്നിരിക്കുന്നു. നിങ്ങൾ അങ്ങനെ ആ മാറ്റങ്ങൾ പിന്നീട് തിരസ്കരിക്കാനും സാധിക്കും GIMP, ആരംഭിക്കുകയും ഇമേജിൽ മാറ്റങ്ങൾ രേഖപ്പെടുത്തുന്നത് നിർത്തുന്നതിനും അറിയിക്കൂ എന്ന് പൊരുത്തപ്പെട്ട പ്രസ്താവനകൾ അവരെ തോന്നുന്നുവോ.

നിങ്ങൾ തികച്ചും ഒരു പുതിയ ചിത്രം സൃഷ്ടിക്കുന്നു, അത് നിലവിലുള്ള ചിത്രം മാറ്റുന്നതിൽ കാരണം അത് അനുവദനീയമല്ല ഈ ഫംഗ്ഷനുകളും ഉപയോഗിക്കാൻ അർത്ഥവുമില്ല. എന്നിരുന്നാലും, നിലവിലുള്ള ഇമേജ് മാറ്റുന്നതിൽ ചെയ്യുമ്പോൾ തീർച്ചയായും നിങ്ങൾ ഈ ഫംഗ്ഷനുകളും ഉപയോഗിക്കാൻ ആഗ്രഹിക്കുന്നു.

ഈ ഫംഗ്ഷനുകൾ ഉപയോഗിക്കുമ്പോൾ ഒരു സ്ക്രിപ്റ്റ് പഴയപടിയാക്കുന്നതിൽ ഏതാണ്ട് ഇനിയിപ്പോൾ പ്രവർത്തിക്കുന്നു.

3.6.2. സ്ക്രിപ്റ്റ് കുറച്ചുകൂടി വികസിപ്പിക്കുന്നു

ഇപ്പോൾ ഞങ്ങൾ ടെക്സ്റ്റ് ബോക്സുകൾ സൃഷ്ടിക്കാൻ ഒരു വേണമെന്നുമില്ല-ഡാൻഡി സ്ക്രിപ്റ്റ് ഉണ്ട്, അത് രണ്ട് സവിശേഷതകൾ പറയട്ടെ

  • നിലവിൽ, ഇമേജ് കൃത്യമായി ടെക്സ്റ്റ് ചുറ്റും fit വലുപ്പം ആണ് - അവിടെ ഡ്രോപ്പ് ഷാഡോകളും അല്ലെങ്കിൽ പ്രത്യേക ഇഫക്റ്റുകൾ പോലെ, ഒന്നും മുറി തുടർന്ന് (സ്ക്രിപ്റ്റുകളുണ്ട് സ്വയം ആവശ്യമെങ്കിൽ ചിത്രത്തിന്റെ വലുപ്പം ചെയ്യും പോലും). പാഠത്തെ ചുറ്റും ഒരു ബഫർ ചേർക്കുക അനുവദിക്കുക, പോലും ഉപയോക്താവ് ഫലമായി പാഠത്തിന്റെ വലുപ്പം ഒരു ശതമാനത്തില് ചേർക്കാൻ എത്ര ബഫർ വ്യക്തമാക്കുക ചെയ്യട്ടെ.

  • ഈ സ്ക്രിപ്റ്റ് എളുപ്പത്തിൽ ടെക്സ്റ്റ് പ്രവർത്തിക്കുന്ന മറ്റ് സ്ക്രിപ്റ്റുകളിൽ ഉപയോഗിക്കാം. അത് ചിത്രത്തിന്റെ ലെയറുകൾക്കുമായുള്ള തിരിച്ചെത്തി, അതിനാൽ മറ്റ് സ്ക്രിപ്റ്റുകൾ ഈ സ്ക്രിപ്റ്റ് വിളിച്ച് ഞങ്ങൾ സൃഷ്ടിക്കാൻ ഇമേജും പാളികൾ ഉപയോഗിക്കാൻ കഴിയും അങ്ങനെ ന്റെ അതിന് അവധി അനുവദിക്കുക.

3.6.3. നിർവചനങ്ങളും, രജിസ്ട്രേഷൻ ഫംഗ്ഷൻ പരിഷ്ക്കരിക്കുന്നത്

ഉപയോക്തൃ ബഫറിന്റെ തുക വ്യക്തമാക്കുക ചെയ്യട്ടെ, ഞങ്ങളുടെ ചടങ്ങിൽ രജിസ്ട്രേഷൻ ചടങ്ങിൽ ഒരു പാരാമീറ്റർ ചേർക്കും:

  (define (script-fu-text-box inTest inFont inFontSize inTextColor inBufferAmount)
  (let*
        (
           ; define our local variables
           ; create a new image:

3.6.4. പുതിയ കോഡ് ചേർക്കുന്നു

ഞങ്ങൾ രണ്ട് സ്ഥലങ്ങളിൽ കോഡ് ചേർക്കാൻ പോകാനൊരുങ്ങുന്ന; ഞങ്ങൾ ചിത്രത്തിന്റെ വലുപ്പം തൊട്ടുമുമ്പ്, സ്ക്രിപ്റ്റ് അവസാനം (പുതിയ ചിത്രം, പാളി പാഠം മടങ്ങാൻ).

ഞങ്ങൾ ടെക്സ്റ്റ് ഉയരം വീതി നേടുകയും ശേഷം ഞങ്ങൾ ഉപയോക്താവ് വ്യക്തമാക്കിയ ബഫർ തുക അടിസ്ഥാനമാക്കി ഈ മൂല്യങ്ങൾ വലുപ്പം മാറ്റേണ്ടതുണ്ട്. നാം അത് ഭീഷണിപ്പെടുത്തുന്നു life- അല്ല കാരണം അതു 0-100% വരെയാണ് ഉറപ്പാക്കുന്നതിന് പരിശോധിക്കുമ്പോൾ ഏതെങ്കിലും പിശക് ചെയ്യില്ല, കൂടാതെ ബഫറിന്റെ ശതമാനം ചേർക്കുന്നതിനായി ഉപയോക്താവിനെ “200” പോലുള്ള ഒരു മൂല്യം നൽകുക കഴിയില്ല കാരണങ്ങൾ ഒന്നും ഉള്ളതിനാൽ .

        (set! theBuffer (* theImageHeight (/ inBufferAmount 100) ) )

        (set! theImageHeight (+ theImageHeight theBuffer theBuffer) )
        (set! theImageWidth  (+ theImageWidth  theBuffer theBuffer) )

ഞങ്ങൾ ഇവിടെ ചെയ്യുന്നതിനെക്കുറിച്ച് എല്ലാ ടെക്സ്റ്റ് ഉയരം അടിസ്ഥാനമാക്കി ബഫർ ക്രമീകരണം ഞങ്ങളുടെ പുതിയ ചിത്രത്തിന്റെ ഉയരം വീതി ഇരുവരും രണ്ടുതവണ അത് ചേർക്കുന്നു. (നാം അതിനെ ബഫർ ടെക്സ്റ്റ് ഇരുഭാഗത്തും ചേർക്കും ആവശ്യമാണ് കാരണം രണ്ടുതവണ ഇരുവരും തലങ്ങളും ചേർക്കുക.)

ഇപ്പോൾ ഞങ്ങൾ ഒരു ബഫർ അനുവദിക്കുന്നതിന് ചിത്രം വലുപ്പം എന്നു ഞങ്ങൾ ഇമേജ് പാഠത്തിന്റെ കേന്ദ്രീകരിക്കുന്നതിന് വേണം. ഈ (theBuffer, theBuffer) എന്ന (X, Y) നിർദ്ദേശാങ്കങ്ങൾ ലേക്ക് പിടയുന്നത് ഇത് ചെയ്തത്. ഞാൻ പാളി ചിത്രം വ്യാപ്തിയിൽ ശേഷം ഈ ലൈൻ ചേർത്തു:

        (gimp-layer-set-offsets theText theBuffer theBuffer)

മുന്നോട്ട് പോയി നിങ്ങളുടെ സ്ക്രിപ്റ്റ് സംരക്ഷിക്കാൻ, ഡേറ്റാബേസിൽ പുതുക്കുന്നു ശേഷം അത് പരീക്ഷിക്കുക.

ചെയ്യാൻ നിലയിലാണെങ്കിൽ എല്ലാ ഞങ്ങളുടെ ചിത്ര, പാളി, ടെക്സ്റ്റ് പാളി മടങ്ങിവരും ആണ്. ചിത്രം പ്രദർശിപ്പിക്കുന്നത് ശേഷം ഞങ്ങൾ ഈ വരി ചേർക്കുക:

(പട്ടിക theImage theLayer theText)

ഈ ലിസ്റ്റിന്റെ അത് ഉപയോഗിക്കാൻ താൽപ്പര്യമുള്ള മറ്റ് സ്ക്രിപ്റ്റുകൾ ലഭ്യമാക്കുന്നതിന് ചടങ്ങിൽ അവസാന ലൈനാണ്.

മറ്റൊരു സ്ക്രിപ്റ്റിൽ നമ്മുടെ പുതിയ ടെക്സ്റ്റ് ബോക്സ് സ്ക്രിപ്റ്റ് ഉപയോഗിക്കുക, ഞങ്ങൾ താഴെ എന്തെങ്കിലും എഴുതുക കഴിഞ്ഞില്ല:

        (set! theResult (script-fu-text-box
                         "Some text"
                         "Charter" "30"
                         '(0 0 0)
                         "35"
                        )
        )
        (gimp-image-flatten (car theResult))

അഭിനന്ദനങ്ങൾ, നിങ്ങൾക്ക് സ്ക്രിപ്റ്റ്-ഫു നിങ്ങളുടെ ബ്ലാക്ക് ബെൽറ്റ് നിന്റെ വഴി നൽകുന്നത്!
