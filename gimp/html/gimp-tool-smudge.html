3.15. സ്മഡ്ജ്

ചിത്രം 14.95. Smudge tool


സ്മഡ്ജ് ടൂൾ സജീവ പാളി അല്ലെങ്കിൽ ഒരു നിര ന് നിറങ്ങൾ സ്മഡ്ജ് നിലവിലെ ബ്രഷ് ഉപയോഗിക്കുന്നു. അതു പേരിനു നിറം എടുത്തു നിങ്ങൾക്കു സജ്ജമാക്കാൻ കഴിയും അകലെ അത് പാലിക്കുന്നുണ്ടെന്ന് അടുത്ത നിറങ്ങൾ, അതു ഇളക്കുക അതു ഉപയോഗിക്കുന്നു.

3.15.1. ടൂൾ സജീവമാക്കുന്നു

You can find the Smudge tool in various ways :

  • through Tools → Paint Tools → Smudge. in the image menu,

  • by clicking on the tool icon: in Toolbox,

  • or by pressing the S key on keyboard.

3.15.2. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

മാറ്റം

Shift കീ സ്ഥലങ്ങളിൽ വര മോഡിലേക്ക് സ്മഡ്ജ് ഉപകരണം. ഹോൾഡിംഗ് Shift Button1 ക്ലിക്ക് ചെയ്യുമ്പോൾ നേരായ വരിയിൽ സ്മഡ്ജ് ചെയ്യും. തുടർച്ചയായ ക്ലിക്കുകൾ കഴിഞ്ഞ വരിയുടെ അവസാനം നിന്നും ആ നേരായ വരികളിൽ smudging തുടരും.

Ctrl,

Shift കൂടെ Ctrl ഉപയോഗിച്ച്, നിങ്ങൾ 15 ° കാലടി മാറും രണ്ട് തുടർച്ചയായി വരികൾ തമ്മിലുള്ള കോൺ ഒതുക്കിനിർത്താൻ കഴിയും.

3.15.3. ഓപ്ഷനുകൾ

ചിത്രം 14.96. The Smudge tool in Toolbox


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

Opacity; Brush; Dynamics; Dynamics Options; Fade Options; Apply Jitter; Hard Edge; Rate
പല അല്ലെങ്കിൽ എല്ലാ പെയിന്റ് ഉപകരണങ്ങൾ ബാധകമായ ഉപകരണം ഓപ്ഷനുകൾ ഒരു വിവരണം വേണ്ടി Common Paint Tool Options കാണുക.
