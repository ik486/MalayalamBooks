8.37. മാക്സ് ആർജിബി

8.37.1. പൊതു അവലോകനം

ചിത്രം 16.196. ഫിൽട്ടർ ഉദാഹരണം “Max RGB”

Original image

Filter “Max RGB” applied


ചിത്രത്തിന്റെ ഓരോ പിക്സൽ, ഈ ഫിൽട്ടർ ചുരുങ്ങിയ പരമാവധി / തീവ്രതയോടെ ചാനൽ താങ്ങി. ഫലം മാത്രം മൂന്നു നിറങ്ങൾ, ചുവപ്പ്, പച്ച, നീല, ഒരുപക്ഷേ ശുദ്ധമായ ചാര ഒരു ഇമേജ് ആണ്.

8.37.2. Activate the filter

This filter is found in the image window menu under Colors → Max RGB.

8.37.3. ഓപ്ഷനുകൾ

ചിത്രം 16.197. “Max RGB” options


പ്രിവ്യൂ

ഈ പ്രിവ്യൂ ഡിസ്പ്ലേകൾ തത്സമയം, ഫിൽറ്റർ ചികിത്സ ശേഷം ഫലമായുണ്ടാകുന്ന ചിത്രം.

Parameter Settings

Hold the maximal channels: ഓരോ പിക്സലിനായുള്ള, അരിപ്പ പരമാവധി തീവ്രത ഉണ്ട് പൂജ്യമായി രണ്ടും മറ്റ് കുറയുന്നു ആർജിബി നിറങ്ങളുടെ ചാനൽ തീവ്രത സൂക്ഷിക്കുന്നു. ഉദാഹരണത്തിന്: 220, 158, 175 max-- > 220, 0, 0 രണ്ട് ചാനലുകൾ ഒരേ തീവ്രത ഉണ്ടെങ്കിൽ, രണ്ടും നടക്കാറുണ്ട്: 210, 54, 210 max-- > 210, 0, 210.

Hold the minimal channels: ഓരോ പിക്സലിനായുള്ള, അരിപ്പ മിനിമൽ തീവ്രത ഉണ്ട് പൂജ്യമായി രണ്ടും മറ്റുള്ളവരെ കുറയ്ക്കുക ആർജിബി നിറങ്ങളുടെ ചാനൽ തീവ്രത സൂക്ഷിക്കുന്നു. ഉദാഹരണത്തിന്: 210, 54, 54 min-- > 0, 54, 54: 220, 158, 175 min-- > 0 158, 0. രണ്ടു വൈകല്യം ചാനലുകൾ രണ്ടും നടക്കാറുണ്ട്, ഒരേ തീവ്രത ഞങ്ങൾക്കുണ്ട്.

നേരിയ തീവ്രത മൂന്ന് ചാനലുകളിൽ ഒരേ മുതൽ ഗ്രേ അളവ് മാറില്ല.
