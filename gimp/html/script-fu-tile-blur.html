2.7. Tileable ഇഫക്റ്റ്

2.7.1. പൊതു അവലോകനം

ചിത്രം 17.18. “Tileable” ഫിൽട്ടർ ഉദാഹരണം

യഥാർത്ഥ

“Tileable Blur” ഫിൽട്ടറിന്


ഈ ഉപകരണം ടൈൽ പശ്ചാത്തലങ്ങളിൽ ഉപയോഗിക്കുന്ന ചിത്രങ്ങളിൽ ടൈൽ ഉള്പെടുത്തുക വിനീതമാവുകയും ഉപയോഗിക്കുന്നു. ഇത് പരസ്പരം അടുത്ത ടൈലിങ്ങിനുമേൽ ശേഷം എന്ന് യോജിപ്പിച്ച് ചിത്രങ്ങളും തമ്മിലുള്ള അതിർത്തി ബ്ലറിംഗ് ചെയ്യുന്നു.

സൂചന

നിങ്ങൾ ഇമേജുകൾ മാത്രം അതിരുകൾ പെരുമാറണം ആഗ്രഹിക്കുന്നുവെങ്കിൽ നിങ്ങൾക്ക് മുഴുവൻ ചിത്രം ഫിൽട്ടർ പ്രയോഗിക്കാൻ കഴിയില്ല. താഴെ പിടികൂടാനുള്ള പ്രതീതി ലഭിക്കാൻ പരിഹാരമാണ്:

  1. പാളി (Layer → Duplicate Layer) മാറ്റങ്ങള് അതിൽ പ്രവർത്തിക്കാൻ അത് തിരഞ്ഞെടുക്കുക.

  2. ഈ പാളി ഒരു 20 പിക്സൽ വ്യാസാർദ്ധം “Tileable Blur” ഫിൽട്ടർ പ്രയോഗിക്കുക.

  3. എല്ലാം (Ctrl+A) തിരഞ്ഞെടുത്ത് പിടികൂടാനുള്ള വീതി അതിർത്തി സൃഷ്ടിക്കാൻ നിരക്കു (Selection → Shrink) കുറയ്ക്കും.

  4. Selection → Feather ഉപയോഗിച്ച് നിരക്കു ഒരു കടൽപുറത്തെ ബോർഡർ തരിക.

  5. Ctrl+K ഉപയോഗിച്ച് തിരഞ്ഞെടുപ്പ് ഇല്ലാതാക്കുക.

  6. Layer → Merge down ലെയറുകൾ ലയിപ്പിക്കുക.

2.7.2. ഫിൽട്ടർ സജീവമാക്കുക

നിങ്ങൾ Filters → Blur → Tileable Blur കീഴിൽ ഇമേജ് മെനുവിൽ ഈ ഫിൽട്ടർ കണ്ടെത്താം.

2.7.3. ഓപ്ഷനുകൾ

ചിത്രം 17.19. “Tileable Blur” ഫിൽട്ടർ ഓപ്ഷനുകൾ


വാസാര്ദ്ധം

വലിയ ആരം കൂടുതൽ അടയാളപ്പെടുത്തിയ മങ്ങൽ ആണ്. Horizontal ആൻഡ് Vertical തിരഞ്ഞെടുക്കുന്നതിലൂടെ, തിരശ്ചീന, ലംബ അതിരുകൾ tileable കഴിയും.

ലംബമായി മങ്ങിക്കുന്നത്, തിരശ്ചീനമായി മങ്ങിക്കുന്നത്

ഈ ഓപ്ഷനുകൾ സ്വയം വിശദീകരിക്കുന്ന ആകുന്നു.

ഇഫക്റ്റ് തരം

പ്രയോഗിക്കപ്പെടുന്ന അൽഗോരിതം തിരഞ്ഞെടുക്കുക:

IIR

ഫോട്ടോഗ്രാഫിക് അല്ലെങ്കിൽ സ്കാൻ ചിത്രങ്ങൾ വേണ്ടി.

RLE

കമ്പ്യൂട്ടർ തയ്യാറാക്കിയ ചിത്രങ്ങൾ വേണ്ടി.
