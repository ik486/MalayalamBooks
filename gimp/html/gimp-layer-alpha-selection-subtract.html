7.40. Subtract from Selection

Subtract from Selection കമാൻഡ് ആൽഫാ ചാനൽ നിന്നുള്ള നിലവിലെ പാളി ഒരു നിര സൃഷ്ടിക്കുന്നു. അതാര്യമായസാധാരണഡയലോഗ്പുറകുവശം പിക്സൽ പൂർണ്ണമായി തിരഞ്ഞെടുത്തു, സുതാര്യമായ പിക്സൽ തിരഞ്ഞെടുത്തത് ആകുന്നു, അർദ്ധസുതാര്യ പിക്സൽ ഭാഗികമായും selected.This സെലക്ഷനും ആകുന്നു നിലവിലുള്ള നിരക്കു നിന്ന് subtracted ആണ്. ആൽഫാ ചാനലിന് സ്വയം മാറ്റിയില്ല.

7.40.1. Activate the Command

  • You can access this command from the image menubar through Layer → Transparency → Subtract from Selection.

ചിത്രം 16.105.  Applying “Subtract from Selection”
