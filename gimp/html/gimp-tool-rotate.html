ചിത്രം 14.119. The Rotate tool in Toolbox


4.5.1. പൊതു അവലോകനം

ഈ ഉപകരണം സജീവ പാളി, ഒരു നിര അല്ലെങ്കിൽ ഒരു പാത തിരിക്കുന്നതിന് ഉപയോഗിക്കുന്നു. ഈ ഉപകരണം ഉപയോഗിച്ച് ഇമേജ് അല്ലെങ്കിൽ നിരക്കു ക്ലിക്ക് ചെയ്യുമ്പോൾ ഒരു Rotation Information ഡയലോഗ് തുറന്നു നോക്കും. നിങ്ങൾക്ക് അവിടെ ഭ്രമണം അച്ചുതണ്ട്, ബിന്ദു അടയാളപ്പെടുത്തി, ഭ്രമണവും കോൺ സജ്ജമാക്കാൻ കഴിയും. നിങ്ങൾ ചിത്രം അല്ലെങ്കിൽ റൊട്ടേഷൻ പോയിന്റിൽ മൗസ് പോയിന്റർ ഇഴച്ച് അതേ ചെയ്യാൻ കഴിയും.

4.5.2. ടൂൾ സജീവമാക്കുന്നു

You can access the Selection Tool in different ways:

  • from the image menu bar Tools → Transform Tools → Rotate,

  • by clicking the tool icon: in the Toolbox,

  • by using the Shift+R key combination.

4.5.3. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

Ctrl,

Holding Ctrl will constrain the rotation angle to 15 degrees increments.

4.5.4. ഓപ്ഷനുകൾ

ചിത്രം 14.120. Rotation tool options


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

ദിശ മാറ്റുക

ഏതു വഴി അല്ലെങ്കിൽ ദിശ ഒരു പാളി തിരിക്കുമ്പോൾ ദിശ സെറ്റുകൾ സുന്ദരമാക്കൂ. ഒറ്റ പ്രതീക്ഷിക്കും ആയി സാധാരണ മോഡ് പാളി തിരിയും. ഒരു പാളി വലതുവശത്ത് 10 ഡിഗ്രി തിരിച്ചു എങ്കിൽ, പാളി പോലുള്ള റെൻഡർ ചെയ്യപ്പെടും. ഈ പെരുമാറ്റം പരിഹാര ഭ്രമണം വിരുദ്ധമാണെന്നാണ്.

പരിഹാര റൊട്ടേഷൻ പ്രാഥമികമായി നേരായ ഇല്ല എന്ന് ഡിജിറ്റൽ ചിത്രങ്ങൾ അറ്റകുറ്റം ഉപയോഗിക്കുന്നു. ചിത്രം 13 ഡിഗ്രി എങ്കിൽ askew തുടർന്ന്, ആ കോണിൽ തിരിക്കുക ശ്രമിക്കുക ആവശ്യമില്ല. പരിഹാര റൊട്ടേഷൻ ഉപയോഗിക്കുന്നതിലൂടെ നിങ്ങൾ കാഴ്ച്ചത്തകരാറുള്ളവർക്കും റൊട്ടേറ്റ് ചിത്രം ഉപയോഗിച്ച് പാളി ച്ചുകൊണ്ട് കഴിയും. രൂപാന്തരം തലതിരിയാത്ത, അല്ലെങ്കിൽ പിന്നോക്കവിഭാഗങ്ങളും നിവർത്തിച്ചിരിക്കുന്നു, ചിത്രം പിശക് തിരുത്താൻ മതിയായ കോണിൽ കൂടെ കറക്കണോ ചെയ്യും.

Constraints

15 Degrees (Ctrl) 15 ഡിഗ്രി ഗുണിതാമാണോ മലക്കുകളോട് ഭ്രമണം അതിടയാക്കും.

4.5.5. The Rotation Information window

ചിത്രം 14.121. The Rotation Information dialog window


കോൺ

ഇവിടെ നിങ്ങൾക്ക് റൊട്ടേഷൻ ആംഗിൾ -180 ° + ന് 180 ഡിഗ്രി, അതായത് 360 ° സജ്ജമാക്കാൻ കഴിയും.

Center X/Y

This option allows you to set the position of the rotation center, represented by a cross surrounded by a circle in the image. A click-and-drag on this point also allows you to move this center even outside the image. Default unit of measurement is pixel, but you can change it by using the drop-down list.
