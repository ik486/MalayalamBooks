6.3. വർണ്ണം പിക്കർ

ചിത്രം 14.180. പണിസഞ്ചിയിൽ കളർ പിക്കർ (കണ്ണ് dropper ഐക്കൺ)


കളർ പിക്കർ ടൂൾ നിങ്ങളുടെ സ്ക്രീനിൽ തുറന്നു ഏതെങ്കിലും ഇമേജ് ഒരു നിറം തിരഞ്ഞെടുക്കുന്നതിനായി ഉപയോഗിച്ച. ഒരു ചിത്രം ഒരു പോയിന്റ് ക്ലിക്കുചെയ്യുന്നതിലൂടെ, പോയിന്റർ കീഴിൽ സ്ഥിതിചെയ്യുന്നു കാര്യത്തിലേക്ക് സജീവ നിറം മാറ്റാൻ കഴിയും. സ്വതവേ, പ്രയോഗം സജീവ പാളി പ്രവർത്തിക്കുന്നു, എന്നാൽ Sample Merge ഐച്ഛികം എല്ലാ പാളികൾ കോമ്പിനേഷൻ തത്ഫലമായി, അതു സ്വരൂപത്തിൽ പോലെ നിറം പിടിച്ചെടുക്കാൻ അനുവദിക്കുന്നു. Only colors in visible layers are used. ഒരു Info window ചിത്രത്തിൽ ക്ലിക്ക് തുറക്കുന്നു.

6.3.1. ടൂൾ സജീവമാക്കുന്നു

നിങ്ങൾ നിരവധി മാർഗത്തിൽ ഈ ഉപകരണം പോകാനാകും:

  • Tools → Color Picker. വഴി ഇമേജ് മെനുവിൽ,

  • പണിസഞ്ചിയിൽ റ്റൂൾ ക്ലിക്കുചെയ്ത്,

  • O കീബോർഡ് കുറുക്കുവഴി അമർത്തി,

  • ഒരു പെയിന്റ് ഉപയോഗിക്കുമ്പോൾ Ctrl കീ അമർത്തി. വർണ്ണ-പിക്കർ ഡയലോഗ് ഈ പ്രവർത്തന വേളയിൽ തുറന്നു ചെയ്തിട്ടില്ലാത്തതിനാൽ ഉപകരണം കീ പ്രകാശനം മാറ്റമില്ല. എങ്കിലും Pointer window ഉപയോഗിച്ച് വിവരങ്ങൾ നേടാൻ കഴിയും.

6.3.2. കീ മോഡിഫയറുകൾ (സ്ഥിരസ്ഥിതികളിലേക്ക്)

Ctrl,

തിരഞ്ഞെടുക്കാവുന്നതാണ് മോഡ് Set foreground color സജ്ജമാക്കുമ്പോൾ ആണെങ്കിൽ, Ctrl കീ അമർത്തിയാൽ Set background color മോഡിലേക്ക് ഉപകരണം മാറുന്നു. തിരഞ്ഞെടുക്കാവുന്നതാണ് മോഡ് Set background color ലേക്ക് സജ്ജീകരിച്ച് കീ Set foreground color ലേക്ക് മോഡ് മാറുന്നു. തിരഞ്ഞെടുക്കാവുന്നതാണ് മോഡ് Pick only എപ്പോഴാണ്, കീ ഒന്നും ചെയ്യാൻ ഇല്ല.

മാറ്റം

നിങ്ങൾ ഒരു പിക്സൽ ക്ലിക്ക് ചെയ്യുമ്പോൾ Shift കീ അമർത്തിയാൽ, Color Picker Information window തുറന്നു നോക്കും.

കുറിപ്പു്

Pointer Information ശാശ്വതമായി നിങ്ങൾ ഒരേ വിവരങ്ങൾ നൽകുന്നു. എന്നാൽ താക്കീത് അത് Sample merged സ്ഥിര വേണം.

6.3.3. ഓപ്ഷനുകൾ

ചിത്രം 14.181. കളർ പിക്കർ ഓപ്ഷനുകൾ


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

സാമ്പിൾ ലയിപ്പിച്ചു

സാമ്പിൾ ലയിച്ച ചെക്ക്-ബോക്സ് പ്രാപ്തമാക്കിയാൽ എല്ലാ ദൃശ്യമായ പാളികൾ നിന്നുള്ള ഒരു പോലെ നിറം വിവരങ്ങൾ എടുക്കും. സാമ്പിൾ ലയനം സംബന്ധിച്ച കൂടുതൽ വിവരങ്ങൾക്ക് ഗ്ലോസ്സറി, Sample Merge ലഭ്യമാണ്.

സാമ്പിൾ ശരാശരി

Radius സ്ലൈഡർ തിരഞ്ഞെടുപ്പിനായി ഒരു ശരാശരി നിറം നിർണ്ണയിക്കാൻ ഉപയോഗിക്കുന്ന സ്ക്വയർ ഭാഗത്തിന്റെ വലിപ്പം മറക്കുക. നിങ്ങൾ പാളി ക്ലിക്കുചെയ്ത് നിലനിർത്തുമ്പോൾ, മൌസ് പോയിന്റർ സ്ക്വയർ അല്ലെങ്കിൽ ആരം വലിപ്പം കാണിക്കുന്നു.

മോഡ് തിരഞ്ഞെടുക്കുക
മാത്രം തിരഞ്ഞെടുക്കുക

തിരഞ്ഞെടുത്ത പിക്സൽ നിറം ഇൻഫർമേഷൻ ഡയലോഗിൽ കാണിക്കും, എന്നാൽ മറ്റുതരത്തിൽ ഉപയോഗിച്ചിട്ടില്ല.

ഫോർഗ്രൌണ്ട് കളർ സജ്ജമാക്കുക

പുരോതലനിറം, പണിസഞ്ചി കളർ ഏരിയ കാണുന്നത് പോലെ നിങ്ങൾ ക്ലിക്ക് പിക്സൽ നിറം ലേക്ക് സജ്ജമാക്കും.

പശ്ചാത്തല വർണ്ണം സജ്ജമാക്കുക

പശ്ചാത്തല നിറം, പണിസഞ്ചി കളർ ഏരിയ കാണുന്നത് പോലെ നിങ്ങൾ ക്ലിക്ക് പിക്സൽ നിറം ലേക്ക് സജ്ജമാക്കും.

പാലറ്റിൽ ചേർക്കുക

ഈ ഓപ്ഷൻ ബോക്സ് ശരിയടയാളമിട്ടാൽ വീഴ്ത്തി നിറം സജീവ വർണ്ണ പാലറ്റ് അയച്ചു. Palette Editor കാണുക.

വിവര ജാലകം ഉപയോഗിക്കുക

ഈ ഓപ്ഷൻ എപ്പോഴാണ് ചെക്കുചെയ്തിട്ടുണ്ടെന്നും, വിവരം വിൻഡോ സ്വയം തുറന്നു നോക്കും. Shift കീ താൽക്കാലികമായി ഈ സാധ്യത ടോഗിൾ അനുവദിക്കുന്നു.

ചിത്രം 14.182. കളർ പിക്കർ വിവര ജാലകം
