8.36. ചൂടുള്ള

8.36.1. പൊതു അവലോകനം

ഈ കമാൻഡ് തിരിച്ചറിയുകയും പാൽ അല്ലെങ്കിൽ NTSC ടിവി സ്ക്രീനിൽ പ്രദർശിപ്പിക്കാൻ പ്രശ്നങ്ങൾ കാരണമായേക്കും ഏത് പിക്സലുകൾ മാറ്റം.

8.36.2. Activate the command

You can access the command from the image menu bar through Colors → Hot.

ഈ കമാൻഡ് മാത്രം ആർജിബി മോഡിൽ ചിത്രങ്ങൾ പ്രവർത്തിക്കുന്നു, സജീവവും പാളി ഒരു ആല്ഫാ ചാനൽ ഇല്ല മാത്രമേ. അല്ലാത്തപക്ഷം മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്.

8.36.3. ഓപ്ഷനുകൾ

ചിത്രം 16.195. “Hot” options


മോഡ്

പാൽ അല്ലെങ്കിൽ NTSC: ടിവിയിലേക്ക് മോഡ് തിരഞ്ഞെടുക്കേണ്ടി.

ആക്ഷൻ

നിങ്ങൾ തിരഞ്ഞെടുക്കാം:

  • Reduce Luminency

  • Reduce Saturation

  • Blacken: ഈ കറുത്ത ഹോട്ട് പിക്സലുകൾ തിരിക്കും.

Create a new layer

ഈ ഓപ്ഷൻ ഉപയോഗിച്ച് പ്രവൃത്തി പകരം ചിത്രത്തിന്റെ ഒരു പുതിയ പാളി നടപ്പിലാക്കും. ഈ നിങ്ങൾക്ക് സമാധാനവും നൽകുന്നതാണ്
