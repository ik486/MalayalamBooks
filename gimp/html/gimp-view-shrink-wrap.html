5.5. Shrink Wrap

ഇത് നിലവിലെ സൂം ഘടകം ചിത്ര പോലെ കൃത്യമായി ഒരേ വലിപ്പം അങ്ങനെ Shrink Wrap കമാൻഡ് വിൻഡോ മാറ്റുന്നു. ചിത്രം പൂർണ്ണമായും സ്ക്രീനിൽ അനുയോജിക്കുന്നില്ലെങ്കിൽ, അങ്ങനെ ചിത്രത്തിന്റെ ഏറ്റവും വലിയ സാധ്യത ഭാഗം കാണിച്ചിരിക്കുന്നു ഇമേജ് വിൻഡോ വിശാലമായിരിക്കുന്നു. പ്രിഫറന്സ് ഡയലോഗിന്റെ Image Window പേജിൽ “Resize window on zoom” ആൻഡ് “Resize window on image size change” ഓപ്ഷനുകൾ നിർത്തുന്നില്ലെങ്കിൽ GIMP സ്വയം ഇത് ചെയ്യാൻ കാര്യം ശ്രദ്ധിക്കുക.

കുറിപ്പു്

ഇവിടെ സ്വഭാവസവിശേഷതകൾ നിവൃത്തിച്ചതുമില്ല എന്ന് GIMP സ്വയം എന്നാൽ “window manager”, നിങ്ങളുടെ കമ്പ്യൂട്ടറിന്റെ ഓപ്പറേറ്റിംഗ് സിസ്റ്റത്തിന്റെ ഒരു ഭാഗം കൊണ്ട് ശ്രദ്ധിക്കുക. അതുകൊണ്ടാണ് വിവരിച്ച പ്രവർത്തനക്ഷമത നിങ്ങളുടെ കമ്പ്യൂട്ടറിൽ വിവിധ ആയിരിക്കാം, അല്ലെങ്കിൽ ഏറ്റവും മോശം കേസിൽ, ഒട്ടും ലഭ്യമല്ലാത്ത.

5.5.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through View → Shrink Wrap,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Ctrl+J ഉപയോഗിച്ച്.
