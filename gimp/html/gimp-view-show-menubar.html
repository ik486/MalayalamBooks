5.19. Show Menubar

Show Menubar കമാൻഡ് പ്രാപ്തരാക്കുകയും മെനുബാർ പ്രദർശിപ്പിക്കുന്നത് നിഷ്ക്രിയമാക്കുന്നു. നിങ്ങൾ full-screen mode പ്രവര്ത്തിക്കുന്നയാളാണെങ്കില് അത് അപ്രാപ്തമാക്കാൻ ഉപയോഗപ്പെടുന്നു. മെനുബാർ പ്രദർശിപ്പിക്കില്ല, നിങ്ങൾ ചിത്രത്തിൽ മെനുബാർ എൻട്രികൾ ആക്സസ് വലത് ക്ലിക്ക് ചെയ്യാം.

നിങ്ങൾ Image Window Appearance ഡയലോഗിൽ മെനുബാർ സ്ഥിര സജ്ജമാക്കാൻ കഴിയും.

5.19.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through View → Show Menubar.
