4.18. To Path

To Path കമാൻഡ് ഒരു പാതയിലേക്ക് ഒരു നിര ആക്കാനുള്ള. ചിത്രം മാറ്റാൻ തോന്നുന്നില്ല, എന്നാൽ നിങ്ങൾ Paths Dialog പുതിയ പാത കാണാൻ കഴിയും. പണിസഞ്ചിയിൽ Path tool ഉപയോഗിക്കുന്നതിലൂടെ, കൃത്യമായും നിരക്കു രൂപരേഖ പൊരുത്തപ്പെടാൻ കഴിയും. കണ്ടെത്താവുന്നതാണ് Paths dialog വിഭാഗത്തിൽ പാതകൾ സംബന്ധിച്ച കൂടുതൽ വിവരങ്ങൾ.

4.18.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menu bar through Select → To Path.

  • നിങ്ങൾക്ക് Selection Editor നിന്നോ നിങ്ങൾ Advanced Options ധാരാളം പ്രദാനം ചെയ്ത Paths Dialog നിന്ന് അത് ആക്സസ് ചെയ്യാൻ കഴിയും.
