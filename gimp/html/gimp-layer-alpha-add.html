7. “Layer” മെനു  

7.33. ആൽഫ ചാനൽ ചേർക്കുക

Add Alpha Channel: ഒരു ആല്ഫാ ചാനൽ യാന്ത്രികമായി ഉടൻ ഒരു തലം നിങ്ങളുടെ ഇമേജ് ചേർക്കുക ചാനൽ ഡയലോഗിൽ ചേർത്തു. അത് ചിത്രത്തിന്റെ സുതാര്യത പ്രതിനിധീകരിക്കുന്നു. നിങ്ങളുടെ ചിത്രം ഒരേയൊരു പാളി ഉണ്ടെങ്കിൽ, ഈ പശ്ചാത്തലത്തിൽ പാളി ആൽഫ ചാനൽ ഉണ്ട്. ഈ സാഹചര്യത്തിൽ, ഈ കമാൻഡ് ഉപയോഗിച്ച് ആൽഫാ ചാനൽ ചേർക്കാൻ കഴിയും.

7.33.1. Activate the Command

  • You can access this command from the image menubar through Layer → Transparency → Add alpha Channel.

  • കൂടാതെ, Layer Dialog ന്, നിങ്ങൾ അതു സന്ദർഭം പോപ്പ്-അപ്പ് മെനുവിലെ Add Alpha Channel വഴി ആക്സസ് ചെയ്യാൻ കഴിയും.
