5.9. Xach-പ്രഭാവം
5. വെളിച്ചവും ഷാഡോ ഫിൽട്ടറുകൾ  

5.9. Xach-പ്രഭാവം

5.9.1. പൊതു അവലോകനം

ചിത്രം 17.120. “Xach-Effect” ഫിൽട്ടർ ഉദാഹരണം

യഥാർത്ഥ ചിത്രം

“Xach-Effect” പ്രയോഗിച്ചു


ഈ ഫിൽട്ടർ തിരഞ്ഞെടുത്ത പ്രദേശത്തെ അഥവാ ആൽഫാ ചാനലിലേക്ക് ഗവണ്മന്റ് അർദ്ധസുതാര്യ 3D പ്രഭാവം ചേർക്കുന്നു. ഈ 3D പ്രഭാവം കൈവരിച്ചിട്ടുള്ള

  1. നിരക്കു ഹൈലൈറ്റ്: പുതിയ പാളി (“Highlight”) സജീവ പാളി, ഹൈലൈറ്റ് വർണ്ണം നിറഞ്ഞു മുകളിൽ സൃഷ്ടിക്കപ്പെടും. അപ്പോൾ ഒരു layer mask അൺമാസ്ക് പിക്സൽ ഭാഗികമായെങ്കിലും സുതാര്യമായ പോന്നു പാളി ചേർക്കും.

    ലെയർ മാസ്ക് ഉപയോഗിച്ച് ഹൈലൈറ്റ് പാളി

  2. ഹൈലൈറ്റ് നിറം നിരക്കു ഇടതു മുകളിൽ അരികുകളും പെയിന്റിംഗ്: “Highlight” പാളി അപ്പ് വിട്ട് ഒരു പിക്സൽ ദീർഘിപ്പിക്കുന്നതാണ് എന്നു വേണ്ടി. ഈ ചെറിയ പ്രദേശങ്ങൾ സുതാര്യമല്ല ആയിരിക്കും.

  3. നിരക്കു താഴെ വലത് വശത്ത് ഒരു drop shadow ഉണ്ടാക്കുന്നു.

നിങ്ങൾ ഈ സ്ഥിരസ്ഥിതി ക്രമീകരണങ്ങൾ മാറ്റമുണ്ടാകാം ഉദാഹരണത്തിന് ഹൈലൈറ്റ് അല്ലെങ്കിൽ നിഴൽ വ്യത്യസ്ത നിറങ്ങൾ തിരഞ്ഞെടുത്ത് തുക ഓഫ്സെറ്റുകളുണ്ടു് എന്ന ദിശകൾ മാറ്റുക.

5.9.2. ഫിൽട്ടർ സജീവമാക്കുക

ഫിൽട്ടർ Filters → Light and Shadow → Xach-Effect കീഴിൽ ഇമേജ് വിൻഡോ മെനുവിലെ കാണപ്പെടുന്നത്.

5.9.3. ഓപ്ഷനുകൾ

ഓപ്ഷനുകൾ രണ്ടു ഗ്രൂപ്പുകൾ, ഓരോ ഹൈലൈറ്റ് അല്ലെങ്കിൽ നിഴൽ നിയന്ത്രിയ്ക്കാനുള്ള എന്നിവയും സ്വഭാവം ഒരു ചെക്ക്ബോക്സ് ഉണ്ട്.

ചിത്രം 17.121. “Xach-Effect” ഓപ്ഷനുകൾ


ഹൈലൈറ്റ് X ഓഫ്സെറ്റ്, ഓഫ്സെറ്റ് വൈ ഹൈലൈറ്റ്

നിരക്കു ഇടതു മുകളിൽ എഡ്ജ് ഹൈലൈറ്റ് നിറം ചായം. ഓഫ്സെറ്റ് ഹൈലൈറ്റ് അതാത് പ്രദേശത്തെ വലിപ്പം (വീതി അല്ലെങ്കിൽ ഉയരം) ആണ്. ഓഫ്സെറ്റ് 0 താഴെയാണെങ്കിൽ (ഈ സ്വതവേ), ഇടത് (X ഓഫ്സെറ്റ് < 0) അല്ലെങ്കിൽ മുകളിൽ (Y ഓഫ്സെറ്റ് < 0) പ്രദേശത്തെ നിറത്തിലുള്ള ചെയ്യും. ഓഫ്സെറ്റ് 0 വലിയവൻ, ശരിയായ (X ഓഫ്സെറ്റ് > 0) അല്ലെങ്കിൽ താഴെ (Y ഓഫ്സെറ്റ് > 0) പ്രദേശത്തെ ചായം ചെയ്യും.

ഹൈലൈറ്റ് വർണ്ണം

ഇത് തിരഞ്ഞെടുത്ത പ്രദേശം ഹൈലൈറ്റ് നിറം ആണ്. ഇത് വെളുത്ത സ്ഥിര, പക്ഷേ Swatch ബട്ടൺ ക്ളിക്ക് ഒരു കളർ സെലക്ടർ കൊണ്ട് വരികയും, നിങ്ങൾ മറ്റേതെങ്കിലും നിറം തിരഞ്ഞെടുക്കാം.

അതാര്യത ഹൈലൈറ്റ്

തിരഞ്ഞെടുക്കേണ്ട നിറം നിറഞ്ഞ ഒരു ഭാഗികമായും സുതാര്യമായ മേഖലയിലൂടെ മൂടി ചെയ്യും. ഈ ഐച്ഛികം സുതാര്യത നില സജ്ജീകരിക്കാൻ അനുവദിക്കുന്നു. ഒരു layer mask ഉപയോഗിക്കും എന്നതിനാൽ, മൂല്യം 0 മുതൽ (സുതാര്യതയോടെ) 255 (ഫുൾ ഒപാസിറ്റിയിൽ) വരെയാണ്.

66 ഹൈലൈറ്റ് അതാര്യത സ്ഥിരസ്ഥിതികളെ 26% തുല്യമാണ് ഏത്.

നിഴൽ ഓപ്ഷനുകൾ വലിച്ചിടുക

ഈ ഉപാധികൾ ബന്ധപ്പെട്ട Drop Shadow ഓപ്ഷനുകൾ (വലിപ്പംമാറ്റിക്കൊണ്ടിരിക്കുന്നജാലകങ്ങളിലുള്ളവകാണിയ്ക്കുക ഇല്ലാതെ) പ്രവർത്തിക്കുന്നു. ചുരുക്കത്തിൽ:

നിഴൽ വർണ്ണ ഡ്രോപ്പ്

ഒരു കളർ സെലക്ടർ തുറക്കാൻ ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുക.

ഒപാസിറ്റി വലിച്ചിടുക

അതാര്യത (0% - 100%) നിഴൽ അടങ്ങുന്ന പാളി.

നിഴൽ മങ്ങൽ ആരം Drop

നിഴൽ ബാധകമാകും ഏത് Gaussian blur ഫിൽറ്റർ ഉപയോഗിക്കുന്ന ആരം.

ഡ്രോപ്പ് ഷാഡോ X ഓഫ്സെറ്റ്, ഡ്രോപ്പ് ഷാഡോ Y ഓഫ്സെറ്റ്

മാർഗനിർദേശവും തുക, ഏത് നിഴല് നിരക്കു നിന്ന് നീക്കും.

നിരക്കു നിലനിർത്തുക

പരിശോധിച്ചാൽ, അരിപ്പ പ്രയോഗിച്ചു വരുമ്പോൾ സജീവ നിരക്കു സജീവ തുടരും.
