2.3. പാഥുകൾ സംഭാഷണം

നിങ്ങൾ ഒരു പാത എന്താണെന്ന് അറിയില്ലെങ്കിൽ ദയവായി വിഭാഗം 5, “ പാഥുകൾ ” കാണുക.

ചിത്രം 15.12. “Paths” ഡയലോഗ്


“Paths” ഡയലോഗ് പാതകൾ നിയന്ത്രിക്കാൻ, നിങ്ങൾ സൃഷ്ടിച്ചപ്പോൾ ഇല്ലാതാക്കാനോ അനുവദിച്ചുകൊണ്ട്, രക്ഷിക്കും നിന്നും തിരഞ്ഞെടുക്കലുകൾ മുതലായവ നല്കുന്നു ഉപയോഗിക്കുന്നു

2.3.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Paths” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • Windows → Dockable Dialogs → Paths: ഇമേജ് മെനുവിൽ നിന്ന്.

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Paths തിരഞ്ഞെടുത്ത്,

Windows മെനുവിൽ, കുറഞ്ഞത് ഒരു ഡയലോഗ് തുറന്നുതന്നെയിരിക്കുന്ന മാത്രമേ ഉണ്ട് detached windows ഒരു ലിസ്റ്റ് ഉണ്ട്. Windows → Paths: ഈ സാഹചര്യത്തിൽ, ഇമേജ്-മെനുവിൽ നിന്നും “Paths” ഡയലോഗ് ഉണ്ടാകാം.

2.3.2. പാഥുകൾ ഡയലോഗ് ഉപയോഗിച്ച്

ഓരോ പാത്ത് ഒറ്റ ചിത്രം ഉൾപ്പെടുന്നത്: പാതകളെ വെറും പാളികൾ പോലെ ചിത്രങ്ങൾ ഘടകങ്ങൾ ഉണ്ട്. വഴികളിലേക്ക് ഡയലോഗ് നിലവിൽ സജീവ ചിത്രം പെടുന്ന എല്ലാ പാതകളെ ഒരു ലിസ്റ്റ് കാണിക്കുന്നു: മാറുന്നതിൽ ചിത്രങ്ങൾ ഡയലോഗ് പാതകളുടെ ഒരു വ്യത്യസ്ത ലിസ്റ്റ് കാണിക്കാൻ കാരണമാകുന്നു. വഴികളിലേക്ക് ഡയലോഗ് ഒരു “Layers, Channels, and Paths” ഡോക്കിൽ എംബഡഡ് എങ്കിൽ, ഡോക്ക് മുകളിൽ ചിത്രം മെനു സജീവ ചിത്രത്തിന്റെ പേര് കാണാൻ കഴിയും. (അല്ലെങ്കിൽ, നിങ്ങൾ ഒരു ഇമേജ് മെനു ഡോക്കിലേയ്ക്ക് ടാബ് മെനുവിൽ നിന്ന് “Show Image Menu” തിരഞ്ഞെടുത്തുകൊണ്ട് ചേർക്കാൻ കഴിയും.)

നിങ്ങൾ ലയർ ഡയലോഗ് പരിചയമുണ്ട്, നിങ്ങൾക്ക് പാഥുകൾ ഡയലോഗ് നിരവധി മാർഗത്തിൽ സമാനമായ കാരണം, ഒരു തല ആരംഭം. ഓരോ പാത നാലു ഇനങ്ങൾ കൂടെ, ചിത്രം നിലനിൽക്കുന്ന എല്ലാ പാതകളെ ഒരു ലിസ്റ്റ് കാണിക്കുന്നു:

പാത ദൃശ്യപരത

ഒരു “open eye” ഐക്കൺ ഇത് അല്ലെങ്കിൽ പാത്ത് കാണാവുന്ന, ശൂന്യമായ ഇടമോ എങ്കിൽ. “Visible” പാത്ത് ഒരു പോറൽ ചിത്രം ഡിസ്പ്ലേ പാത്രമായി എന്നാണ്. അതു എന്തേ അല്ലെങ്കിൽ മറ്റുതരത്തിൽ റെൻഡർ അല്ലാതെ പാത്ത് യഥാർത്ഥത്തിൽ ചിത്രം പിക്സൽ ഡാറ്റ കാണിക്കില്ല. കണ്ണ്-symbol- ബഹിരാകാശത്ത് ക്ലിക്കുചെയ്യുന്നത് പാത ദൃശ്യപരത ടോഗിൾ ചെയ്യുന്നു.

ചെയിൻ പാതകൾ

ഇത് അല്ലെങ്കിൽ പാത രൂപാന്തരപ്പെടുത്തിയോ-ലോക്ക് ശൂന്യമായ ഇടമോ, ഒരു “chain” ചിഹ്നം കണ്ണ്-ചിഹ്നം-സ്പേസ് വലത് കൊടുത്തിരിക്കുന്നു. “Transform-locked” അത് എല്ലാ രൂപാന്തരണങ്ങളെ (തോതൊപ്പിക്കലോടൊത്ത്, ഭ്രമണം, തുടങ്ങിയവ) ഒരേ വിധത്തിൽ ബാധിച്ചു എന്ന് ഘടകങ്ങൾ (പാളികൾ, ചാനലുകൾ തുടങ്ങിയവ) ഒരു സെറ്റ് ഫോമിന്റെ ഭാഗമായ എന്നാണ് അവരിൽ ആരെങ്കിലും പ്രയോഗിച്ചു. ചെയിൻ-ചിഹ്നം-ബഹിരാകാശത്ത് ക്ലിക്കുചെയ്യുന്നത് പാതയുടെ രൂപാന്തരപ്പെടുത്തിയോ-ലോക്ക് നില ടോഗിൾ ചെയ്യുന്നു.

പ്രിവ്യൂ ചിത്രം

പാത ഒരു സ്കെച്ച് കാണിക്കുന്ന ഒരു ചെറിയ പ്രിവ്യൂ-ഐക്കൺ. നിങ്ങൾ ഐക്കൺ ക്ലിക്ക് ചെയ്ത് ഒരു ഇമേജ് കയറി ഡ്രാഗ് ഉണ്ടെങ്കിൽ, ഇത് ആ സ്വരൂപത്തിൽ പാത്ത് ഒരു പകർപ്പ് സൃഷ്ടിക്കും.

പാത പേര്

ചിത്രം അദ്വിതീയ ആയിരിക്കണം പാത്ത് പേര്. പേര് ഇരട്ട ക്ലിക്ക് നിങ്ങൾ അതു എഡിറ്റ് അനുവദിക്കും. നിങ്ങൾ സൃഷ്ടിക്കുന്ന പേര് ഇതിനകം നിലവിലുണ്ട് എങ്കിൽ, ഒരു നമ്പർ സവിശേഷമാക്കുന്നതിനായി കൂട്ടിച്ചേർക്കാനുള്ള (ഉദാ “#1”) ചെയ്യും.

ലിസ്റ്റ് നോൺ-ശൂന്യമാണെങ്കിൽ, ഏതു നിമിഷത്തിലും അവയവങ്ങളിൽ ഒന്നു അടിയിലായി ഡയലോഗ് മെനു അല്ലെങ്കിൽ ബട്ടണുകൾ ഉപയോഗിച്ച് അവതരിപ്പിക്കാനും ഏതെങ്കിലും ഓപ്പറേഷൻസ് വിഷയമാകും ഏത് ചിത്രം ന്റെ active path, ഇതാണ്: സജീവ പാതയിൽ എടുത്തുകാട്ടിയ കാണിച്ചിരിക്കുന്നു ലിസ്റ്റ്. എൻട്രികൾ ഏതെങ്കിലും ക്ലിക്കുചെയ്യുന്നത് സജീവ പാത ആക്കും.

ലിസ്റ്റിൽ ഏതെങ്കിലും എൻട്രി വലത്-ക്ലിക്ക് Paths Menu ചെയ്യുന്നു. നിങ്ങൾക്ക് ഡയലോഗ് ടാബ് മെനുവിൽ നിന്നും വഴികളിലേക്ക് മെനു ആക്സസ് ചെയ്യാൻ കഴിയും.

2.3.3. ബട്ടണുകൾ

എല്ലാ ഡയലോഗ് പാതകൾ ചുവടെയുള്ള ബട്ടണുകൾ വഴികളിലേക്ക് മെനു (വലത്-ക്ലിക്കുചെയ്ത് ആക്സസ് ഒരു പാത ലിസ്റ്റ് എൻട്രി) തുടർന്നും എൻട്രികൾ രൂപീകരിച്ചിരിക്കുന്നത് എന്നാൽ അവരിൽ ചിലരെ നിങ്ങൾ ബട്ടൺ അമർത്തിയാൽ സമയത്ത് മോഡിഫയർ കീകൾ ഇറക്കി പിടിച്ച് സംന്വാദിക്കാവുന്ന അധിക ഓപ്ഷനുകൾ ഉണ്ട്.

2.3.4. “Paths” സന്ദർഭ മെനു

ചിത്രം 15.13. “Paths” സന്ദർഭ മെനു


വഴികളിലേക്ക് മെനു വഴികളിലേക്ക് ഡയലോഗിൽ പട്ടികയിൽ ഒരു പാഥ് എൻട്രി ന്, അല്ലെങ്കിൽ പാതയിൽ ഡയലോഗ് ടാബ് മെനുവിൽ നിന്ന് മികച്ച എൻട്രി ( "വഴികളിലേക്ക് മെനു") തിരഞ്ഞെടുത്താൻ വലത്-ക്ലിക്കുചെയ്ത് കൊണ്ടുവന്ന കഴിയും. ഈ മെനു നിങ്ങൾ പാതകൾ ബാധിക്കുന്ന ഓപ്പറേഷൻസ് ഏറ്റവും ആക്സസ് നൽകുന്നു.

പാത ടൂൾ

Path Tool പാതകൾ സൃഷ്ടിക്കുന്നതും കൃത്രിമം ഉപയോഗിക്കുന്ന Path tool, സജീവമാക്കുന്നതിനായി, ഒരു ബദൽ മാർഗ്ഗം. ഇത് പണിസഞ്ചി നിന്ന്, അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി B (Bézier വേണ്ടി) ഉപയോഗിച്ച് സജീവമാക്കാം.

വഴി പ്രത്യേകതകൾ തിരഞ്ഞെടുക്കുക

Edit Path Attributes നിങ്ങൾ പാത്ത് പേര് മാറ്റാൻ അനുവദിക്കുന്ന ഒരു ചെറിയ ഡയലോഗ് ചെയ്യുന്നു. നിങ്ങൾക്ക് ഈ double- ക്ലിക്ക് ചെയ്ത് നാമം പട്ടികയിൽ വഴികളിലേക്ക് ഡയലോഗിൽ ചെയ്യാൻ കഴിയും.

പുതിയ പാത

New Path, ഒരു പുതിയ പാത സൃഷ്ടിക്കുന്നു വഴികളിലേക്ക് ഡയലോഗിൽ ലിസ്റ്റിൽ ചേർക്കുന്നു, അതു ഇമേജ് സജീവ പാത ചെയ്യുന്നു. അതു നിങ്ങൾക്കു പാത്ത് ഒരു പേര് അനുവദിക്കുന്ന ഒരു ഡയലോഗ് ചെയ്യുന്നു. നിങ്ങൾ ഒന്നും അത് ഉപയോഗിക്കാൻ കഴിയും കുറച്ച് നൽകാൻ പാത ഉപകരണം ഉപയോഗിക്കേണ്ടി വരും പുതിയ പാത, യാതൊരു നങ്കൂരം പോയിന്റുമായി സൃഷ്ടിക്കപ്പെട്ടിരിക്കുന്നത്.

പാത ഉന്നയിക്കുക

Raise Path വഴികളിലേക്ക് ഡയലോഗിൽ പട്ടികയിൽ ഒറ്റ സ്ലോട്ട് കൂടുതൽ പാത നീക്കുന്നു. പട്ടികയിൽ ഒരു പാതയുടെ സ്ഥാനം യാതൊരു ഫങ്ഷണൽ പ്രാധാന്യമുള്ള, അതിനാൽ ഈ ലളിതമായി നിങ്ങൾ ഓർഗനൈസ് കാര്യങ്ങൾ സൂക്ഷിക്കാൻ സഹായിക്കുന്നതിന് സൗകര്യമാണ്.

ലോവർ പാത

Lower Path വഴികളിലേക്ക് ഡയലോഗിൽ പട്ടികയിൽ ഒറ്റ സ്ലോട്ട് താഴത്തെ പാത നീക്കുന്നു. പട്ടികയിൽ ഒരു പാതയുടെ സ്ഥാനം യാതൊരു ഫങ്ഷണൽ പ്രാധാന്യമുള്ള, അതിനാൽ ഈ ലളിതമായി നിങ്ങൾ ഓർഗനൈസ് കാര്യങ്ങൾ സൂക്ഷിക്കാൻ സഹായിക്കുന്നതിന് സൗകര്യമാണ്.

പാത ഡ്യൂപ്ലിക്കേറ്റ്

“Duplicate Path”, സജീവ പാത ഒരു പകർപ്പ് സൃഷ്ടിക്കുന്നു അതിന് തനതായ പേര് നൽകുന്നു, വഴികൾ ഡയലോഗിൽ ലിസ്റ്റിൽ ചേർക്കുന്നു, അതു ഇമേജ് സജീവ പാത ചെയ്യുന്നു. പകർപ്പ് യഥാർത്ഥ പാത പ്രകടമായിരുന്നു മാത്രമേ ദൃശ്യമാകൂ.

പാത ഇല്ലാതാക്കുക

Delete Path നിലവിലെ തിരഞ്ഞെടുത്ത പാതയിൽ ഇല്ലാതാക്കുന്നു.

ദൃശ്യം പാഥുകൾ ലയിപ്പിക്കുക

Merge Visible Paths ദൃശ്യമാകുന്ന ചിത്രം എല്ലാ പാതകൾ എടുക്കും (ആ, വഴികൾ ഡയലോഗിൽ “open eye” ചിഹ്നങ്ങൾ കാണിക്കാൻ എല്ലാം), ഒരേ പാതയുടെ ഘടകങ്ങൾ അവരെ തിരിക്കുക. നിങ്ങൾ സ്ട്രോക്ക് അവരെ എല്ലാവരും അങ്ങനെ തന്നേ മുതലായവ ചെയ്യണമെങ്കിൽ ഈ അസൗകര്യം വരാം

തിരഞ്ഞെടുക്കുന്നതിലേക്ക് പാത; തിരഞ്ഞെടുക്കലിലേക്ക് ചേർക്കുക; തിരഞ്ഞെടുത്തതിന്റെ നിന്ന് കുറയ്ക്കേണ്ട; തിരഞ്ഞെടുക്കൽ ഉപയോഗിച്ച് കൂട്ടിമുട്ടുന്ന

ഈ കമാൻഡുകൾ എല്ലാം ഒരു നിര സജീവ പാത പരിവർത്തനം, തുടർന്ന് വ്യക്തമാക്കിയ വഴികളിൽ നിലവിലുള്ള നിരക്കു അത് സംയോജിപ്പിക്കുക. (“Path to Selection” നിലവിലുള്ള നിരക്കു ഉപേക്ഷിച്ച് മാർഗത്തിൽ നിന്ന് രൂപം ഒറ്റ അതിന് പകരം.) ആവശ്യമെങ്കിൽ, മാർഗ്ഗത്തിൽ ഏതൊരു അടയ്ക്കാത്ത ഘടകങ്ങൾ ഒരു വര ആദ്യ ആങ്കർ പോയിന്റ് അവസാന ആങ്കർ പോയിന്റ് ബന്ധിപ്പിച്ചുകൊണ്ട് അടച്ചുകഴിഞ്ഞു. തത്ഫലമായുണ്ടാകുന്ന നിരക്കു വേണ്ടി “marching ants” അടുത്തുനിന്ന് പാത പിന്തുടരുക, എന്നാൽ എഴുത്തുകുത്തുകളും തികഞ്ഞ പ്രതീക്ഷിക്കുന്നില്ല.

പാതയിലേക്കും തിരഞ്ഞെടുക്കൽ

ഈ പ്രവർത്തനം നിരവധി വഴികളിൽ ആക്സസ് ചെയ്യാം:

  • , ഒരു ഇമേജ് മെനുബാർ നിന്നും Select → To Path ആയി

  • , വഴികൾ ഡയലോഗ് മെനുവിൽ നിന്നും Selection to Path ആയി.

  • വഴികളിലേക്ക് ഡയലോഗ് താഴെയുള്ള Selection to Path ബട്ടൺ നിന്ന്.

Selection to Path ചിത്രം ൻറെ നിരക്കു നിന്ന് ഒരു പുതിയ പാത സൃഷ്ടിക്കുന്നു. മിക്കപ്പോഴും ഫലമായുണ്ടാകുന്ന പാത്ത് അടുത്തുനിന്ന് തിരഞ്ഞെടുപ്പിന്റെ “marching ants” പിന്തുടരും; എങ്കിലും കത്തുകളിൽ സാധാരണയായി തികഞ്ഞ കഴിയില്ല.

ഒരു ഏകമാന പാതയിലേക്ക് ഒരു ദ്വിമാന നിരക്കു മാസ്ക് മാറ്റുമ്പോൾ ചില പകരം മാറിമറിഞ്ഞത് അൽഗോരിതങ്ങൾ കേന്ദ്രീകരിക്കുന്നു: നിങ്ങൾ ചുവടെയുള്ള Selection to Path ബട്ടൺ അമർത്തിക്കൊണ്ട് Shift കീ പിടിച്ച് ലഭ്യമാക്കുന്ന അതു Advanced Options ഉപയോഗിച്ച് നടക്കുന്ന വഴി, ഭേദഗതി പാഥുകൾ ഡയലോഗ്. നിങ്ങൾക്ക് എല്ലാ നിഗൂഢ പേരുകൾ, 20 വ്യത്യസ്ത ഓപ്ഷനുകൾ ചരങ്ങളും സജ്ജമാക്കാൻ അനുവദിക്കുന്ന വിപുലമായ ഓപ്ഷനുകൾ ഡയലോഗ്, ചെയ്യുന്നു. വിപുലമായ ഓപ്ഷനുകൾ ശരിക്കും ഡെവലപ്പർമാർക്ക് മാത്രമേ ഉദ്ദേശിച്ചിട്ടുള്ള, അവരോട് സഹായിക്കാൻ ഈ ഡോക്യുമെന്റേഷൻ പാഠത്ത് പോകുന്നു. പൊതുവെ പറഞ്ഞാൽ, Selection to Path അതിൽ നിങ്ങൾ പ്രതീക്ഷിക്കുന്നില്ല എന്തു ചെയ്യും, നിങ്ങൾ (നിങ്ങൾ ആഗ്രഹിക്കുന്ന പക്ഷം) എങ്ങനെ അതു ചെയ്തു വിഷമിക്കേണ്ടതില്ല ആവശ്യമില്ല.

സ്ട്രോക്ക് പാത

ഈ പ്രവർത്തനം നിരവധി വഴികളിൽ ആക്സസ് ചെയ്യാം:

  • , ഒരു ഇമേജ് മെനുബാർ നിന്നും Edit → Stroke Path ആയി

  • , വഴികൾ ഡയലോഗ് മെനുവിൽ നിന്നും Stroke Path ആയി.

  • വഴികളിലേക്ക് ഡയലോഗ് താഴെയുള്ള Paint along the path ബട്ടൺ നിന്ന്.

  • പാത ടൂളിനായി ടൂൾ ഓപ്ഷനുകൾ ൽ Stroke Path ബട്ടണിൽ നിന്ന്.

“Stroke Path” ലൈൻ ശൈലികളും അതാണെന്ന് ഓപ്ഷനുകൾ വൈവിധ്യമാർന്ന അനുമതി, ചിത്രത്തിന്റെ സജീവ പാളി സജീവമായ പാത റെൻഡർ. കൂടുതൽ വിവരങ്ങൾക്ക് Stroking വിഭാഗം കാണുക.

വഴിയും പകർത്തുക

Copy Path പകർപ്പുകൾ വഴികളിലേക്ക് ക്ലിപ്ബോർഡ് ലേക്ക് സജീവ പാത്ത്, മറ്റൊരു ചിത്രം ഒട്ടിക്കുകയും പ്രാപ്തരാക്കുകയെന്ന.

സൂചന

നിങ്ങൾക്ക് പകർത്തി ടാർഗെറ്റ് ചിത്രം പ്രദർശന കടന്നു വഴികളിലേക്ക് ഡയലോഗ് നിന്ന് അതിന്റെ ഐക്കൺ ഇഴച്ച് ഒരു പാത്ത് ഒട്ടിക്കാൻ കഴിയും.
കുറിപ്പു്

നിങ്ങൾ ഒരു ഇമേജ് ഒരു വഴിയും പകർത്തുക, അത് കാണാൻ കഴിയില്ല. നിങ്ങൾ പാത ഡയലോഗിൽ ദൃശ്യമാക്കാൻ ഞങ്ങൾക്കുണ്ട്.
ഒട്ടിക്കുക പാത

Paste Path പാത ക്ലിപ്ബോർഡ് ഉളളടക്കം നിന്ന് ഒരു പുതിയ പാത സൃഷ്ടിക്കുന്നു, വഴികൾ ഡയലോഗിൽ ലിസ്റ്റിൽ ചേർക്കുന്നു, അതു ഇമേജ് സജീവ പാത ചെയ്യുന്നു. പാതയൊന്നും മുമ്പ് ക്ലിപ്പ്ബോർഡിലേക്ക് പകർത്തി എങ്കിൽ, മെനു എൻട്രി സംവേദനക്ഷമരഹിത ആയിരിക്കും.

ഇംപോർട്ട് പാത

Import Path എസ്.വി.ജി. ഫയലിൽ നിന്നും ഒരു പുതിയ പാത സൃഷ്ടിക്കുന്നു: അത് നിങ്ങൾക്ക് ഫയൽ നാവിഗേറ്റ് അനുവദിക്കുന്ന ഒരു ഫയൽ തിരഞ്ഞെടുക്കൽ ഡയലോഗ് മാറികിട്ടി. എസ്.വി.ജി. ഫയലുകൾ വിവരങ്ങൾ അവർ എങ്ങനെ ജിമ്പ് പാതകൾ നൽകിക്കൊണ്ടിരിക്കുന്നത് വേണ്ടി Paths വിഭാഗം കാണുക.

കയറ്റുമതി പാത

Export Path ഒരു ഫയൽ പാത്ത് സംരക്ഷിക്കാൻ അനുവദിക്കുന്നു: അതു നിങ്ങൾ ഫയൽ പേരും സ്ഥലവും വ്യക്തമാക്കാൻ അനുവദിക്കുന്ന ഡയലോഗ് സംരക്ഷിക്കുക ഒരു ഫയൽ മാറികിട്ടി. നിങ്ങൾക്ക് പിന്നീട് Import Path കമാൻഡ് ഉപയോഗിച്ച് ഏത് ജിമ്പ് ഇമേജ് ഈ പാത ചേർക്കാൻ കഴിയും. പാതകൾ സംരക്ഷിക്കുന്നത് ഉപയോഗിക്കുന്ന ഫോർമാറ്റ് എസ്.വി.ജി.; ഈ അത്തരം Sodipodi അല്ലെങ്കിൽ Inkscape വെക്റ്റർ-ഗ്രാഫിക്സ് പ്രോഗ്രാമുകളും നിങ്ങൾ സംരക്ഷിക്കുക പാതകളെ ഇംപോർട്ട് കഴിയും എന്നാണ്. എങ്ങനെയാണ് അവർ ജിമ്പ് പാതകളെ നൽകിക്കൊണ്ടിരിക്കുന്നത് എസ്.വി.ജി. ഫയലുകൾ കൂടുതൽ വിവരങ്ങൾക്ക് Paths വിഭാഗം കാണുക.
