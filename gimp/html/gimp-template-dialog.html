4. ചിത്രം മാനേജ്മെന്റ് ബന്ധപ്പെട്ട ഡയലോഗുകൾ  

4.4. ടെംപ്ലേറ്റുകൾ സംഭാഷണം

ചിത്രം 15.68. ടെംപ്ലേറ്റുകൾ ഡയലോഗ്


ടെംപ്ലേറ്റുകൾ സൃഷ്ടിച്ച ഒരു ചിത്രം ഫോർമാറ്റ് ടെംപ്ലേറ്റുകൾ ആകുന്നു. ജിമ്പ് നിങ്ങൾ ഫലകങ്ങൾ ധാരാളം പ്രദാനം നിങ്ങളുടെ സ്വന്തമാക്കുന്നു സൃഷ്ടിക്കാൻ കഴിയും. ഒരു പുതിയ ഇമേജ് സൃഷ്ടിക്കുമ്പോൾ, നിലവിലുള്ള ഫലകങ്ങൾ ലിസ്റ്റിലേക്ക് ആക്സസ് പക്ഷേ നീ അവരെ നിയന്ത്രിക്കാൻ കഴിയില്ല. “Templates” ഡയലോഗ് നിങ്ങൾ ഈ ഫലകങ്ങൾ നിയന്ത്രിക്കുന്നതിന് അനുവദിക്കുന്നു.

4.4.1. ഡയലോഗ് സജീവമാക്കുന്നു

“Templates” ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം വിഭാഗം 2.3, “ഡയലോഗുകൾ, ഡോക്കിങ്” കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • Windows → Dockable Dialogs → Templates: ഇമേജ് മെനുവിൽ നിന്ന്.

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Templates തിരഞ്ഞെടുത്ത്.

4.4.2. ടെംപ്ലേറ്റുകൾ ഡയലോഗ് ഉപയോഗിച്ച്

അവയുടെ ഐക്കണിൽ ക്ലിക്കുചെയ്ത് ഒരു ടെംപ്ലേറ്റ് തിരഞ്ഞെടുക്കൂ. റൈറ്റ് ക്ലിക്ക് ബട്ടണുകൾ അതേ ഫംഗ്ഷനുകൾ വാഗ്ദാനം ഒരു പ്രാദേശിക മെനു വെളിപ്പെടുത്തുന്നു.

4.4.2.1. ഗ്രിഡ് / പട്ടിക മോഡുകൾ

“Templates” ഡയലോഗ് ടാബ് മെനുവിൽ View as Grid ആൻഡ് View as List തമ്മിലുള്ള തിരഞ്ഞെടുക്കാം. ഗ്രിഡ് മോഡിൽ, ഫലകങ്ങൾ ഔട്ട് ഒരേപോലുള്ള ഐക്കണുകളുടെ ചതുരാകൃതിയിലുള്ള അണിനിരന്നു (ഞങ്ങൾ പിന്നീട് നോക്കാം അവരെ ഒരു പ്രത്യേക ഐക്കൺ കൊടുത്തു പക്ഷം) കിടക്കുന്നു. തിരഞ്ഞെടുത്ത ടെംപ്ലേറ്റ് പേര് മാത്രം പ്രദർശിപ്പിച്ചിരിക്കുന്നു. പട്ടിക മോഡിൽ, അവർ ലംബമായി വരിവരിയായി ചെയ്യുന്നു; ഐക്കണുകൾ വളരെ തനിപ്പകർപ്പാണോയെന്ന്; എല്ലാ പേരുകളും പ്രദർശിപ്പിച്ചിരിക്കുന്നു.

ഈ ടാബ് മെനുവിൽ, Preview Size ഐച്ഛികം ലഘുചിത്രങ്ങൾ വലിപ്പം മാറ്റാൻ അനുവദിക്കുന്നു.

സൂചന

ഒരു ലിസ്റ്റ് കാഴ്ചയിൽ Ctrl+F ഒരു തിരയൽ ഫീൽഡ് തുറക്കുന്നു. പട്ടിക ആയി കാണുക; ഗ്രിഡായി കാണുക കാണുക
4.4.2.2. ചുവടെയുള്ള ബട്ടണുകൾ

ഡയലോഗ് ചുവടെയുള്ള ബട്ടണുകൾ പല വഴികളിൽ ടെംപ്ലേറ്റുകൾ ഓപ്പറേറ്റ് അനുവദിക്കുന്നു:

തിരഞ്ഞെടുത്ത ടെംപ്ലേറ്റ് നിന്നും ഒരു പുതിയ ചിത്രം സൃഷ്ടിക്കുക

ഈ ബട്ടൺ ക്ലിക്കുചെയ്യുന്നത് തിരഞ്ഞെടുത്ത ടെംപ്ലേറ്റ് മാതൃകയിൽ ഡയലോഗ് Create a new image തുറക്കുന്നു.

ഒരു പുതിയ ടെംപ്ലേറ്റ് സൃഷ്ടിക്കുക

ഈ ബട്ടൺ ക്ലിക്കുചെയ്യുന്നത്, New template ഡയലോഗ്, എഡിറ്റ് ഫലകം ഡയലോഗ് ഒരേപോലുള്ള തുറക്കുന്നു താഴെ ഞങ്ങൾ കാണും എന്നു.

തിരഞ്ഞെടുത്ത ടെംപ്ലേറ്റ് ഡ്യൂപ്ലിക്കേറ്റ്

ഈ ബട്ടൺ ക്ലിക്കുചെയ്യുന്നത് നാം ഇപ്പോൾ പഠിക്കാൻ പോകുന്ന എഡിറ്റ് ഫലകം ഡയലോഗ് തുറക്കുന്നു.

തിരഞ്ഞെടുത്ത ടെംപ്ലേറ്റ് എഡിറ്റുചെയ്യുക

ഈ ബട്ടൺ ക്ലിക്കുചെയ്യുന്നത് Edit Template ഡയലോഗ് തുറക്കുന്നു.

തിരഞ്ഞെടുത്ത ടെംപ്ലേറ്റ് ഇല്ലാതാക്കുക

എന്താണെന്ന് ഊഹിക്കുക?

സൂചന

ഓരോ ടെംപ്ലേറ്റ് നിങ്ങളുടെ സ്വകാര്യ GIMP ഡയറക്ടറിയിൽ ഒരു templaterc ഫയൽ ശേഖരിക്കും. നിങ്ങൾ ചില ഇല്ലാതാക്കി ഫലകങ്ങൾ വീണ്ടെടുക്കേണ്ട, നിങ്ങൾക്ക് GIMP ന്റെ സിസ്റ്റം അറ etc/gimp/2.0 ഡയറക്ടറിയിൽ മാസ്റ്റർ templaterc ഫയലിൽ നിന്നും നിങ്ങളുടെ ഫയൽ പകർത്തുക അല്ലെങ്കിൽ ഉൾപ്പെടുത്താൻ കഴിയും ടെംപ്ലേറ്റ് എൻട്രികൾ.

4.4.3. ഫലകം എഡിറ്റ്

ചിത്രം 15.69. ഫലകം ഡയലോഗ് എഡിറ്റ്


ഡയലോഗ് നിങ്ങൾ തിരഞ്ഞെടുത്ത ടെംപ്ലേറ്റ് പ്രത്യേകതകൾ സജ്ജീകരിക്കാൻ അനുവദിക്കുന്നു.

ഡയലോഗ് ചുവടെ Edit Template ബട്ടൺ ക്ളിക്ക് ഈ എഡിറ്റർ ആക്സസ് ചെയ്യാൻ കഴിയും.

ഓപ്ഷനുകൾ

പേര്

ഈ ടെക്സ്റ്റ് ബോക്സിൽ നിങ്ങൾക്ക് പ്രദർശിപ്പിച്ച ടെംപ്ലേറ്റ് പേര് പരിഷ്കരിക്കാനും കഴിയും.

ഐക്കൺ

ഈ ഐക്കണിൽ ക്ലിക്കുചെയ്യുന്നതിലൂടെ, ഐക്കണുകൾ ഒരു ലിസ്റ്റ് തുറക്കുക. തിരഞ്ഞെടുത്ത ടെംപ്ലേറ്റ് നാമം ചിത്രീകരിക്കാൻ അവരിൽ ഒരാൾ തിരഞ്ഞെടുക്കാം.

ചിത്ര വലുപ്പം

ഇവിടെ നിങ്ങൾ വീതി പുതിയ ചിത്രത്തിന്റെ ഉയരം വെച്ചു. സ്ഥിര യൂണിറ്റുകൾ പിക്സൽ, എങ്കിലും നിങ്ങൾ .ഹാളിനൊരുവശത്തായി മെനു ഉപയോഗിച്ച്, നിങ്ങൾക്ക് ഇഷ്ടമാണെങ്കിൽ മറ്റ് ചില യൂണിറ്റ് മാറാനാകും. നിങ്ങൾ ചെയ്യേണ്ടത്, ലഭിക്കുന്ന പിക്സൽ വലിപ്പം നിർണ്ണയിക്കുന്നത് X, Y റെസലൂഷൻ (നിങ്ങൾ വിപുലമായ ഓപ്ഷനുകൾ മാറ്റില്ല കഴിയും) വഴി, നിങ്ങൾ View മെനുവിൽ മാറ്റാൻ കഴിയും “Dot for Dot” അസ്തമന, വഴി ശ്രദ്ധിയ്ക്കുക.

കുറിപ്പു്

ഒരു ചിത്രത്തിന്റെ ഓരോ പിക്സൽ മെമ്മറി സംഭരിച്ചിരിക്കുന്ന, ഓർമ്മിക്കുക. നിങ്ങൾ പിക്സൽ ഉയർന്ന സാന്ദ്രത വലിയ ഫയലുകൾ സൃഷ്ടിക്കുന്നത് എങ്കിൽ, GIMP ഇമേജിൽ ലേക്ക് അപേക്ഷിക്കുന്ന ആണെങ്കിൽ ഓരോ ചടങ്ങിൽ കുറച്ച് സമയം ആവശ്യമാണ്.
ഛായാചിത്രം / ലാൻഡ്സ്കേപ്പ് ബട്ടണുകൾ

ഈ ബട്ടണുകൾ അർദ്ധകായ സ്കേപ്പ് മോഡ് തമ്മിൽ ടോഗിൾ. മൂർത്തമായി, അവയുടെ ഫലം വീതിയും ഉയരവും മൂല്യങ്ങളെ കൈമാറാൻ ആണ്. X, Y റെസലൂഷൻ (വിപുലമായ ഓപ്ഷനുകൾ ൽ) വ്യത്യസ്തമാണ് എങ്കിൽ, ഈ മൂല്യങ്ങൾ പുറമേ മാറ്റിക്കളഞ്ഞു ചെയ്യുന്നു. വലതുവശത്ത്, ഇമേജ് സൈസ്, ഇമേജ് മിഴിവും കളർ സ്പേസ് പ്രദർശിപ്പിക്കുന്നു.

വിപുലമായ ഓപ്ഷനുകൾ

ചിത്രം 15.70. “Advanced Options” ഡയലോഗ്


ഈ കൂടുതൽ വിപുലമായ ഉപയോക്താക്കൾക്ക് പലിശ പ്രധാനമായും ആകും എന്നു ഉപാധികൾ.

X, Y ചിത്രം

ഈ മൂല്യങ്ങൾ പ്രധാനമായും പ്രിന്റിംഗ് ബന്ധപ്പെട്ട് പ്ലേ വരികയുമില്ല അവർ പിക്സലിൽ ചിത്രത്തിന്റെ വലുപ്പം ബാധിക്കുന്നില്ല, പക്ഷേ അച്ചടിച്ചാൽ അവർ കടലാസിൽ അതിന്റെ വലിപ്പം നിർണ്ണയിക്കാൻ. പിന്നീട് 100% സൂമിലോ ജിമ്പ് ശരിയായ ശാരീരിക വലുപ്പത്തിൽ മോണിറ്ററിൽ ചിത്രം പ്രദർശിപ്പിക്കാൻ, പിക്സൽ കണക്കാക്കുന്നതുമല്ല പോലെ ശ്രമിക്കുന്നു “Dot for Dot” View മെനുവിൽ സ്വിച്ച് ഓഫ് ചെയ്യുകയാണെങ്കിൽ: അവർ ഇമേജ് മോണിറ്റർ പ്രദർശിപ്പിക്കണം വഴി ബാധിക്കും തലങ്ങളും ചിത്രം. ഡിസ്പ്ലേ കൃത്യവും, മോണിറ്റർ പാകപ്പെടുത്തി ചെയ്തു അല്ലാതെ, എന്നാൽ വന്നേക്കാം. ഈ ചെയ്തതു കഴിയും ഒന്നുകിൽ ജിമ്പ് ഇൻസ്റ്റോൾ ചെയ്യുമ്പോൾ, അല്ലെങ്കിൽ മുൻഗണനകൾ ഡയലോഗ് Display tab നിന്ന്.

Colorspace

നിങ്ങൾ ഒരു ആർജിബി ചിത്രം അല്ലെങ്കിൽ ഒരു ഗ്രേസ്കെയിലാക്കി ചിത്രം ഒന്നുകിൽ പുതിയ ഇമേജ് തയ്യാറാക്കാം. ഈ വഴിയിൽ നേരിട്ട് ഗണമാണോയെന്നത് ചിത്രം സൃഷ്ടിക്കാൻ കഴിയില്ല, പക്ഷേ ഒന്നും സൃഷ്ടിച്ചുകഴിയുമ്പോൾ ശേഷം ഇൻഡെക്സ്ഡ് മോഡിലേക്ക് ചിത്രം പരിവർത്തനം നിന്നും നിങ്ങളെ തടയുന്നു.

നിറയ്ക്കുക

നിങ്ങൾ പുതിയ ചിത്രം പശ്ചാത്തലത്തിൽ പാളി നിറെക്കും ഖര നിറം നാലു മാറ്ഗ്ഗങ്ങളുണ്ട്:

  • Foreground color, മെയിൻ പണിസഞ്ചിയിൽ കാണിച്ചിരിക്കുന്നതുപോലെ.

  • Background color, മെയിൻ പണിസഞ്ചിയിൽ കാണിച്ചിരിക്കുന്നതുപോലെ.

  • White, പലപ്പോഴും ഉപയോഗിക്കുന്ന.

  • Transparent. ഈ ഓപ്ഷൻ തിരഞ്ഞെടുത്തിരിക്കുന്നു എങ്കിൽ, പുതിയ ചിത്രത്തിൽ പശ്ചാത്തല പാളി ഒരു ആല്ഫാ ചാനലുമായി സൃഷ്ടിക്കപ്പെടും; മറ്റുതരത്തിൽ അല്ല.

അഭിപ്രായം

നിങ്ങൾക്ക് ഇവിടെ ഒരു വിവരണാത്മക അഭിപ്രായം എഴുതാം. ടെക്സ്റ്റ് ഒരു “parasite” ചിത്രം അറ്റാച്ചുചെയ്യും, ചില ഫയൽ ഫോർമാറ്റുകൾ (പക്ഷേ അവയെല്ലാം) ചിത്രം സഹിതം സംരക്ഷിക്കും.
