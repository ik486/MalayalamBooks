6.15. സ്കെയിൽ ചിത്രം

Scale Image കമാൻഡ് ഇത് അടങ്ങുന്ന പിക്സലുകളുടെ എണ്ണം മാറ്റുന്നതിലൂടെ ശക്തന്മാരായ ചിത്രത്തിന്റെ ശാരീരിക വലുപ്പം കുറയ്ക്കുന്നു. അത് ചിത്രത്തിന്റെ ഉള്ളടക്കങ്ങളുടെ വലിപ്പ മാറ്റങ്ങള് അതിനനുസരിച്ച് ക്യാൻവാസ് മാറ്റുന്നു.

മുഴുവൻ ചിത്രത്തിൽ പ്രവർത്തിക്കുന്നു. നിങ്ങളുടെ ഇമേജ് വ്യത്യസ്ത വലിപ്പത്തിലുള്ള പാളികൾ ഉണ്ടെങ്കിൽ, ഇമേജ് ചെറിയ making അവരിൽ ചിലർ ഒന്നും ഇറങ്ങി, ഒരു പാളി ഒരു പിക്സൽ വൈഡ് അല്ലെങ്കിൽ ഉയർന്ന കുറവായി കഴിയാത്ത സ്ഥിതിക്ക് അകലുകയും കഴിഞ്ഞില്ല. ഇങ്ങനെ സംഭവിക്കുകയാണെങ്കിൽ, നിങ്ങൾ ഓപ്പറേഷൻ മുമ്പുള്ള മുന്നറിയിപ്പ് ചെയ്യും.

നിങ്ങൾ ഒരു പ്രത്യേക പാളി സ്കെയിൽ ആഗ്രഹിക്കുന്നുണ്ടെങ്കിൽ, Scale Layer കമാൻഡ് ഉപയോഗിക്കുക.

കുറിപ്പു്

If scaling would produce an image larger than the “Maximum new image size” set in the Environment page of the Preferences dialog (which has a default of 128 Mb), you are warned and asked to confirm the operation before it is performed. You may not experience any problems if you confirm the operation, but you should be aware that very large images consume a lot of resources and extremely large images may take more resources than you have, causing GIMP to crash or not perform well.

6.15.1. Activate the Command

  • You can access this command from the image menubar through Image → Scale Image.

ചിത്രം വലിപ്പം

അതു നിങ്ങളുടെ സ്ക്രീനിൽ ലോഡ് ചെയ്യുകയും ചെയ്ത ശേഷം, റാം, ഇമേജ് ഫയലിൽ പ്രദർശിപ്പിക്കുമ്പോഴല്ല, അല്ലെങ്കിൽ കടലാസിൽ അതു അച്ചടിച്ച ശേഷം: നിങ്ങൾ ഒരു ഇമേജ് നാലു സ്ഥലങ്ങളിൽ ഒന്നാണ് സ്ഥിതി കഴിയും ഓർക്കണം. അതു നേരിട്ട് ചിത്രം (റാം അല്ലെങ്കിൽ ഒരു ഫയലിൽ) ആവശ്യമാണ് മെമ്മറി അളവ് ബാധിക്കുന്നു അങ്ങനെ ചിത്രം ചെയറിന്, പിക്സലുകളുടെ എണ്ണം (വിവരങ്ങൾ തുക) ചിത്രവും അടങ്ങിയിരിക്കുന്നു.

എന്നിരുന്നാലും പ്രിന്റിംഗ് വലുപ്പം പുറമേ അടിസ്ഥാനപരമായി എത്ര പിക്സൽ കടലാസ് ഓരോ ഇഞ്ച് ന് ഉണ്ടാകും നിശ്ചയിക്കുന്നത് ചിത്രം, പ്രമേയത്തിലെ ആശ്രയിച്ചിരിക്കുന്നു. നിങ്ങൾ ചിത്രം കയറലും അതിൽ പിക്സലുകളുടെ എണ്ണം മാറ്റാതെ തന്നെ അച്ചടി വലിപ്പം മാറ്റാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ നിങ്ങൾക്ക് Print Size ഡയലോഗ് ഉപയോഗിക്കണം. സ്ക്രീൻ വലുപ്പം പിക്സൽ എണ്ണത്തിലുള്ള, മാത്രമല്ല സ്ക്രീൻ റെസലൂഷൻ, സൂം ഫാക്റ്റർ ആൻഡ് Dot for Dot ഓപ്ഷൻ ക്രമീകരണം മാത്രമല്ല ആശ്രയിച്ചിരിക്കുന്നു.

നിങ്ങൾ അതിന്റെ യഥാർത്ഥ വലുപ്പത്തിൽ അപ്പുറം ഒരു ചിത്രം വലുതാക്കുന്നു എങ്കിൽ GIMP interpolation നഷ്ടപ്പെടുന്നുണ്ട് പിക്സൽ കണക്കാക്കുന്നു, പക്ഷേ ഏതെങ്കിലും പുതിയ വിശദാംശങ്ങൾ ചേർക്കുക ഇല്ല. കൂടുതൽ നിങ്ങൾക്ക് ഒരു ഇമേജ് വലുതാക്കുന്നു കൂടുതൽ മങ്ങിയ അതു മാറുന്നു. ഒരു റെസിഡന്റ് ചിത്രം രൂപം നിങ്ങൾ തിരഞ്ഞെടുക്കുന്ന interpolation രീതി ആശ്രയിച്ചിരിക്കുന്നു. നിങ്ങൾ ഒരു ഇമേജ് സ്കെയിൽ ശേഷം Sharpen അരിപ്പ ഉപയോഗിച്ച് ഭാവം മെച്ചപ്പെടുത്താൻ എങ്കിലും അതു നിങ്ങളെ സ്കാൻ ഉയർന്ന റെസലൂഷൻ ഉപയോഗിക്കുക, ഡിജിറ്റൽ ഫോട്ടോകളെടുക്കാനുള്ള അല്ലെങ്കിൽ മറ്റ് മാർഗ്ഗങ്ങളിലൂടെ ഡിജിറ്റൽ ചിത്രങ്ങൾ നല്ലത്. റാസ്റ്റർ ചിത്രങ്ങൾ അന്തർലീനമായിട്ടുള്ള നന്നായി സ്കെയിൽ ചെയ്യരുത്.

നിങ്ങൾക്ക് ഒരു വെബ് പേജിൽ അത് ഉപയോഗിക്കാൻ ഉദ്ദേശിക്കുന്നുവെങ്കിൽ നിങ്ങളുടെ ചിത്രം കുറയ്ക്കാൻ വരാം. നിങ്ങൾക്ക് മിക്ക ഇന്റർനെറ്റ് ഉപയോക്താക്കളെ പൂർണ്ണമായും ഒരു വലിയ ചിത്രം പ്രദർശിപ്പിക്കാൻ കഴിയില്ല താരതമ്യേന ചെറിയ സ്ക്രീനുകൾ ഉണ്ടെന്ന് പരിഗണിക്കണം. പല സ്ക്രീനുകൾ 1024x768 അല്ലെങ്കിൽ അതിലും കുറവ് ഒരു ചിത്രം ഉണ്ട്.

പിക്സൽ ചേർത്തുകൊണ്ടോ നീക്കംചെയ്ത് “Resampling” വിളിക്കുന്നു.

Width; Height

നിങ്ങൾ Scale കമാൻഡ് ക്ലിക്ക് ചെയ്യുമ്പോൾ, ഡയലോഗ് പിക്സലിൽ യഥാർത്ഥ ചിത്രത്തിന്റെ തലങ്ങളും പ്രദർശിപ്പിക്കുന്നു. നിങ്ങൾ Width നിങ്ങൾ പിക്സൽ ചേർത്തുകൊണ്ടോ നീക്കംചെയ്ത് നിങ്ങളുടെ ഇമേജ് നൽകാൻ ആഗ്രഹിക്കുന്ന Height സജ്ജമാക്കാൻ കഴിയും. വീതിയും ഉയരവും ബോക്സുകൾ അടുത്ത ചെയിൻ ഐക്കൺ പിരിയാത്ത ആണെങ്കിൽ, വീതിയും ഉയരവും പരസ്പരം അതേ അനുപാതത്തിൽ താമസിക്കും. നിങ്ങൾ അത് ക്ലിക്കുചെയ്ത് ചെയിൻ പൊട്ടി, നിങ്ങൾക്ക് സ്വതന്ത്രമായി അവർക്ക് കഴിയും, എന്നാൽ ഈ ചിത്രം വളച്ചൊടിക്കാൻ ചെയ്യും.

എന്നിരുന്നാലും, നിങ്ങൾ പിക്സലിൽ അളവുകൾ സജ്ജമാക്കാൻ ഇല്ല. ഡ്രോപ്-ഡൗൺ മെനുവിൽ നിന്നും വിവിധ യൂണിറ്റുകൾ തിരഞ്ഞെടുക്കാം. നിങ്ങൾ യൂണിറ്റ് ശതമാനം തിരഞ്ഞെടുക്കുകയാണെങ്കിൽ, നിങ്ങൾ അതിന്റെ യഥാർത്ഥ വലുപ്പത്തിൽ ആപേക്ഷിക ഇമേജ് വലുപ്പം സജ്ജമാക്കാൻ കഴിയും. നിങ്ങൾക്ക് പോലുള്ള ഇഞ്ച് അല്ലെങ്കിൽ മില്ലിമീറ്റർ ഫിസിക്കൽ യൂണിറ്റുകൾ, ഉപയോഗിക്കാം. ആ ചെയ്യുകയാണെങ്കിൽ, X resolution ആൻഡ് Y resolution നിലങ്ങൾ ഉചിതമായ മൂല്യങ്ങൾ അവർ പിക്സൽ ഫിസിക്കല് ​​യൂണിറ്റുകൾ, ഇമേജ് തലങ്ങളും തമ്മിലുള്ള പരിവർത്തനം ഉപയോഗിക്കുന്ന കാരണം സജ്ജമാക്കണം.

X resolution; Y resolution

നിങ്ങൾ X resolution ആൻഡ് Y resolution മേഖലകളിലെ ചിത്രത്തിനായുള്ള അച്ചടി റെസലൂഷൻ സജ്ജമാക്കാൻ കഴിയും. ഡ്രോപ്പ്-ഡൌൺ മെനു ഉപയോഗിച്ച് അളവിന്റെ യൂണിറ്റുകൾ മാറ്റാൻ കഴിയും.

ഗുണമേന്മയുള്ള

ഇമേജ് വലുപ്പം മാറ്റാൻ, ഒന്നുകിൽ ചില പിക്സൽ നീക്കംചെയ്യും ചെയ്യണം അല്ലെങ്കിൽ പുതിയ പിക്സൽ ചേർക്കണം. നിങ്ങൾ ഉപയോഗിക്കുന്ന പ്രക്രിയ ഫലം ഗുണമേന്മ നിർണ്ണയിക്കുന്നു. Interpolation ഡ്രോപ്പ് ഡൗൺ ലിസ്റ്റ് ഒരു സ്കെയിൽ സ്വരൂപത്തിൽ പിക്സൽ നിറം interpolating ലഭ്യമായ രീതികൾ നിരക്കു നൽകുന്നു:

Interpolation

  • None: ഇല്ല interpolation ഉപയോഗിക്കുന്നു. അവർ ഔട്ടും ചെയ്യുമ്പോൾ പോലെ പിക്സൽ, ലളിതമായി വികസിക്കും അല്ലെങ്കിൽ നീക്കം ചെയ്യുന്നു. ഈ രീതി വളരെ ഫാസ്റ്റ് നിലവാരം കുറഞ്ഞ, എന്നാൽ.

  • Linear: ഈ രീതി താരതമ്യേന ഫാസ്റ്റ്, എന്നാൽ ഇപ്പോഴും ഗംഭീരമായി നല്ല ഫലങ്ങൾ നൽകുന്നു.

  • Cubic: മികച്ച ഫലങ്ങൾ ഉത്പാദിപ്പിക്കുന്നത് രീതി, മറിച്ച് പതുക്കെയുള്ള രീതി.

  • Sinc (Lanczos 3): ജിമ്പ്-2.4 പുതിയ ഈ രീതി പ്രധാനപ്പെട്ട resizings കുറവാണ് മങ്ങൽ നൽകുന്നു.

കുറിപ്പു്

ഇതും കാണുക നിങ്ങൾ ഒരു പാളി, ഒരു നിര അല്ലെങ്കിൽ ഒരു പാത്ത് സ്കെയിൽ അനുവദിക്കുന്നു Scale tool.
