6.19. ദൃശ്യം ലെയറുകൾ ലയിപ്പിക്കുക
6. “Image” മെനു  

6.19. ദൃശ്യം ലെയറുകൾ ലയിപ്പിക്കുക

Merge Visible Layers കമാൻഡ് ഒരു ലെയറിലേക്ക് ദൃശ്യമാണ് ഏത് പാളികൾ ലയിപ്പിക്കുന്നു. ദൃശ്യം പാളികൾ ഒരു “eye” ഐക്കൺ ഡയലോഗ് ലേയറുകളിൽ കാണിക്കേണ്ടതാണ് ഏത് ഭാഗ്യവാന്മാർ.

കുറിപ്പു്

ഈ കമാൻഡ് ഉപയോഗിച്ച്, യഥാർത്ഥ കാണാവുന്ന പാളികൾ അപ്രത്യക്ഷമാകും. New From Visible കമാൻഡ് ഉപയോഗിച്ച്, ഒരു പുതിയ പാളി സ്റ്റാക്ക് മുകളിലുള്ള തയ്യാറാക്കി, യഥാർത്ഥ കാണാവുന്ന പാളികൾ നിലനിൽക്കുകയുള്ളൂ.

6.19.1. Activate the Command

  • You can access this command from the image menubar through Image → Merge Visible Layers,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Ctrl+M ഉപയോഗിച്ച്.

6.19.2.  Description of the “Layers merge Options” Dialog

ചിത്രം 16.71. The “Layers Merge Options” Dialog


ഫൈനൽ ലയിപ്പിച്ചു ലെയർ ആയിരിക്കണം:

ദൃശ്യം പാളികൾ പാളികളുടെ ഡയലോഗിൽ ഒരു “eye” ഐക്കൺ ഉപയോഗിച്ച് അടയാളപ്പെടുത്തിയിരിക്കും ഏത് പാളികൾ ആകുന്നു.

  • Expanded as necessary: അന്തിമ പാളി ലയിപ്പിച്ച പാളികൾ എല്ലാ ഉൾക്കൊള്ളാൻ മാത്രം വലുതാണ്. GIMP ഒരു പാളി ഇമേജ് കൂടുതലായിരിക്കണം എന്ന് ഓർക്കുക.

  • Clipped to image: അന്തിമ പാളി ഇമേജ് അതേ വലുപ്പം ആണ്. GIMP ലെ പാളികൾ ചിത്രത്തിൽ തന്നെ കൂടുതലായിരിക്കണം കഴിയുമെന്ന് ഓർമിക്കുക. ചിത്രം വലുതാണ് ആ സ്വരൂപത്തിൽ ഏതെങ്കിലും പാളികൾ ഈ ഓപ്ഷൻ വഴി ചെത്തിയെടുത്തത് ചെയ്യുന്നു.

  • Clipped to bottom layer: അന്തിമ പാളി ചുവടെ പാളി അതേ വലുപ്പം ആണ്. ചുവടെ പാളി ദൃശ്യമായ പാളികൾ ചില ചെറുതാണ് എങ്കിൽ, അവസാന പാളി ചുവടെ പാളി വലുപ്പവും സ്ഥാനത്തേക്ക് ചെത്തിയെടുത്തത് ചെയ്ത് കസവുകൊണ്ടുള്ളതു.

Merge within active group only

This self-explanatory option is enabled when a layer group exists.

Discard invisible layers

ഈ ഓപ്ഷൻ എപ്പോഴാണ് ചെക്കുചെയ്തിട്ടുണ്ടെന്നും, നോൺ ദൃശ്യമായ പാളികൾ പാളി സ്റ്റാക്ക് നിന്നും നീക്കംചെയ്തു.
