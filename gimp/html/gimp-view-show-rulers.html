5.20. Show Rulers

Show Rulers കമാൻഡ് പ്രാപ്തരാക്കുകയും ഭരണാധികാരികൾ പ്രദർശിപ്പിക്കുന്നത് നിഷ്ക്രിയമാക്കുന്നു. നിങ്ങൾ full-screen mode പ്രവര്ത്തിക്കുന്നയാളാണെങ്കില് അതു അവരെ അപ്രാപ്തമാക്കാൻ ഉപയോഗപ്പെടുന്നു.

നിങ്ങൾ Image Window Appearance ഡയലോഗിൽ ഭരണാധികാരികൾ സ്ഥിര സജ്ജമാക്കാൻ കഴിയും.

5.20.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through View → Show Rulers,

  • അല്ലെങ്കിൽ കീബോർഡ് കുറുക്കുവഴി Shift+Ctrl+R ഉപയോഗിച്ച്.
