പാഠം 8. സംയോജിത ചിത്രങ്ങൾ  

4. ലെയർ ഗ്രൂപ്പുകൾ

ഈ സാധ്യത GIMP-2.8 കൂടെ പ്രത്യക്ഷനായി.

നിങ്ങൾ മരത്തിൽ-പോലുള്ള വഴിയിൽ സമാനതകൾ ആ ഗ്രൂപ്പ് പാളികൾ ചെയ്യാനാകും. അതുകൊണ്ട്, പാളി ലിസ്റ്റ് നിയന്ത്രിക്കാൻ എളുപ്പം മാറുന്നു.

ഒരു പാളി ഗ്രൂപ്പ് സൃഷ്ടിക്കുക

നിങ്ങൾ പാളി ഡയലോഗ് ചുവടെയുള്ള Create a new layer group ബട്ടൺ ക്ളിക്ക് ഒരു പാളി ഗ്രൂപ്പ് സൃഷ്ടിക്കാൻ കഴിയും,

Layer → New Layer Group വഴി അല്ലെങ്കിൽ പാളി ഡയലോഗ് സന്ദർഭ മെനു വഴി.

ഈ ശൂന്യമായ പാളി ഗ്രൂപ്പ് നിലവിലെ പാളി മുകളിൽ ദൃശ്യമാകുന്നു. നിരവധി മഹത്തുക്കൾ സൃഷ്ടിക്കുമ്പോൾ അത് ബാല്യത്തിന്റെ പേര് (പേര് ഇരട്ട-ക്ലിക്കുചെയ്യുക അല്ലെങ്കിൽ F2, അല്ലെങ്കിൽ എഡിറ്റ് ചെയ്യാനുള്ള, നിങ്ങൾ ശരിയായ ലെയർ ഡയലോഗ് തിരയാം നേടുകയും സന്ദർഭ മെനുവിൽ Edit Layer Attributes ഉപയോഗിക്കുക) നൽകാൻ പ്രധാനമാണ്, അല്ലാഞ്ഞാൽ ആശയക്കുഴപ്പത്തിലാണ് ലഭിക്കും .

നിങ്ങൾ നിരവധി പാളി ഗ്രൂപ്പുകൾ സൃഷ്ടിക്കാൻ കഴിയും നീ അവരെ embbed കഴിയും, മറ്റൊന്ന് ഒരു പാളി ഗ്രൂപ്പ് ഉൾപ്പെടുന്നു എന്ന്.

ഒരു പാളി ഗ്രൂപ്പ് പാളികൾക്ക് ചേർക്കുന്നു

നിങ്ങൾ click- ഒരു പാളി ഗ്രൂപ്പിലേക്ക് existing layers കഴിയുന്നതാണ്-വലിച്ചോ അവരെ. PLACEHOLDER,-101561

നിലവിലെ പാളി ഗ്രൂപ്പിന് ഒരു new layer ചേർക്കുന്നതിന്, പാളി ഡയലോഗ് താഴെയുള്ള Create a new layer ക്ലിക്കുചെയ്യുക അല്ലെങ്കിൽ ഇമേജ് മെനുവിൽ New Layer കമാൻഡ് ഉപയോഗിക്കുക.

ഒരു പാളി ഗ്രൂപ്പ് ശൂന്യമല്ലെന്ന്, ഒരു ചെറിയ “>” ഐക്കൺ ലഭ്യമാകുന്നു. അതിൽ ക്ലിക്കുചെയ്യുന്നതിലൂടെ, മടക്കിക്കളയുന്നു കഴിയും ലെയർ ലിസ്റ്റിൽ / മടക്കാത്ത. PLACEHOLDER,-101566

നിങ്ങൾ പാളികൾ അവരെ മാറ്റുന്നതിലൂടെ ഒരു പാളി ഗ്രൂപ്പിലേക്ക് ഒരുമിച്ച് ചേർക്കാൻ, അവരെ മാത്രം ഇട്ടു ദൃശ്യമായതും “New From Visible ” കമാൻഡ് ഉപയോഗിച്ച് കഴിയും. എല്ലാ കാണാവുന്ന പാളികൾ പുറത്ത് അകത്തും പാളി ഗ്രൂപ്പുകൾ, സജീവ പാളി ഗ്രൂപ്പ് ചേർക്കും.

ഒരു പാളി ഗ്രൂപ്പ് നിഷിപ്തമായിരിക്കുന്നു ലെയറുകളെയും ചെറുതായി എളുപ്പത്തിൽ അറിയുന്നു പാളികൾ ഗ്രൂപ്പിന്റെ ഭാഗമായ അനുവദിച്ചുകൊണ്ട് വലതുവശത്ത് ഇൻഡന്റ് ചെയ്യുന്നു.

ഉയർത്തി ലോവർ ലെയർ ഗ്രൂപ്പുകൾ

സാധാരണ പാളികൾ കൊണ്ട് ചെയ്യുന്നതുപോലെ നിങ്ങൾ പാളി ഡയലോഗിൽ പാളി ഗ്രൂപ്പുകൾ ഉളവാക്കുവാൻ താഴ്ന്ന കഴിയും: ക്ലിക്ക്-ആൻഡ്-വലിച്ചിടലിനുമായി, അമ്പടയാളം ഉപയോഗിച്ച് താഴേക്കും കീകൾ പാളി ഡയലോഗ് ചുവടെ.

ഒരു പാളി ഗ്രൂപ്പ് ഡ്യൂപ്ലിക്കേറ്റ്

നിങ്ങൾ ഒരു പാളി ഗ്രൂപ്പ് ഡ്യൂപ്ലിക്കേറ്റ് കഴിയും: Create a duplicate of the layer ബട്ടണില് ക്ലിക്ക് ചെയ്യുകയോ റൈറ്റ് ക്ലിക്ക് ചെയ്ത് പോപ് അപ് സന്ദർഭ മെനു Duplicate Layer കമാൻഡ് തിരഞ്ഞെടുക്കുക.

ലെയർ ഗ്രൂപ്പുകൾ നീക്കുക

നിങ്ങൾ ക്ലിക്കിൽ-ആൻഡ്-വലിച്ചോ move a layer group to another image കഴിയും. നിങ്ങൾക്ക് പകർത്തി ഒട്ടിക്കുക Ctrl-C Ctrl-V ഉപയോഗിച്ച് അത് അപ്പോൾ, നിങ്ങൾ (പാളി ഡയലോഗ് ചുവടെയുള്ള അവതാരകൻ ബട്ടൺ) ആങ്കർ വേണം ഒരു ഫ്ലോട്ടിംഗ് നിര നേടുകയും.

നിങ്ങൾക്ക് move a layer group to the canvas കഴിയും: ഈ ഗ്രൂപ്പ് in ഗ്രൂപ്പ് തനിപ്പകർപ്പുകളൊന്നുമില്ല. ഡ്യൂപ്ലിക്കേറ്റ് പാളി ഗ്രൂപ്പിൽ ചെയിൻ എല്ലാ പാളികൾ ആക്റ്റിവേറ്റ് നീക്കുക ഉപകരണം, പിന്നെ, ചിത്രം, പാളി നീക്കുക. ഒരു ചിത്രം മൾട്ടി ലെയർ വസ്തുക്കൾ വർദ്ധിപ്പിക്കും ഒരു മാർഗ്ഗം കൂടിയാണ്.

ഒരു പാളി ഗ്രൂപ്പ് ഇല്ലാതാക്കുക

ഒരു പാളി ഗ്രൂപ്പ് ഇല്ലാതാക്കാൻ, പാളി ഡയലോഗ് താഴെയുള്ള റെഡ്ക്രോസ് ബട്ടണിൽ ക്ലിക്ക് ചെയ്യുകയോ റൈറ്റ് ക്ലിക്ക് ചെയ്ത് Delete layer തിരഞ്ഞെടുക്കുക.

ഉൾച്ചേർക്കുക ലെയർ ഗ്രൂപ്പുകൾ

ഒരു പാളി ഗ്രൂപ്പ് സജീവമാകുമ്പോൾ, നിങ്ങൾ “Add New Layer Group” കമാൻഡ് ഉപയോഗിച്ച് അതിനുള്ളിൽ മറ്റൊരു ഗ്രൂപ്പ് ചേർക്കാൻ കഴിയും. പരിധിയും കാണുന്നു, എംബഡഡ് പാളി ഗ്രൂപ്പുകൾ എണ്ണം കാരണം, മെമ്മറി ഒഴിവാക്കപ്പെട്ടിരിക്കുന്നു.

ലെയർ മോഡുകൾ ഗ്രൂപ്പുകള്

ഒരു പാളി ഗ്രൂപ്പ് പ്രയോഗിച്ചു ഒരു പാളി മോഡ് മാത്രം ഈ ഗ്രൂപ്പിൽ ലെയറുകളും പ്രവർത്തിക്കുന്നു. ഒരു പാളി ഗ്രൂപ്പ് മുകളിൽ ഒരു പാളി മോഡ് പുറത്ത്, തലം ഗ്രൂപ്പുകൾ അകത്ത് കീഴില് എല്ലാ പാളികൾ പ്രവർത്തിക്കുന്നു.

യഥാർത്ഥ ചിത്രം

ചിത്രം 8.33. അല്ലെങ്കിൽ ലെയർ ഗ്രൂപ്പ് ഔട്ട് ലെയർ മോഡ്

മാത്രം സമചതുരവും ത്രികോണം ഗ്രേ ഔട്ട്; ഞങ്ങൾ in സാച്ചുറേഷൻ മോഡിൽ പാളി ഗ്രൂപ്പ് ഒരു വെളുത്ത പാളി ചേർത്തു.

പുറമേ പശ്ചാത്തല പാളി എല്ലാ പാളികൾ താഴ്ഭാഗത്ത് ഗ്രേ ഔട്ട് ചെയ്യുന്നു, നാം സാച്ചുറേഷൻ മോഡിൽ പാളി ഗ്രൂപ്പിന്റെ ഒരു വെളുത്ത പാളി out ചേർത്തു.


അതാര്യത

ഒരു പാളി ഗ്രൂപ്പ് സജീവമാകുമ്പോൾ, അതാര്യത മാറ്റങ്ങൾ ഗ്രൂപ്പിലെ എല്ലാ പാളികൾ പ്രയോഗിച്ചു.

ലെയർ മാസ്ക്

You cannot add a layer mask to a layer group (the corresponding option is grayed out). But, as with normal layers, you can add a layer mask to a layer in the group to mask a part of the layer.

നാം ത്രികോണം പാളി ഒരു വെളുത്ത (പൂർണ്ണ അതാര്യത) ലെയർ മാസ്ക് ചേർത്തു.
