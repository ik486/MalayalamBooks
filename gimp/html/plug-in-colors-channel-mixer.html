യഥാർത്ഥ ചിത്രം

“Channel Mixer” പ്രയോഗിച്ചു


ഈ കമാൻഡ് RGB ചാനലുകളുടെ മൂല്യങ്ങൾ കൂടിച്ചേർന്നുണ്ടായ. ഇത് ഉപയോഗിച്ചോ അല്ലാതെയോ ആൽഫ ചാനൽ ചിത്രങ്ങൾ പ്രവർത്തിക്കുന്നു. അതു ചെന്നൈയില് മോഡ് പ്രിവ്യൂ ഉണ്ട്.

8.14.2. കമാൻഡ് സജീവമാക്കുക

നിങ്ങൾ Colors → Components → Channel Mixer മുഖേന ഈ കമാൻഡ് കണ്ടെത്താൻ കഴിയും.

8.14.3. ഓപ്ഷനുകൾ

ചിത്രം 16.146. “Channel Mixer” കമാൻഡ് ഓപ്ഷനുകൾ


ഔട്ട്പുട്ട് ചാനൽ

ഈ മെനുവിൽ നിന്ന് നിങ്ങൾക്ക് ഇളക്കുക ചാനൽ തിരഞ്ഞെടുക്കുക. തിരഞ്ഞെടുപ്പുകൾ റെഡ്, ഗ്രീൻ, അല്ലെങ്കിൽ നീല. Monochrome ഓപ്ഷൻ ചെക്കുചെയ്തിട്ടുള്ളപ്പോൾ .പിന്നെ അത്.

റെഡ്, ഗ്രീൻ, ബ്ലൂ

ഈ മൂന്ന് സ്ലൈഡറുകൾ ഔട്ട്പുട്ട് ലേക്ക്, ചുവപ്പ് പച്ച, നീല ചാനലിന്റെ സംഭാവനയാണ് വെച്ചു. നെഗറ്റീവ് കഴിയും. ഈ സ്ലൈഡറുകൾ -200 മുതൽ 200. അവർ ഔട്ട്പുട്ട് ചാനലിലേക്ക് ആട്രിബ്യൂട്ടുചെയ്യും ഏത് ശതമാനം പ്രതിനിധീകരിക്കാൻ ഗ്രാജുവേറ്റ് ചെയ്യുന്നു. 100% ചിത്രം പഠിച്ച പിക്സൽ ചാനൽ മൂല്യം തുല്യമാണ്.

മോണോക്രോംആക്കുക

ഈ ഓപ്ഷൻ ഒരു ചാര-സ്കെയിലിൽ ആർജിബി ഇമേജിലേക്ക് ആർജിബി ഇമേജ് ആക്കാനുള്ള. ചാനൽ മിക്സർ കമാൻഡ് പലപ്പോഴും ഈ ലക്ഷ്യത്തോടെയാണ് പലപ്പോഴും മറ്റ് വഴികൾ (ഗ്ലോസറിയിൽ Grayscale കാണുക) അതിനെക്കാൾ മികച്ച ഫലം നൽകുന്നു കാരണം, കാഴ്ച ഉപയോഗിക്കുന്നു. Output Channel മെനു സംവേദനക്ഷമരഹിത സൃഷ്ടിക്കുന്നു.

കുറിപ്പു്

21%, 72%, 7% ക്രമീകരണങ്ങൾ നിങ്ങൾ ഒരേ ചാര പ്രതല (മൂല്യം) ചിത്രം / മോഡിൽ ഗ്രേസ്കെയിൽ കമാൻഡ് പോലെ കൊടുക്കും. (30%, 59%, v2.2 11% ആയിരുന്നു).
പ്രതല സംരക്ഷിക്കുക

കണക്കുകൂട്ടലുകളും വളരെയധികം വ്യക്തമായ വളരെ ഉയർന്ന മൂല്യങ്ങൾ ഒരു ചിത്രത്തിൽ കാരണമായേക്കാം. അവരുടെ മദ്ധ്യേ ഒരു നല്ല വിഷ്വൽ അനുപാതം സൂക്ഷിക്കുന്നതിനുമുള്ള സമയത്ത് ഈ ഓപ്ഷൻ നിറം ചാനലുകളുടെ luminosities കുറയ്ക്കുകയും. അതിനാൽ, മൊത്തത്തിലുള്ള പ്രതല മാറ്റാതെ തന്നെ നിറങ്ങൾ ആപേക്ഷിക ഭാരം മാറ്റാൻ കഴിയും.

8.14.5. ചാനൽ മിക്സർ എങ്ങനെ പ്രവർത്തിക്കുന്നു?

ആർജിബി മോഡിൽ

ഈ മോഡിൽ, നിങ്ങൾ ഒരു Output Channel തിരഞ്ഞെടുക്കേണ്ടി. ഈ ചാനൽ പരിഷ്ക്കരിക്കും ഏത് ഒന്നാണ്. ഡയലോഗ് വിൻഡോയിൽ, അതിന്റെ സ്ഥിര മൂല്യം യഥാർത്ഥ ചിത്രം ചാനൽ മൂല്യം പൊരുത്തപ്പെടുന്ന 100% ആണ്. വർദ്ധിപ്പിച്ചു അല്ലെങ്കിൽ കുരയ്ക്കാൻ സാധിക്കും. സ്ലൈഡർ അറ്റത്ത് എന്തുകൊണ്ട് -200 200 ആകുന്നു അത്.

മൂന്ന് ആർജിബി സ്ലൈഡറുകൾ ഓരോ ചാനലിന്റെയും ഒരു ശതമാനം വേണ്ടതു. ചിത്രം ഓരോ പിക്സലിനായുള്ള, ഈ ശതമാനം നിന്ന് ഓരോ ചാനലിനായി കേക്കു മൂല്യങ്ങളുടെ ആകെത്തുക ഔട്ട്പുട്ട് ചാനൽ ലഭിക്കും. ഒരു ഉദാഹരണം ഇതാ:

ചിത്രം 16.147. യഥാർത്ഥ ചിത്രം അതിന്റെ ചാനലുകൾ

ചുവപ്പ്, പച്ച, നീല പിക്സൽ ആർജിബി മൂല്യങ്ങൾ, ചാര സ്ക്വയറുകളുടെ പ്രദർശിപ്പിച്ചിരിക്കുന്നു. കമാൻഡ് (0 എപ്പോഴും 0 നൽകുന്ന ഏതൊരു ശതമാനം ഗുണിച്ചാൽ) വഴി സംബന്ധിച്ചിടത്തോളം അല്ല കറുത്ത ദീർഘചതുരം കറുത്ത (; 0 0 0) കാരണം, പ്രത്യേക ആണ്. ഫലം കവിയരുത് 255 വേണ്ടാ നെഗറ്റീവ് കഴിയും.


ചിത്രം 16.148. ഔട്ട്പുട്ട് ചാനൽ ഉപഭോഗം. ഗ്രീൻ ചാനൽ +50

ചുവന്ന ചത്വരത്തിൽ, പിക്സൽ മൂല്യങ്ങൾ 230 ആകുന്നു; 10; 10. ആപേക്ഷികമായ മൂല്യങ്ങൾ 1 ആകുന്നു; 0.5; 0. കണക്കുകൂട്ടൽ ഫലം 230 * 1 + 10 * 0,5 + 10 * 0 = 235 ആണ്. ഒരേ ന്യായവാദം പച്ച, നീല പുറവുമുള്ള സാധുവാണ്.

ചുവന്ന നിറം അടങ്ങുന്ന ചാര സ്ക്വയർ, ൽ, കണക്കുകൂട്ടൽ ഫലം 255. മുകളിൽ ആണ് 255. ഒരു നെഗറ്റീവ് മൂല്യം ചുരുക്കിയിരിക്കുന്നു 0 കുറയും.


ചിത്രം 16.149. ഔട്ട്പുട്ട് ചാനൽ ഉപഭോഗം. ഗ്രീൻ ചാനൽ + 50%. സംരക്ഷിക്കുക പ്രതല ഓപ്ഷൻ പരിശോധിക്കുന്നു.

റെഡ് ഔട്ട്പുട്ട് ചാനലിലേക്ക് ആരോപിക്കപ്പെട്ടു മൂല്ല്യങ്ങൾ വളരെ വ്യക്തമായ ചിത്രം തടയുകയായിരുന്നു കുറവാണ്.


മോണോക്രോംആക്കുക മോഡിൽ

ഈ ഓപ്ഷൻ എപ്പോഴാണ് ചെക്കുചെയ്തിട്ടുണ്ടെന്നും, ചിത്രത്തിന്റെ പ്രിവ്യൂ ഗ്രേസ്കെയിലാക്കി ഓഫാകും, എന്നാൽ ഇമേജ് കമാൻഡ് നടപടി സാധൂകരിച്ചു വരെ ഇപ്പോഴും മൂന്നു ചാനലുകളിൽ ഒരു ആർജിബി ചിത്രം.

ചിത്രം 16.150. മോണോക്രോംആക്കുക ഓപ്ഷൻ. റെഡ്: 100% ഗ്രീന്: 50% ബ്ലൂ: 0%. പ്രതല തന്നിഷ്ടങ്ങൾക്കനുസരിച്ച് സംരക്ഷിക്കുക.

ഓരോ സ്ക്വയർ ൽ പിക്സൽ ഒരു ചാര തലത്തിൽ യഥാർത്ഥ ചിത്രം റെഡ് ചാനൽ മൂല്യം തുല്യമാണ് (പശ്ചാത്തല എല്ലാ സ്ക്വയറുകളുടെ കാണുന്നതിനായി ശേഷം പിങ്ക് ചായം ചെയ്തു) മാറിക്കഴിഞ്ഞു ചെയ്തിരിക്കുന്നു.


ഇവിടെ Preserve Luminosity ചെന്നൈയില് മോഡിൽ പ്രവർത്തിക്കുന്ന ഇങ്ങനെ: “ For example, suppose the sliders were Red:75%, Green:75%, Blue:0%. With Monochrome on and the Preserve Luminosity option off, the resulting picture would be at 75%+75%+0% =150%, very bright indeed. A pixel with a value of, say, R,G,B=127,100,80 would map to 127*0.75+100*0.75+80*0=170 for each channel. With the Preserve Luminosity option on, the sliders will be scaled so they always add up to 100%. In this example, that scale value is 1/(75% +75%+0%) or 0.667. So the pixel values would be about 113. The Preserve Luminosity option just assures that the scale values from the sliders always adds up to 100%. Of course, strange things happen when any of the sliders have large negative values ” (പ്ലഗ്-ഇൻ രചയിതാവ് സ്വയമായി).

കുറിപ്പു്

Which channel will you modify? ഇത് നിങ്ങൾ ചെയ്യേണ്ടത് ആശ്രയിച്ചിരിക്കുന്നു. തത്വത്തിൽ, റെഡ് ചാനൽ നന്നായി തീവ്രത പരിഷ്ക്കരണങ്ങൾ അനുയോജ്യമായ. ഗ്രീൻ ചാനൽ നന്നായി ശബ്ദം വിശദാംശങ്ങൾ മാറ്റങ്ങൾ ബ്ലൂ ചാനൽ, ധാന്യം മാറ്റങ്ങൾ ചേർച്ചപ്പെടുത്തുകയാണ്. നിങ്ങൾ Decompose കമാൻഡ് ഉപയോഗിക്കാം.
