3. പുതിയ തലങ്ങളെ ഉണ്ടാക്കുന്നു
പാഠം 8. സംയോജിത ചിത്രങ്ങൾ  

3. പുതിയ തലങ്ങളെ ഉണ്ടാക്കുന്നു

ഒരു ചിത്രം പുതിയ പാളികൾ സൃഷ്ടിക്കാൻ നിരവധി മാർഗ്ഗങ്ങളുണ്ട്. ഇവിടെ ഏറ്റവും പ്രധാനപ്പെട്ട അക്കൂട്ടർ:

  • ചിത്രം മെനുവിൽ Layer → New Layer തെരഞ്ഞെടുക്കുന്നു. ഇത് നിങ്ങൾ പുതിയ പാളി അടിസ്ഥാന സജ്ജമാക്കാൻ അനുവദിക്കുന്ന ഒരു ഡയലോഗിന് നൽകുന്നു; അതു സഹായത്തിന് New Layer dialog വിഭാഗം കാണുക.

  • ചിത്രം മെനുവിൽ Layer → Duplicate Layer തെരഞ്ഞെടുക്കുന്നു. ഇത് വെറും സജീവ പാളി മുകളിൽ, നിലവിലെ സജീവ പാളി ഒരു തികഞ്ഞ പകർപ്പാണ് ഒരു പുതിയ പാളി, സൃഷ്ടിക്കുന്നു.

  • എപ്പോഴാണ് “cut” അല്ലെങ്കിൽ “copy” എന്തെങ്കിലും, തുടർന്ന് Ctrl+V അല്ലെങ്കിൽ Edit → Paste ഉപയോഗിച്ച് പേസ്റ്റ്, കാരണമാകുന്ന താൽക്കാലിക പാളി ഒരു തരത്തിൽ ഒരു “floating selection” ആണ്. നിങ്ങൾക്ക് മറ്റെന്തെങ്കിലും ചെയ്യാൻ കഴിയും മുമ്പ്, ഒന്നുകിൽ ഒരു നിലവിലുള്ള പാളി ഫ്ലോട്ടിങ് നിരക്കു അവതാരക, അല്ലെങ്കിൽ ഒരു സാധാരണ പാളി അതിനെ പരിവർത്തനം ചെയ്യണം. രണ്ടാമതു പറഞ്ഞ ചെയ്യാൻ, പുതിയ പാളി ഒട്ടിക്കാൻ മെറ്റീരിയൽ ഉൾക്കൊള്ളാൻ മാത്രം വെറും വലിയ വലിപ്പമുള്ള ചെയ്യും.
