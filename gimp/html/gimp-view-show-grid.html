5.12. Show Grid

Show Grid കമാൻഡ് ഉപയോഗിച്ച് പ്രാപ്തമാക്കും, ഗ്രിഡ് പ്രദർശിപ്പിക്കുന്നത് പ്രവർത്തനരഹിതമാക്കാനാകും. അത് തയ്യാറാക്കാൻ ചെയ്യുമ്പോൾ, ഗ്രിഡ് ചിത്രം ഓവർലേകളും നിങ്ങൾ തിരഞ്ഞെടുത്ത ചിത്രം ഘടകങ്ങൾ ച്ചുകൊണ്ട് ലേക്ക് സുഗമമാക്കുന്നു.

നിങ്ങൾ Image Window Appearance ഡയലോഗിൽ ഗ്രിഡ് സ്ഥിര സജ്ജമാക്കാൻ കഴിയും.

സൂചന

പുറമേ Configure Grid കമാൻഡും Snap to Grid കമാൻഡ് കാണുക.
