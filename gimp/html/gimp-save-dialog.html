5.5. ഫയൽ സംരക്ഷിക്കുക

The Save command saves your image to disk. With GIMP-2.8, this command saves in XCF format only. If you try to save to a format other than XCF, you get an error message:

Starting from GIMP-2.8.8, the error dialog sports a link that jumps directly to the export command dialog. Please see വിഭാഗം 1.1, “സംരക്ഷിക്കുക / കയറ്റുമതി ചിത്രങ്ങൾ”.

നിങ്ങൾ ഇതിനകം ചിത്രം സംരക്ഷിച്ചു ചെയ്തിട്ടുണ്ടെങ്കിൽ, മുൻ ഇമേജ് ഫയൽ നിലവിലുള്ള പതിപ്പുമായുള്ള എഴുതപ്പെട്ടു. നിങ്ങൾ ഇതിനകം ചിത്രം സംരക്ഷിച്ചു എങ്കിൽ, Save കമാൻഡ് സംരക്ഷിക്കുക ചിത്രം ഡയലോഗ് തുറക്കുന്നു.

നിങ്ങളുടെ ചിത്രം സംരക്ഷിച്ചു പണമെങ്കിൽ പുറത്ത് പോകുകയാണെങ്കിൽ, GIMP നിങ്ങൾ ശരിക്കും അങ്ങനെ ചെയ്യാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, “Confirm closing of unsaved images” ഓപ്ഷൻ മുൻഗണനകൾ ഡയലോഗ് Environment പേജ് ചെക്കുചെയ്തിട്ടുണ്ടെന്നതും ചോദിച്ച്.

ചിത്രം 15.76. ചിത്രം ഡയലോഗ് സംരക്ഷിക്കുക


5.5.1. സംഭാഷണം സജീവമാക്കുക

  • നിങ്ങൾ File → Save വഴി ഇമേജ് മെനു ബാറിലെ ഈ കമാൻഡ് ആക്സസ് ചെയ്യാൻ കഴിയും,

  • അല്ലെങ്കിൽ കീബോർഡ് നിന്ന് കുറുക്കുവഴി Ctrl+S ഉപയോഗിച്ച്.

  • Use Ctrl+Shift+S to save the opened image with a different name.

5.5.2. ചിത്രം സംരക്ഷിക്കൽ ഡയലോഗ്

ഈ ഫയൽ ബ്രൌസർ ഉപയോഗിച്ച് നിങ്ങൾക്ക് പേര് നേരിട്ട് പെട്ടിയിൽ ഫയൽനാമം എഡിറ്റ് (സ്വതവേ “Untitled.xcf” ആണ്) അല്ലെങ്കിൽ പേര് ലിസ്റ്റിൽ ഒരു ഫയൽ തിരഞ്ഞെടുത്ത് കഴിയും. മാത്രമേ ഞങ്ങൾ എക്സ്.സി.എഫ് ഫോർമാറ്റ് അനുവദിച്ചിട്ടിട്ടുണ്ട് ആവർത്തിക്കുക. നിങ്ങൾക്ക് Save in Folder ഇമേജ് ലക്ഷ്യസ്ഥാനം പരിഹരിക്കണം. ആവശ്യമെങ്കിൽ നിങ്ങൾക്ക് പുതിയ ഒരു ഫോൾഡർ സൃഷ്ടിക്കാൻ കഴിയും.

ഫയൽ തെരഞ്ഞെടുക്കു. നിങ്ങൾ ഈ ഉപാധി വികസിപ്പിക്കുകയും ചെയ്താൽ, നിങ്ങളുടെ എക്സ്.സി.എഫ് ഫയൽ ഒരു ഫോർമാറ്റിൽ തെരഞ്ഞെടുക്കാം:
