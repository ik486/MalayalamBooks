5.10. Desaturate

Desaturate കമാൻഡ് ഉപയോഗിക്കുന്നതിലൂടെ, ചാര ഇതേ ഷേഡുകൾ ലേക്കുള്ള സജീവ ലെയറിലെ നിറങ്ങൾ എല്ലാ പരിവർത്തനം ചെയ്യാനാകും. ഈ രണ്ടു അർത്ഥത്തിലും ഗ്രേസ്കെയിലാക്കി ചിത്രം പരിവർത്തനം വ്യത്യസ്തമാണ്. ഒന്നാമത്തേത്, സജീവ പാളി രണ്ടാമത്തെ പ്രവർത്തിക്കുന്ന, പാളി ന് നിറങ്ങൾ ഇപ്പോഴും മൂന്നു ഘടകങ്ങളുമായി ആർജിബി മൂല്യങ്ങളും അവ. ഇത് നിങ്ങൾ പിന്നീട് നിറം ഉപയോഗിച്ച്, പാളി, അല്ലെങ്കിൽ വ്യക്തിഗത ഭാഗങ്ങൾ പെയിന്റ് സാധിക്കും.

കുറിപ്പു്

ഈ കമാൻഡ് മാത്രം RGB ചിത്രങ്ങൾ പാളികൾ പ്രവർത്തിക്കുന്നു. ചിത്രം ഗ്രേസ്കെയിൽ അല്ലെങ്കിൽ ഇൻഡെക്സ്ഡ് മോഡിൽ ഉണ്ടെങ്കിൽ, അത് ഒന്നും ചെയ്യാൻ കഴിയില്ല.

5.10.1. Activating the Command

You can get to this tools in several ways:

  • In the image menu through Tools → Color Tools → Desaturate or Colors → Desaturate,

  • by clicking on the tool icon in Toolbox, if this tool has been installed there.For this, please refer to വിഭാഗം 1.7, “പണിസഞ്ചി”.
