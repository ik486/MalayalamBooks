6.9. റൊട്ടേഷൻ

നിങ്ങൾ Image മെനുവിന്റെ Transform സബ്മെനു ന് റൊട്ടേഷൻ ആജ്ഞകൾ ഉപയോഗിച്ച്, ഇമേജ് 90 ° ഘടികാരദിശയിൽ അല്ലെങ്കിൽ എതിർഘടികാരദിശയിൽ തിരിക്കുക, അല്ലെങ്കിൽ അതു 180 ° തിരിക്കുകയോ കഴിയും. ഈ കമാൻഡുകൾ അർദ്ധകായ ലാൻഡ്സ്കേപ്പ് ഓറിയന്റേഷൻ മാറാൻ ഉപയോഗിക്കാൻ കഴിയും. അവർ മുഴുവൻ ചിത്രം പ്രവർത്തിക്കാം. നിങ്ങൾ മറ്റൊരു കോണിൽ ചിത്രം തിരിക്കുന്നതിന് ആഗ്രഹിക്കുന്ന ഒരു നിര തിരിക്കുന്നതിനോ ഒരു പാളി തിരിക്കുന്നതിന് എങ്കിൽ Rotate Tool ഉപയോഗിക്കുക. നിങ്ങൾക്ക് Layer Transform മെനു ഉപയോഗിച്ച് ഒരു പാളി തിരിക്കാനോ കഴിയും.

6.9.1. Activate the Commands

നിങ്ങൾ വഴി ഇമേജ് മെനുബാർ നിന്ന് ഈ മൂന്നു കമാൻഡുകൾ ആക്സസ് ചെയ്യാൻ കഴിയും

  • Image → Transform → Rotate 90 degrees CW,

  • Image → Transform → Rotate 90 degrees CCW ആൻഡ്

  • Image → Transform → Rotate 180°.
