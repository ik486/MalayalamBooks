4.4. വിള

ചിത്രം 14.115. Crop tool


ക്രോപ്പ് ഉപകരണം ഒരു ചിത്രം മുറിക്കുന്നതിന് അല്ലെങ്കിൽ ക്ലിപ്പ് ഉപയോഗിക്കുന്നു. അത് ചിത്രത്തിന്റെ ദൃശ്യമായതും അദൃശ്യനായ എല്ലാ പാളികൾ പ്രവർത്തിക്കുന്നു. ഈ ഉപകരണം പലപ്പോഴും അതിർത്തികൾ നീക്കം, അല്ലെങ്കിൽ ഒരു കൂടുതൽ കേന്ദ്രീകൃതമായ തൊഴിലാളി പ്രദേശം നിങ്ങൾക്ക് നൽകാൻ അനാവശ്യ പ്രദേശങ്ങൾ ഉന്മൂലനം ഉപയോഗിക്കുന്നു. ഇത് നിങ്ങളുടെ ചിത്രത്തിന്റെ യഥാർത്ഥ തലങ്ങളും പൊരുത്തപ്പെടാത്ത ഒരു നിർദ്ദിഷ്ട ഇമേജ് സൈസ് ആവശ്യമാണെങ്കിൽ ഉപയോഗപ്പെടുന്നു.

വെറും നിരക്കു ഉപകരണങ്ങളും പോലെ, പുതിയ ക്രോപ്പ് ഉപകരണം v2.4 പുറത്തിറക്കിയിരുന്നു മെച്ചപ്പെടുത്തി. വലുപ്പ യഥാർത്ഥത്തിൽ പകരം വ്യാപ്തി പ്രവർത്തനക്ഷമത നീക്കാൻ രണ്ടും നല്കുന്നതിന്റെ ക്രോപ്പ് വലുപ്പം കൈകാര്യം. ഉപകരണം ജിമ്പ് ഉപകരണങ്ങൾ ഉപയോഗിച്ച് കൂടുതൽ സ്വാഭാവികമായും സുസ്ഥിരമായി പെരുമാറുന്ന. നീക്കാൻ, പ്രദേശത്തെ ദീർഘചതുരം ക്ലിക്കിന്റെ വലിച്ചിടുക. വലുപ്പം ഒരേ സമയം വശങ്ങളും കോണിലും ഹാൻഡിൽ-ബാറുകൾ വലിച്ചോ ഒരേസമയം ഒന്നോ രണ്ടോ മഴു സാധ്യമാണ്. പുറത്ത് ഒരു നല്ല passepartout പ്രാബല്യത്തോടെ ഇരുണ്ടു കഴിയും മെച്ചപ്പെട്ട എത്ര അവസാന വിള പോലെ നോക്കും എന്ന ആശയം ലഭിക്കാൻ. കത്രിക്കൽ സാധൂകരിക്കാനാകുന്നതിലും, വിള റെക്ടാംഗിളിനുള്ളിൽ ക്ലിക്ക് അല്ലെങ്കിൽ Enter കീ അമർത്തുക.

മൌസ് ചലിക്കുന്ന ക്രോസ്-മുടി മാറുന്നു ചെയ്യുമ്പോൾ, ക്രോപ്പ് നീക്കാൻ കീബോർഡ് ആരോ കീകൾ ഉപയോഗിക്കാം. Shift കീ അമർത്തിപ്പിടിക്കുക 25 പിക്സൽ ഇന്ക്രിമെന്റ് പ്രകാരം നീക്കുന്നതിന് അനുവദിക്കുന്നു.

You can use Guides to position the crop area. Make sure that the View → Snap to Guides option is checked.

4.4.3. ടൂൾ ഓപ്ഷനുകൾ

ചിത്രം 14.116. “Crop” ടൂളിനായി ടൂൾ ഓപ്ഷനുകൾ


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

Current Layer Only

ഈ ഓപ്ഷൻ വിള മാത്രമേ സജീവ പാളി ബാധിക്കും ആക്കും.

Allow Growing

ഈ ഓപ്ഷൻ വിള അനുവദിക്കുന്നു അല്ലെങ്കിൽ ചിത്രം (അല്ലെങ്കിൽ പാളി) പുറത്തുള്ള സ്ഥാനം വലുപ്പം, പോലും ക്യാൻവാസ്. അതിനാൽ, തത്ഫലമായി ചിത്രത്തിലേക്ക് നിങ്ങൾ ആഗ്രഹിക്കുന്ന വലിപ്പം നൽകാൻ കഴിയും. വിള യാതൊരു മെറ്റീരിയൽ ഉണ്ടെങ്കിൽ സുതാര്യത ഉപയോഗിക്കും.

ചിത്രം 14.117. “Allow Growing” ഉദാഹരണത്തിന്

ഒരു വലിയ ക്യാൻവാസിൽ ഒരു ചിത്രം

ഓപ്ഷൻ പരിശോധിക്കുന്നു. ക്രോപ്പ് കാൻവാസിൽ പുറത്ത് വ്യാപിച്ചിരിക്കുന്നു.

തത്ഫലമായി ചിത്രം.


Expand from Center

ഈ ഓപ്ഷൻ എപ്പോഴാണ് ചെക്കുചെയ്തിട്ടുണ്ടെന്നും, ക്രോപ്പ് ആദ്യ പിക്സൽ നിങ്ങൾ സെന്റര് എടുത്ത ക്ലിക്കുചെയ്തു നിന്ന് വികസിക്കുന്നു. ക്രോപ്പ് ഡ്രോയിംഗ് നിങ്ങൾക്ക് Ctrl ഉപയോഗിച്ച് ഈ ഓപ്ഷൻ ടോഗിൾ ചെയ്യാൻ കഴിയും.

Fixed

നിങ്ങൾക്ക് ക്രോപ്പ് ഡ്രോയിംഗ് സമയത്ത് Ctrl കീ പിടിച്ചുകൊണ്ട് ഈ ഓപ്ഷൻ ആക്സസ് ചെയ്യാൻ കഴിയും. ഈ ഐച്ഛികം ക്രോപ്പ് ആദരവ് ഫിക്സഡ് അളവുകൾ, അല്ലെങ്കിൽ അവരുടെ അനുപാതം ഡ്രോയിംഗ് ഉണ്ടാക്കുവാൻ നിരവധി പ്രദാനം:

  • Aspect ratio: ഡിഫോൾട്ട് സാധ്യത തുടർന്ന്. ക്രോപ്പ് വരയ്ക്കുന്നത് സമയത്ത് വീതിയും ഉയരവും അവർ യഥാർത്ഥ ഇമേജിൽ സമാനമായ അനുപാതം പ്രമാണിച്ചു.

  • Width / Height: വീതി അല്ലെങ്കിൽ ഉയരം ഇതുമാത്രമാണ് തുടരും. ഈ മാനം മൂല്യം താഴെ കൊടുത്തിരിക്കുന്ന പെട്ടിയിൽ സജ്ജമാക്കാൻ കഴിയൂ; അതു 100 പിക്സൽ സ്ഥിര.

  • Size: വീതിയും ഉയരവും പരിഹരിക്കപ്പെടുന്നതായിരിക്കും. അവരുടെ മൂല്യങ്ങൾ ഉദാഹരണത്തിന് ഫോം “150x100” ൽ, താഴെ കൊടുത്തിരിക്കുന്ന പെട്ടിയിൽ സജ്ജമാക്കാൻ കഴിയൂ. ക്രോപ്പ് ഉടൻ ഇമേജിൽ ക്ലിക്ക് ഈ മൂല്യങ്ങൾ സ്വീകരിക്കാം. വലതുവശത്ത്, രണ്ടു ബട്ടണുകൾ നിങ്ങൾ ഒരു Landscape (widthwise) അല്ലെങ്കിൽ Portrait (നേരുള്ളവനും) ക്രോപ്പ് ഫോർമാറ്റ് തിരഞ്ഞെടുക്കുക.

സ്ഥാനം

ഈ രണ്ടു ടെക്സ്റ്റ് ബോക്സുകൾ തത്സമയം ക്രോപ്പ് മുകളിലെ ഇടത് കോണിൽ സ്ഥാനം (, ഇടതുഭാഗത്ത് തിരശ്ചീന വലതുഭാഗത്ത് ലംബമായ) കാണിക്കാനും വളരെ സ്വമേധയാ മാറ്റാനാകും. ഇത് പിക്സൽ ഇങ്ങനെയായിരുന്നു, എന്നാൽ നിങ്ങൾ യൂണിറ്റ് px ബട്ടൺ ഡ്രോപ്പ്-ഡൌൺ ലിസ്റ്റ് നന്ദി മാറ്റാനാകും. കോർഡിനേറ്റ് വംശജനായ കാൻവാസിൽ (അല്ല ചിത്രത്തിന്റെ) മുകളിലെ ഇടത് കോണിൽ ആണ്.

വലുപ്പം

ഈ രണ്ടു ടെക്സ്റ്റ് ബോക്സുകൾ തത്സമയം വിള-ദീർഘചതുരം വലിപ്പം (, ഇടതുഭാഗത്ത് തിരശ്ചീന വലതുഭാഗത്ത് ലംബമായ) കാണിക്കാനും വളരെ സ്വമേധയാ മാറ്റാനാകും. ഇത് പിക്സൽ ഇങ്ങനെയായിരുന്നു, എന്നാൽ നിങ്ങൾ യൂണിറ്റ് px ബട്ടണിന്റെ drop- ഡൌണ് ലിസ്റ്റ് നന്ദി മാറ്റാനാകും.

ഹൈലൈറ്റ്

ഈ ഓപ്ഷൻ ക്രോപ്പ് ഉയർത്തിക്കാട്ടുന്ന ഉദ്ദേശിച്ചിട്ടുള്ള ഇരുണ്ട പുറത്ത് ടോഗിൾ ചെയ്യുന്നു.

Autoshrink

ഓട്ടോ ബട്ടണിൽ നിന്ന് അളവുകൾ വരയ്ക്കാൻ ഏത് നിന്നുള്ള സജീവ പാളിയിൽ, ഒരു അതിർത്തി കണ്ടെത്താൻ ശ്രമിക്കും ചുരുക്കുക. ഈ ഓപ്ഷൻ മാത്രം പശ്ചാത്തലവും കുത്തനെ വ്യത്യസ്ഥമാക്കുന്നു ഒറ്റപ്പെട്ട വസ്തുക്കൾ നല്ലവണ്ണം പ്രവർത്തിക്കുന്നു.

ചിത്രം 14.118. “Autoshrink” ഉദാഹരണത്തിന്

അതിൽ മൂർച്ചയുള്ള പരിധി ഒരു ക്രോപ്പ് ഒരു ചിത്രം.

Autoshrink ബട്ടൺ ക്ലിക്ക് ചെയ്ത ശേഷം, ക്രോപ്പ് മൂർച്ചയുള്ള പരിധികൾ fit ചുരുക്കിയതും ചെയ്തു.


Shrink Merged

ഈ ഓപ്ഷൻ ഓട്ടോ ചുരുക്കുക അല്ലെങ്കിൽ ഇതേ പ്രവർത്തിക്കുന്നു. അത് വെറും സജീവ പാളി നിന്ന് അധികം എല്ലാ visible പാളികൾ നിന്ന് പിക്സൽ വിവരം ഉപയോഗിക്കുന്നു.
