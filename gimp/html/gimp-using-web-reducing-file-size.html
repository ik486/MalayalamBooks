2.2. ഇനിയും കൂടുതൽ ഫയൽ വലിപ്പം കുറയ്ക്കൽ

നിങ്ങൾ കുറച്ചുകൂടി നിങ്ങളുടെ ചിത്രത്തിൻറെ വലുപ്പം കുറയ്ക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ നിങ്ങൾക്ക് ഇൻഡെക്സ്ഡ് മോഡിലേക്ക് നിങ്ങളുടെ ചിത്രം പരിവർത്തനം കഴിഞ്ഞില്ല. നിറങ്ങൾ എല്ലാ മാത്രം 256 മൂല്യങ്ങളിൽ കുറയും എന്നാണ്. യഥാർത്ഥ മിനുസമാർന്ന ഘനമായ സാധാരണയായി ബന്ധനങ്ങൾ ഒരു പരമ്പര പരിവർത്തനം കാരണം ഇൻഡെക്സ്ഡ് മോഡിലേക്ക് മിനുസമുള്ള നിറം സംക്രമണം അല്ലെങ്കിൽ ഘനമായ ഉപയോഗിച്ച് ചിത്രങ്ങൾ പരിവർത്തനം ചെയ്യരുത്. പരിവർത്തനം നടന്ന് സാധാരണ നാടൻ വീണതും കാരണം ഇൻഡെക്സ്ഡ് മോഡ് ഫോട്ടോഗ്രാഫുകൾ ശുപാർശ ചെയ്തിട്ടില്ല.

ചിത്രം 6.8. ഇൻഡെക്സ്ഡ് ചിത്രം

ഗണമാണോയെന്നത് ചിത്രം ഒരു ബിറ്റ് വീണതും നോക്കാൻ കഴിയും. ഇടത് ചിത്രം ശരിയായ ചിത്രം 300 ശതമാനം സൂം, അതിന്റെ യഥാർത്ഥ വലുപ്പത്തിൽ Wilber ആണ്.
