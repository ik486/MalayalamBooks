4.3. നീങ്ങുക

ചിത്രം 14.113. The Move tool in Toolbox


The Move Tool is used to move layers, selections, paths or guides. It works also on texts.

4.3.1. ടൂൾ സജീവമാക്കുന്നു

You can access the Move Tool in different ways:

  • From the image menu bar Tools → Transform Tools → Move,

  • By clicking the tool icon: .

  • കീബോർഡ് കുറുക്കുവഴി M ഉപയോഗിക്കുന്നതിലൂടെ.

  • ഒരു ഗൈഡിനെ സൃഷ്ടിക്കുമ്പോൾ നീക്കുക പ്രയോഗം സ്വയം സജീവമാകുന്നു.

    കുറിപ്പു്

    space ബാർ താഴേയ്ക്കാക്കുമ്പോൾ താൽക്കാലികമായി നീക്കുക സജീവ പ്രയോഗം മാറ്റുന്നു. നീക്കുക ഉപകരണമായി നീണ്ട സ്പേസ് ബാർ അമർത്തിപ്പിടിച്ചു പോലെ സജീവമായിരിക്കുന്ന. യഥാർത്ഥ ടൂൾ സ്പേസ് ബാർ പ്രകാശനം സജീവമാക്കുന്നു. ഈ പെരുമാറ്റം Switch to Move tool ഓപ്ഷൻ Edit → Preferences → Image Windows → Space Bar പ്രവർത്തനക്ഷമമാക്കിയതിനാൽ മാത്രമേ നിലവിലുണ്ട്.

4.3.2. ഓപ്ഷനുകൾ

ചിത്രം 14.114. Move Tool options


സാധാരണ, ഉപകരണം ഓപ്ഷനുകൾ ഉടൻ ഒരു ഉപകരണം സജീവമാക്കാൻ പോലെ പണിസഞ്ചി കീഴിൽ ഘടിപ്പിച്ചിട്ടുള്ള ഒരു ജാലകം പ്രദർശിപ്പിക്കും. അഥവാ അങ്ങനെയല്ലെങ്കിൽ, നിങ്ങൾ തിരഞ്ഞെടുത്ത ഉപകരണം ഓപ്ഷൻ വിൻഡോ തുറക്കുന്ന Windows → Dockable Windows → Tool Options വഴി ഇമേജ് ബാറില് നിന്നും ആക്സസ് ചെയ്യാൻ കഴിയും.

ഈ ഉപാധികൾ Transform tools common options വിവരിച്ചിരിക്കുന്നു.

നീങ്ങുന്നത് ചോയ്സ് ഉപകരണം നിറ്ത്തിയ ശേഷം നിലനിൽക്കുകയാണെങ്കിൽ ഓർമിക്കുക.

Tool toggle (Shift)

നീക്കുന്നത് “Layer” ചെയ്താൽ

ഒരു പാളി അല്ലെങ്കിൽ ഗൈഡ് തിരഞ്ഞെടുക്കുക

ഇത് നിലവിലെ പാളി ഭാഗമാക്കാൻ മൂലകം മേൽ പോകുമ്പോഴും നിരവധി പാളികൾ കൊണ്ട് ഒരു ഇമേജിൽ, മൌസ് പോയിന്റർ ഒരു അടയാളസൂചി തിരിയുന്നു. അപ്പോൾ നിങ്ങൾ click- കഴിയും ഒപ്പം-വലിച്ചിടുക. മൗസ് സൂചികയെ ഒരു ചെറിയ കൈ ആകൃതി (നിങ്ങൾ not എന്തു സജീവ പാളി ഒരു ഘടകത്തെ ട്ട് കാണിക്കുന്ന) ഉണ്ട്, പകരം ഒരു സജീവമല്ലാത്ത പാളി നീക്കും (നീക്കുന്നതിനിടെ സജീവ പാളി മാറുന്നു).

ഒരു ഗൈഡ് നിങ്ങളുടെ ചിത്രത്തിൽ നിലനിൽക്കുന്നുണ്ടെങ്കിൽ, അത് മൌസ് പോയിന്റര് പോകുമ്പോഴും ചുവപ്പ് തിരിക്കും. അപ്പോൾ അതു സജീവമാക്കി, അത് നീക്കാനാകും.

സജീവ പാളി നീക്കുക

മാത്രം നിലവിലെ പാളി നീക്കും. നിങ്ങൾ എളുപ്പത്തിൽ തെറ്റായ പാളി തിരഞ്ഞെടുക്കാനും എവിടെ സുതാര്യമായ പ്രദേശങ്ങൾ ഒരു പാളി, നീക്കാൻ ചെയ്യണമെങ്കിൽ ഈ ക്രമീകരണം സഹായിക്കും.

നീക്കുന്നത് “Selection” ചെയ്താൽ

നിരക്കു ന്റെ രൂപരേഖയിൽ നീക്കും (??? കാണുക).

നീക്കുന്നത് “Path” ചെയ്താൽ

ഒരു പാത തിരഞ്ഞെടുക്കുക

ഡിഫോൾട്ട് ആണ്. അതു visible path അടിക്കുമ്പോൾ മൗസ് പോയിന്റർ ഒരു ചെറിയ കൈ തിരിയുന്നു. അപ്പോൾ നിങ്ങൾ ക്ലിക്കിൽ-ആൻഡ്-വലിച്ചോ അതു (നീക്കുന്നതിനിടെ സജീവ പാത്ത് ആയിരിക്കും) വഴി ഈ പാത നീക്കാൻ കഴിയൂ.

സജീവ പാത്ത് നീക്കുക

മാത്രം നിലവിലെ പാത്ത് നീക്കും. നിങ്ങൾ Path Dialog നിലവിലെ പാത്ത് മാറ്റാനാകും.

4.3.3. നീക്കുക ഉപകരണം പ്രവർത്തനങ്ങളുടെ സംഗ്രഹം

Moving a selection

നീക്കുക ഉപകരണം നിരക്കു ഔട്ട്ലൈൻ നീക്കാൻ അനുവദിക്കുന്നു. നീക്കുക മോഡ് “Layer” ആണെങ്കിൽ, നിങ്ങൾക്ക് Ctrl+Alt കീകൾ പിടിക്കുക വേണം.

നീക്കുക മോഡ് തിരഞ്ഞെടുപ്പ് എങ്കിൽ, നിങ്ങൾക്ക് ക്ലിക്ക്-ആൻഡ്-വലിച്ചു കഴിയും ക്യാൻവാസ് ആരും എവിടെയും നിരക്കു ഔട്ട്ലൈൻ നീക്കാൻ. നിങ്ങൾക്ക് കൃത്യമായി തിരഞ്ഞെടുക്കലുകൾ നീക്കാൻ ആരോ കീകൾ ഉപയോഗിക്കാം. പിന്നെ, ഇറങ്ങി 25 പിക്സൽ വർദ്ധനവിൽ ചെയ്ത ശേഷം Shift കീ നീക്കങ്ങൾ holding.

നിങ്ങൾ നീക്കാൻ ഉപകരണം ഉപയോഗിച്ച് ഒരു നിര നീക്കാൻ ചെയ്യുമ്പോൾ, നിരക്കു കേന്ദ്രത്തിൽ ഒരു ക്രോസും അടയാളപ്പെടുത്തി. View → Snap to Guides (or Grid) ഓപ്ഷൻ എങ്കിൽ ഈ കുരിശും നിരക്കു ബൗണ്ടറികൾ ഗൈഡുകൾ അല്ലെങ്കിൽ ഗ്രിഡിലേക്ക് സ്നാപ്പ്: ഈ എളുപ്പം തിരഞ്ഞെടുക്കലുകൾ പരിപൂര്ണമാണ് ചെയ്യുന്നു.

മറ്റ് സാധ്യതകൾ വേണ്ടി Moving selections കാണുക.

ഒരു പാളി നീക്കുന്നു

നീക്കുക മോഡ് “Layer” ആയിരിക്കണം. നിങ്ങൾ ഒന്നോ അതിലധികമോ പാളികൾ, Point to Layer (or Guide) ഉണ്ടെങ്കിൽ പിന്നെ നിങ്ങൾ Move the Active Layer ​​ആൻഡ് തമ്മിൽ തിരഞ്ഞെടുക്കാം.

Moving Grouped Layers

പാളികൾ (ചെറിയ ചെയിൻ ചിഹ്നം ഉപയോഗിച്ച്) ഗ്രൂപ്പുചെയ്തിരിക്കുന്നു എങ്കിൽ എല്ലാവരും നീങ്ങും, ഏതുമാകട്ടെ പാളിയുടെ നിലവിൽ സജീവമാണ്.

Moving a guide

നിങ്ങൾ ഒരു ഭരണാധികാരി നിന്ന് ഒരു ഗൈഡ് pull ചെയ്യുമ്പോൾ, നീക്കുക പ്രയോഗം സ്വയം സജീവമാകുന്നു. മറ്റൊരു ഉപകരണം ഉപയോഗിച്ച് തുടർന്നാണ് കേസ് അല്ല നിങ്ങൾ സ്വയം അത് സജീവമാക്കുന്നതിന് ഞങ്ങൾക്കുണ്ട്. മൗസ് സൂചികയെ ഒരു ഗൈഡ് അടിക്കുമ്പോൾ, ഈ ഗൈഡ് ചുവന്ന തിരിഞ്ഞ് ഇങ്ങനെ നിങ്ങൾ അതു നീക്കാൻ ക്ലിക്ക്-ആൻഡ്-വലിച്ചു കഴിയും.

Moving a path

പാത ടൂൾ ഡയലോഗ് സ്വന്തം ഇളക്കം ഫംഗ്ഷൻ ഉണ്ട്: ??? കാണുക. എന്നാൽ നിങ്ങൾക്ക് നീക്കുക ഉപകരണം ഉപയോഗിക്കാം. നീക്കുക മോഡ് “Path” ലേക്ക് സജ്ജമാക്കണം. പാത്ത് അദൃശ്യനായ മാറുന്നു ശ്രദ്ധിക്കുക; അതു പാത ഡയലോഗിൽ ദൃശ്യമാക്കുന്നതിനിടയാക്കും. നിങ്ങൾ നീക്കും പാത്ത് തിരഞ്ഞെടുക്കാൻ അല്ലെങ്കിൽ സജീവ പാത നീക്കാൻ കഴിയൂ.

Moving a text

ഓരോ ടെക്സ്റ്റ് സ്വന്തം പാളി ഉണ്ട് പാളികൾ പോലെ നീക്കാൻ കഴിയില്ല. വിഭാഗം 6.6, “ടെക്സ്റ്റ്” കാണുക.
