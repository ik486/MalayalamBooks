7.24. Delete Layer Mask

Delete Layer Mask കമാൻഡ് സജീവ പാളി സ്വയം വരുത്താതെ, സജീവ പാളി പാളി മാസ്ക് ഇല്ലാതാക്കുന്നു. സജീവ പാളി ഒരു പാളി മാസ്ക് ഇല്ലെങ്കിൽ, മെനു എൻട്രി കൈമാറ്റം ചെയ്ത് ആവശ്യം തന്നെയാണ്.

7.24.1. കമാൻഡ് സജീവമാക്കുന്നു

  • You can access this command from the image menubar through Layer → Mask → Delete Layer Mask,

  • or from the pop-up menu you get by right-clicking on the active layer in the Layers Dialog.
