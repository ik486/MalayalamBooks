3. ഫയലുകൾ തുറക്കുന്നു
പാഠം 5. കടന്നു ജിമ്പ് ഇമേജുകൾ നേടുന്നു  

3. ഫയലുകൾ തുറക്കുന്നു

GIMP നിലവിലുള്ള ചിത്രം തുറക്കുന്ന നിരവധി മാർഗ്ഗങ്ങളുണ്ട്:

3.1. ഫയൽ തുറക്കുക

നിലവിലുള്ള ചിത്രം തുറക്കാൻ ഏറ്റവും വ്യക്തമായ വഴി മെനു ആകുന്നു. നിങ്ങൾ ഫയൽ നാവിഗേറ്റ് അതിന്റെ പേരില് ക്ളിക്ക് അനുവദിച്ചുകൊണ്ട് Open Image ഡയലോഗ് തുറക്കാൻ File → Open ഉപയോഗിക്കുക. നിങ്ങൾ തുറക്കാൻ ഫയല് പേരും സ്ഥലവും അറിയാമെങ്കിൽ ഈ രീതി നന്നായി പ്രവർത്തിക്കുന്നു. Open Image ഡയലോഗ് ഒരു പ്രിവ്യൂ പാൻ ഉണ്ട് ആണെങ്കിലും, അതു സുഖ (എളുപ്പമല്ല) ഒരു ലഘുചിത്രം അടിസ്ഥാനമാക്കി ഒരു ചിത്രം കണ്ടുപിടിക്കുക എന്നതാണ്.

കുറിപ്പു്

ഒരു ഫയല്തുറക്കുന്നതിനിടെപ്രശ്നംസംഭവിച്ചിരിക്കുന്നു, GIMP ഫയൽ തരം നിർണ്ണയിക്കാൻ വേണം. നിർഭാഗ്യവശാൽ അത്തരം . jpg പോലെ ഫയൽ എക്സ്റ്റൻഷൻ, അല്ല വിശ്വസനീയമായ: ചിത്രം വിപുലീകരണങ്ങൾ സിസ്റ്റത്തിൽ നിന്ന് സിസ്റ്റത്തിലേക്ക് വ്യത്യാസപ്പെട്ടിരിക്കും; ഏതെങ്കിലും ഫയൽ ഏത് വിപുലീകരണവും ഞങ്ങൾക്കുണ്ട് പുനർനാമകരണം കഴിയും; ഒരു ഫയൽ പേര് ഒരു വിപുലീകരണം കുറവുണ്ടോ എന്തുകൊണ്ടാണ് പല കാരണങ്ങൾ ഉണ്ട്. GIMP ആദ്യം അതിലെ ഉള്ളടക്കങ്ങളും പരിശോധിക്കുന്നതിലൂടെ ഒരു ഫയൽ തിരിച്ചറിയാൻ ശ്രമിച്ചാൽ: സാധാരണയായി ഉപയോഗിക്കുന്ന ഫയൽ ഫോർമാറ്റുകൾ ഏറ്റവും അംഗീകരിക്കാൻ അവരെ അനുവാദമുള്ള “magic headers” ഞങ്ങൾക്കുണ്ട്. മാജിക് ഫലമൊന്നും അവയോരോന്നും മാത്രം GIMP വിപുലീകരണം ഉപയോഗിച്ച് അവലംബിക്കാൻ ഇല്ല.

ചിത്രം 5.4. “Open Image” ഡയലോഗ്


GIMP 2.2 നിങ്ങൾ ഒരു ഫയൽ വേഗം സഹായിക്കുന്നതിന് നിരവധി സവിശേഷതകൾ നൽകുന്ന ഒരു പുതിയ Open Image ഡയലോഗ് അവതരിപ്പിച്ചു. ഒരുപക്ഷേ ഏറ്റവും പ്രധാനപ്പെട്ട നിങ്ങൾ പലപ്പോഴും ഉപയോഗിക്കുന്ന ഫോൾഡറുകൾ, “bookmarks”, അല്ലെങ്കിൽ Places സൃഷ്ടിക്കാൻ കഴിവ് ആണ്. ബുക്ക്മാർക്കുകൾ ലിസ്റ്റിൽ ഡയലോഗ് ഇടത് വശത്ത് ദൃശ്യമാകുന്നു. അത്തരം “Desktop” പോലെ മുകളിൽ പശുക്കൾ, സ്വയം നൽകിയിരിക്കുന്നു. ലിസ്റ്റിൽ നിലവിലെ ഡയറക്ടറി ചേർക്കാൻ Add ബട്ടൺ ഉപയോഗിക്കുക. തിരഞ്ഞെടുത്ത ബുക്ക്മാർക്ക് നീക്കം Remove ബട്ടൺ ഉപയോഗിക്കുക. ഡയറക്ടറി നേരിട്ട് നാവിഗേറ്റ് ഒരു ബുക്ക്മാർക്ക് ഇരട്ട ക്ലിക്കുചെയ്യുക.

ഡയലോഗ് കേന്ദ്രത്തിൽ തിരഞ്ഞെടുത്ത ഡയറക്ടറിയുടെ ഉള്ളടക്കങ്ങളുടെ ഒരു അടങ്ങുന്നു. സബ്ഡയറക്ടറികൾ ഫയലുകൾ അവരെ താഴെ, പട്ടികയിൽ മുകളിൽ കാണിക്കില്ല. സ്വതവേ, ഡയറക്ടറിയിൽ എല്ലാ ഫയലുകളും പ്രദര്ശിപ്പിക്കുന്നു, എന്നാൽ നിങ്ങൾ ഡയറക്ടറി നന്ദി കീഴെ ദൃശ്യമാകുന്ന ഫയൽ തരം നിരക്കു മെനു ഉപയോഗിച്ച് ഒരു പ്രത്യേക തരത്തിലുള്ള ഇമേജ് ഫയലുകൾ ലേക്കുള്ള ലിസ്റ്റിംഗ് നിയന്ത്രിക്കാൻ കഴിയും.

ലിസ്റ്റിംഗ് ഒരു ഇമേജ് ഫയൽ എൻട്രി തിരഞ്ഞെടുക്കുമ്പോൾ പ്രിവ്യൂ ചിത്രം കുറിച്ചുള്ള ചില അടിസ്ഥാന വിവരങ്ങൾ സഹിതം, ഡയലോഗ് വലത്തു ഭാഗത്തു ദൃശ്യമാകുന്നു. തയ്യാറാകുമ്പോൾ തന്നെ അവ വരുമ്പോൾ പ്രിവ്യൂകൾ കാഷെചെയ്ത ശ്രദ്ധിക്കുക, പ്രിവ്യൂ തെറ്റായിരിക്കാം കാരണമായേക്കാവുന്നത് നിങ്ങൾക്ക് ചെയ്യാൻ കഴിയുന്ന ചില കാര്യങ്ങൾ ഉണ്ട്. ഈ സംഭവിക്കാൻ നിങ്ങൾ കരുതുന്നു എങ്കിൽ, പ്രിവ്യൂ പ്രദേശത്ത് Ctrl കീ ക്ലിക്കു താഴേയ്ക്കാക്കുമ്പോൾ സൃഷ്ടിക്കപ്പെടുന്നതെന്ന് ഒരു പുതിയ പ്രിവ്യൂ നിർബന്ധിതമാക്കാം.

സ്വതവേ, ഒരു Location വാചക ബോക്സ് ഫയൽ ഡയലോഗ് തുറക്കുക അതില്ല. അത് മാറി ചെയ്യാവൂ Ctrl+L കീ ഈ ടെക്സ്റ്റ് ബോക്സ് ടോഗിൾ ചെയ്യുന്നു. മറ്റൊരുവിധത്തിൽ, നിങ്ങൾ ടെക്സ്റ്റ് ബോക്സ് ടോഗിൾ മുകളിലെ ഇടത് മൂലയിൽ പേപ്പർ പെൻസിൽ ഐക്കൺ ക്ലിക്ക് ചെയ്യാം.

കുറിപ്പു്

നിങ്ങൾ ലിസ്റ്റിൽ നിന്ന് ഒരു ഫയൽ പേര് തിരഞ്ഞെടുക്കുക, താഴത്തെ വലത് കോണിലുള്ള അല്ലെങ്കിൽ ഡയലോഗിൽ “Open” ബട്ടൺ ക്ലിക്ക്, അത് GIMP സ്വയം നിങ്ങൾക്ക് ഫയൽ തരം നിർണ്ണയിക്കാൻ ഏതാണ്ട് എപ്പോഴും സത്യമാണ്. അപൂർവമായി ന് ഫയൽ തരം അസാധാരണമായ, പേരിലെ അർത്ഥവത്തായ വിപുലീകരണം പോയിട്ടില്ല പ്രധാനമായും എങ്കിൽ GIMP ശരിയായി ഫയൽ തരം തിരിച്ചറിയാൻ പരാജയപ്പെടാം. ഈ ആവശ്യമെങ്കിൽ സ്വമേധയാ ഫയൽ തരം വ്യക്തമാക്കാൻ ഡയലോഗ് ചുവടെയുള്ള Select File Type ഉപയോഗിക്കുക. സാധാരണയായി, എങ്കിലും, GIMP ഒരു ഇമേജ് ഫയൽ തുറക്കാൻ പരാജയപ്പെടുകയാണെങ്കിൽ, അതു വഷളായി അല്ലെങ്കിൽ ഒരു പിന്തുണയ്ക്കുന്ന ഫോർമാറ്റ് ആണ്.
