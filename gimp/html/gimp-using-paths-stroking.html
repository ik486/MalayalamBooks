5.5. ഒരു പാത അതാണെന്ന്

ചിത്രം 7.12. അതാണെന്ന് പാതകൾ

മുകളിൽ ദൃഷ്ടാന്തം നാലു പാതകൾ, ഓരോന്നും പല വിധത്തിൽ എന്തേ.


ഗോപനെ പാത പ്രയോഗം വേണ്ടി ചിത്ര മെനു അല്ലെങ്കിൽ റൈറ്റ് ക്ലിക്ക് വിഭവസൂചികയ്ക്കുള്ളഡയലോഗ് വഴികൾ, അല്ലെങ്കിൽ ടൂൾ ഓപ്ഷനുകൾ ഡയലോഗിൽ “Stroke Path” ബട്ടണിൽ നിന്ന് Edit → Stroke Path ഉപയോഗിച്ച് അവർ stroked പക്ഷം ചിത്രം പിക്സൽ ഡാറ്റ രൂപം ഭേദം ഇല്ല.

ഈ വിധേനയും ഏതെങ്കിലും “Stroke Path” തിരഞ്ഞെടുക്കുന്നത് നിങ്ങൾ അതാണെന്ന് നടക്കുന്ന രീതി നിയന്ത്രിക്കാൻ അനുവദിക്കുന്ന ഒരു ഡയലോഗ് ചെയ്യുന്നു. വരിയിലെ ശൈലികളുടെ വൈവിധ്യമാർന്ന തിരഞ്ഞെടുക്കാൻ കഴിയും, അല്ലെങ്കിൽ മുതലായവ പോലുള്ള ക്ലോൺ പ്രയോഗം അസാധാരണമായ പശുക്കൾ, സ്മഡ്ജ് ടൂൾ, എറേസർ ഉൾപ്പെടെ പെയിന്റ് ഉപകരണങ്ങൾ ഏതെങ്കിലും സ്ട്രോക്ക്, ചെയ്യാനാകും

ചിത്രം 7.13. Stroke Path ഡയലോഗ്


നിങ്ങൾ കൂടുതൽ ഒരു പാത്ത് ഒന്നിലധികം തവണ അതാണെന്ന് വഴി അതാണെന്ന് ഇഫക്റ്റുകൾ ശ്രേണി വർദ്ധിപ്പിക്കാൻ, അല്ലെങ്കിൽ വ്യത്യസ്ത വീതി മാതൃകയിൽ അല്ലെങ്കിൽ നയപ്രഖ്യാപന ഉപയോഗിച്ച് കഴിയും. ഈ വഴിയിൽ രസകരമായ ഇഫക്റ്റുകൾ ലഭിക്കുന്നതിന് സാധ്യതകൾ ഏറെക്കുറെ പരിധിയുമില്ല.
