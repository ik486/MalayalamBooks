3. പെയിന്റ് ഉപകരണങ്ങൾ  

3.3. ബ്രഷ് ഉപകരണങ്ങൾ (പെൻസിൽ, അന്തസ്സായിത്തന്നെയാണു, എയർബ്രഷ്)

ചിത്രം 14.57. Painting example

മൂന്ന് സ്ട്രോക്കുകൾ ഒരേ ചുറ്റും ഫസി ബ്രഷ് (ഇടതുഭാഗത്ത് മുകളിലെ കാണിച്ചിരിക്കുന്ന ഔട്ട്ലൈൻ) ചായം, പെൻസിൽ (ഇടത്), അന്തസ്സായിത്തന്നെയാണു (മധ്യം), ഒപ്പം എയർബ്രഷ് (വലത്) ഉപയോഗിച്ച്.


ഈ ഗ്രൂപ്പിൽ ഉപകരണങ്ങൾ GIMP ന്റെ അടിസ്ഥാന പെയിന്റിംഗ് ഉപകരണങ്ങൾ, അവർ ഈ വിഭാഗത്തിൽ ഒരുമിച്ചു ചർച്ച ആസ്തി പൊതുവായി മതിയായ സവിശേഷതകൾ. എല്ലാ പെയിന്റ് ഉപകരണങ്ങൾ പൊതുവായുള്ള സവിശേഷതകൾ Common Features വിഭാഗത്തിൽ വിവരിച്ചിരിക്കുന്നു. ഒരു വ്യക്തിഗത ടൂൾ ഉപയോഗിക്കുക സവിശേഷതകൾ ആ ഉപകരണം പറ്റിച്ചേർന്നു വിഭാഗത്തിൽ വിവരിച്ചിരിക്കുന്നു.

പെൻസിൽ ഈ ഗ്രൂപ്പിൽ പ്രയോഗങ്ങളുടെ പ്രാകൃതമായ; അതു ഹാർഡ്, നോൺ-അപരത്വമാണ് വിരുദ്ധ വെഫ്റ്റ് ചെയ്യുന്നു. അന്തസ്സായിത്തന്നെയാണു ഇന്റർമീഡിയറ്റ്; അതു ഒരുപക്ഷേ ഗ്രൂപ്പിന്റെ ഏറ്റവും സാധാരണയായി ഉപയോഗിക്കുന്ന. എയർബ്രഷ് ഏറ്റവും വഴങ്ങുന്ന നിയന്ത്രിക്കാവുന്ന ആണ്. ഈ വഴക്കം പുറമേ എന്നാൽ അന്തസ്സായിത്തന്നെയാണു അധികം ഉപയോഗിക്കാൻ കുറച്ചുകൂടി കൂടുതൽ ബുദ്ധിമുട്ടാണ് ചെയ്യുന്നു.

ഈ ഉപകരണങ്ങൾ എല്ലാ അടിസ്ഥാന പാലറ്റ് നിന്ന് അല്ലെങ്കിൽ ഒരു ഗ്രേഡിയന്റ് നിന്ന് ഒന്നുകിൽ അതേ നയപ്രഖ്യാപന, വർണ്ണങ്ങൾ തിരഞ്ഞെടുത്തതിന് സമാന ഓപ്ഷനുകൾ പങ്ക്. എല്ലാ മോഡുകൾ വൈവിധ്യമാർന്ന ചിത്രകല കഴിവുള്ളവയാണ്.

3.3.1. Key modifiers

Ctrl,

പിക്സൽ നിറം ലേക്ക് (Toolbox Color Area പ്രദർശിപ്പിച്ച അല്ലാത്തത്) പാളി ഏതെങ്കിലും പിക്സൽ മേലെ ക്ലിക്ക് പുരോതല നിറം സജ്ജീകരിക്കുന്നു: Ctrl കീ അമർത്തിപ്പിടിക്കുക ഒരു Color Picker ഈ ഉപകരണങ്ങൾ ഓരോ മാറ്റുന്നു.

മാറ്റം

ഈ കീ വര മോഡിലേക്ക് ഈ ഉപകരണങ്ങൾ മതിയാവില്ല. ഹോൾഡിംഗ് Shift Button 1 ക്ലിക്ക് ചെയ്യുമ്പോൾ ഒരു വര സൃഷ്ടിക്കും. തുടർച്ചയായ ക്ലിക്കുകൾ കഴിഞ്ഞ വരിയുടെ അവസാനം നിന്നും ആ നേരായ ഡ്രോയിംഗ് തുടരും.
