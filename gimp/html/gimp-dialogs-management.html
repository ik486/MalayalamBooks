4. ചിത്രം മാനേജ്മെന്റ് ബന്ധപ്പെട്ട ഡയലോഗുകൾ

4.1. ബഫറുകൾ സംഭാഷണം

ചിത്രം 15.62. ബഫറുകൾ ഡയലോഗ് (ഒരു പട്ടിക)


ബഫറുകൾ ഇമേജ് ഡാറ്റ, വെട്ടി അല്ലെങ്കിൽ ഒരു വരയ്ക്കാനാകുന്നത് (ഒരു പാളി, ലെയർ മാസ്ക് മുതലായവ) ഭാഗമായി പകർത്താനും വരുമ്പോൾ സൃഷ്ടിച്ച താൽക്കാലിക റിപ്പോസിറ്ററികൾ ആകുന്നു. തീർച്ചയായും,, Edit → Buffer → Copy Named അല്ലെങ്കിൽ Edit → Buffer → Cut Named ഒരു ഡയലോഗിൽ ഡാറ്റ സംഭരിക്കാൻ ഒരു ബഫർ പേര് നൽകാൻ ആവശ്യപ്പെടുന്ന മാറികിട്ടി സൃഷ്ടിക്കാനും പേരുള്ള ബഫറുകളെയും എണ്ണത്തിൽ ഹാർഡ് പരിധി ഇല്ല ആണെങ്കിലും: ഈ ബഫർ ഒരു പ്രമാണം രണ്ടു വഴികളിൽ രക്ഷിക്കാൻ കഴിയും. ഓരോ ഒറ്റ മെമ്മറി ഒരു പങ്ക് വിനിയോഗിക്കുന്നു.

“Buffers” ഡയലോഗ് നിലവിലുള്ള എല്ലാ പേരുള്ള ബഫറുകളെയും ഉള്ളടക്കങ്ങൾ കാണിക്കുന്നു, നിങ്ങൾ പല വഴികളിൽ അവരെ ഓപ്പറേറ്റ് അനുവദിക്കുന്നു. നിങ്ങൾ മുകളിൽ, ഗ്ലോബൽ ബഫർ ഉള്ളടക്കങ്ങൾ കാണിക്കുന്നു, എന്നാൽ ഈ കേവലം ഒരു ഡിസ്പ്ലേ; നിങ്ങൾ അതു കൊണ്ട് ഒന്നും ചെയ്യാൻ കഴിയില്ല.

ശ്രദ്ധിക്കുക

Named ബഫറുകളെയും സെഷനുകളിലുടനീളം സംരക്ഷിച്ചില്ല. ഉള്ളടക്കം സംരക്ഷിക്കാനുള്ള ഏക വഴി ചിത്രങ്ങൾ അവരെ ഒട്ടിക്കുക എന്നതാണ്.

4.1.1. ഡയലോഗ് സജീവമാക്കുന്നു

ഈ ഡയലോഗ് ഒരു dockable ഡയലോഗ് ആണ്; അതു കൈകാര്യം ന് സഹായത്തിനായി വിഭാഗം ??? കാണും.

താങ്കൾക്ക് ആക്സസ് ചെയ്യാൻ കഴിയും:

  • ഒരു ഇമേജ് മെനുവിൽ നിന്ന്: Windows → Dockable Dialogs → Buffers;

  • ഏതെങ്കിലും dockable ഡയലോഗിൽ ടാബ് മെനുവിൽ നിന്നും ക്ലിക്കുചെയ്ത് Add Tab → Buffers തിരഞ്ഞെടുത്ത്.

Windows മെനുവിൽ, കുറഞ്ഞത് ഒരു ഡയലോഗ് തുറന്നുതന്നെയിരിക്കുന്ന മാത്രമേ ഉണ്ട് detached windows ഒരു ലിസ്റ്റ് ഉണ്ട്. Windows → Buffers: ഈ സാഹചര്യത്തിൽ, ഇമേജ്-മെനുവിൽ നിന്നും “Buffers” ഡയലോഗ് ഉണ്ടാകാം.

4.1.2. ബഫറുകൾ ഡയലോഗ് ഉപയോഗിച്ച്

ചിത്രം 15.63. ബഫറുകൾ മെനു


ഡിസ്പ്ലേ പ്രദേശത്തെ ഒരു ബഫർ ക്ലിക്കുചെയ്യുന്നത് സജീവ ബഫർ ഞാൻ ചെയ്യുന്നു. ഇ., ഡയലോഗ് താഴെയുള്ള അല്ലെങ്കിൽ ബഫറുകൾ മെനു ഉപയോഗിച്ച് വധിക്കപ്പെട്ട പേസ്റ്റ് കമാൻഡുകൾ ബട്ടണുകൾ ഉപയോഗിക്കും ഒന്നോ. ഇരട്ട-ക്ലിക്കുചെയ്ത് ഒരു ബഫർ ന് അതിന്റെ ഉള്ളടക്കം ഒരു ഫ്ലോട്ടിംഗ് നിരക്കു സജീവ ചിത്രത്തിലേക്ക് ഒട്ടിക്കാൻ കാരണമാകുന്ന; ഈ “Paste Buffer” കമാൻഡ് ഒരു ദ്രുത വഴി.

ഡയലോഗ് ചുവടെ നാലു ബട്ടണുകൾ. തങ്ങളാൽ ഓപ്പറേഷൻസ് നിങ്ങളെയും സജീവ ബഫർ റൈറ്റ് ക്ലിക്ക് കിട്ടാറുള്ള ബഫറുകൾ മെനു നിന്നും ആക്സസ് ചെയ്യാം.

ചിത്രം 15.64. ബഫറുകൾ ഡയലോഗ് (ഗ്രിഡ് കാഴ്ച)


“Buffers” ഡയലോഗ് ടാബ് മെനുവിൽ View as Grid ആൻഡ് View as List തമ്മിലുള്ള തിരഞ്ഞെടുക്കാം. ഗ്രിഡ് മോഡിൽ, ബഫറുകളെയും ചതുരാകൃതിയിലുള്ള അണിനിരന്നു നിരത്തിയിരിക്കുന്നത്. പട്ടിക മോഡിൽ അവർ ലംബമായി, ഓരോ വരി ബഫറിന്റെ ഉള്ളടക്കങ്ങളുടെ ഒരു ലഘുചിത്രം അതിന്റെ പേര്, അതിന്റെ പിക്സൽ തലങ്ങളും കാണിക്കുന്ന കൂടിയ ചെയ്യുന്നു.

സൂചന

Ctrl+F ഒരു തിരയൽ ഫീൽഡ് തുറക്കുന്നു. പട്ടിക ആയി കാണുക; ഗ്രിഡായി കാണുക കാണുക

ഡയലോഗ് ന്റെ ടാബ് മെനുവിലെ “Preview Size” സബ്മെനു ഉപയോഗിച്ച് ഡയലോഗിൽ ബഫർ പ്രിവ്യൂകളുടെ വലിപ്പം മാറ്റാൻ കഴിയും.

4.1.2.1. ചുവടെയുള്ള ബട്ടണുകൾ

ഡയലോഗ് ചുവടെ നിങ്ങൾ ബട്ടണുകൾ ഒരു ദമ്പതികൾ കണ്ടെത്താൻ:

ബഫർ ഒട്ടിക്കുക

ഈ കമാൻഡ് ഒരു ഫ്ലോട്ടിംഗ് നിരക്കു സജീവമാണ് ചിത്രം തെരഞ്ഞെടുക്കപ്പെട്ട് ബഫർ ഉളളടക്കം pastes. ഈ സാധാരണ Paste കമാൻഡ് തമ്മിലുള്ള ഏക വ്യത്യാസം അത് ആഗോള ക്ലിപ്ബോർഡ് ബഫർ അധികം തിരഞ്ഞെടുത്ത ബഫർ ഉപയോഗിക്കുന്നു എന്നതാണ്.

ബഫർ ഒട്ടിക്കുക

ഈ കമാൻഡ് ഒരു ഫ്ലോട്ടിംഗ് നിരക്കു സജീവമാണ് ചിത്രം ൻറെ നിരക്കു തെരഞ്ഞെടുക്കപ്പെട്ട് ബഫർ ഉളളടക്കം pastes. ഈ സാധാരണ Paste Into കമാൻഡ് തമ്മിലുള്ള ഏക വ്യത്യാസം അത് ആഗോള ക്ലിപ്ബോർഡ് ബഫർ അധികം തിരഞ്ഞെടുത്ത ബഫർ ഉപയോഗിക്കുന്നു എന്നതാണ്.

ന്യൂ ആയി ബഫർ ഒട്ടിക്കുക

ഈ കമാൻഡ് തിരഞ്ഞെടുത്ത ബഫർ ഉളളടക്കം ഒരു പുതിയ സിംഗിൾ-പാളി ഇമേജ് ഉണ്ടാക്കുന്നു. ഈ സാധാരണ Paste as New കമാൻഡ് തമ്മിലുള്ള ഏക വ്യത്യാസം അത് ആഗോള ക്ലിപ്ബോർഡ് ബഫറിന്റെ ഉള്ളടക്കം അധികം തിരഞ്ഞെടുത്ത ബഫർ ഉപയോഗിക്കുന്നു എന്നതാണ്.

ബഫർ ഇല്ലാതാക്കുക

ഈ കമാൻഡ് തിരഞ്ഞെടുത്ത പേരുള്ള ബഫർ ഇല്ലാതാക്കുന്നു, ആരും ചോദ്യങ്ങൾ ചോദിച്ചു. നിങ്ങൾ ഗ്ലോബൽ ബഫർ ഇല്ലാതാക്കാൻ കഴിയില്ല.

4.1.2.2. സന്ദർഭ മെനു

ചിത്രം 15.65. “Buffers” സന്ദർഭ മെനു


ഈ കമാൻഡുകൾ ബട്ടണുകൾ മുകളിൽ വിശദമാക്കിയ ചെയ്യുന്നു.
