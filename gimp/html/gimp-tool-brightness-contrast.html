5.5. മിഴിവ്-തീവ്രത

മിഴിവ്-തീവ്രത പ്രയോഗം സജീവ അല്ലെങ്കിൽ പാളി നിരക്കു വേണ്ടി മിഴിവും കോൺട്രാസ്റ്റ് അളവ് മറക്കുക. ഈ ഉപകരണം ഉപയോഗിക്കാൻ എളുപ്പമാണ്, എന്നാൽ താരതമ്യേന unsophisticated ആണ്. മാനത്തിൽ കർവ് ഉപകരണങ്ങൾ നിങ്ങളെ മാറ്റങ്ങൾമൂലം ഒരേ തരത്തിലുള്ള അനുവദിക്കുന്നു, മാത്രമല്ല നിങ്ങൾ കടുത്ത നിറങ്ങളിൽ നിന്ന് വ്യത്യസ്തമായി നിറങ്ങളില് പെരുമാറണം കഴിവ് നൽകുന്നത്. പൊതുവെ പറഞ്ഞാൽ, ബിസി ഉപകരണം ഏതാനും നിമിഷങ്ങൾക്കുള്ളിൽ ഒരു "വേഗത്തിലും വൃത്തികെട്ട" അഡ്ജസ്റ്റ്മെന്റ് ചെയ്യുന്നത് വലിയ ആണ്, പക്ഷേ ചിത്രം പ്രധാനമാണ് നിങ്ങൾക്ക് അത് കഴിയുന്നത്ര മനോഹരമായിരിക്കും ആഗ്രഹിക്കുന്ന, മറ്റ് ഉപകരണങ്ങൾ ഒന്ന് ഉപയോഗിക്കും.

ജിമ്പ് 2.4 ൽ ഈ ഉപകരണം ഓപ്പറേറ്റിങ് ഒരു പുതിയ മാർഗം ചേർത്തു: ചിത്രം അകത്ത് മൌസ് ക്ളിക്ക് ഇടത്തും മൗസ് ബട്ടൺ നിലനിർത്തിക്കൊണ്ടുതന്നെ വലിച്ചോ. മൌസ് നീക്കുന്നു ലംബമായി തെളിച്ചം മാറ്റുന്നു; തിരശ്ചീനമായി ചലിക്കുന്ന തീവ്രത മാറ്റുന്നു. നിങ്ങൾ ഫലം സംതൃപ്തരാണെന്ന് ചെയ്യുമ്പോൾ, ഡയലോഗും OK ബട്ടൺ അമർത്തുക, അല്ലെങ്കിൽ നിങ്ങളുടെ കീബോർഡിലെ Return കീ.

5.5.1. ടൂൾ സജീവമാക്കുന്നു

You can get to the Brightness-Contrast tool in two ways:

  • ചിത്രം-മെനുവിൽ വഴി: Tools → Color Tools → Brightness- Contrast അല്ലെങ്കിൽ Colors → Brightness-Contrast,

  • by clicking the tool icon: in Toolbox, provided that you have installed color tools in Toolbox. For this, please refer to വിഭാഗം 1.7, “പണിസഞ്ചി”.

Edit these settings as Levels

എളുപ്പം നിങ്ങളുടെ പ്രവർത്തനത്തിലാക്കാൻ, ഈ ബട്ടൺ നിങ്ങളെ സമാന ക്രമീകരണങ്ങൾ ഉപയോഗിച്ച് Levels ഉപകരണം തിരിയുന്നു അനുവദിക്കുന്നു.

പ്രിവ്യൂ

പ്രിവ്യൂ ചെക്ക്-ബോക്സും പുതിയ നില ക്രമീകരണങ്ങൾ ഉടനെ കാണാൻ കഴിയുന്ന ചലനാത്മകമായി അങ്ങനെ മിഴിവും കോൺട്രാസ്റ്റ് മാറ്റങ്ങളും ചെയ്യുന്നു.
